<a name="set-up-bar-code-mask-characters"></a>バーコード マスク文字の設定
-------------------------------

バーコード マスクがバーコードを作成し、直ちにPOSの(POS)にスキャンするバーコードを識別するのに使用されます。 では、文字で作成するバーコードのフォームを示すプレースホルダとして機能する構成されます。 バーコード マスクを設定するには、バーコード マスク文字を設定する必要があります。 ** [小売りとコマース** ** &gt; **在庫管理]、[実行 &gt; **バーコード、および分類する** &gt; **マスク文字**。 バーコード マスク文字を作成します**新しい**。 バーコード マスク文字は、次のデータを表示するように作成できます。

|                      |                                                                                                                 |
|----------------------|-----------------------------------------------------------------------------------------------------------------|
| **Field**            | **Description**                                                                                                 |
| **Product**          | プロジェクトIDのプレースホルダー。                                                                                     |
| **Any number**       | バーコードにハードコードな数を指定するのに使用します。                                                  |
| **Check digit**      | バーコードの妥当性を確認する、バーコード マスクのバーコード フォームをチェック ディジットを使用することを示します。 |

| **Color digit**      | 色を含む製品のために作成されたバーコードの色を示します。                               |
| **Style digit**      | フォームを含む製品のために作成されたバーコードのフォームを示します。                             |
| **EAN license code** | [EANライセンス コードに発行[EANライセンスのプレースホルダー。                                                       |
| [**価格**]            | 価格で埋め込まれているバーコードの価格を表します。                                                                   |

## <a name="define-bar-code-masks"></a>バーコード マスクを定義します。
バーコード マスク文字が必要なバーコード マスクに指定された後、在庫管理、** [小売りとコマース** &gt; ** **実行 &gt; **バーコード、および分類する** &gt; **バーコードのマスクの設定**。 このページでは、指定された文字を前に使用するバーコード マスクを定義できます。 これらのバーコード マスクは、バーコードを生成するときに使用され、POS.でスキャンするバーコードのIDができます。

1.  新しいバーコード マスクを作成します**新しい**。
2.  値を** [マスクIDと** ** **説明]フィールドに入力し、[バーコード マスクを選択**タイプ**フィールドを選択します。
3.  **概要**セクションでは、の値を**バーコード標準**一つが必要な場合、フィールドにつき、次を指定するバーコードの接頭語を選択します。
4.  **バーコード マスク セグメント**セクションで、作成するバーコードに使用するバーコード区分を追加します。
