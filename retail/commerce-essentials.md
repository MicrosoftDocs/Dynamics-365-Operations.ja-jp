# <a name="commerce-essentials-overview"></a>Commerce エッセンシャルの概要

[!include[banner](includes/banner.md)]


この記事は、Microsoft Dynamics 365 for Operations の Commerce エッセンシャル、販売時点管理 (POS) および商取引のコンフィギュレーション オプションに関する情報を提供します。 Commerce エッセンシャルは、小売業者がすばやく Microsoft Dynamics ベースの小売用ソリューションを稼働させ、小売固有の機能に焦点を合わせるのを助ける、すぐに使える小売用ワークロードを提供します。 

<a name="overview"></a>概要
--------

ビジネス ソフトウェアの環境には、それぞれが極めて特殊な機能を実行する多くのシステムが含まれます。 多くの場合、小売業者は、Microsoft Dynamics 365 for Operations の小売管理機能を利用しますが、既存の財務および人事管理の (HR) システムとは置き換えません。 これらの小売業者のために、小売固有のワークロードが必要です。 Commerce エッセンシャルは、Dynamics 365 for Operations のコンフィギュレーション オプションとしてこれらの小売固有の機能を提供します。 コンフィギュレーションが完了したら、小売固有でないソリューションをすべて非表示にし、小売組織の管理という重要なことにフォーカスできるようにします。 Commerce エッセンシャルは、Microsoft Dynamics AX 2012 R3 CU8 のコンフィギュレーション オプションとして最初に導入されました。 それ以降、従来型の組織だけでなく、無指向性チャンネルの小売業者に対応できる解決策となるよう、改良が多数加えられてきました。

## <a name="configuration"></a>構成
[Commerce エッセンシャル] をコンフィギュレーションするには、[**システム管理**] &gt; [**設定**] &gt; [**ライセンス コンフィギュレーション**] へ移動し、[**小売**] まで下にスクロールします。 [**小売**] ノードを展開して、[**Commerce エッセンシャル**] のチェック ボックスをオンにします。 [**保存**] をクリックし、変更を確認し、ページを更新します。 完全なソリューションに再度切り替えるには、[**Commerce エッセンシャル**] &gt; [**本社の設定**] &gt; [**ライセンス コンフィギュレーション**] へ移動し、[**小売**] まで下にスクロールします。 [**小売**] ノードを展開して、[**Commerce エッセンシャル**] のチェック ボックスをオフにします。 [**保存**] をクリックし、変更を確認し、ページを更新します。

## <a name="what-is-included"></a>含まれているもの。
[Commerce エッセンシャル] には、Dynamics 365 for Operations 小売ソリューションのコアに組み込まれた、コール センター機能を除く小売のバック オフィス機能のすべてが含まれます。 すべてのチャンネル、作業者、製品、店舗、レジスター、およびデバイス管理機能が、1 つのナビゲーション ノードに含まれています。 また、すべての完全な小売ワークスペースが含まれており、すぐに使えるロール固有のランディング ページを提供します。 [Commerce エッセンシャル] は、Retail Modern POS (MPOS) および電子商取引チャンネルから利益を得られる、従来型のまたはモバイル シナリオを持つ小売業者を対象にしています。 [Commerce エッセンシャル] を使用して、小売業者は 1 つの場所でこれらのチャンネルを管理できるため、本当の無指向性チャンネルの機能を得られます。

## <a name="how-is-commerce-essentials-different"></a>[Commerce エッセンシャル] は、どこが異なっているのか。
統合ナビゲーションや小売専用ページの提供に加えて、[Commerce エッセンシャル] は、フィールドや管理などのすべての非小売ページ要素を非表示にします。 小売業者が手元にあるタスクに集中するのに役立つ、合理化されたユーザー インターフェイスという結果になります。 すべての製品のライセンスがある小売業者は、さらに小売指向のユーザーの開始点として [Commerce エッセンシャル] を使用できます。 HR 詳細機能などの他の要素は、設定されているライセンス契約の条項に従い、必要に応じて、[Commerce エッセンシャル] に追加できます。 [Commerce エッセンシャル] が提供する合理化および簡略化の一部として、会計エクスポート機能により、小売業者は　[Commerce エッセンシャル] で未収の合計を既存の会計システムの勘定番号へマップすることができます。 小売システムが財務データを生成するときに、このデータは [Commerce エッセンシャル] から小売業者の会計システムに簡単にエクスポートできます。 さらに高度な統合機能が必要な場合、[Commerce エッセンシャル] には、Dynamics 365 for Operations に含まれている完全な [インポート / エクスポート] のフレームワークも含まれます。 既定で含まれる何百ものデータ エンティティにより、[Commerce エッセンシャル] には、複数の特殊なシステムが使用されている複数階層の環境に容易に適合する柔軟性があります。
