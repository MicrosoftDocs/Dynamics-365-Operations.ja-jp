# <a name="submit-and-approve-project-budgets"></a>プロジェクト予算の送信および承認

[!include[task guide banner](../../includes/task-guide-banner.md)]

この手順ではプロジェクトの予算を作成して送信する方法を示します。 

プロジェクト予算を作成するとき、プロジェクトの収益および原価の見積を入力してから、実際のプロジェクト トランザクションの管理に使用することができます。 プロジェクトの予算作成では、元の予算と修正内容をプロジェクト ワークフローに送って認証を申請する必要があります。 ワークフローを使用すると、プロセスの管理を強化できるうえ、変更履歴レコードを作成できます。

このタスクは、USSI データ セットを使用して作成されました。

1. [プロジェクト管理および会計] > [プロジェクト] > [すべてのプロジェクト] の順に選択します。
2. 一覧で、目的のレコードを見つけ、選択します。
3. 一覧で、選択された行のリンクをクリックします。
4. [アクション] ペインで [計画] をクリックします。
5. [プロジェクト予算] をクリックします。
6. [説明] フィールドに値を入力します。
7. [原価] セクションを展開
8. [新規] をクリックします。
9. [トランザクション タイプ] フィールドでオプションを選択します。
10. [カテゴリ] フィールドで値を入力または選択します。
11. [元の予算] フィールドに、数値を入力します。
12. [収益] セクションを展開します。
13. [新規] をクリックします。
14. 一覧で、選択された行をマークします。
15. [トランザクション タイプ] フィールドでオプションを選択します。
16. [カテゴリ] フィールドで値を入力または選択します。
17. [元の予算] フィールドに、数値を入力します。
18. [保存] をクリックします。
19. [ワークフロー] をクリックします。
20. [送信] をクリックします。
21. [コメント] フィールドで、値を入力します。
22. [送信] をクリックします。
