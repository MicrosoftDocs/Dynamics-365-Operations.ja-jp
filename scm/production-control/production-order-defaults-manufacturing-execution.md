# <a name="production-order-defaults-in-manufacturing-execution"></a>製造実行の生産パラメーター既定値

[!include[banner](../includes/banner.md)]




作業者が生産ジョブに登録する前に、[**製造オーダーの既定**] ページのすべての設定を注意深く考慮する必要があります。 会社がマルチサイト機能を使用する場合は、サイトごとに異なる製造オーダーを既定することができます。 生産管理と統合した注文の既定値は、**製造オーダーの既定値**ページの次のタブで設定できます:

-   **一般** – 製造実行の生産ジョブにおける一般の注文既定値。
-   **開始** – 生産ジョブや工程の開始時に使用する注文の既定値。
-   **工程** – 生産プロセス中に、生産ジョブや生産工程、および工程に関するフィードバックに適用される注文既定値。
-   **完了レポート** – 製造オーダーの最後の工程の完了後、品目を報告する際に適用される注文既定値。
-   **数量の検証** – 製造オーダーの開始数量およびフィードバック数量を検証する場合の注文既定値。

## <a name="types-of-production-jobs"></a>生産ジョブのタイプ
**工程**タブの**ジョブ登録**ページで、登録する必要がある生産ジョブのタイプを選択します。 通常、作業者は設定ジョブとプロセス ジョブを登録します。 ただし、ジョブのスケジューリングが適用されている場合は、製造オーダーの処理時にも作業者が登録する必要がある他のジョブ タイプ (配送ジョブなど) を選択できます。 **重要:** 関連するすべてのジョブ タイプが選択されていることを確認してください。 それ以外の場合は、**ジョブ登録**ページで登録できるジョブが現れない場合があります。 **工順グループ**ページの**ジョブ管理**列で行った選択に従って、選択を調整します。 工順グループで**ジョブ管理**が選択されている場合、このジョブ タイプが製造オーダーで完了報告されます。 この報告は、ジョブが [製造実行] で完了報告されたときに発生します。 [**ジョブ管理**] が選択されているすべてのジョブ タイプで工程が完了報告されると、[製造実行] の工程も完了報告されます。 **注意:** 一部のジョブ タイプは生産仕訳帳から手動で報告できます。 この場合、ジョブ タイプには**ジョブ管理**を選択しますが、**製造オーダーの既定値**ページの**工程**タブでは、登録するジョブ タイプを選択しないでおきます。

## <a name="bom-consumption-and-picking-list-journals"></a>BOM 消費とピッキング リスト仕訳帳
材料は、生産中に自動または手動で消費されるように設定できます。 自動消費は、部品表 (BOM) の明細行で設定されている部品消費ルールによって、また、製造オーダーの既定値の**自動 BOM 消費**フィールドの設定によって、制御されます。 自動消費は、製造オーダーが開始または完了報告された時に発生するように設定できます。 また、ジョブレベルで、ジョブが開始または完了した時に実行するようにもできます。

### <a name="controlling-material-consumption-when-a-production-order-is-started"></a>製造オーダーの開始時の材料消費を制御

製造オーダーの開始時の材料消費は、**開始**タブの**自動 BOM 消費**フィールドで制御できます。 使用可能な値は次のとおりです。

-   **部品消費ルール** – 製造オーダーの開始時に、製造 BOM 明細行で設定された部品消費ルールに従って、材料の数量消費を実行します。 部品消費ルールが**開始**に設定されている材料明細行だけが生産の開始時に実行されます。
-   **常時** – 開始数量に比例する材料の数量が常に消費されます。
-   **なし** – 材料の数量消費は実行されません。

### <a name="controlling-material-consumption-when-a-production-job-is-started-or-completed"></a>製造ジョブの開始時または完了時の材料消費を制御

製造ジョブの開始時または完了時の材料消費は、**工程**タブの**自動 BOM 消費**フィールドで制御できます。 使用可能な値は次のとおりです。

-   **部品消費ルール** – 製造ジョブの数量の開始時または完了時に、製造 BOM 明細行で設定された部品消費ルールに従って、材料の数量消費を実行します。 部品消費ルールが**開始**または**完了**に設定されている材料明細行だけが消費されます。
-   **常時** – ジョブの開始数量に比例する材料の数量が常に消費されます。
-   **なし** – 材料の数量消費は実行されません。

### <a name="controlling-material-consumption-when-a-production-order-is-reported-as-finished"></a>製造オーダーの完了報告時の材料消費を制御

製造オーダーの完了レポートの処理時の材料消費は、**完了レポート**タブの**自動 BOM 消費**フィールドで制御できます。 使用可能な値は次のとおりです。

-   **部品消費ルール** – 製造オーダーの完了レポート時に、製造 BOM 明細行で設定された部品消費ルールに従って、材料の数量消費を実行します。 部品消費ルールが**完了**に設定されている材料明細行だけが消費されます。
-   **常時** – ジョブの完了レポート時の数量に比例する材料の数量が常に消費されます。
-   **なし** – 材料の数量消費は実行されません。
