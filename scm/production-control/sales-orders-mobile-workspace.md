# <a name="sales-orders-mobile-workspace-for-microsoft-dynamics-365-for-operations-app"></a>工程のアプリケーションのMicrosoft Dynamics 365の販売注文の携帯電話ワークスペース

販売注文携帯電話ワークスペースを使用すると、場所、およびいつでも販売注文で最後にどのできます。 

<a name="prerequisites"></a>必要条件
-------------

| 前提条件                                                         | 説明                                                                                                                                                                   |
|----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 工程のモバイル者のプラットフォームのMicrosoft Dynamics 365について読み取り | [アクションの携帯電話プラットフォーム] (/dynamics365/operations/dev itpro/mobile apps/mobileプラットフォーム) 365 for Operations                                                              |
| Dynamics 365 for Operations                                          | 工程バージョン1611では、Microsoft Dynamics 365および工程のプラットフォームの更新は3 (2016年12月11) のMicrosoft Dynamicsがある環境を使用していることを確認します。 |
| 3215650 KB変更                                                    | 工程にMicrosoft Dynamics 365で設定されたワークスペースを有効にするように変更をインストールします。                                                                       |
| インストールされるアクションのアプリケーション365 for Operationsを持つモバイル デバイス | 携帯電話アプリケーションの店舗からアクションのアプリケーション365 for Operationsをダウンロードします。                                                                                                      |

## <a name="overview"></a>概要
この者のモバイル アプリケーション ワークスペースは、Dynamics 365 for Operationsにアクセスして、販売注文などに、注文のステータス、顧客の連絡先情報と注文の受け手の連絡先に関する詳細情報を表示できるようになります。 携帯電話ワークスペースは、販売注文の簡単なビューを提供します。 顧客から販売注文を表示できますかまたはすべての販売注文を表示、または特定の販売注文に関する情報を表示します。 携帯電話ワークスペースは、詳細な販売注文の分析に役立つとしての表示を提供します。

### <a name="view-all-sales-orders"></a>すべての販売注文を表示します。

このビューは、すべての販売注文を一覧表示します。

-   表示する販売注文を選択、次のフィルタの一つを使用します。
    -   販売注文によって検索します。
    -   顧客IDによって検索します。
    -   顧客の名前で検索します。
    -   ステータスによって検索します。
    -   リリース ステータスによって検索します。
    -   作成日に検索および時間を表示します。

<!-- -->

-   販売注文を選択すると、特定の注文の詳細を表示できます。 具体的には確認できます:
    -   顧客の名前と住所情報
    -   "指定出荷日"および"確認済の出荷日などのさまざまな販売注文日、
    -   注文の受け手の連絡先情報
    -   顧客の連絡先情報
    -   注文明細行
    -   販売注文の出荷されたかいつどのように示す出荷

### <a name="view-orders-for-a-customer-"></a>顧客の表示の購買** **

このビューは、顧客ごとに1の販売注文を一覧表示します。

-   顧客の注文を表示、次のフィルタの一つを使用します。
    -   名前で検索
    -   勘定ごとの検索

<!-- -->

-   顧客を選択すると、表示できます:
    -   顧客の名前とグループ
    -   顧客の連絡先情報
    -   販売注文に関する顧客の販売注文と詳細:
        -   顧客の名前と住所情報
        -   各販売注文日
        -   注文の受け手の連絡先情報
        -   顧客の連絡先情報
        -   注文明細行
        -   販売注文の出荷されたかいつどのように示す出荷

## <a name="get-started"></a>はじめに
モバイル デバイスの販売注文の携帯電話ワークスペースを使用して開始するには、次の手順に従います。

1.  携帯電話アプリケーションの店舗で、工程のアプリケーションのMicrosoft Dynamics 365をダウンロードしてインストールします。
2.  デバイスのアプリケーションを開始します。
3.  Dynamics 365 for Operations URLを入力します。
4.  に署名するための会社を入力します。 たとえば、入力しますUSMF ** **。
5.  ユーザーが署名する、最初に、作業勘定、Microsoft Dynamics 365のユーザー名とパスワードの入力が求められます。 資格情報を入力します。 署名した後に、会社については、ワークスペースが表示されます。

携帯電話アプリケーション ワークスペースを表示するには、工程のアプリケーション365 for Operationsに必要なワークスペースを公開する必要があります。

1.  Dynamics 365 for Operationsを開始します。
2.  **システム** ** ** &gt; 設定、管理実行 &gt; **システム パラメータ**。
3.  [**携帯電話アプリケーションを管理します。**。
4.  携帯電話プラットフォームに発行する場合は、ワークスペースを選択します。
5.  選択します**発行ワークスペース**。
6.  発行されたワークスペースを表示するデバイスを更新します。

## <a name="view-information-about-sales-orders-for-a-customer"></a>顧客の販売注文に関する情報を表示します。
1.  モバイル デバイスで、**販売注文**ワークスペースを選択します。
2.  **選択します。顧客の注文を表示します。**
3.  目的の顧客を検索使用**勘定**または**顧客名**情報。
4.  顧客を選択します。
5.  **連絡先情報**または選択します**販売注文**。
6.  **販売注文**オンにすると、顧客の販売注文の一覧が表示されます。
7.  **販売注文**選択します。
8.  販売注文明細行、出荷、顧客の連絡先情報と注文の受け手の連絡先に関する情報を表示できます。
