# <a name="inventory-on-hand-mobile-workspace-for-microsoft-dynamics-365-for-operations-app"></a>工程のアプリケーションのMicrosoft Dynamics 365の手持在庫の携帯電話ワークスペースの在庫します。

手持在庫の携帯電話ワークスペースは、引当、使用可能な在庫に移動して追跡はいつでも、手順で表示されます。 

<a name="prerequisites"></a>必要条件
-------------

| 前提条件                                                         | 説明                                                                                                                                        |
|----------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| 工程のモバイル者のプラットフォームのMicrosoft Dynamics 365について読み取り | [アクションの携帯電話プラットフォーム] (/dynamics365/operations/dev itpro/mobile apps/mobileプラットフォーム) 365 for Operations                                   |
| Dynamics 365 for Operations                                          | 工程バージョン1611では、Microsoft Dynamics 365および工程のプラットフォームの更新は3 (2016年12月11) のMicrosoft Dynamicsがある環境 |
| 3215650 KB変更                                                    | 工程にMicrosoft Dynamics 365で設定されたワークスペースを有効にするように変更をインストールします。                                       |
| インストールされるアクションのアプリケーション365 for Operationsを持つモバイル デバイス | 携帯電話アプリケーションの店舗からアクションのアプリケーション365 for Operationsをダウンロードします。                                                                           |

## <a name="introduction"></a>はじめに
通常、会社の在庫の複数の出荷日と複数の入庫があります。 これらの移動が連続して手持在庫ステータスを変更します。 手持在庫の携帯電話ワークスペースは選択モバイル デバイスの在庫データに対する最新の情報を得ることができるように、クロス会社の手持在庫ステータスを表示できるようになります。 に関係なく、倉庫、購買、販売、製造、または管理で作業するかどうか、または他のロールが必要です。、手持在庫データはいつでも、手順でアクセスできます。 携帯電話ワークスペースは機能に手持在庫ステータスの簡単なビューを提供し、機能、現在の材料の引当や引当されていない手持在庫に手持在庫を表示できるようになります。 また、手持在庫を照会するには、品目番号を入力することができ、手持在庫の製品またはバリアントのフィルタ処理された検索を実行します。 具体的には、携帯電話ワークスペースはこれらの機能があります:

-   手持在庫ステータスの表示と製品が見つからないように副産物の番号または製品名が取得できます。
-   選択した製品に、次の情報を表示できます:
    -   サイト別の手持在庫
    -   倉庫別の手持在庫
    -   場所別の手持在庫
    -   バッチごとの手持在庫 (バッチ管理されている製品の場合) 
    -   在庫ステータス別の手持在庫

<!-- -->

-   製品の手持在庫は、次の方法で表示されます:
    -   現物在庫によって (このビューは、合計金額を表します)。
    -   引当済現物数によって (このビューは引当金額を表します)。
    -   引当可能現物数によって (このビューは予約がない使用できる金額を表します)。

## <a name="get-started"></a>はじめに
モバイル デバイスで開始する場合:

1.  携帯電話アプリケーションの店舗から、工程のアプリケーションのMicrosoft Dynamics 365をダウンロードしてインストールします。
2.  デバイスのアプリケーションを開始します。
3.  Dynamics 365 for Operations URLを入力します。
4.  に署名するための会社を入力します。 たとえば、入力しますUSMF ** **。
5.  ユーザーが署名する、最初に、作業勘定、Microsoft Dynamics 365のユーザー名とパスワードの入力が求められます。 資格情報を入力します。 署名した後に、会社については、ワークスペースが表示されます。

携帯電話アプリケーション ワークスペースを表示するには、工程のアプリケーション365 for Operationsに必要なワークスペースを公開する必要があります。

1.  Dynamics 365 for Operationsを開始します。
2.  **システム** ** ** &gt; 設定、管理実行 &gt; **システム パラメータ**。
3.  [**携帯電話アプリケーションを管理します。**。
4.  携帯電話プラットフォームに発行する場合は、ワークスペースを選択します。
5.  選択します**発行ワークスペース**。
6.  発行されたワークスペースを表示するデバイスを更新します。

## <a name="view-the-onhand-inventory-for-a-product"></a>製品のonhandの在庫を表示します。
1.  モバイル デバイスで、**手持在庫**ワークスペースを選択します。
2.  **選択します。品目があり、手持在庫を確認します。**。 オフラインで使用するアプリケーションに読み込まれる製品の一覧が表示されます。 既定では、50品目が読み込まれます、この番号を変更できます。 詳細については、潜在顧客読取りのハンドブックを表示します。
3.  品目のリストにない場合、**詳細を検索します。** Dynamics 365 for Operationsのオンラインを検索するします。 検索の副産物の名前に副産物の番号またはスイッチを取得します。
4.  製品を選択します。 品目の画像がある場合は、画像が表示されます。
5.  手持在庫ステータスを表示する場合は、次のオプションの一つを選択します:
    -   サイトごとの手持在庫を表示します。
    -   倉庫ごとの手持在庫を表示します。
    -   場所ごとの手持在庫を表示します。
    -   バッチごとの手持在庫を表示します (バッチ管理されている製品の場合) 
    -   在庫ステータスごとの手持在庫を表示します。

    製品の手持在庫は、次の方法で表示されます:
    -   現物在庫によって (このビューは、合計金額を表します)。
    -   引当済現物数によって (このビューは引当金額を表します)。
    -   引当可能現物数によって (このビューは予約がない使用できる金額を表します)。
