[!include[banner](../includes/banner.md)]


<span data-ttu-id="35a46-104">標準原価データの更新は、1 バージョン アプローチと 2 バージョン アプローチという 2 つの異なるアプローチを使用して管理できます。</span><span class="sxs-lookup"><span data-stu-id="35a46-104">Updates to standard cost data can be managed by using two different approaches -  the one-version approach and the two-version approach.</span></span>

<span data-ttu-id="35a46-107">2 バージョン アプローチは、オリジナルの原価のレコードを含む 1 つのバージョンとすべての原価の更新のレコードを含む 2 番目のバージョンを使用します。</span><span class="sxs-lookup"><span data-stu-id="35a46-107">The two-version approach uses アプローチの概念を使用すると、複数の差分更新を識別できます。</span><span class="sxs-lookup"><span data-stu-id="35a46-108">A primary advantage of the two-version approach is the clear delineation and tracking of cost updates in a アプローチを使用すると、複数の差分更新を識別できます。この場合、差分更新ごとに、差分原価レコードを格納する別個の原価バージョンが用意されます。</span><span class="sxs-lookup"><span data-stu-id="35a46-109">The two-version approach can be used to identify multiple example, updates that reflect new items or error corrections.</span></span> <span data-ttu-id="35a46-112">単一の原価バージョンが、当年の標準原価を表すと仮定します。</span><span class="sxs-lookup"><span data-stu-id="35a46-112">Assume that a single costing version represents the standard costs for the current year.</span></span> <span data-ttu-id="35a46-113">このバージョンの識別子は 2016-STD です。</span><span class="sxs-lookup"><span data-stu-id="35a46-113">The identifier for this version is 2016-STD.</span></span> <span data-ttu-id="35a46-114">バージョン 2016-STD には、すべての品目の現在の有効原価が格納されています。</span><span
