# <a name="create-boms-february-2016-only"></a>BOM の作成 (2016 年 2 月のみ)

[!include[task guide banner](../../includes/task-guide-banner.md)]

このタスクでは、完成品と半完成品の部品表の構造の作成について説明します。 これは、BOM 計算シリーズの 4 番目のタスクです。 このタスクの作成に使用するデモ データの会社は USMF です。


## <a name="create-bom-for-a-semi-finished-product"></a>半完成品の BOM の作成
1. [製品情報管理] > [製品] > [リリースされた製品] の順に移動します。
2. 一覧で、選択された行のリンクをクリックします。
    * 品目番号 BOM_2 を選択します。  
3. アクション ペインで、[エンジニア] をクリックします。
4. [BOM バージョン] をクリックします。
5. [新規] をクリックします。
6. [BOM と BOM バージョン] をクリックします。
7. [名前] フィールドに値を入力します。
    * たとえば、[BOM_2] を入力します。  
8. [サイト] フィールドで値を入力または選択します。
    * たとえば、サイト 1 を入力または選択します。  
9. [OK] をクリックします。
10. [新規] をクリックします。
11. [品目番号] フィールドに値を入力します。
    * この例では、「ITEM_C」を入力します。  
12. [倉庫] フィールドで、値を入力または選択します。
    * たとえば、11 を入力または選択します。  
13. [ヘッダー] をクリックします。
14. [承認] をクリックして、部品表を承認します。
15. [OK] をクリックします。
16. [承認] をクリックします。
    * [承認] ボタンは、[BOM バージョン] セクションのツールバーにあります。 表示されていない場合は、部品表のページの右上にあるヘッダーをクリックして、[承認] を表示します。  
17. [OK] をクリックします。
18. [有効化] をクリックします。
19. ページを閉じます。
20. ページを閉じます。
21. ページを閉じます。

## <a name="create-bom-for-a-finished-product"></a>完成品の BOM の作成
1. 一覧で、選択された行のリンクをクリックします。
    * 品目番号 BOM_1 を選択します。  
2. アクション ペインで、[エンジニア] をクリックします。
3. [BOM バージョン] をクリックします。
4. [新規] をクリックします。
5. [BOM と BOM バージョン] をクリックします。
6. [名前] フィールドに値を入力します。
    * たとえば、[BOM_1] を入力します。  
7. [サイト] フィールドで値を入力または選択します。
    * たとえば、サイト 1 を入力または選択します。  
8. [OK] をクリックします。
9. [新規] をクリックします。
10. [品目番号] フィールドに値を入力します。
    * この例では、「ITEM_A」を入力します。  
11. [倉庫] フィールドで、値を入力または選択します。
    * この例では、「11」を選択します。  
12. [新規] をクリックします。
13. [品目番号] フィールドに値を入力します。
    * この例では、「ITEM_B」を入力します。  
14. [倉庫] フィールドで、値を入力または選択します。
    * たとえば、11 を入力または選択します。  
15. [新規] をクリックします。
16. [品目番号] フィールドに値を入力します。
    * この例では、「BOM_2」を入力します。  
17. 一覧で、選択された行をマークします。
18. [倉庫] フィールドで、値を入力または選択します。
    * この例では、倉庫 11 を入力または選択します。  
19. [ヘッダー] をクリックします。
20. [承認] をクリックして、部品表を承認します。
21. [OK] をクリックします。
22. [承認] をクリックします。
    * [承認] ボタンは、[BOM バージョン] セクションのツールバーにあります。 表示されていない場合は、部品表のページの右上にあるヘッダーをクリックして、[承認] を表示します。  
23. [OK] をクリックします。
24. [有効化] をクリックします。
25. ページを閉じます。
26. ページを閉じます。
27. ページを閉じます。
