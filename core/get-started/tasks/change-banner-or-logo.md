# <a name="change-the-banner-or-logo"></a>バナーまたはロゴの変更

[!include[task guide banner](../../includes/task-guide-banner.md)]

次の手順は、システム管理者が法人用に表示するバナーまたはロゴ画像を更新するために使用できるステップを一覧にしています。 この手順の作成に使用するデモ データの会社は USMF です。

1. [組織管理] > [組織] > [法人] の順に移動します。
2. 左側のリストで、バナーまたはロゴを更新する法人を選択します。 これが既に選択されている場合、次の手順を実行します。
3. [画像] セクションを展開します。 これが既に展開されている場合、次の手順を実行します。
4. [編集] をクリックします。
5. [ダッシュボードの会社の画像タイプ] フィールドで、「バナー」を選択します。
6. [変更] をクリックします。
    * ヒント : バナーの画像の理想的な解像度は 1920 x 281 ピクセルです。 ロゴの画像の理想的な幅は 350 ピクセルです。  
    * アップロードが完了すると、[ダッシュボードの画像] セクションに新しいバナーまたはロゴの画像が表示されているはずです。  
7. [保存] をクリックします。
8. [共通] > [共通] > [既定のダッシュボード] に移動します。
    * ダッシュボードに新しいバナーまたはロゴの画像が表示されているはずです。  
    * 会社のピッカーを使用して、バナーをアップロードしたものに、会社を変更する必要があるかもしれません。
