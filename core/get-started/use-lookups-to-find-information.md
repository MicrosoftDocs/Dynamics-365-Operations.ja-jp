# <a name="use-lookups-to-find-information"></a>情報の検索、参照を使用します。

、Microsoft Dynamics 365では、多数のフィールドに適切な仕訳帳または目的の値を検索が容易に役立つ参照があります。 複数の拡張機能は、これらのコントロールを使用可能により、ユーザーに費やされたにする参照が表示されます。 このトピックでは、これらの新しい参照機能について情報と、システムの参照から業務の使用を取得するために、必要な情報が表示されます。
