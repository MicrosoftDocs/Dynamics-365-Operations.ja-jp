# <a name="set-up-injury-and-illness-information"></a>けが/病気の情報を設定します

[!include[task guide banner](../../includes/task-guide-banner.md)]

雇用者は、従業員がいつ危険な仕事場により、傷害を負ったか、また疾病を患ったかを把握している必要があります。 [障害と疾病] ページで情報を設定し、職場における障害または疾病の報告を促進できます。 ここでは、治療、原価、結果のタイプなど、傷害と疾病のタイプを設定できます。 この手順の作成に使用するデモ データの会社は USMF です。

1. [人事管理] > [作業者] > [けがおよび病気] > [けがおよび病気の設定] へ移動します。
2. [新規] をクリックします。
3. [けが/病気のタイプ] フィールドで、値を入力します。
    * 例: 挫傷  
4. [説明] フィールドに値を入力します。
    * 例 : 骨折  
5. [身体部位] タブをクリックします。
6. [新規] をクリックします。
7. [身体部位] フィールドに値を入力します。
    * 例 : 手首  
8. [説明] フィールドに値を入力します。
    * 例 : 手首  
9. [処置のタイプ] タブをクリックします。
10. [新規] をクリックします。
11. [処置のタイプ] フィールドに値を入力します。
    * 例: そえ木  
12. [説明] フィールドに値を入力します。
    * 例 : そえ木を当てる  
13. [コスト タイプ] タブをクリックします。
14. [新規] をクリックします。
15. [コスト タイプ] フィールドに値を入力します。
    * 例 : X線  
16. [説明] フィールドに値を入力します。
    * 例 : X線  
17. [結果タイプ] タブをクリックします。
18. [新規] をクリックします。
19. [結果タイプ] フィールドに値を入力します。
    * 例 : 療法  
20. [説明] フィールドに値を入力します。
    * 例 : 理学療法  
21. [重要度レベル] タブをクリックします。
    * カスタマイズ可能な重要度レベルを作成できます。 例 : 重症度 1 が軽傷を、重症度 3 が重症を示すようにできます。  
22. [報告先機関] タブをクリックします。
    * 報告先機関とは、インシデントを報告する先の機関です。 既定のチェック・ボックスをオンにして、傷害と疾病の報告先である既定の機関を選択します。  
23. [保存] をクリックします。
