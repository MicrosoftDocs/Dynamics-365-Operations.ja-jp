# <a name="develop-a-succession-plan"></a>継続的計画の作成

[!include[task guide banner](../../includes/task-guide-banner.md)]

組織が育ち、継続的計画を考慮すると、同様のスキルを持つ別の人物を見つけたい場合があります。  スキル マッピングは、既存の従業員および応募者を分析して、評価した従業員のスキル セットに一致するかどうかを確認することができます。 この手順の作成に使用するデモ データの会社は USMF です。

1. [人事管理] > [コンピテンシー] > [スキル分析] > [スキル マッピング プロファイル] の順に移動します。
2. [新規] をクリックします。
3. [スキル マッピング] フィールドで、スキル マッピングの名前を入力します。  例: 従業員。
4. [説明] フィールドに値を入力します。
5. [日付] フィールドに日付を入力します。
6. [プロファイルの取得] をクリックします。
7. [個人] をクリックします。
8. [個人] フィールドに名前を入力するか、またはドロップ ダウンから選択します。  例: Cassie Hicks。
9. [OK] をクリックします。
10. 証明書のクイック タブを展開し、スキル マッピングに含まれる証明書を表示または編集します。
11. [スキル] クイック タブを展開し、含めるスキルを表示または編集します。
12. 一覧で、最初の行をマークします。  例: 会計
13. [オプション] チェック ボックスをオンにします。
14. [重要性] フィールドで、オプションを選択します。
    * スキルをオプションとしてマークする場合、スキルの重要性レベルを選択する必要があります。  
15. 一覧で行 2 を選択します。
16. [オプション] チェック ボックスをオンにします。
17. [重要性] フィールドで、オプションを選択します。
18. 一覧で行 3 を選択します。
19. [オプション] チェック ボックスをオンにします。
20. [重要性] フィールドで、オプションを選択します。
21. 一覧で行 4 を選択します。
22. [オプション] チェック ボックスをオンにします。
23. [重要性] フィールドで、オプションを選択します。
24. [教育] のクイック タブを展開し、スキル マッピングに含める教育のコンピテンシーを表示または編集します。
25. [実行] をクリックします。
26. [OK] をクリックします。
27. [結果] をクリックします。
28. [レポート] をクリックします。
    * レポートでは、最適なマッチがレポートの先頭に一覧表示されます。  ギャップのある要素が一覧されていることを確認できます。  これは、スキル マッピングに表示されるレベルと、従業員に割り当てられたスキル レベルとの差です。
