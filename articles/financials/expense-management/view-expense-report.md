# <a name="view-an-expense-report"></a>経費精算書の表示

[!include [banner](../includes/banner.md)]

総勘定元帳または調達でトランザクションを表示するクエリを実行すると、クエリが大量の情報を返す可能性があります。 したがって、検索を絞り込むこともできます。 必要な情報を取得するには、表示されている経費精算書を確認するトランザクションにドリルダウンできます。

経費精算書に関して、トランザクションから元の清算書 (経費管理に保存されている) にドリルダウンできます。 経費の合計金額に加えて、明細化されたトランザクション、付加価値税 (VAT) の復旧に適用できるトランザクション、および経費に関連付けられている受取も表示できます。

トランザクションが表示された元の経費精算書にドリルダウンするには、トランザクションを選択し、次に**元の文書**を選択します。

個別のトランザクションを表示すると、トランザクションに関連する追加データを表示できます。 たとえば、明細化されるホテル経費に対して、インターネット アクセス、通話、ルーム サービスなどの経費の詳細の一覧を表示できます。
