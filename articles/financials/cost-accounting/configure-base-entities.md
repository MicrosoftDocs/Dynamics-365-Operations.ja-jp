# <a name="configure-base-entities"></a>基準エンティティのコンフィギュレーション 

[!include[banner](../includes/banner.md)]


原価会計は独立モジュールで、以下のエンティティが Dynamics 365 for Finance and Operations, Enterprise Edition で正しくコンフィギュレーションされている必要があります。

-  通貨
-  為替レート
-  会計カレンダー  

原価会計モジュールを使用する前に、通貨、為替レート、および会計カレンダーを会社用に定義する必要があります。 

これらのエンティティの詳細については、e ラーニング トレーニング コース「[モジュール 2 通貨および為替レートの設定](https://mbspartner.microsoft.com/AX/CourseModules/1215)」を受講してください (CustomerSource アカウントが必要です)。
