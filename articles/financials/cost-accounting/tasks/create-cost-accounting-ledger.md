# <a name="create-a-cost-accounting-ledger"></a>原価会計元帳の作成

[!include[task guide banner](../../includes/task-guide-banner.md)]

原価会計元帳は、レポート単位の全体を示しています。 原価要素の分析コード、統計分析コード、会計カレンダー、および通貨によって定義されています。 法人の概念として、認識不可能です。 法人とそのデータは、多くの原価会計元帳と関連付けることができます。 この記録では、USP2 デモ データ会社を使用します。

1. [原価計算] > [元帳の設定] > [原価会計元帳] へ、移動します。
2. [新規] をクリックします。
3. [名前] フィールドに値を入力します。
4. [原価要素分析コード] フィールドで、値を入力または選択します。
5. [会計カレンダー] フィールドで、値を入力または選択します。
6. [会計通貨] フィールドで、値を入力または選択します。
7. [為替レート] フィールドで、値を入力または選択します。
8. [統計分析コード] フィールドで、値を入力または選択します。
9. [保存] をクリックします。
