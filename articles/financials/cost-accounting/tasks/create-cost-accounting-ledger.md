# <a name="create-a-cost-accounting-ledger"></a><span data-ttu-id="33a18-103">原価会計元帳の作成</span><span class="sxs-lookup"><span data-stu-id="33a18-103">Create a cost accounting ledger</span></span>

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="33a18-104">原価会計元帳は、レポート単位の全体を示しています。</span><span class="sxs-lookup"><span data-stu-id="33a18-104">A cost accounting ledger represents the overall reporting unit.</span></span> <span data-ttu-id="33a18-105">原価要素の分析コード、統計分析コード、会計カレンダー、および通貨によって定義されています。</span><span class="sxs-lookup"><span data-stu-id="33a18-105">It is defined by a cost element dimension, statistical dimension, fiscal calendar, and currency.</span></span> <span data-ttu-id="33a18-106">法人の概念として、認識不可能です。</span><span class="sxs-lookup"><span data-stu-id="33a18-106">It is agnostic to the concept of legal entities.</span></span> <span data-ttu-id="33a18-107">法人とそのデータは、多くの原価会計元帳と関連付けることができます。</span><span class="sxs-lookup"><span data-stu-id="33a18-107">A legal entity and its data can be associated with many cost accounting ledgers.</span></span> <span data-ttu-id="33a18-108">この記録では、USP2 デモ データ会社を使用します。</span><span class="sxs-lookup"><span data-stu-id="33a18-108">This recording uses the USP2 demo data company.</span></span>

1. <span data-ttu-id="33a18-109">[原価計算] > [元帳の設定] > [原価会計元帳] へ、移動します。</span><span class="sxs-lookup"><span data-stu-id="33a18-109">Go to Cost accounting > Ledger setup > Cost accounting ledgers.</span></span>
2. <span data-ttu-id="33a18-110">[新規] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="33a18-110">Click New.</span></span>
3. <span data-ttu-id="33a18-111">[名前] フィールドに値を入力します。</span><span class="sxs-lookup"><span data-stu-id="33a18-111">In the Name field, type a value.</span></span>
4. <span data-ttu-id="33a18-112">[原価要素分析コード] フィールドで、値を入力または選択します。</span><span class="sxs-lookup"><span data-stu-id="33a18-112">In the Cost element dimension field, enter or select a value.</span></span>
5. <span data-ttu-id="33a18-113">[会計カレンダー] フィールドで、値を入力または選択します。</span><span class="sxs-lookup"><span data-stu-id="33a18-113">In the Fiscal calendar field, enter or select a value.</span></span>
6. <span data-ttu-id="33a18-114">[会計通貨] フィールドで、値を入力または選択します。</span><span class="sxs-lookup"><span data-stu-id="33a18-114">In the Accounting currency field, enter or select a value.</span></span>
7. <span data-ttu-id="33a18-115">[為替レート] フィールドで、値を入力または選択します。</span><span class="sxs-lookup"><span data-stu-id="33a18-115">In the Exchange rate type field, enter or select a value.</span></span>
8. <span data-ttu-id="33a18-116">[統計分析コード] フィールドで、値を入力または選択します。</span><span class="sxs-lookup"><span data-stu-id="33a18-116">In the Statistical dimension field, enter or select a value.</span></span>
9. <span data-ttu-id="33a18-117">[保存] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="33a18-117">Click Save.</span></span>
