# <a name="create-and-assign-a-cost-distribution-policy-to-a-cost-control-unit"></a>原価配分ポリシーの作成と原価管理単位への割り当て

[!include[task guide banner](../../includes/task-guide-banner.md)]

[原価配分ルール] は、集団コスト センターごとに集計される原価を配分するために使用されます。 コスト経理担当は、原価が配賦基準に基づいて、コスト センターに配分されるか確認します。 ポリシーと対応ルールは、原価管理単位に割り当てられます。 このタスク ガイドでは、原価配分ポリシーと対応ルールを作成する方法を示す例を使用します。


## <a name="create-a-policy"></a>ポリシーの作成
1. [原価会計] > [ポリシー] > [原価配布ポリシー] へ移動します。
2. [新規] をクリックします。
3. [ポリシー名] フィールドで、値を入力します。
4. [説明] フィールドに値を入力します。
5. [原価オブジェクト分析コード階層] フィールドで、値を入力または選択します。
    * [組織] を選択します。  
6. [原価要素分析コード階層] フィールドで、値を入力または選択します。
    * [CDS P/L] を選択します。  
7. [統計分析コード] フィールドで、値を入力または選択します。
    * [統計要素] を選択します。  
8. [保存] をクリックします。

## <a name="create-rules-for-the-policy"></a>ポリシーのルールを作成
1. [新規] をクリックします。
2. 一覧で、選択された行をマークします。
3. [原価オブジェクト分析コード階層ノード] フィールドで、値を入力または選択します。
    * 階層を展開し、094 を選択します。  
4. [原価要素分析コード階層ノード] フィールドで、値を入力または選択します。
    * [その他の営業経費] を選択し、[605110 クリーニング] を選択します。  
5. [原価動作] フィールドで、オプションを選択します。
    * [固定値] を選択します。  
6. [配賦基準] フィールドで、値を入力または選択します。
7. [新規] をクリックします。
8. 一覧で、選択された行をマークします。
9. [原価オブジェクト分析コード階層ノード] フィールドで、値を入力または選択します。
    * 階層を展開し、094 を選択します。  
10. [原価要素分析コード階層ノード] フィールドで、値を入力または選択します。
    * [その他の営業経費] を選択し、[605150 賃貸] を選択します。  
11. [原価動作] フィールドで、オプションを選択します。
    * [固定値] を選択します。  
12. [配賦基準] フィールドで、値を入力または選択します。
13. [保存] をクリックします。

## <a name="assign-rules-to-a-cost-control-unit"></a>原価管理単位へルールを割り当て
1. 原価管理単位の [ポリシーの割り当て] をクリックします。
2. [新規] をクリックします。
3. 一覧で、選択された行をマークします。
4. [会計日から有効] フィールドで、値を入力します。
    * 有効な会計年度で、[9 月 1 日] を選択します。  
5. [原価管理単位] フィールドで、値を入力または選択します。
6. [保存] をクリックします。
