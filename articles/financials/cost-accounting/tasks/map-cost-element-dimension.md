# <a name="map-a-cost-element-dimension"></a>原価要素分析コードのマップ

[!include [task guide banner](../../includes/task-guide-banner.md)]

原価管理者は、この手順を使用して、MXMF 法人の原価要素分析コードに原価要素分析コードをマップします。 この記録では、USP2 デモ データ会社を使用します。

1. [原価計算] > [分析コード] > [コスト要素分析コード] の順序で進みます。
2. 一覧で、目的のレコードを見つけ、選択します。
    * この例の場合、[原価要素] を選択します。  
3. [分析コード マッピング] をクリックします。
4. この分析コードから、[コンフィギュレーション マッピング] をクリックします。
5. [新規] をクリックします。
6. [移動先分析コード] フィールドでは、値を入力または選択します。
    * この例の場合、[MXMF 原価要素] を選択します。  
7. [新規] をクリックします。
8. 一覧で、選択された行をマークします。
9. [移動元分析コード メンバー] フィールドで、値を入力または選択します。
    * この例の場合、分析コード メンバー 606400 [電話 & ファックスの費用] を選択します。  
10. [移動先分析コード メンバー] フィールドでは、値を入力または選択します。
    * この例の場合、分析コード メンバー6001004 [Telefono] を選択します。  
11. [保存] をクリックします。
