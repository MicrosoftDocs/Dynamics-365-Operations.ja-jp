<span data-ttu-id="3db8c-105">この手順では、割増償却の提案方法を説明します。</span><span class="sxs-lookup"><span data-stu-id="3db8c-105">Use this procedure to learn how to propose additional depreciation.</span></span> <span data-ttu-id="3db8c-106">始める前に、固定資産が取得され、通常の減価償却が転記されていることを確認します。</span><span class="sxs-lookup"><span data-stu-id="3db8c-106">Before you begin, be sure that the fixed asset has been acquired and the ordinary depreciation has been posted.</span></span>  <span data-ttu-id="3db8c-107">また、[固定資産コンフィギュレーション キー] が選択されていることを確認します。</span><span class="sxs-lookup"><span data-stu-id="3db8c-107">Also, be sure that the Fixed Assets configuration key is selected.</span></span>
