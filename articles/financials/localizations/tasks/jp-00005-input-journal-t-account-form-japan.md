[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="affa6-104">日本では、仕訳入力には一般的に T 字勘定形式が使用されます。</span><span class="sxs-lookup"><span data-stu-id="affa6-104">In Japan, the T-account format is commonly used to input journal entries.</span></span> 



<span data-ttu-id="affa6-105">この手順では、T 字勘定形式を使用した仕訳入力の作成について説明します。</span><span class="sxs-lookup"><span data-stu-id="affa6-105">This procedure walks you through creating a journal entry using the T-account format.</span></span> 



<span data-ttu-id="affa6-106">この手順では、JPMF デモ会社のデータを使用します。</span><span class="sxs-lookup"><span data-stu-id="affa6-106">This task uses the JPMF demo company data.</span></span>

1. <span data-ttu-id="affa6-107">[総勘定元帳] > [総勘定元帳] > [仕訳入力] > [一般仕訳帳] の順に移動します。</span><span class="sxs-lookup"><span data-stu-id="affa6-107">Go to General ledger > Journal entries > General journals.</span></span>

3. <span data-ttu-id="affa6-109">[名前] フィールドに値を入力します。</span><span class="sxs-lookup"><span data-stu-id="affa6-109">In the Name field, type a value.</span></span>

9. <span data-ttu-id="affa6-115">[説明] フィールドで値を入力します。</span><span class="sxs-lookup"><span data-stu-id="affa6-115">In the Description field, type a value.</span></span>

12. <span data-ttu-id="affa6-118">[勘定] フィールドで、任意の値を指定します。</span><span class="sxs-lookup"><span data-stu-id="affa6-118">In the Account field, specify the desired values.</span></span>
13. <span data-ttu-id="affa6-119">[説明] フィールドに値を入力します。</span><span class="sxs-lookup"><span data-stu-id="affa6-119">In the Description field, type a value.</span></span>
