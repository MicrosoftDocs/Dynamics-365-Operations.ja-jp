# <a name="set-up-payment-fee-for-japan-japan"></a>日本の支払手数料の設定 (日本)

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

このタスクでは、日本の支払手数料の設定について説明します。



このタスクは デモ データ会社 JPMF を使用して作成されました。


## <a name="create-a-bank-group"></a>銀行グループの作成
1. [現金および銀行管理] > [設定] > [銀行グループ] の順に移動します。
2. [新規] をクリックします。
3. [銀行グループ] フィールドに値を入力します。
4. [名前] フィールドに値を入力します。
5. [一般] セクションを展開します。
6. [銀行コード] フィールドに値を入力します。
7. [支店コード] フィールドに値を入力します。
8. [保存] をクリックします。

## <a name="create-a-vendor-payment-fee-group"></a>仕入先の支払手数料グループの作成
1. [買掛金勘定] > [支払の設定] > [ベンダー支払手数料グループ] の順に移動します。
2. [新規] をクリックします。
3. [ベンダー支払手数料グループ] フィールドに値を入力します。
    * 例: 「PaymFee」と入力します。  
4. [説明] フィールドに値を入力します。
5. [保存] をクリックします。

## <a name="specify-a-payment-fee-group-for-a-vendor"></a>仕入先の支払手数料のグループを指定します。
1. [買掛金勘定] > [仕入先] > [すべての仕入先] の順に移動します。
2. 一覧で、選択された行のリンクをクリックします。
3. [編集] をクリックします。
4. [支払] セクションを展開します。
5. [ベンダー支払手数料グループ] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
6. 一覧で、選択された行のリンクをクリックします。
    * 例: 「VendPayFee」 を選択します。  
7. [保存] をクリックします。

## <a name="create-bank-rules-for-payment-fee"></a>銀行手数料組合せの作成
1. [買掛金勘定] > [支払の設定] > [銀行手数料組合せ] の順に移動します。
2. [新規] をクリックします。
3. [ID] フィールドに値を入力します。
4. [名前] フィールドに値を入力します。
5. [一般] セクションを展開します。
6. [追加] をクリックします。
    * 1 つ以上のフィルターを追加し、銀行ルールを作成します。  
7. [サード パーティ銀行グループ] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
    * ベンダーの銀行基準を入力してフィルター処理します。。  
8. 一覧で、目的のレコードを見つけ、選択します。
9. 一覧で、選択された行のリンクをクリックします。
    * 例: 「銀行コード」 を選択します。  
10. [法人の銀行グループ] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
11. 一覧で、選択された行のリンクをクリックします。
    * 支払手数料生成ルールの基準を追加するには、手順を繰り返します。  
    * 1 つの銀行ルールの基準の適用には、「AND」論理を使用します。  
12. [保存] をクリックします。

## <a name="create-a-payment-fee"></a>支払手数料の作成
1. [買掛金勘定] > [支払の設定] > [支払手数料] の順に移動します。
2. [新規] をクリックします。
3. [手数料 ID] フィールドに値を入力します。
4. [名前] フィールドに値を入力します。
5. [手数料説明] フィールドに値を入力します。
6. [請求] フィールドで [元帳] を選択します。
7. [勘定科目] フィールドで、任意の値を指定します。
8. [支払手数料勘定] フィールドで、任意の値を指定します。
9. [仕訳帳タイプ] フィールドで、オプションを選択します。
    * 例: 「仕入れ先出金」を選択します。  
10. [保存] をクリックします。
11. [支払手数料の設定] をクリックします。
12. [グループ化] フィールドで、オプションを選択します。
    * 例: 「グループ」を選択します。  
13. [関連銀行] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
14. 一覧で、選択された行のリンクをクリックします。
    * 例: 「0001_009」を選択します。  
15. [ベンダー グループ タイプ] フィールドで、オプションを選択します。
    * 「グループ」を選択すると、コンフィギュレーションがより簡単になります。  
16. [ベンダー勘定/グループ] フィールドに値を入力します。
    * 例: 「VendPayFee」と入力します。  
17. [支払方法] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
18. 一覧で、選択された行のリンクをクリックします。
    * 例: 「銀行」 を選択します。  
19. [支払手数料の銀行ルール ID] フィールドに値を入力します。
    * 例: 「DiffBank」と入力します。  
20. [一般] タブをクリックします。
21. [支払通貨] フィールドに値を入力します。
    * 例: 「JPY」と入力します。  
22. [最小] フィールドに数値を入力します。
    * 例: 「30000」と入力します。  
23. [割合/金額] フィールドで、オプションを選択します。
    * 例: 「金額」を選択します。  
24. [手数料金額] フィールドに数値を入力します。
    * フィールド「630」の設定  
25. [手数料の通貨] フィールドに値を入力します。
    * 例: 「JPY」と入力します。  
26. [売上税グループ] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
27. 一覧で、選択された行のリンクをクリックします。
    * 例: 「JP」を選択します。  
28. [品目売上税グループ] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
29. 一覧で、選択された行のリンクをクリックします。
    * 例: 「課税対象」 を選択します。  
30. [保存] をクリックします。
