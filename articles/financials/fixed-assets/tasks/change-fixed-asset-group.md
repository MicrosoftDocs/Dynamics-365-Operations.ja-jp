# <a name="change-a-fixed-asset-group"></a>固定資産グループの変更

[!include [task guide banner](../../includes/task-guide-banner.md)]

固定資産は正しい固定資産グループに割り当てる必要があります。 固定資産グループは、次の場合に使用されます。

 - 照会およびレポートの作成

 - 新しい固定資産の設定

 - 元帳の統合、および適切な勘定科目への固定資産トランザクションの転記

このガイドでは、USMF デモ会社を使用します。

1. [固定資産] > [固定資産] > [固定資産] に移動します。
2. 固定資産グループを変更する固定資産を選択します。
3. [固定資産グループの変更] をクリックします。
4. [新規グループ] フィールドで、値を入力または選択します。
5. 選択した固定資産に固定資産番号を割り当てる場合は、このオプションを選択します。
    * [固定資産番号] フィールドは、[新しい固定資産番号] オプションを選択した場合に使用できます。   固定資産に自動番号付けが設定されている場合、次に使用可能な固定資産番号がフィールドに表示されます。 数値は変更できます。   手動での番号付けが設定されている場合、このフィールドは空白で、新しい固定資産番号を入力する必要があります。     
6. [OK] をクリックします。
7. [はい] をクリックします。
