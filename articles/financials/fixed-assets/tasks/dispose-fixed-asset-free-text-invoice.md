[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="be465-104">ここでは、自由書式の請求書を使用した固定資産の廃棄方法を説明します。</span><span class="sxs-lookup"><span data-stu-id="be465-104">This procedure shows how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="be465-105">[売掛金勘定] > [請求書] > [すべての自由書式の請求書] の順に移動します。</span><span class="sxs-lookup"><span data-stu-id="be465-105">Go to Accounts receivable > Invoices > All free text invoices.</span></span>
2. <span data-ttu-id="be465-106">[新規] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="be465-106">Click New.</span></span>
3. <span data-ttu-id="be465-107">[顧客口座] フィールドで値を入力または選択します。</span><span class="sxs-lookup"><span data-stu-id="be465-107">In the Customer account field, enter or select a value.</span></span>
4. <span data-ttu-id="be465-108">既定の請求書日付を検証し、該当する場合は編集します。</span><span class="sxs-lookup"><span data-stu-id="be465-108">Validate the default Invoice date and edit if applicable.</span></span>
5. <span data-ttu-id="be465-109">通貨などの、残りの既定のヘッダー フィールドを検証し、該当する場合は編集します。</span><span class="sxs-lookup"><span data-stu-id="be465-109">Validate remaining default header fields, such as Currency and edit if applicable.</span></span>

8. <span data-ttu-id="be465-112">既定の売上税グループと品目売上税グループを検証します。</span><span class="sxs-lookup"><span data-stu-id="be465-112">Validate the default Sales tax group and Item sales tax group.</span></span>
9. <span data-ttu-id="be465-113">単価または固定資産の売却の金額を入力します。</span><span class="sxs-lookup"><span data-stu-id="be465-113">Enter the Unit price or hte Amount of the sale of the fixed asset.</span></span>
10. <span data-ttu-id="be465-114">明細行の詳細をクリックします。</span><span class="sxs-lookup"><span data-stu-id="be465-114">Click Line details.</span></span>  
11. <span data-ttu-id="be465-115">売却する固定資産番号を選択します。</span><span class="sxs-lookup"><span data-stu-id="be465-115">Select the Fixed asset number to be sold.</span></span>
12. <span data-ttu-id="be465-116">[転記] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="be465-116">Click Post.</span></span>
