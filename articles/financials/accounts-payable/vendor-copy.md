# <a name="copy-vendors-by-using-shared-number-sequences"></a><span data-ttu-id="d9e70-103">共有番号順序を使用して、仕入先をコピーする</span><span class="sxs-lookup"><span data-stu-id="d9e70-103">Copy vendors by using shared number sequences</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="d9e70-104">共有番号順序を使用して、仕入先 ID を割り当てることができます。</span><span class="sxs-lookup"><span data-stu-id="d9e70-104">You can use shared number sequences to assign vendor IDs.</span></span> <span data-ttu-id="d9e70-105">また、共有番号順序を使用して、同じ仕入先 ID を維持したまま、法人間で仕入先をコピーすることもできます。</span><span class="sxs-lookup"><span data-stu-id="d9e70-105">Shared number sequences also let you copy vendors from one legal entity to another legal entity but use the same vendor IDs in both legal entities.</span></span>

## <a name="setup"></a><span data-ttu-id="d9e70-106">セットアップ</span><span class="sxs-lookup"><span data-stu-id="d9e70-106">Setup</span></span>

<span data-ttu-id="d9e70-107">この機能が有効化されるのは、共有番号順序を使用して仕入先 ID を割り当てる場合です。</span><span class="sxs-lookup"><span data-stu-id="d9e70-107">The feature is activated when you use a shared number sequence to assign vendor IDs.</span></span> <span data-ttu-id="d9e70-108">仕入先のコピー先となるすべての法人内で、同じ番号順序を使用する必要があります。</span><span class="sxs-lookup"><span data-stu-id="d9e70-108">You must use the same number sequence in every legal entity that you want to copy a vendor to.</span></span> <span data-ttu-id="d9e70-109">仕入先番号順序を変更するには、各法人に対する **買掛金パラメーター** ページを使用します。</span><span class="sxs-lookup"><span data-stu-id="d9e70-109">You change the vendor number sequence on the **Accounts payable parameters** page for each legal entity.</span></span> <span data-ttu-id="d9e70-110">**買掛金** \> **設定** \> **買掛金パラメーター** を選択し、**番号順序** タブを選択します。</span><span class="sxs-lookup"><span data-stu-id="d9e70-110">Select **Accounts payable** \> **Setup** \> **Accounts payable parameters**, and then select the **Number sequences** tab.</span></span>

<span data-ttu-id="d9e70-111">各仕入先グループに対して仕入先番号順序を指定することもできます。</span><span class="sxs-lookup"><span data-stu-id="d9e70-111">You can also set up vendor number sequences for each vendor group.</span></span> <span data-ttu-id="d9e70-112">また、これらの番号順序を共有する必要があります。</span><span class="sxs-lookup"><span data-stu-id="d9e70-112">These number sequences must also be shared.</span></span> <span data-ttu-id="d9e70-113">仕入先グループに対する番号順序が最初に使用されます。</span><span class="sxs-lookup"><span data-stu-id="d9e70-113">The number sequence for a vendor group is used first.</span></span> <span data-ttu-id="d9e70-114">仕入先グループに対して番号順序を指定しない場合、**買掛金パラメーター** ページで指定した番号順序が使用されます。</span><span class="sxs-lookup"><span data-stu-id="d9e70-114">If no number sequence is specified for a vendor group, the number sequence that is specified on the **Accounts payable parameters** page is used.</span></span>

<span data-ttu-id="d9e70-115">手動仕入先 ID を使用する場合、法人間で仕入先をコピーすることもできます。</span><span class="sxs-lookup"><span data-stu-id="d9e70-115">You can also copy vendors between legal entities if you use manual vendor IDs.</span></span> <span data-ttu-id="d9e70-116">ただし、仕入先 ID が既に存在している法人に仕入先をコピーしようとした場合、コピー プロセスは開始しません。</span><span class="sxs-lookup"><span data-stu-id="d9e70-116">However, if you try to copy a vendor to a legal entity where the vendor ID already exists, the copy process won't be started.</span></span>
