# <a name="key-invoice-data-into-the-ap-system-using-invoice-pool"></a>請求管理グループを使用した AP システムへの請求書データの入力

[!include [task guide banner](../../includes/task-guide-banner.md)]

このタスク ガイドでは、請求書を作成する仕入帳を使用する方法を示します。  その後、請求書を発注書と一致させるために、また仕入先請求書のページにある経費を確定するために、請求書管理グループを使用します。


## <a name="create-a-purchase-order"></a>発注書の作成
1. [買掛金勘定] > [発注書] > [発注書] の順に移動します。
2. [新規] をクリックして新しい発注書を作成します。
3. [仕入先] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
4. 一覧から仕入先を選択します。 例としては、仕入先「1001」です。
5. [OK] をクリックします。
6. [品目番号] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
7. サービス品目番号を一覧で見つけます。 たとえば、「S0001」を選択します。
8. 品目番号をクリックし、それを選択します。
    * 正味金額は 75.00 です。  これは、請求書上に表示される金額です。  
9. [アクション] ペインで、[購買] をクリックします。
10. [確認] をクリックします。

## <a name="create-and-post-and-invoice"></a>請求書の作成と転記
1. [買掛金勘定] > [請求書] > [仕入帳] の順に移動します。
2. [新規] をクリックします。
3. ルックアップを開いて、使用する仕入帳の名前を選択します。
4. 使用する仕入帳の名前を選択します。
5. 登録を開いて経費明細行を入力するには [明細行] をクリックします。
6. ルックアップで、仕入先を選択します。 たとえば、「仕入先 1001」をクリックします。
7. [請求書] フィールドに請求書番号を入力します。
8. [説明] フィールドで値を入力します。
9. [貸方] フィールドに数値を入力します。
10. [発注書] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
11. 先ほど作成した発注書を選択します。
12. [承認者] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
13. 承認者を強調表示して [選択] をクリックすると、その承認者を選択します。
14. [転記] をクリックします。
15. フォームを閉じます。
16. フォームを閉じます。

## <a name="open-an-invoice-from-the-pool-and-match-it-to-a-purchase-order-to-complete-the-invoice-process"></a>プールから請求書を開き、発注書と照合させ、請求書プロセスを完了させます。
1. [買掛金勘定] > [請求書] > [請求管理グループ] の順に移動します。
2. [発注書] をクリックして、プール内の発注書から仕入先請求書を作成します。
3. 確認する請求書を選択します。
4. [照合状態の更新] をクリックして、照合を完了させます。
5. [アクション] ウィンドウで、[オプション] をクリックします。
6. [ビューの変更] をクリックします。
7. [グリッド ビュー] をクリックします。
8. [転記] をクリックします。
9. フォームを閉じます。
10. [買掛金勘定] > [仕入先] > [仕入先] の順に移動します。
11. 発注書に記載されていた仕入先を選択します。 たとえば、「仕入先 1001」を選択します。
12. 一覧で、選択された行のリンクをクリックします。
13. [アクション] ウィンドウで、[仕入先] をクリックします。
14. [トランザクション] をクリックします。
15. 作成した請求書を選択します。
    * 仕入帳昇給額は取り消されて、適切な経費勘定に転記されます。
