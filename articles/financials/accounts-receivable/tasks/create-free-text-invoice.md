# <a name="create-a-free-text-invoice"></a>自由書式の請求書の作成

[!include[task guide banner](../../includes/task-guide-banner.md)]

このタスク ガイドでは、自由書式の請求書の作成について説明します。 このタスクでは、USMF というデモ会社を使用します。

1. [売掛金勘定] > [請求書] > [すべての自由書式の請求書] の順に移動します。
2. [新規] をクリックします。
3. [顧客口座] フィールドで値を選択します。
    * 請求先/元 ID は、既定では、顧客口座に使用される口座と同じになります。   
    * 請求書が転記されていない場合、勘定状態は [処理中] から始まります。   
    * 請求書番号は、請求書の転記時に割り当てられます。  
    * SEPA 委任状を使用している場合、口座引落の委任状は、顧客口座の選択時に自動的に委任状から作成されます。  
4. [説明] フィールドに値を入力します。
5. 主勘定のフィールドに、分析コードなしで勘定番号を指定します。
    * また、主勘定について 1 文字以上を入力し、ルックアップを使用して勘定を検索することもできます。 このガイドの後のほうで、分析コードを入力します。  
6. [明細行の詳細] クイック タブを展開すると、自分の主勘定に分析コードを追加できます。
7. [財務分析コード行] タブをクリックします。
    * 分析コードは、選択した明細行のみに有効です。    
    * 売上税グループは、顧客から入力されます。 顧客に売上税グループがない場合、主勘定の売上税グループが使用されます。  
    * 品目の売上税グループは、主勘定から作成されます。 主勘定に品目売上税グループがない場合、総勘定元帳の売上税パラメーターの品目売上税グループが使用されます。    
8. [数量] フィールドに数値を入力します。
    * 数量はオプションです。  
9. [単価] フィールドに数値を入力します。
    * 単価はオプションです。  
    * 金額は、数量に単価を乗じて計算されます。 ただし、金額を入力してその計算を上書きできます。  
10. 請求書の売上税の計算を表示するには、[売上税] をクリックします。
    * このページで売上税金額を確認するか、[調整] タブで金額を上書きできます。  
11. [OK] をクリックします。
12. 請求書に手数料を追加するには、[手数料] をクリックします。 
13. [諸費用コード] フィールドに値を入力します。
14. [諸費用金額] フィールドに数値を入力します。
15. ページを閉じます。
16. 請求書集計表の詳細および合計を表示するには、[合計] をクリックします。
17. [閉じる] をクリックします。
18. [転記] をクリックして請求書を転記します。 転記する前にキャンセルできます。
    * 請求書の印刷タイミングの変更: [現在] を選択すると、各請求書の更新時に印刷します。[後] を選択すると、すべての請求書を更新した後に印刷します。  
    * 顧客の与信限度額の転記前チェックの方法を変更する場合は、与信限度額のタイプを変更します。  
    * 請求書を印刷する場合、[はい] を選択します。  
    * 請求書を転記する場合、[はい] を選択します。 転記せずに請求書を印刷することもできます。  
19. [OK] をクリックします。
