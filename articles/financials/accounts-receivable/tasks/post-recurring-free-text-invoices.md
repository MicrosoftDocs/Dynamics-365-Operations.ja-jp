# <a name="generate-and-post-recurring-free-text-invoices"></a>定期的な自由書式の請求書の生成と転記

[!include [task guide banner](../../includes/task-guide-banner.md)]

定期請求書は、顧客に対して同じ金額を定期的に請求する際に使用します。 このレコードでは、USMF デモ会社を使用します。 記録は A/R 請求書の管理および処理を行う担当者を対象としています。


## <a name="generate-recurring-invoices"></a>定期請求書の生成

## <a name="post-recurring-invoices"></a>定期請求書の転記
1. [売掛金勘定] > [請求書] > [定期請求書] > [定期請求書の転記] の順に移動します。
    * 既に生成されている定期請求書に表示および印刷されるこのページを使用します。  
2. 一覧で、選択された行のリンクをクリックします。
    * 定期請求書グループを選択します。  
3. [合計] をクリックします。
    * 定期請求書グループの合計を確認します。  
4. [閉じる] をクリックします。
    * 各行の下には、定期的な自由書式の請求書が記載されます。 行を選択し、[詳細] ボタンをクリックし、自由書式の請求書の詳細を表示します。  
5. [検証] をクリックします。
    * 選択した請求書にエラーがないことを確認してください。しかし、請求書は転記しません。  
6. [転記] をクリックします。
    * 選択した請求書を転記します。
