# <a name="export-letter-of-credit"></a>輸出信用状

[!include [task guide banner](../../includes/task-guide-banner.md)]

この手順は、[輸出信用状] のプロセスを説明します。

信用状は銀行が発行する契約であり、銀行は、買主と売主の契約が満たされた場合に買主に代わって確実に支払を行うことに同意します。



この手順の前に、「銀行融資と転記プロファイルの設定」手順および「信用状 _ 銀行融資契約の作成」手順を実行します。 この手順を正常に実行するためには、USMF のデモ会社が選択されている必要があります。




## <a name="create-sales-order-for-export-letter-of-credit"></a>[輸出信用状] の販売注文の作成
1. [売掛金勘定] > [注文] > [すべての販売注文] の順に移動します。
2. [新規] をクリックします。
3. [顧客口座] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
4. 一覧で、目的のレコードを見つけ、選択します。
5. 一覧で、選択された行のリンクをクリックします。
6. [全般] セクションを展開または折りたたみます。
7. [サイト] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
    * 発行される品目が保管されているサイトを選択します。  
8. 一覧で、選択された行のリンクをクリックします。
9. [倉庫] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
    * 発行される品目が保管されている [倉庫] を選択します。  
10. 一覧で、選択された行のリンクをクリックします。
    * 注記 : [銀行ドキュメント タイプ] フィールドは「信用状」の値と共に選択されているべきです。  
11. [銀行ドキュメント タイプ] フィールドで「信用状」を選択します。
12. [配送] セクションを展開または折りたたみます。
    * [配送日の管理] = [なし] を選択します。  
13. [入荷希望日] フィールドに日付を入力します。
14. [OK] をクリックします。
15. [品目番号] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
    * 必要な品目を [出庫 / 売却済] として選択します。  
16. 一覧で、目的のレコードを見つけ、選択します。
17. 一覧で、選択された行のリンクをクリックします。
18. [単価] フィールドに数値を入力します。
19. [明細行の詳細] セクションを展開または折りたたみます。
20. [出荷] タブをクリックします。
21. [指定出荷日] フィールドに日付を入力します。
22. [確定出荷日] フィールドに日付を入力します。
23. [アクション] ウィンドウで、[管理] をクリックします。
24. [信用状] をクリックします。
25. [銀行ドキュメント番号] フィールドに値を入力します。
26. [有効期限] フィールドに日時を入力します。
27. [銀行詳細] セクションを展開または折りたたみます。
28. [発行銀行] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
29. 一覧で、選択された行のリンクをクリックします。
30. [通知銀行] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
31. 一覧で、目的のレコードを見つけ、選択します。
32. 一覧で、選択された行のリンクをクリックします。
33. [販売注文の出荷のフェッチ] をクリックします。
34. [銀行ドキュメントの発行] をクリックします。
35. ページを閉じます。

## <a name="post-packing-slip"></a>[梱包明細] の転記
1. アクション ウィンドウで、[ピッキングと梱包] をクリックします。
2. [梱包明細の転記] をクリックします。
3. [パラメーター] セクションを展開または折りたたみます。
4. [数量] フィールドで [すべて] を選択します。
5. [設定] セクションを展開または折りたたみます。
6. [梱包明細日] フィールドに日付を入力します。
7. [出荷番号] を選択します。
8. 一覧で、選択された行のリンクをクリックします。
9. [OK] をクリックします。
10. [OK] をクリックします。

## <a name="post-sales-invoice"></a>売上請求書の転記
1. アクション ウィンドウで、[請求書] をクリックします。
2. [請求書] をクリックします。
3. [概要] セクションを展開または折りたたみます。
4. [出荷番号] を選択します。
5. 一覧で、選択された行のリンクをクリックします。
6. [設定] セクションを展開または折りたたみます。
7. [請求日] フィールドに日付を入力します。
8. [OK] をクリックします。
9. [OK] をクリックします。

## <a name="shipment-document-submitted-status"></a>出荷ドキュメントの送信済ステータス
1. [アクション] ウィンドウで、[管理] をクリックします。
2. [信用状] をクリックします。
3. [明細] セクションを展開または折りたたみます。
    * 注記 : 「ドキュメント送信」フィールドは「はい」に設定されます。  

## <a name="verify-export-letter-of-credit"></a>[輸出信用状] の確認
1. [現金および銀行管理] > [信用状] > [輸出信用状] および [輸入取立] の順に移動します。
2. 一覧で、目的のレコードを見つけ、選択します。
3. 一覧で、選択された行のリンクをクリックします。
    * [輸出信用状] が「請求済」の [出荷状態] であることを確認します。  

## <a name="customer-payment"></a>顧客支払
1. [売掛金勘定] > [支払] > [支払仕訳帳] の順に移動します。
2. [新規] をクリックします。
3. 一覧で、選択された行をマークします。
4. [名前] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
5. 一覧で、選択された行のリンクをクリックします。
6. [行] をクリックします。
7. [日付] フィールドに日付を入力します。
8. [勘定] フィールドで、任意の値を指定します。
9. [決済] をクリックします。
10. [合計] のヘッダーにあるチェック ボックスをオンにします。
    * 注記:「表示」フィールドを「信用状」に設定します。  
11. 一覧で、目的のレコードを見つけ、選択します。
12. [マーク] のチェック ボックスをオンまたはオフにします。
13. [OK] をクリックします。
14. [支払] タブをクリックします。
    * [銀行ドキュメント番号] と [出荷番号の詳細] を確認します。  
15. [転記] をクリックします。

## <a name="verify-export-letter-of-credit-after-payment"></a>支払後の [輸出信用状] の確認
1. [現金および銀行管理] > [信用状] > [輸出信用状] および [輸入取立] の順に移動します。
2. 一覧で、目的のレコードを見つけ、選択します。
3. 一覧で、選択された行のリンクをクリックします。
    * [出荷ステータス] = [支払領収済] および [残高金額] = 0.00 となることを確認します。
