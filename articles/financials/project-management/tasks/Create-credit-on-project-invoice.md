# <a name="create-a-credit-note-on-project-invoices"></a>プロジェクト請求書の貸方票の作成

[!include [task guide banner](../../includes/task-guide-banner.md)]

1. [プロジェクト管理および会計] > [プロジェクト] > [すべてのプロジェクト] の順に選択します。 
2. 一覧で、目的のレコードを見つけ、選択します。 
3. 貸方票に使用されるプロジェクト ID を選択します。 一覧で、選択された行のリンクをクリックします。 
4. [アクション] ウィンドウで、[管理] をクリックします。 
5. [請求仕訳帳] をクリックします。 
6. 貸方票に対してトランザクションを選択します。 一覧で、選択された行をマークします。 
7. [機能] をクリックします。 
8. 貸方票の選択をクリックします。 
9. 貸方票の作成対象となるトランザクションは、チェック ボックスをオンにして選択します。
10. [OK] をクリックします。 

この手順で貸方票を仮発行請求書に使用できるようになります。
