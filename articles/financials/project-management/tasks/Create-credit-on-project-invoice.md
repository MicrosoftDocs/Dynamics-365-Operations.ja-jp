# <a name="create-a-credit-note-on-project-invoices"></a><span data-ttu-id="cf171-103">プロジェクト請求書の貸方票の作成</span><span class="sxs-lookup"><span data-stu-id="cf171-103">Create a credit note on project invoices</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

1. <span data-ttu-id="cf171-104">[プロジェクト管理および会計] > [プロジェクト] > [すべてのプロジェクト] の順に選択します。</span><span class="sxs-lookup"><span data-stu-id="cf171-104">Go to Project management and accounting > Projects > All projects.</span></span> 
2. <span data-ttu-id="cf171-105">一覧で、目的のレコードを見つけ、選択します。</span><span class="sxs-lookup"><span data-stu-id="cf171-105">In the list, find and select the desired record.</span></span> 
3. <span data-ttu-id="cf171-106">貸方票に使用されるプロジェクト ID を選択します。</span><span class="sxs-lookup"><span data-stu-id="cf171-106">Select the Project ID that is used for the Credit Note.</span></span> <span data-ttu-id="cf171-107">一覧で、選択された行のリンクをクリックします。</span><span class="sxs-lookup"><span data-stu-id="cf171-107">In the list, click the link in the selected row.</span></span> 
4. <span data-ttu-id="cf171-108">[アクション] ウィンドウで、[管理] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="cf171-108">On the Action Pane, click Manage.</span></span> 
5. <span data-ttu-id="cf171-109">[請求仕訳帳] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="cf171-109">Click Invoice journals.</span></span> 
6. <span data-ttu-id="cf171-110">貸方票に対してトランザクションを選択します。</span><span class="sxs-lookup"><span data-stu-id="cf171-110">Select the transaction for credit note.</span></span> <span data-ttu-id="cf171-111">一覧で、選択された行をマークします。</span><span class="sxs-lookup"><span data-stu-id="cf171-111">In the list, mark the selected row.</span></span> 
7. <span data-ttu-id="cf171-112">[機能] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="cf171-112">Click Functions.</span></span> 
8. <span data-ttu-id="cf171-113">貸方票の選択をクリックします。</span><span class="sxs-lookup"><span data-stu-id="cf171-113">Click Select for credit note.</span></span> 
9. <span data-ttu-id="cf171-114">貸方票の作成対象となるトランザクションは、チェック ボックスをオンにして選択します。</span><span class="sxs-lookup"><span data-stu-id="cf171-114">Select the transaction for credit note by marking the check box.</span></span>
10. <span data-ttu-id="cf171-115">[OK] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="cf171-115">Click OK.</span></span> 

<span data-ttu-id="cf171-116">この手順で貸方票を仮発行請求書に使用できるようになります。</span><span class="sxs-lookup"><span data-stu-id="cf171-116">After this step, the Credit Note will be available for Invoice Proposal.</span></span>
