# <a name="create-and-validate-journals"></a>仕訳帳の作成および検証

[!include [task guide banner](../../includes/task-guide-banner.md)]

このタスク ガイドでは、仕訳帳および仕訳帳明細行を作成し、検証します。 このタスクでは、USMF というデモ会社を使用します。  



1. [総勘定元帳] > [総勘定元帳] > [仕訳入力] > [一般仕訳帳] の順に移動します。
2. [新規] をクリックします。
3. [名前] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
4. 一覧で、目的のレコードを見つけ、選択します。
5. [行] をクリックします。
6. [勘定] フィールドに [勘定タイプ] に基づいた適切な勘定を入力します。
7. [説明] フィールドで値を入力します。
8. 借方または貸方勘定の金額を入力します。 このタスク ガイドでは、借方金額を想定しています。
9. [相手勘定] フィールドに [相手勘定タイプ] に基づいた適切な勘定を入力します。
10. [検証] をクリックします。
11. [検証] をクリックします。
12. [転記] をクリックします。
13. [伝票] をクリックします。
