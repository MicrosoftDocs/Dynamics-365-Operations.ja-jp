# <a name="define-audit-policies-for-source-documents"></a>元伝票の監査ポリシーの定義

[!include[task guide banner](../../includes/task-guide-banner.md)]

この手順は、監査ポリシー ルールを設定および実行する方法を示します。 この例では、ホテル経費タイプの経費精算書を使用しています。 この手順では、USMF というデモ会社を使用します。 監査担当者ロールには、これらのタスクを実行する適切なアクセス許可があります。

1. [監査ワークベンチ] > [設定] > [ポリシー ルール タイプ] の順に移動します。
2. [新規] をクリックします。
3. [ルール名] フィールドに値を入力します。
4. [説明] フィールドに値を入力します。
5. [クエリ名] フィールドで [経費精算書の明細行] を選択
6. クエリ タイプ フィールドで [集計] を選択
7. [法人] フィールドで [法人] を選択
8. [文書日付の参照] フィールドで [変更日時] を選択
9. [保存] をクリックします。
10. [監査ワークベンチ] > [設定] > [監査ポリシー] の順に移動します。
11. [新規] をクリックします。
12. [名前] フィールドに値を入力します。
13. [ポリシーの組織] のセクションを展開します。
14. ツリーで、「Contoso Entertainment System USA」を選択します。
15. [追加] をクリックします。
16. ツリーで、「Contoso Consulting USA」を選択します。
17. [追加] をクリックします。
18. ツリーで、「Contoso Retail USA」を選択します。
19. [追加] をクリックします。
20. [ポリシーの組織] のセクションを折りたたみます。
21. [ポリシー ルール] のセクションを展開します。
22. 一覧で、以前に作成したポリシー ルールを見つけて選択します。
23. [ポリシー ルールの作成] をクリックします。
24. [有効日] フィールドに日時を入力します。
25. [フィルター] をクリックします。
26. リストで、[経費カテゴリ] の行を選択し、詳細を [ホテル] に設定
27. [基準] フィールドで、値を入力または選択します。
28. [集計] タブをクリックします。
29. [追加] をクリックします。
30. リストで、トランザクション金額フィールドの値を選択
31. [フィールド] フィールドで、値を入力または選択します。
32. 集計関数のフィールドで、「合計」を選択します。
33. [グループ化] タブをクリックします。
34. [追加] をクリックします。
35. 一覧で、[従業員] の値を選択 
36. [追加] をクリックします。
37. 一覧で、[経費カテゴリ] の値を選択
38. [フィールド] フィールドで、値を入力または選択します。
39. [HAVING] タブをクリックします。
40. [追加] をクリックします。
41. [トランザクション金額] を選択
42. [フィールド] フィールドで、値を入力または選択します。
43. 集計関数のフィールドで、「合計」を選択します。
44. [基準] フィールドで、「>2000」と入力します。
45. [OK] をクリックします。
46. [テスト] をクリックします。
47. [ドキュメントの選択の開始日] フィールドで日時を入力します。
48. [ドキュメントの選択の終了日] フィールドで日時を入力します。
49. [テストの実行] をクリックします。
50. [アクション] ペインで、[監査ポリシー] をクリックします。
51. [追加のオプション] をクリックします。
52. [開始日] フィールドで、日付と時刻を入力します。
53. [終了日] フィールドに日時を入力します。
54. [バッチ] をクリックします。
55. [バックグラウンドで実行] セクションを展開します。
56. [バッチ処理] フィールドで [はい] を選択します。
57. [OK] をクリックします。
58. [監査ワークベンチ] > [監査ケース] の順に移動します。
59. 一覧で、目的のレコードを見つけ、選択します。
60. 一覧で、選択された行のリンクをクリックします。
61. [関連] セクションを展開します。
62. 一覧で、目的のレコードを見つけ、選択します。
63. 一覧で、選択された行のリンクをクリックします。
