<span data-ttu-id="a7b10-105">会計年度末には、新しい年に開始残高を転送するために年度末決算処理を実行する必要があります。</span><span class="sxs-lookup"><span data-stu-id="a7b10-105">At the end of a fiscal year, you must run the year-end close process to transfer opening balances to the new year.</span></span> <span data-ttu-id="a7b10-106">ほとんどの組織は年度末決算処理を複数回実行します。</span><span class="sxs-lookup"><span data-stu-id="a7b10-106">Most organizations will run the year-end close process multiple times.</span></span> <span data-ttu-id="a7b10-107">1 回目は、残高を新しい会計年度に移動させます。</span><span class="sxs-lookup"><span data-stu-id="a7b10-107">The first time would be to get the balances moved into the new fiscal year.</span></span> <span data-ttu-id="a7b10-108">年度末決算は、残高を調整エントリから新しい会計年度に移動するため、その後必要に応じて何回でも実行することができます。</span><span class="sxs-lookup"><span data-stu-id="a7b10-108">The year-end close can then be run again, as many times are required, to move the balances from adjusting entries into the new fiscal year.</span></span>
