# <a name="container-module"></a><span data-ttu-id="f82ff-103">コンテナー モジュール</span><span class="sxs-lookup"><span data-stu-id="f82ff-103">Container module</span></span>

<span data-ttu-id="f82ff-104">このトピックでは、コンテナー モジュールと、Microsoft Dynamics 365 Commerce のサイト ページにそれを追加する方法について説明します。</span><span class="sxs-lookup"><span data-stu-id="f82ff-104">This topic covers container modules and describes how to add them to site pages in Microsoft Dynamics 365 Commerce.</span></span>

## <a name="overview"></a><span data-ttu-id="f82ff-105">概要</span><span class="sxs-lookup"><span data-stu-id="f82ff-105">Overview</span></span>

<span data-ttu-id="f82ff-106">コンテナー モジュールは、その中の他のモジュールをホストするモジュールです。</span><span class="sxs-lookup"><span data-stu-id="f82ff-106">A container module is a module that hosts other modules inside it.</span></span> <span data-ttu-id="f82ff-107">コンテナー モジュールの主な目的は、それに対して設定されているプロパティを通じて、内部に含まれるモジュールのレイアウトを定義することです。</span><span class="sxs-lookup"><span data-stu-id="f82ff-107">The primary purpose of a container module is to define, through the properties that are set for it, the layout of the modules that it contains.</span></span> <span data-ttu-id="f82ff-108">たとえば、それらのモジュールは 2 列、3 列、4 列、または 6 列のレイアウトで左右に並んで表示できます。</span><span class="sxs-lookup"><span data-stu-id="f82ff-108">For example, those modules can appear side by side in a two-column, three-column, four-column, or six-column layout.</span></span> <span data-ttu-id="f82ff-109">また、コンテナーの幅に限定することも、画面全体に表示することもできます。</span><span class="sxs-lookup"><span data-stu-id="f82ff-109">They can also be limited to the width of the container, or they can fill the screen.</span></span> <span data-ttu-id="f82ff-110">ヘッダーは、各コンテナー モジュールに追加することもできます。</span><span class="sxs-lookup"><span data-stu-id="f82ff-110">A heading can also be added to every container module.</span></span>

<span data-ttu-id="f82ff-111">コンテナー、2 スロットのコンテナー、3 スロットのコンテナーの 3 つのコンテナー モジュールがサポートされています。</span><span class="sxs-lookup"><span data-stu-id="f82ff-111">Three container modules are supported: container, container with 2-slots, and container with 3-slots.</span></span> <span data-ttu-id="f82ff-112">任意のタイプのモジュールを、これらのコンテナー内に配置できます。</span><span class="sxs-lookup"><span data-stu-id="f82ff-112">Modules of any type can be put inside these containers.</span></span>
