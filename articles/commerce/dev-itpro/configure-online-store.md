# <a name="configure-online-stores"></a>オンライン ストアの構成

[!include [banner](../includes/banner.md)]

この記事では、オンライン ストアの一元的な構成と管理に役立つ記事へのリンクを示します。

次の表に一覧表示されている記事は、クライアントでコマース コンポーネントと小売りオンライン ストアをコンフィギュレーションするのに役立ちます。

## <a name="configure-an-online-store"></a>オンライン ストアのコンフィギュレーション

| タスク                                                | 詳細                                                                                                                                                                                                                                                                                                                                                   | 記事                                                                                                                                                                                                                                                                                                                                                                                                                                   |
|-----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| コンポーネントの構成をします。                        | コマース操作の情報を設定および管理します。 この情報には、店舗、税、製品、ギフト カード、プロモーション、割引が含まれます。                                                                                                                                                                                                          | [小売の設定および管理](/dynamicsax-2012/appuser-itpro/setting-up-and-maintaining-retail) (Microsoft Dynamics AX 2012 用 TechNet コンテンツ)                                                                                                                                                                                                                                                                                          |
| チャネル ナビゲーション階層をコンフィギュレーションします。    | オンライン ストアを通じて提供する製品のカテゴリ構造を設定するために、チャンネル ナビゲーション カテゴリ階層を作成します。 カテゴリ階層を定義し、カテゴリに製品、製品属性グループ、および属性値を割り当てます。 その後、オンライン ストアにカテゴリ階層を割り当てます。                            | [小売階層の設定](/dynamicsax-2012/appuser-itpro/set-up-a-retail-hierarchy)</br> (AX 2012 用 TechNet コンテンツ)</br> [属性および属性タイプ](/dynamicsax-2012/appuser-itpro/set-up-attributes-and-attribute-types)の設定 (AX 2012 用 TechNet コンテンツ)</br> [小売属性グループ](/dynamicsax-2012/appuser-itpro/set-up-retail-attribute-groups)の設定 (AX 2012 用 TechNet コンテンツ) |
| 組織階層にオンライン ストアを追加します。 | 作成したオンライン ストアで製品の品揃えを割り当てたり、注文を満たしたり、またはそのオンライン ストアの情報を含むレポートを生成したりするには、1 つ以上の組織階層に店舗を割り当てる必要があります。 少なくとも、オンライン ストアを製品の品揃えを含む組織階層に割り当てる必要があります。 | [オンライン ストアの設定](/dynamicsax-2012/appuser-itpro/set-up-an-online-store) (AX 2012 用 TechNet コンテンツ)                                                                                                                                                                                                                                                                                                     |
| オンライン ストアに荷渡方法を追加します。          | オンライン ストアが提供する配送方法を選択します。                                                                                                                                                                                                                                                                                                 | [オンライン ストアの設定](/dynamicsax-2012/appuser-itpro/set-up-an-online-store) (Microsoft AX 2012 用 TechNe コンテンツ)                                                                                                                                                                                                                                                                                                     |
| 属性をマップし、メタデータを追加します。                   | 各カテゴリまたはチャネル製品の属性が Microsoft SharePoint サイトのオンライン ストアでどのように動作するかを示すオプションを選択します。                                                                                                                                                                                              | [オンライン ストアの設定](/dynamicsax-2012/appuser-itpro/set-up-an-online-store) (AX 2012 用 TechNet コンテンツ)                                                                                                                                                                                                                                                                                                     |
