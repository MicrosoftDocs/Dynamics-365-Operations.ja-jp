# <a name="browser-requirements-for-dynamics-365-commerce-sites"></a>Dynamics 365 Commerce サイトのブラウザー要件

[!include [banner](includes/banner.md)]

このトピックでは、Microsoft Dynamics 365 Commerce サイトのブラウザー要件について説明します。

Dynamics 365 Commerce サイトをもっともサポートする Web ブラウザーは次のとおりです。 各 Web ブラウザーの最新版を検索するには、ソフトウェア メーカーの Web サイトに移動します。

- [Microsoft Edge](https://www.microsoft.com/edge)(現在のバージョンから 1 を引いた値まで)
- [Mozilla Firefox](https://www.mozilla.org/firefox/new) (現在のバージョンから 1 を引いた値まで)
- [Google Chrome](https://www.google.com/chrome) (現在のバージョンから 1 を引いた値まで)
- [Apple Safari](https://support.apple.com/downloads/safari) (現在のバージョンから 1 を引いた値まで)

> [!NOTE]
> Internet Explorer はサポートされていません。 Internet Explorer を使用する場合、一部の機能を使用できないため、ユーザーのエクスペリエンスは標準以下になる場合があります。
