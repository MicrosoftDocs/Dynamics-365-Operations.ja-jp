# <a name="digital-asset-management-overview"></a>デジタル資産管理の概要

[!include [banner](includes/banner.md)]

この記事では、Microsoft Dynamics 365 Commerce サイト ビルダーのデジタル資産管理の概要を示します。

Dynamics 365 Commerce サイト ビルダーのメディア ライブラリー機能には、次のようなリッチデジタル資産管理機能が含まれています。
- 画像資産
- ビデオ資産
- Office ドキュメントや PDF ファイルなど、その他のタイプのバイナリ資産
- ローカライズされた画像
- フォルダー アップロード
- 画像のクロッピング
- 画像の焦点のカスタマイズ
- オムニ チャネル画像資産 (製品、製品バリアント、カタログ、カテゴリ、作業者、顧客の画像)

デジタル資産はチャネル固有であり、既定ではオムニ チャネル資産を除いて、環境全体で使用されます。 

サイト内の左側のナビゲーション ウィンドウの **メディア ライブラリー** 下にあるサイト ビルダー内に、デジタル資産と管理機能が配置されています。
