[!include [banner](includes/banner.md)]

<span data-ttu-id="7f3ba-104">このトピックでは、Microsoft Dynamics 365 Commerce サイト ビルダーの画像のトリミング方法について説明します。</span><span class="sxs-lookup"><span data-stu-id="7f3ba-104">This topic describes how to crop images in Microsoft Dynamics 365 Commerce site builder.</span></span>

<span data-ttu-id="7f3ba-105">Commerce サイト ビルダーのメディア ライブラリーを使用すると、画像をトリミングして、さまざまな種類のモジュールとビューポートに対して最適化することができます。</span><span class="sxs-lookup"><span data-stu-id="7f3ba-105">The Commerce site builder Media Library allows you to crop images to optimize them for different module types and viewports.</span></span>

## <a name="crop-an-image"></a><span data-ttu-id="7f3ba-106">画像のトリミング</span><span class="sxs-lookup"><span data-stu-id="7f3ba-106">Crop an image</span></span>

<span data-ttu-id="7f3ba-107">サイト ビルダーの画像をトリミングするには、次の手順を実行します。</span><span class="sxs-lookup"><span data-stu-id="7f3ba-107">To crop an image in site builder, follow these steps.</span></span>

1. <span data-ttu-id="7f3ba-108">Commerce サイト ビルダーの左側のナビゲーション ウィンドウで、**メディア ライブラリー** を選択します。</span><span class="sxs-lookup"><span data-stu-id="7f3ba-108">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="7f3ba-109">メイン ウィンドウで、変更する画像を選択します。</span><span class="sxs-lookup"><span data-stu-id="7f3ba-109">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="7f3ba-110">コマンド バーで、**編集** を選択します。</span><span class="sxs-lookup"><span data-stu-id="7f3ba-110">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="7f3ba-111">画像を選択して、**編集モード** を入力します。</span><span class="sxs-lookup"><span data-stu-id="7f3ba-111">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="7f3ba-112">**編集モード** で、**モジュール別に画像編集** を選択します。</span><span class="sxs-lookup"><span data-stu-id="7f3ba-112">Under **Edit Mode**, select **Edit View by Module**.</span></span>
1. <span data-ttu-id="7f3ba-113">**モジュール** ドロップダウン メニューから、モジュール タイプを選択します。</span><span class="sxs-lookup"><span data-stu-id="7f3ba-113">From the **Module** drop-down menu, select the module type.</span></span>
1. <span data-ttu-id="7f3ba-114">**ビュー タイプ** ドロップダウン メニューから、画像の種類を選択します。</span><span class="sxs-lookup"><span data-stu-id="7f3ba-114">From the **View type** drop-down menu, select the view type.</span></span>
1. <span data-ttu-id="7f3ba-115">**配置** ドロップダウン メニューから、画像配置を選択します。</span><span class="sxs-lookup"><span data-stu-id="7f3ba-115">From the **Placement** drop-down menu, select the image placement.</span></span>
1. <span data-ttu-id="7f3ba-116">**ビューポート** ドロップダウン メニューから、ビューポート サイズを選択します。</span><span class="sxs-lookup"><span data-stu-id="7f3ba-116">From the **Viewport** drop-down menu, select the viewport size.</span></span>
1. <span data-ttu-id="7f3ba-117">画像は、トリミング領域を表す区分に重ねて付けられます。</span><span class="sxs-lookup"><span data-stu-id="7f3ba-117">The image is overlaid with the area representing the crop region.</span></span> <span data-ttu-id="7f3ba-118">必要に応じて、トリミング領域を移動し、サイズを変更します。</span><span class="sxs-lookup"><span data-stu-id="7f3ba-118">Move and resize the crop region as needed.</span></span> <span data-ttu-id="7f3ba-119">縦横比は自動的に管理されます。</span><span class="sxs-lookup"><span data-stu-id="7f3ba-119">The aspect ratio will be maintained automatically.</span></span>
1. <span data-ttu-id="7f3ba-120">完了したら、コマンド バーで **保存** を選択し、**編集完了** を選択します。</span><span class="sxs-lookup"><span data-stu-id="7f3ba-120">When you're done, on the command bar, select **Save**, and then select **Finish editing**.</span></span> 

<span data-ttu-id="7f3ba-121">カスタム トリミングの完了後、画像の変更はすぐに有効になります。</span><span class="sxs-lookup"><span data-stu-id="7f3ba-121">After custom cropping is completed, image modifications will take effect almost immediately.</span></span>
