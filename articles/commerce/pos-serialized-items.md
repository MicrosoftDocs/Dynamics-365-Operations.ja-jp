## <a name="serial-number-configurations"></a><span data-ttu-id="072f6-109">シリアル番号の構成</span><span class="sxs-lookup"><span data-stu-id="072f6-109">Serial number configurations</span></span>

<span data-ttu-id="072f6-110">ある品目に対して、シリアル番号の使用が設定された追跡用分析コードのグループが割り当てられている場合、その品目はシリアル品目と見なされます。</span><span class="sxs-lookup"><span data-stu-id="072f6-110">An item is considered a serialized item if it's assigned a tracking dimension group that is set up to allow for serial numbers.</span></span> <span data-ttu-id="072f6-111">**追跡用分析コード グループ** ページの Commerce Headquarters にて、**有効化** オプションを選択して在庫プロセスにおけるシリアル番号管理を有効化します。</span><span class="sxs-lookup"><span data-stu-id="072f6-111">In Commerce headquarters, on the **Tracking dimension groups** page, select the **Active** option to enable serial numbers for the inventory process.</span></span> <span data-ttu-id="072f6-112">販売プロセスにおけるシリアル番号管理を有効にするには、 **販売プロセスで有効化** を選択します。</span><span class="sxs-lookup"><span data-stu-id="072f6-112">To enable serial numbers for the sales process, select the **Active in sales process** option.</span></span>

<span data-ttu-id="072f6-113">**追跡用分析コード** クイックタブで、**ブランク入庫を許可する** オプションが有効になっている場合は、シリアル番号は在庫入庫の過程では任意入力となります。</span><span class="sxs-lookup"><span data-stu-id="072f6-113">On the **Tracking dimensions** FastTab, if the **Blank receipt allowed** option is turned on, the serial number is an optional input during the inventory receipt process.</span></span> <span data-ttu-id="072f6-114">このオプションがオフになっている場合は、シリアル番号が必須となります。</span><span class="sxs-lookup"><span data-stu-id="072f6-114">If the option is turned off, the serial number is required.</span></span>

<span data-ttu-id="072f6-115">**シリアル番号** クイックタブで、**シリアル番号管理** オプションがオンになっている場合は、ユニットごとに一意のシリアル番号を使用する必要があります。</span><span class="sxs-lookup"><span data-stu-id="072f6-115">On the **Serial numbers** FastTab, if the **Serial number control** option is turned on, the serial number must be unique per unit.</span></span> <span data-ttu-id="072f6-116">この場合、重複するシリアル番号を使用することはできません。</span><span class="sxs-lookup"><span data-stu-id="072f6-116">In other words, duplicated serial numbers aren't allowed.</span></span>

<span data-ttu-id="072f6-118">POSアプリケーションの**在庫入庫**処理を使用すると、シリアル品目に対して次のタスクを実行できます：</span><span class="sxs-lookup"><span data-stu-id="072f6-118">The **Inbound inventory** operation in the POS application lets users perform the following tasks for serialized items:</span></span>

- <span data-ttu-id="072f6-120">発注書または振替発注を介して店舗でシリアル品目を受領する際には、当該品目に事前登録されている番号を確認してください。</span><span class="sxs-lookup"><span data-stu-id="072f6-120">Validate preregistered serial numbers against serialized items when those items are received in the store via a purchase order or transfer order.</span></span>

> [!NOTE]
> <span data-ttu-id="072f6-121">**在庫入庫** 処理を使用してシリアル品目を登録、または確認するには、当該品目に追跡用分析コードのグループが割り当てられている必要があります。これは、**有効化** オプションでシリアル番号の使用が設定するものであり、**販売プロセスで有効化** オプションではありません。</span><span class="sxs-lookup"><span data-stu-id="072f6-121">To use the **Inbound inventory** operation to register or validate a serialized item, the item must be assigned a tracking dimension group that is set up to allow for serial numbers for the **Active** option, not the **Active in sales process** option.</span></span>

<span data-ttu-id="072f6-122">入荷プロセスを開始するには、**在庫入庫**処理の、**入庫中** ビューで品目のバーコードをスキャン、または品目 ID を入力することで開始できます。</span><span class="sxs-lookup"><span data-stu-id="072f6-122">To begin the receiving process, in the **Inbound inventory** operation, you can start in the **Receiving now** view by scanning the item bar code or entering the item ID.</span></span> <span data-ttu-id="072f6-123">また、品目を手動で選択して、**フル オーダー リスト** ビューで開始することもできます。</span><span class="sxs-lookup"><span data-stu-id="072f6-123">Alternatively, you can start in the **Full order list** view by manually selecting the item.</span></span>

<span data-ttu-id="072f6-124">選択または入庫している品目がシリアル品目の場合、その品目の**詳細**ウィンドウには、**シリアル番号管理**ページへの**シリアル番号の管理**リンクが表示されます。</span><span class="sxs-lookup"><span data-stu-id="072f6-124">If the item that is being selected or received is a serialized item, the **Details** pane for the item contains a **Manage serial number** link that opens the **Serial number management** page.</span></span> <span data-ttu-id="072f6-125">別の方法で**シリアル番号の管理** ページを開くには、注文の詳細ビューのアプリ バーで **シリアル番号** を選択するか、入荷のプロセス中に表示されるダイアログ ボックスで**シリアル番号の管理**を選択してください。</span><span class="sxs-lookup"><span data-stu-id="072f6-125">Alternatively, you can open the **Serial number management** page either by selecting **Serial number** on the app bar of the order details view or by selecting **Manage serial number** in the dialog box that prompts you during the receiving process.</span></span> <span data-ttu-id="072f6-126">**シリアル番号の管理**ページには、登録や確認が保留中となっているすべての未処理のシリアル番号が一覧表示されます。</span><span class="sxs-lookup"><span data-stu-id="072f6-126">The **Serial number management** page lists all open serial number lines that are pending registration or validation.</span></span> <span data-ttu-id="072f6-127">このページには 2 つのタブが存在する場合があり、それぞれ現在の処理中の品目と、注文で過程でシリアル化された品目が表示されます。</span><span class="sxs-lookup"><span data-stu-id="072f6-127">There might be two tabs on this page: one for the current item and one for all the serialized items in the order.</span></span>

<span data-ttu-id="072f6-128">**シリアル番号の管理**ページの**状態**フィールドには、それぞれのシリアル番号が置かれている現在のステージに関する情報が表示されます：</span><span class="sxs-lookup"><span data-stu-id="072f6-128">The **Status** field on the **Serial number management** page provides information about the current stage that each serial number is in:</span></span>

- <span data-ttu-id="072f6-129">**未登録** : シリアル番号が指定されていないか、事前登録されたシリアル番号がまだ確認されていません。</span><span class="sxs-lookup"><span data-stu-id="072f6-129">**Not registered** – The serial number hasn't been provided, or the preregistered serial number hasn't yet been validated.</span></span>
- <span data-ttu-id="072f6-130">**登録**：シリアル番号が登録され、店舗のチャンネル データベースのローカルで保存されているか、事前登録されたシリアル番号が確認されています。</span><span class="sxs-lookup"><span data-stu-id="072f6-130">**Registering** – The serial number has been registered and saved locally to the store's channel database, or the preregistered serial number has been validated.</span></span> <span data-ttu-id="072f6-131">**登録**の状態となっているシリアル番号のみが、受領の完了時にコマース バックオフィスに送信されます。</span><span class="sxs-lookup"><span data-stu-id="072f6-131">Only serial numbers that have a status of **Registering** will be submitted to Commerce headquarters when you finish receiving.</span></span>

<span data-ttu-id="072f6-133">発注書で**シリアル番号の管理**オプションを使用すると、シリアル品目の入荷プロセス中にダイアログ ボックスを表示して入力を促すことができます。</span><span class="sxs-lookup"><span data-stu-id="072f6-133">For a purchase order, a dialog box that prompts you during the receiving process of a serialized item offers the **Manage serial number** option.</span></span> <span data-ttu-id="072f6-134">このオプションを選択すると、**シリアル番号の管理**ページが開き、シリアル番号の登録を開始できます。</span><span class="sxs-lookup"><span data-stu-id="072f6-134">You can select that option to open the **Serial number management** page and start to register serial numbers.</span></span> <span data-ttu-id="072f6-135">入庫が登録される前であれば、入庫処理の過程ではこの手順を省略して、後で入力することもできます。</span><span class="sxs-lookup"><span data-stu-id="072f6-135">You can also skip this step during the receiving process and provide the input later, before the receipt is posted.</span></span>

<span data-ttu-id="072f6-136">既定では、現在処理中の品目のタブが表示されます。</span><span class="sxs-lookup"><span data-stu-id="072f6-136">By default, the tab for the current item is shown.</span></span> <span data-ttu-id="072f6-137">すべてのシリアル番号の明細行には、空のシリアル番号値が割り当てられ、状態は**未登録**となります。</span><span class="sxs-lookup"><span data-stu-id="072f6-137">All serial number lines have an empty serial number value and a status of **Not registered**.</span></span> <span data-ttu-id="072f6-138">シリアル番号のバーコードをスキャンするか、またはアプリ バーの **シリアル番号**を選択して、ダイアログ ボックスにシリアル番号を連続入力することができます。</span><span class="sxs-lookup"><span data-stu-id="072f6-138">You can scan serial number bar codes, or you can select **Serial number** on the app bar to continuously enter serial numbers in the dialog box.</span></span> <span data-ttu-id="072f6-139">入力したシリアル番号が一覧に表示され、シリアル番号明細行の状態が**登録済**に変更され ます。</span><span class="sxs-lookup"><span data-stu-id="072f6-139">The serial numbers that you enter appear in the list, and the status of the serial number lines is changed to **Registering**.</span></span> <span data-ttu-id="072f6-140">一覧に登録できるシリアル番号の最大数は、入庫の数量と同数です。</span><span class="sxs-lookup"><span data-stu-id="072f6-140">The maximum number of serial numbers that you can register in the list equals the receiving quantity.</span></span> <span data-ttu-id="072f6-141">入力に誤りがある場合は、**詳細**ウィンドウで、**編集**または**クリア**を選択して、入力したシリアル番号を変更することができます。</span><span class="sxs-lookup"><span data-stu-id="072f6-141">If you make a mistake, you can select **Edit** or **Clear** in the **Details** pane to make changes to the serial numbers that you entered.</span></span>
