# <a name="add-product-recommendation-lists-to-e-commerce-pages"></a>E コマース ページへの製品推奨リストの追加

[!include [banner](includes/preview-banner.md)]
[!include [banner](includes/banner.md)]

このトピックでは、E コマース ページに製品推奨リストを追加する方法について説明します。

1. 既存の E コマースの作成ページで、**モジュールの追加**を選択します。
2. **製品コレクション** モジュールを検索します。
3. モジュール ウィザードで、ページに追加するリストを選択します。
4. 変更を保存します。

ページに追加できるリストのタイプの詳細については、[製品推奨の概要](product-recommendations.md) を参照してください。
