# <a name="add-recommendation-lists-to-a-site"></a>サイトへの推奨リストの追加


[!include [banner](includes/banner.md)]

このトピックでは、E コマース ページに製品推奨リストを追加する方法について説明します。

1. 既存の E コマース サイト ビルダー ページで、**モジュールの追加**を選択します。
2. **製品コレクション** モジュールを検索します。
3. モジュール ウィザードで、ページに追加するリストを選択します。
4. 変更を保存します。

ページに追加できるリストのタイプの詳細については、[製品収集モジュールの概要](product-collection-module-overview.md) を参照してください。
