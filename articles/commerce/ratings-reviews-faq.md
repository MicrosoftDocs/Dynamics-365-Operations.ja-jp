# <a name="ratings-and-reviews-faq"></a>評価とレビューに関するよく寄せられる質問

[!include [banner](includes/banner.md)]
[!include [banner](includes/preview-banner.md)]

このトピックでは、Microsoft Dynamics 365 Commerce 評価とレビュー ソリューションに関するよく寄せられる質問に対する回答を示します。

### <a name="how-do-i-hide-the-ratings-refiner-that-appears-on-search-results-and-category-pages-if-my-e-commerce-site-doesnt-use-the-ratings-and-reviews-solution"></a>eコマース サイトで評価とレビュー ソリューションを使用しない場合、検索結果やカテゴリ ページに表示される評価絞り込み機能を非表示にするにはどうすればよいですか?

評価とレビュー ソリューションを使用しない eコマース チャネルで検索結果やカテゴリ ページで評価の絞り込み機能を非表示にする手順については、[評価とレビュー ソリューションが無効な場合に、評価絞り込み機能が検索結果やカテゴリ ページに表示される](troubleshoot/hide-ratings-refiner.md)を参照してください。

### <a name="how-do-i-enforce-the-manual-publishing-of-ratings-and-reviews"></a>評価とレビューを手動で公開するにはどうすればよいですか?

自動公開を回避して、評価とレビューの手動公開を有効にして実行する方法については、[モデレーターによる評価とレビューの手動公開を有効にする](manual-publish-rating-reviews.md)を参照してください。
