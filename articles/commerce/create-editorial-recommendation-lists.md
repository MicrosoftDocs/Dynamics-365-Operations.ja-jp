<span data-ttu-id="0893c-104">このトピックでは、小売業者が Microsoft Dynamics 365 Commerce の顧客のために、手動の製品リストを作成および管理する方法を説明します。</span><span class="sxs-lookup"><span data-stu-id="0893c-104">This topic explains how merchandizers can create and manage manual product lists for Microsoft Dynamics 365 Commerce customers.</span></span>

<span data-ttu-id="0893c-105">キュレーション リストとは、ユーザーによって作成され、精選された個々のコンテンツの集合のことです。</span><span class="sxs-lookup"><span data-stu-id="0893c-105">Curated lists are collections of individual content created and curated by people.</span></span>  

## <a name="create-a-new-list"></a><span data-ttu-id="0893c-106">新しいリストの作成</span><span class="sxs-lookup"><span data-stu-id="0893c-106">Create a new list</span></span>

<span data-ttu-id="0893c-107">精選された製品推奨事項リストを作成するには、次の手順に従います。</span><span class="sxs-lookup"><span data-stu-id="0893c-107">To create a curated product recommendation list, follow these steps.</span></span>

1. <span data-ttu-id="0893c-108">**小売** &gt; **製品推奨事項** &gt; **推奨事項リスト**に移動します。</span><span class="sxs-lookup"><span data-stu-id="0893c-108">Go to **Retail** &gt; **Product recommendations** &gt; **Recommendation lists**.</span></span>
1. <span data-ttu-id="0893c-109">**新規** を選択します。</span><span class="sxs-lookup"><span data-stu-id="0893c-109">Select **New**.</span></span>
1. <span data-ttu-id="0893c-110">**リスト ID** フィールドに値を入力します。</span><span class="sxs-lookup"><span data-stu-id="0893c-110">In the **List Id** field, enter a value.</span></span>
1. <span data-ttu-id="0893c-111">**リスト名**フィールドに値を入力します。</span><span class="sxs-lookup"><span data-stu-id="0893c-111">In the **List name** field, enter a value.</span></span>
    - <span data-ttu-id="0893c-112">**リスト名**は、**製品収集**モジュールの精選されたリスト セクションに表示されるリストのタイトルです。</span><span class="sxs-lookup"><span data-stu-id="0893c-112">The **List name** is the title of the list that will appear in the curated lists section of the **Product collection** module.</span></span>
1. <span data-ttu-id="0893c-113">リストに製品を追加するには、**製品の追加**を選択します。</span><span class="sxs-lookup"><span data-stu-id="0893c-113">To add products to the list, select **Add products**.</span></span>
1. <span data-ttu-id="0893c-114">リスト上で製品の表示順を変えるには、**表示順**列に値を入力します。</span><span class="sxs-lookup"><span data-stu-id="0893c-114">To change the order of the products in the list, enter a value in the **Display order** column.</span></span>
    - <span data-ttu-id="0893c-115">2 つの製品が同じ表示順の値を持つ場合、それら 2 つの最終的な並び順は、バック オフィスとは異なることがあります。</span><span class="sxs-lookup"><span data-stu-id="0893c-115">If two products have the same display order value, then the final order of those two results may differ from the back office.</span></span>
1. <span data-ttu-id="0893c-116">**保存**を選択してリストを保存します。</span><span class="sxs-lookup"><span data-stu-id="0893c-116">Select **Save** to save the list.</span></span>
