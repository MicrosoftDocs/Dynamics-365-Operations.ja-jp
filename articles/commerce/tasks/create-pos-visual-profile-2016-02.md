# <a name="create-point-of-sale-pos-visual-profiles"></a>販売時点管理 (POS) 視覚プロファイルの作成

[!include [banner](../includes/banner.md)]

この手順では、新しい POS (販売時点管理) の視覚プロファイルの作成方法を説明します。 視覚プロファイルには、POS レジスターの外観を決定する基本情報が含まれます。 複数の視覚プロファイルを作成し、特定のレジスタで使用するプロファイルを割り当てることができます。 この手順では、デモ データの会社 USRT を使用します。

1. Retail と Commerce > チャネル設定 > POS 設定 > POS プロファイル > 視覚プロファイルの順に移動します。
2. [新規] をクリックします。
3. [プロファイル番号] フィールドに値を入力します。
4. [説明] フィールドに値を入力します。
5. [アプリケーション タイプ] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
6. 一覧で、選択された行のリンクをクリックします。
7. [テーマ] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
8. 一覧で、選択された行のリンクをクリックします。
9. [アクセント色] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
10. 一覧で、目的のレコードを見つけ、選択します。
11. 一覧で、選択された行のリンクをクリックします。
12. [バックグラウンドでログイン] セクションの拡張を切り替えます。
13. [横の画像 ID] フィールドで、画像 ID を選択するか、または入力します。
14. [ポートレイト画像 ID] フィールドで、画像 ID を選択または入力します。
15. [バックグラウンド] セクションの展開を切り替えます。
16. RequestPopup 画像 ID。
17. 一覧で、選択された行のリンクをクリックします。
18. [保存] をクリックします。
