# <a name="manage-assortments-november-2016"></a>品揃えの管理 (2016 年 11 月)

[!include [banner](../includes/banner.md)]

この手順では、新製品の品揃えを作成し発表する方法およびデモデータ企業USRTの使用方法を示します。 


1. [カテゴリと製品の管理] をクリックします。

## <a name="create-an-assortment"></a>品揃えの作成
1. [品揃え] タブをクリックします。
2. [新規] をクリックします。
3. [品揃え] をクリックします。
    * 品揃え ID が必要であり、固有値である必要があります。  
4. [品揃え名] フィールドに値を入力します。
5. [有効日] フィールドに日付を入力します。
6. [有効期限] フィールドに日付を入力します。
7. Commerce チャネル セクションを展開します。
8. [行の追加] をクリックします。
9. ツリーで、「Contoso Retail\Electronics\Boston」を選択します。
10. [追加] をクリックします。
11. [OK] をクリックします。
12. [製品] セクションを展開します。
13. [明細行の追加] をクリックします。
14. [カテゴリ] フィールドで値を入力または選択します。
15. [保存] をクリックします。

## <a name="publish-an-assortment"></a>品揃えの公開
1. [発行] をクリックします。
2. [はい] をクリックします。
