# <a name="add-a-favicon"></a><span data-ttu-id="d008b-103">お気に入りの追加</span><span class="sxs-lookup"><span data-stu-id="d008b-103">Add a favicon</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="d008b-104">このトピックでは、サイトにお気に入りを追加する方法について説明します。</span><span class="sxs-lookup"><span data-stu-id="d008b-104">This topic explains how to add a favicon to your site.</span></span>

## <a name="overview"></a><span data-ttu-id="d008b-105">概要</span><span class="sxs-lookup"><span data-stu-id="d008b-105">Overview</span></span>

<span data-ttu-id="d008b-106">お気に入りは小さなグラフィック ファイルで、Web ブラウザー タブ、アドレス バー、閲覧の履歴、およびブックマークやお気に入りなど、その他の場所内に表示されます。</span><span class="sxs-lookup"><span data-stu-id="d008b-106">A favicon is a small graphics file that is shown on a web browser tab, in the Address bar, in the browsing history, and in bookmarks or favorites, among other places.</span></span> <span data-ttu-id="d008b-107">サイトにお気に入りを追加することをお勧めします。それによりブランドを代表して強化し、顧客が閲覧する他のサイトと区別するのに役立ちます。</span><span class="sxs-lookup"><span data-stu-id="d008b-107">We recommend that you add a favicon to your site, because it represents and reinforces your brand, and helps distinguish your site from other sites that your customers visit.</span></span>

<span data-ttu-id="d008b-108">サイトにはさまざまなサイズとファイル タイプのお気に入りを追加できますが、このトピックでは、単一のお気に入りを追加する方法について説明します。</span><span class="sxs-lookup"><span data-stu-id="d008b-108">Although you can add multiple favicons of various sizes and file types to your site, this topic shows how to add a single favicon.</span></span> <span data-ttu-id="d008b-109">ただし、同じプロセスと場所を使用して、お気に入りをさらに追加することができます。</span><span class="sxs-lookup"><span data-stu-id="d008b-109">However, the same process and location are used to add more favicons.</span></span>

## <a name="upload-a-favicon-to-your-sites-asset-collection"></a><span data-ttu-id="d008b-110">サイトのアセット コレクションにお気に入りをアップロードする</span><span class="sxs-lookup"><span data-stu-id="d008b-110">Upload a favicon to your site's asset collection</span></span>

<span data-ttu-id="d008b-111">お気に入りをサイトのアセット コレクションにアップロードするには、次の手順に従います。</span><span class="sxs-lookup"><span data-stu-id="d008b-111">To upload a favicon to your site's asset collection, follow these steps.</span></span>

1. <span data-ttu-id="d008b-112">左のナビゲーション ウィンドウで、**メディア ライブラリー** を選択します。</span><span class="sxs-lookup"><span data-stu-id="d008b-112">In the left navigation pane, select **Media Library**.</span></span>
1. <span data-ttu-id="d008b-113">コマンド バーで、**アップロード \> メディア項目のアップロード** を選択します。</span><span class="sxs-lookup"><span data-stu-id="d008b-113">On the command bar, select **Upload \> Upload Media Items**.</span></span>
1. <span data-ttu-id="d008b-114">[ファイル エクスプローラー] ウィンドウで、アップロードするお気に入り画像ファイルを選択し、**開く** を選択します。</span><span class="sxs-lookup"><span data-stu-id="d008b-114">In the File Explorer window, browse to the favicon image file that you want to upload, select it, and then select **Open**.</span></span>
1. <span data-ttu-id="d008b-115">**メディア項目のアップロード** ダイアログ ボックスで、必要なタイトルと代替テキストを入力します。</span><span class="sxs-lookup"><span data-stu-id="d008b-115">In the **Upload Media Item** dialog box, enter the required title and alt text.</span></span>
1. <span data-ttu-id="d008b-116">画像をアップロードしてすぐに公開する場合は、**アップロード後にメディア項目を公開する** チェック ボックスをオンにします。</span><span class="sxs-lookup"><span data-stu-id="d008b-116">If you want to publish the image immediately after upload, select the **Publish media items after upload** check box.</span></span>

    > [!NOTE]
    > <span data-ttu-id="d008b-117">**アップロード後にメディア項目を公開する** オプションを選択しない場合は、後で **メディア項目** ページからお気に入りを手動で公開する必要があります。</span><span class="sxs-lookup"><span data-stu-id="d008b-117">If you don't select the **Publish media items after upload** check box, you must return to **Media items** page and manually publish the favicon later.</span></span>

1. <span data-ttu-id="d008b-118">**OK** を選択します。</span><span class="sxs-lookup"><span data-stu-id="d008b-118">Select **OK**.</span></span>
1. <span data-ttu-id="d008b-119">右側のプロパティ ウィンドウで、お気に入りのパブリック URL をコピーします。</span><span class="sxs-lookup"><span data-stu-id="d008b-119">In the property pane on the right, copy the public URL of the favicon.</span></span> <span data-ttu-id="d008b-120">この URL は後続の手順で使用します。</span><span class="sxs-lookup"><span data-stu-id="d008b-120">You will use this URL later.</span></span>

<span data-ttu-id="d008b-122">お気に入りの HTML を作成するには、次の HTML 文字列を使用します。</span><span class="sxs-lookup"><span data-stu-id="d008b-122">To create the HTML for the favicon, use the following HTML string.</span></span> <span data-ttu-id="d008b-123">**href** 属性に関しては、**Public\_URL\_for\_your\_favicon** を前述のコピーしたパブリック URL に置き換えます。</span><span class="sxs-lookup"><span data-stu-id="d008b-123">For the **href** attribute, replace **Public\_URL\_for\_your\_favicon** with the public URL that you copied earlier.</span></span>

## <a name="create-a-fragment-that-contains-a-metatag-for-your-favicon"></a><span data-ttu-id="d008b-124">お気に入りのメタタグを含むフラグメントを作成する</span><span class="sxs-lookup"><span data-stu-id="d008b-124">Create a fragment that contains a metatag for your favicon</span></span>

<span data-ttu-id="d008b-125">お気に入りのメタタグを含むフラグメントを作成するには、次の手順に従ってください。</span><span class="sxs-lookup"><span data-stu-id="d008b-125">To create a fragment that contains a metatag for your favicon, follow these steps.</span></span>

1. <span data-ttu-id="d008b-126">**フラグメント** に移動し、**新規** を選択します。</span><span class="sxs-lookup"><span data-stu-id="d008b-126">Go to **Fragments**, and select **New**.</span></span>
1. <span data-ttu-id="d008b-127">**新規ページ フラグメント** ダイアログボ ックスで、ページ フラグメントの基になるモジュールに **メタタグ** を選択します。</span><span class="sxs-lookup"><span data-stu-id="d008b-127">In the **New fragment** dialog box, select **Metatags** as the module that the fragment is based on.</span></span>
1. <span data-ttu-id="d008b-128">フラグメントの名前を入力し、**OK** を選択します。</span><span class="sxs-lookup"><span data-stu-id="d008b-128">Enter a name for the fragment, and then select **OK**.</span></span>
1. <span data-ttu-id="d008b-129">フラグメント階層ツリーで、**既定のメタタグ** 子を選択し ます。</span><span class="sxs-lookup"><span data-stu-id="d008b-129">In the fragment hierarchy tree, select the **Default metatags** child.</span></span>
1. <span data-ttu-id="d008b-130">右側のウィンドウで、 **Metaタグ** 配下の **追加** を選択、前述の作成済みの HTML 文字列を入力します。</span><span class="sxs-lookup"><span data-stu-id="d008b-130">In the right pane, under **Meta Tags**, select **Add**, and then enter the HTML string that you created earlier for the favicon.</span></span> 
1. <span data-ttu-id="d008b-131">**編集の完了** を選択し、 **発行** を選択してフラグメントを公開します。</span><span class="sxs-lookup"><span data-stu-id="d008b-131">Select **Finish editing**, and then select **Publish** to publish the fragment.</span></span>

## <a name="add-the-metatag-fragment-to-the-html-head-section-of-your-pages"></a><span data-ttu-id="d008b-132">ページの HTML 内の head セクションにメタタグのページ フラグメントを追加する</span><span class="sxs-lookup"><span data-stu-id="d008b-132">Add the metatag fragment to the HTML head section of your pages</span></span>

<span data-ttu-id="d008b-133">ページの HTML 内の **head** セクションにメタタグのページ フラグメントを追加するには、次の手順に従ってください。</span><span class="sxs-lookup"><span data-stu-id="d008b-133">To add the metatag fragment to the HTML **head** section of your pages, follow these steps.</span></span>

1. <span data-ttu-id="d008b-134">**テンプレート** に移動して、お気に入りを追加するページのテンプレートを開き、続いて **編集** を開きます。</span><span class="sxs-lookup"><span data-stu-id="d008b-134">Go to **Templates**, open the template for the pages that you want to add your favicon to, and then select **Edit**.</span></span>
1. <span data-ttu-id="d008b-135">テンプレート階層ツリーで、**HTML head** コンテナーの右にある省略符号 (**...**) ボタンを選択し、 **フラグメントの追加** を選択します。</span><span class="sxs-lookup"><span data-stu-id="d008b-135">In the template hierarchy tree, select the ellipsis (**...**) button to the right of the **HTML head** container, and then select **Add fragment**.</span></span>
1. <span data-ttu-id="d008b-136">**フラグメントの選択** ダイアログ ボックスで、以前に作成した メタタグ フラグメントを選択し、**OK** を選択します。</span><span class="sxs-lookup"><span data-stu-id="d008b-136">In the **Select fragment** dialog box, select the metatag fragment that you created earlier, and then select **OK**.</span></span>
1. <span data-ttu-id="d008b-137">**編集の完了** を選択し、 **発行** を選択してテンプレートを公開します。</span><span class="sxs-lookup"><span data-stu-id="d008b-137">Select **Finish editing**, and then select **Publish** to publish the template.</span></span>

> [!NOTE]
> <span data-ttu-id="d008b-138">サイトで複数のテンプレートを使用している場合は、メタタグのラグメントをすべてのテンプレートに追加する必要があります。</span><span class="sxs-lookup"><span data-stu-id="d008b-138">If your site uses more than one template, you must add the metatags fragment to all of them.</span></span>

<span data-ttu-id="d008b-139">メタタグ フラグメントを追加したテンプレートに基づくページをプレビューすると、[ブラウザー] タブにお気に入りが表示されます。</span><span class="sxs-lookup"><span data-stu-id="d008b-139">When you preview pages that are based on the template that you added the metatags fragment to, you should now see the favicon on the browser tab.</span></span>
