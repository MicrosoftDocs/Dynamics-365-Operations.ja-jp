# <a name="online-and-offline-point-of-sale-pos-operations"></a>オンラインおよびオフラインでの販売時点管理 (POS) の操作

[!include [banner](includes/banner.md)]
[!include [banner](includes/preview-banner.md)]

ユーザーが販売時点管理 (POS) で行うほとんどのアクションは、操作と見なされます。 操作は、Dynamics 365 Commerce バック オフィスで構成および管理されます。 多くの操作を POS ボタングリッドのボタンに追加できます。 その後、ユーザーは、操作を呼び出すためのボタンを選択し、その機能を実行することができます。 ボタン グリッド デザイナーのボタン グリッドに追加する操作を使用できるようにするには、最初に Commerce headquarters (**小売とコマース \> チャネル設定 \> POS 設定 \> POS \> POS Operations**) の POS 操作ビューで操作をユーザー操作としてマークする必要があります。

他の操作はメインの POS アプリケーションの一部であり、画面上のボタンから呼び出されるか、他のワークフローやプロセスの一部として呼び出されます。

いくつかの操作は、現在 Modern POS またはクラウド POS では使用できません。 これらの操作の一部は、追加の拡張と構成を必要とするロケール固有の操作です。 その他は、現在サポートされていない Microsoft Dynamics AX 2012 の機能です。

以下の表は、Modern POS およびクラウド POS で使用可能な操作に関する詳細を示しています。 この表は、アプリケーションがオペレーションを呼び出すことができる場所と、POS がオフライン モードのときに使用可能かどうかも指定します。

次の列は、操作を呼び出すことができる場所を指定します。

- **ユーザー操作** – 操作は、POS 画面レイアウトの一部である POS ボタン グリッドのボタンに割り当てることができます。
- **トランザクション画面** – 操作は、POS トランザクション画面で設定されている POS ボタングリッドから呼び出すことができます。
- **ウェルカム画面** － 操作は、POS ウェルカム画面で構成されている POS ボタン グリッドから呼び出すことができます。
