# <a name="commerce-chat-features-overview"></a>Commerce チャット機能の概要

[!include [banner](includes/banner.md)]

Microsoft Dynamics 365 Customer Service 用オムニチャネルと Dynamics 365 Commerce eコマース web サイト付 Power Virtual Agents アプリケーションを統合することで、ビジネスは権限を向上できます。 この記事では、 Customer Service 用オムニチャネルと Power Virtual Agents が Dynamics 365 Commerce eコマースの Web サイトと統合された場合にオンラインで選択できる機能の概要を説明します。

## <a name="omnichannel-for-customer-service"></a>Customer Service 用オムニチャネル

Customer Service 用オムニチャネルは、Dynamics 365 Customer Service の機能を拡張する信頼性の高いアプリケーションです。 これにより、ユーザーは接続して、ライブ チャット、音声、ショート メッセージ サービス (SMS) などのチャンネルを通じて顧客と接続し、関係を持つ組織をすることができます。 Customer Service 用オムニチャネルについての詳細は、[Customer Service 用オムニチャネルの紹介](/dynamics365/customer-service/introduction-omnichannel) を参照してください。 Customer Service 用オムニチャネルを Dynamics 365 Commerce eコマース web サイトと統合する方法についての詳細は、[Customer Service 用オムニチャネルを使った Commerce チャット モジュール](commerce-chat-module.md) を参照してください。

## <a name="power-virtual-agents"></a>Power Virtual Agents

Power Virtual Agents を使用すると、さまざまな要求に対応する強力な AI 強化されたチャットボットを作成できます。 Power Virtual Agents についての詳細は、[Power Virtual Agents 概要](/power-virtual-agents/fundamentals-what-is-power-virtual-agents) を参照してください。 Dynamics 365 Commerce Eコマースの Web サイトと Power Virtual Agents を統合する 方法については 、[Microsoft Power Virtual Agents を使った Commerce チャット](chat-module-pva.md) を参照してください。

> [!NOTE]
> Power Virtual Agents は、Customer Service 用オムニチャネルに既に含まれています。 仮想エージェント (チャットボット) とライブ エージェントの両方の機能が必要な場合は、Customer Service 用オムニチャネルを使用します。
