## <a name="create-layouts-for-an-existing-page"></a><span data-ttu-id="1769e-105">既存のページのレイアウトの作成</span><span class="sxs-lookup"><span data-stu-id="1769e-105">Create layouts for an existing page</span></span>

> [!NOTE]
> <span data-ttu-id="1769e-106">メイン スロットの下に 2 つ以上のモジュールがあるページの場合のみ、既存のページのレイアウトを作成できます。</span><span class="sxs-lookup"><span data-stu-id="1769e-106">You can create layouts for an existing page only if that page has at least two modules under the main slot.</span></span>

<span data-ttu-id="1769e-107">既存のページのレイアウトを作成するには、次の手順に従います。</span><span class="sxs-lookup"><span data-stu-id="1769e-107">To create layouts for an existing page, follow these steps.</span></span>

1. <span data-ttu-id="1769e-108">**ページ**に移動して、リスト内の既存のページを検索します。</span><span class="sxs-lookup"><span data-stu-id="1769e-108">Go to **Pages**, and find the existing page in the list.</span></span> <span data-ttu-id="1769e-109">必要に応じて、検索機能を使用します。</span><span class="sxs-lookup"><span data-stu-id="1769e-109">Use the search feature as you require.</span></span>
1. <span data-ttu-id="1769e-110">ページを選択し、**編集** を選択してチェックアウトし、ページ名を選択して開きます。</span><span class="sxs-lookup"><span data-stu-id="1769e-110">Select the page, select **Edit** to check it out, and then select the page name to open it.</span></span> <span data-ttu-id="1769e-111">モジュールの順序をメモします。</span><span class="sxs-lookup"><span data-stu-id="1769e-111">Make a note of the module order.</span></span>
1. <span data-ttu-id="1769e-112">**新しいレイアウトとして保存**を選択します。</span><span class="sxs-lookup"><span data-stu-id="1769e-112">Select **Save as New Layout**.</span></span>
1. <span data-ttu-id="1769e-113">レイアウト名を入力し、**OK** を選択します。</span><span class="sxs-lookup"><span data-stu-id="1769e-113">Enter a name for the layout, and then select **OK**.</span></span>
1. <span data-ttu-id="1769e-114">**埋め込みレイアウトに変換**を選択します。</span><span class="sxs-lookup"><span data-stu-id="1769e-114">Select **Convert to Embedded Layout**.</span></span>
1. <span data-ttu-id="1769e-115">必要に応じてモジュールの順序を変更し、新しい順序をメモします。</span><span class="sxs-lookup"><span data-stu-id="1769e-115">Change the order of the modules as you require, and make a note of the new order.</span></span>
1. <span data-ttu-id="1769e-116">**新しいレイアウトとして保存**を選択します。</span><span class="sxs-lookup"><span data-stu-id="1769e-116">Select **Save as New Layout**.</span></span>
1. <span data-ttu-id="1769e-117">レイアウト名を入力し、**OK** を選択します。</span><span class="sxs-lookup"><span data-stu-id="1769e-117">Enter a name for the layout, and then select **OK**.</span></span>
1. <span data-ttu-id="1769e-118">**レイアウトの変更**を選択し、作成した最初のレイアウトを選択して、**OK** を選択します。</span><span class="sxs-lookup"><span data-stu-id="1769e-118">Select **Change Layout**, select the first layout that you created, and then select **OK**.</span></span> <span data-ttu-id="1769e-119">モジュールの順序をメモします。</span><span class="sxs-lookup"><span data-stu-id="1769e-119">Make a note of the module order.</span></span> <span data-ttu-id="1769e-120">変更により、レイアウトと共に保存されているモジュールの順序と一致します。</span><span class="sxs-lookup"><span data-stu-id="1769e-120">Change it so that it matches the module order that was saved with the layout.</span></span>
1. <span data-ttu-id="1769e-121">**編集の完了** を選択してページをチェックインし、 **発行** を選択して公開します。</span><span class="sxs-lookup"><span data-stu-id="1769e-121">Select **Finish editing** to check in the page, and then select **Publish** to publish it.</span></span>

> [!NOTE]
> <span data-ttu-id="1769e-123">ページを作成するのに使用されたテンプレートに複数のレイアウトがある場合にのみ、既存のページに異なるレイアウトを選択することができます。</span><span class="sxs-lookup"><span data-stu-id="1769e-123">You can select a different layout for an existing page only if the template that was used to create that page has more than one layout.</span></span>

1. <span data-ttu-id="1769e-125">**ページ**に移動して、リスト内の既存のページを検索します。</span><span class="sxs-lookup"><span data-stu-id="1769e-125">Go to **Pages**, and find the existing page in the list.</span></span> <span data-ttu-id="1769e-126">必要に応じて、検索機能を使用します。</span><span class="sxs-lookup"><span data-stu-id="1769e-126">Use the search feature as you require.</span></span>
1. <span data-ttu-id="1769e-127">ページを選択し、**編集** を選択してチェックアウトし、ページ名を選択して開きます。</span><span class="sxs-lookup"><span data-stu-id="1769e-127">Select the page, select **Edit** to check it out, and then select the page name to open it.</span></span>
1. <span data-ttu-id="1769e-128">**レイアウトの変更**を選択します。</span><span class="sxs-lookup"><span data-stu-id="1769e-128">Select **Change layout**.</span></span>
1. <span data-ttu-id="1769e-129">ページの新しいレイアウトを選択し、**OK** を選択します。</span><span class="sxs-lookup"><span data-stu-id="1769e-129">Select the new layout for the page, and then select **OK**.</span></span> <span data-ttu-id="1769e-130">ページ エディターが更新され、新しいレイアウトが表示されます。</span><span class="sxs-lookup"><span data-stu-id="1769e-130">The page editor is refreshed to show the new layout.</span></span>
