# <a name="create-a-benefits-plan"></a>給付金プランの作成

この記事では、Dynamics 365 Human Resources の給付金プランを設定する方法について説明します。

1. **給付金管理** ワーク スペースの **プラン** で、**給付金プラン** を選択します。

2. **新規** を選択します。

3. **一般** タブで、次のフィールドの値を指定します。

   | フィールド | 説明 |
   | --- | --- |
   | **計画** | プランの固有識別子。 |
   | **説明** | プランの説明。 |
   | **計画タイプ** | 新しいプランを作成する場合は、プラン タイプを指定する必要があります。 プラン タイプは、給付金の特定のタイプを高レベルにグループ化したものです。 各プラン タイプによって、従業員がそのタイプの複数のプランに登録できるかどうか、連絡先が受益者または扶養家族であるかどうかを指定し、補償オプションを定義します。 給付金のニーズを満たすために、新しいカスタムのプラン タイプを作成できます。 主な給付金プランのタイプは次のとおりです。 <ul><li>401K</li><li>ADD</li><li>歯科</li><li>フィットネス</li><li>FSA</li><li>寿命</li><li>LTD</li><li>医療</li><li>PTO</li><li>STD</li><li>眼科</li></ul> |
   | **計画タイプ コード** | プラン タイプのプラン タイプ コード。 |
   | **プログラム** | オプションでプランを割り当てるプログラムを指定します。 |
   | **バンドル** | オプションでプランを割り当てるバンドルを指定します。 |
   | **マスター** | 計画が、配賦先のバンドル内のマスター計画であるかどうかを指定します。 |
   | **ステータス** | 給付金プランの現状態を示します。 既定値は、有効です。 状態を無効に変更すると、そのプランを登録時に選択として使用できなくなります。 |
   | **発効日時** | プラン開始の日時。 既定値は現在のシステム日付です。 |
   | **失効日時** | プランが終了する日付と時刻 (状態が無効に設定されている)。 既定値は 12/31/2154 で、意味がありません。 |

4. **構成** タブで、作成するプラン タイプに応じて、次のフィールドに値を指定します。

   | 計画タイプ | フィールド | 説明 |
   | --- | --- | --- |
   | 医療 (医療、歯科、視力、HMO) | COBRA | プランが、COBRA (連結済オムニバス予算調整法) に適格かどうかを指定します。 |
   | 医療 (医療、歯科、視力、HMO) | HIPAA | プランが、HIPAA (健康保険の携行性とアカウンタビリティ法) に適格かどうかを指定します。 |
   | <ul><li>医療 (医療、歯科、視力、HMO)</li><li>その他 (PTO、フィットネス)</li><li>その他</li><li>長期障碍者</li><li>ADD (基本ライフ、自発的ライフ)</li><li>節約 (たとえば、401 (k))</li><li>FSA</li></ul> | 対象となる税引前 | 税金が適用される前に、プランに出資を加えるかどうかを指定します。 |
   | <ul><li>医療 (医療、歯科、視力、HMO)</li><li>その他 (PTO、フィットネス)</li><li>長期障碍者</li><li>ADD (基本ライフ、自発的ライフ)</li><li>節約 (たとえば、401 (k))</li><li>FSA</li></ul> | 対象となる税引き後 | 税金が適用された後に、プランに出資を加えるかどうかを指定します。 |
   | <ul><li>医療 (医療、歯科、視力、HMO)</li><li>その他 (PTO、フィットネス)</li><li>長期障碍者</li><li>ADD (基本ライフ、自発的ライフ)</li><li>節約 (たとえば、401 (k))</li><li>FSA</li></ul> | 寄稿者 | プランに貢献する人を指定 – 従業員、雇用主、またはその両方。 |
   | <ul><li>長期障碍者</li><li>ADD (基本ライフ、自発的ライフ)</li></ul> | 最小補償 | プランに必要な保険適用範囲の最小金額。 |
   | <ul><li>長期障碍者</li><li>ADD (基本ライフ、自発的ライフ)</li></ul> | 最大補償 | プランに必要な保険適用範囲の最高金額。 |
   | <ul><li>長期障碍者</li><li>ADD (基本ライフ、自発的ライフ)</li></ul> | 補償増分の使用 | 補充金額が有効な増分金額と一致することを検証するかどうか指定します。 |
   | <ul><li>長期障碍者</li><li>ADD (基本ライフ、自発的ライフ)</li></ul> | 増分金額 | プランの保険適用範囲の増加金額。 たとえば、増加金額が 1,000の 場合、従業員は保険の $200,500 を持つことができないので、$201,000 に切り上げるかまたは $200,000 に切り下げる必要があります。 |
   | <ul><li>長期障碍者</li><li>ADD (基本ライフ、自発的ライフ)</li></ul> | 切り上げの方向 | 補償金額が増分金額値を満たしていない場合に、丸める方向 (切り上げまたは切り捨て) を指定します。 |
   | ADD (基本ライフ、自発的ライフ) | 保険可能体の証拠 | 従業員が保険の証拠を入力する必要があるかどうかを指定します。 |
   | ADD (基本ライフ、自発的ライフ) | 量 | 会計通貨での金額。 このフィールドは、保険チェック ボックスの証拠が選択される場合のみ使用できます。 |
   | <ul><li>節約 (たとえば、401 (k))</li><li>FSA</li></ul> | 最小年間負担額 | プランに必要な最小負担額。 |
   | <ul><li>節約 (たとえば、401 (k))</li><li>FSA</li></ul> | 最大年間負担額 | プランに必要な最高負担額。 |
   | 節約 (たとえば、401 (k)) | 雇用主の最大金額 | 従業員が給付金期間中に貯蓄プランに貢献できる最大金額。 このフィールドを使用するには、雇用主の一致チェック ボックスをオンにする必要があります。 |
   | 節約 (たとえば、401 (k)) | 雇用主拠出金 | 雇用主が貯蓄プランに貢献するかどうかを指定します。 |
   | 節約 (たとえば、401 (k)) | 雇用主拠出金率 | 雇用主が満たす従業員貢献度の割合。 |
   | 節約 (たとえば、401 (k)) | 雇用主拠出金上限 | 雇用主が満たす最大割合。 たとえば、雇用主が従業員の給料の 6% までの従業員の貢献の 100% に一致した場合、雇用主の一致は上限 6% です。 |

5. **設定** タブで、次のフィールドの値を指定します。

   | フィールド | 説明 |
   | --- | --- |
   | **登録許可/続行** | 従業員が資格要件を満たす場合、プランに登録できるかどうかを指定します。</br></br>いいえに設定されている場合、適格性を処理するときプランは従業員に使用できません。 |
   | **前年度からの自動登録** | 適格な従業員を前年度に登録した場合に、その従業員を計画に自動的に登録するかどうかを指定します。 |
   | **既定で自動登録** | 既定で登録のプランを選択するかどうかを指定します。 プランは必須ではないので、従業員は既定選択を変更できます。 |
   | **新規登録不可** | 前年度のプランに登録された適格な従業員だけにプランを制限するかどうかを指定します。 |
   | **必須計画** | プランに従業員を自動的に登録するかどうかを指定します。 従業員は登録の選択を変更することはできません。 |
   | **取得日** | プランが会社で作成された日付。 |
   | **仕入先口座** (給付金外注業者) | プランがその計画に対して割増給与を支払う仕入先。 |
   | **名前** (給付金外注業者) | 仕入先の名前。 |
   | **仕入先照会** (給付金外注業者) | プランの仕入先の参照。 たとえば、会社のグループ プラン番号を表示します。 |
   | **代替参照** (給付金外注業者) | プランの仕入先代替参照。 たとえば、会社の口座番号。 |
   | **通貨** (給付金外注業者) | 仕入先への割増給与を支払うために使用される通貨。 |
   | **経費口座** (給付金外注業者) | プラン割増時間の経費勘定として使用される総勘定科目。 |
   | **仕入先口座** (給付金管理者) | 会社が管理者にそのプランを支払う仕入先。 プランが自己管理である場合、このフィールドを空白のままにします。 |
   | **名前** (給付金管理者) | 給付金管理者の仕入先名。 |
   | **仕入先照会** (給付金管理者) | プランの管理者の仕入先参照。 |
   | **代替参照** (給付金管理者) | プランの管理者の仕入先の代替参照。 |
   | **通貨** (給付金管理者) | 給付金管理者を支払うために使用される通貨。 |
   | **経費口座** (給付金管理者) | プランの管理に関連付けられている原価の経費勘定として使用される総勘定科目。 |

6. **フィルター** タブで、必要に応じてフィルター処理を行います。 次のフィールドで、フィルター処理を行います。

   - **事業単位**
   - **部門**
   - **法人エンティティ**
   - **保管場所**
   - **配置**

7. **適格性ルール** タブで、次のフィールドの値を指定します。

   | フィールド | 説明 |
   | --- | --- |
   | **行番号** | 適格性ルールの明細行番号。 |
   | **適格性ルール** | 給付金プランに適用する適格性ルール。 この適格性ルールは、対応するアクション タイプに適用され、指定された補償期間および控除に関連付けられます。 |
   | **アクション タイプ** | 適格性ルールを適用するアクションは次のとおりです: 給付金の登録または給付金の有効期限。 |
   | **補償待機期間** | 待機期間フォームからの値。 補償期間は、適格性ルールおよびアクション タイプに基づいて、従業員が給付金の補償または給付金の有効期限切れを待機する日数または月数を指定します。 |
   | **控除待機期間** | 待機期間フォームからの値。 控除待ち期間は、適格性ルールおよびアクション タイプに基づいて、従業員が給与から給付金の控除を待機する日数または月数を指定します。 |

8. **保存** を選択します。

## <a name="view-enrolled-workers"></a>登録された作業者を表示

選択された給付金プランに割り当てられている作業者が表示できます。

1. **給付金管理** ワーク スペースの **プラン** で、**給付金プラン** を選択します。

2. **登録された作業者** を選択します。

## <a name="attach-coverage-options"></a>補償オプションを関連付けます

選択された給付金プランに補償オプションを追加できます。 補償オプションを指定すると、補償オプションのレートおよび控除の設定が一緒に設定できます。  例: 医療計画の場合、ユーザーはファミリー補償オプションを選択します。  その後、関連付けられているプラン (レート設定にて設定) のファミリー レートと、関連付けられているプラン (レート設定にて設定) の控除を選択する必要があります。 これにより、選択した補償のために雇用主および従業員の費用が提供されます。 その後、従業員 + 1 または従業員の補償における補償範囲または従業員の補償に対するプロセスを繰り返します。

1. **給付金管理** ワーク スペースの **プラン** で、**給付金プラン** を選択します。

2. **補償オプションを添付** を選択します。

## <a name="override-eligibility-rules"></a>適格性ルールの上書き

適格性ルールの例外として、作業者をプランに追加できます。 追加した各作業者は、給付金プランの対象となります。

1. **給付金管理** ワーク スペースの **プラン** で、**給付金プラン** を選択します。

2. **適格性ルールの上書き** を選択します。

## <a name="view-attached-periods"></a>添付された期間の表示

使用可能な給付金期間の一覧を確認できます。

1. **給付金管理** ワーク スペースの **プラン** で、**給付金プラン** を選択します。

2. **期間** を選択します。

## <a name="view-plan-information"></a>プラン情報の表示

プランの説明を入力すると、従業員が給付金の選択に役立てることができます。 ここで入力したプラン情報は、補償オプション リストでプランにポイントがある場合に従業員セルフ サービスに表示されます。

1. **給付金管理** ワーク スペースの **プラン** で、**給付金プラン** を選択します。

2. **プラン情報** を選択します。

## <a name="view-flex-credit-programs"></a>フレックス クレジット プログラムの表示

1. **給付金管理** ワーク スペースの **プラン** で、**給付金プラン** を選択します。

2. **フレックス クレジット プログラム** を選択します。
