# <a name="get-support"></a>サポートの利用

この記事では、Microsoft Dynamics 365 Human Resources のヘルプを入手する方法について説明します。

人事管理のサポートを得るには、次の表にあるリソースを使用します。 これらのリソースは、トラブルシューティング時に従うよう推奨される順序で一覧表示されています。

| タスク | 詳細情報 |
|------|------------------|
| コミュニティに質問します。 | [Dynamics 365 Human Resources コミュニティ](https://community.dynamics.com/365/talent) に移動し、質問に関するヘルプを入手します。 回答を検索するには、**質問**のとなりの検索フィールドに質問を入力します。 質問するには、**質問する**を選択します。 |
| クラウドを利用したサポート ツールの使用。 | [Microsoft Dynamics Lifecycle Services (LCS)](https://lcs.dynamics.com/) において、プロジェクトに移動し、**サポート**タイルを選択します。 そこから、2 つの方法で問題を送信することができます。<ul><li>**アクティブな問題** タブで、送信する問題を選択し、**Microsoft に送信**を選択します。</li><li>**Microsoft に送信** タブで、**新しいインシデントを送信** をクリックし、新しい問題を送信します。</li></ul> |
| アプリ内のサポートを取得します。 | ページの右上隅にある**ヘルプ** ボタン (**?**) を選択します。 **サポート**を選択します。 問題は LCS の**アクティブな問題**上で報告され、管理者は社内サポートを提供するか Microsoft に問題を送信するかを決定できます。 |
| 人事管理のライセンスに関する質問のヘルプを入手します。 | パートナー、または Microsoft 販売担当者に問い合わせます。 |
| 人事管理内の新機能および機能を要求します。 | 人事管理にて**ヘルプ**ボタン (**?**) を選び、**アイデア**を選択します。 また、[Dynamics 365 アプリケーションのアイデア](https://experience.dynamics.com/ideas/) にアクセスし、既存のアイデアの表示、検索、または投票をしたり、新しいアイデアを追加したりすることもできます。 |
