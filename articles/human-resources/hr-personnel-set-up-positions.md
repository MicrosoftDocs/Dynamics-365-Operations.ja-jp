# <a name="set-up-positions"></a>職位の設定



職位は組織階層の下位レベルの重要な要素です。 職位は、職務の個々のインスタンスです。 たとえば、「販売マネージャ (東部)」という職位は、「販売マネージャ」という職務に関連付けられている職位の 1 つです。 職位は部問内に存在し、一人の作業者だけが関連付けられている可能性があります。 このタスクでは、職位の作成に必要な手順を説明します。 この手順は、人事管理の専門家を対象としています。

1. [要員管理] をクリックします。
2. [空いている職位] をクリックします。
3. [新規] をクリックすると、ドロップ ダイアログが開きます。
4. [職務] フィールドで値を入力または選択します。
    * ジョブの説明、肩書き、フルタイム相当の雇用係数は、選択されたジョブから職位に自動的にコピーされます。  
5. [ジョブの変更] を解決します。
6. [職位の作成] をクリックします。
7. [部門] フィールドで、値を入力または選択します。
8. [職位タイプ] フィールドで、値を入力または選択します。
9. [報酬地域] フィールドで、値を入力または選択します。
    * 報酬地域フィールドにより、その職位の従業員に適用される報酬の適格性ルールおよび固定昇給予算が決定します。  
10. [割り当てに使用可能] フィールドで、日付と時刻を入力します。
11. [職位の期間] セクションを展開します。
    * 職位の期間は、事前に入力された入社日および退職日に基づいて既定で入力されます。  
12. [報告先職位] セクションを展開します。
    * 別の職位に報告する職位に作業者を割り当てる場合、2 種類の職位に割り当てられている作業者間の直接の報告関係を作成します。  
13. [新規] をクリックすると、ドロップ ダイアログが開きます。
14. [報告先職位] フィールドで値を入力または選択します。
15. [作成] をクリックします。
16. [作業者割り当て] セクションを展開します。
17. [リレーションシップ] セクションを展開します。
    * 組織でマトリックスの階層または別のカスタム階層を使用すると、職位階層タイプを設定し、設定した各階層の職位への報告リレーションシップを追加することができます。  
18. [追加] をクリックします。
19. 一覧で、選択された行をマークします。
20. [階層名] フィールドで、値を入力または選択します。
21. [報告先職位] フィールドで値を入力または選択します。
22. [給与] セクションを展開します。
23. [支払サイクル] フィールドで、値を入力または選択します。
24. [支払者] フィールドで、値を入力または選択します。
25. [年間基本勤務時間] フィールドで、数値を入力します。
    * これはこの職位の作業者が年ごとに作業すると予測され、定期的に支払われる時間数です。  
26. [労働組合] セクションを展開します。
27. [労働組合] セクションを折りたたみます。
28. [財務分析コード] セクションを展開します。
29. [配送テンプレート] フィールドで、値を入力または選択します。
30. [部門] フィールドで、値を入力または選択します。
31. [保存] をクリックします。
