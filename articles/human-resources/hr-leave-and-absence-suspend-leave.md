# <a name="suspend-leave"></a><span data-ttu-id="90344-103">休暇の中断</span><span class="sxs-lookup"><span data-stu-id="90344-103">Suspend leave</span></span>

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

<span data-ttu-id="90344-104">従業員の休暇を一時停止して、選択した休暇タイプに対して休暇の見越計上処理を停止できます。</span><span class="sxs-lookup"><span data-stu-id="90344-104">You can suspend leave for an employee to stop leave accruals from being processed for selected leave types.</span></span> 

## <a name="suspend-leave-and-absence-for-an-employee"></a><span data-ttu-id="90344-105">従業員の休暇および欠勤を中断する</span><span class="sxs-lookup"><span data-stu-id="90344-105">Suspend leave and absence for an employee</span></span>

1. <span data-ttu-id="90344-106">従業員のレコードで、**休暇** を選択します。</span><span class="sxs-lookup"><span data-stu-id="90344-106">On the employee's record, select **Leave**.</span></span>

2. <span data-ttu-id="90344-107">**休暇の中断** を選択します。</span><span class="sxs-lookup"><span data-stu-id="90344-107">Select **Suspend leave**.</span></span>

3. <span data-ttu-id="90344-108">**新規** を選択します。</span><span class="sxs-lookup"><span data-stu-id="90344-108">Select **New**.</span></span>

4. <span data-ttu-id="90344-109">**休暇の見越計上の中断** ダイアログ ボックスで、中断の **開始日** および **終了日** と共に **休暇タイプ** を選択します。</span><span class="sxs-lookup"><span data-stu-id="90344-109">In the **Suspend leave accrual** dialog box, select the **Leave type** along with the **Start date** and **End date** for the suspension.</span></span>

5. <span data-ttu-id="90344-110">必要に応じて、中断の **コメント** を追加できます。</span><span class="sxs-lookup"><span data-stu-id="90344-110">Optionally, you can add a **Comment** for the suspension.</span></span> 

<span data-ttu-id="90344-111">従業員の休暇が中断されている間に見越計上が処理された場合、中断された休暇タイプの見越計上は行われません。</span><span class="sxs-lookup"><span data-stu-id="90344-111">If accruals are processed while the employee's leave is suspended, no accrual will be made for the suspended leave types.</span></span>
