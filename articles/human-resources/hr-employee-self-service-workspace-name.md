# <a name="change-employee-self-service-workspace-name"></a><span data-ttu-id="d687b-103">従業員セルフ サービス ワークスペース名の変更</span><span class="sxs-lookup"><span data-stu-id="d687b-103">Change Employee self service workspace name</span></span>

<span data-ttu-id="d687b-104">ボランティアやその他の非従業員がいる場合は、**従業員セルフ サービス** ワークスペースの名前を変更できます。</span><span class="sxs-lookup"><span data-stu-id="d687b-104">If you have volunteers or other non-employees, you might want to change the name of the **Employee self-service** workspace.</span></span> <span data-ttu-id="d687b-105">代わりに、このワークスペースを **セルフ サービス** に変更できます。</span><span class="sxs-lookup"><span data-stu-id="d687b-105">You can change this workspace to **Self service** instead.</span></span>

> [!NOTE]
> <span data-ttu-id="d687b-106">**従業員セルフ サービス** ワークスペースの名前を変更すると、Dynamics 365 Human Resources が内部で使用するメニュー項目も変更されます。</span><span class="sxs-lookup"><span data-stu-id="d687b-106">Changing the name of the **Employee self-service** workspace also changes the menu item that is used internally by Dynamics 365 Human Resources.</span></span> <span data-ttu-id="d687b-107">以前に **HcmEmployeeSelfServiceWorkspace** メニュー項目にセキュリティのカスタマイズを適用している場合は、パリティを保持するために、同じ変更を **HcmSelfServiceWorkspace** に適用することをお勧めします。</span><span class="sxs-lookup"><span data-stu-id="d687b-107">If you previously applied security customizations to the **HcmEmployeeSelfServiceWorkspace** menu item, we recommend applying the same changes to **HcmSelfServiceWorkspace** to maintain parity.</span></span>

1. <span data-ttu-id="d687b-108">Human Resources では、**人事管理** を選択し、**リンク** を選択して、**人事管理パラメーター** を選択します。</span><span class="sxs-lookup"><span data-stu-id="d687b-108">In Human Resources, select **Personnel management**, select **Links**, and then select **Human resources parameters**.</span></span>

2. <span data-ttu-id="d687b-109">**従業員セルフ サービス** タブを選択します。</span><span class="sxs-lookup"><span data-stu-id="d687b-109">Select the **Employee self-service** tab.</span></span>

3. <span data-ttu-id="d687b-110">**表示名** の **セルフ サービス** を選択します。</span><span class="sxs-lookup"><span data-stu-id="d687b-110">Under **Display name**, select **Self service**.</span></span>

   ![従業員セルフ サービス ワークスペース名をセルフ サービスに変更](./media/hr-employee-self-service-workspace-name.png)

4. <span data-ttu-id="d687b-112">**保存** を選択します。</span><span class="sxs-lookup"><span data-stu-id="d687b-112">Select **Save**.</span></span>
