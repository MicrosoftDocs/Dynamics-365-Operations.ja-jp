# <a name="change-employee-self-service-workspace-name"></a>従業員セルフサービスのワークスペース名の変更

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

ボランティアやその他の非従業員がいる場合は、**従業員セルフ サービス** ワークスペースの名前を変更できます。 代わりに、このワークスペースを **セルフ サービス** に変更できます。

> [!NOTE]
> **従業員セルフ サービス** ワークスペースの名前を変更すると、Dynamics 365 Human Resources が内部で使用するメニュー項目も変更されます。 以前に **HcmEmployeeSelfServiceWorkspace** メニュー項目にセキュリティのカスタマイズを適用している場合は、パリティを保持するために、同じ変更を **HcmSelfServiceWorkspace** に適用することをお勧めします。

1. Human Resources では、**人事管理** を選択し、**リンク** を選択して、**人事管理パラメーター** を選択します。

2. **従業員セルフ サービス** タブを選択します。

3. **表示名** の **セルフ サービス** を選択します。

   ![従業員セルフ サービス ワークスペースという名前をセルフ サービスに変更します。](./media/hr-employee-self-service-workspace-name.png)

4. **保存** を選択します。
