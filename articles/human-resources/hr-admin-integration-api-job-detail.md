## <a name="properties"></a>プロパティ

| プロパティ<br>**現物名**<br>**_種類_** | 使用 | 説明 |
| --- | --- | --- |
| **ジョブ ID**<br>mshr_jobid<br>*文字列* | 読み取り専用<br>必須 | ジョブの固有 ID。 |
| **市場価格の低しきい値**<br>mshr_marketpricelowthreshold<br>*実数* | | 職位を一意に識別するためのシステム生成の GUID 値。  |
