# <a name="process-type"></a>プロセスのタイプ

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

このトピックでは、Dynamics 365 Human Resources における [いいえ] と [はい] のオプションについて説明します。

## <a name="option-set-process-type"></a>オプション セット: プロセスのタイプ

物理名: mshr_hrmCompProcessType

このオプションは、Human Resources 仮想エンティティのプロセスのタイプに対するオプション セットを提供します。

| 先頭値 | ラベル | 説明 |
| --- | --- | --- |
| 200000000 | None | なし。 |
| 200000001 | 年次 | 年次。 |
| 200000002 | 記念日 | 記念日です。 |
| 200000003 | 非循環型 | 非循環型。 |
| 200000004 | ½ 年 | 半期に 1 回。 |
| 200000005 | 毎四半期 | 毎四半期。 |
| 200000006 | 月次 | 月次。 |
