# <a name="streamlined-employee-navigation-and-entry"></a>合理化された従業員の入力とナビゲーションと入力

Dynamics 365 Human Resources により、従業員および雇用データを効率的に入力できます。 過去、有効、および未来の従業員および契約社員の作業履歴情報をすばやく更新できます。

## <a name="view-options"></a>オプションの表示

**作業者** ページの **オプションの表示** を使用して、1 つの一覧から従業員と契約社員の組み合わせを選択できます。 これらのオプションにより、法人間、または現在サインインしている法人の従業員を表示できます。 また、有効、保留中、および退職した作業者を表示したり、作業者のタイプ (従業員または契約社員) に基づいて結果を制限したりすることもできます。 高度なセキュリティが有効にされている場合、アクセスできる法人の従業員と契約社員のみが表示されます。

リスト ビューの列は、選択に基づいて変更されます。 たとえば、退職した従業員を表示する場合、退職日および理由コードがリストに追加の列として表示されます。 

## <a name="navigation-and-banner"></a>ナビゲーションおよびバナー

バナーには、各作業者の重要な情報が表示されます。 有効な作業者のバナーには、次の列が表示されます。

- **肩書き**
- **部門**
- **職位タイプ**
- **作業者タイプ**
- **マネージャー**
- **個法**

退職した作業者のバナーには、次の列が表示されます。

- **退職日**
- **理由**

保留中の作業者のバナーには、次の列が表示されます。

- **肩書き**
- **部門**
- **職位タイプ**
- **マネージャー**
- **開始日**
- **個法**

**作業者** ページのアクション ペインは、基本オプションを含めるように再編成されています。 情報は次のカテゴリ別に整理されます。 

- 作業
- 個人
- 終了
- 報酬
- メリット
- コンプライアンス

さらに、**作業者** ページにある **リンク** タブを使用すると、作業者に関連する情報すべてにアクセスできる中心的な場所をユーザーに与えることができます。


## <a name="work-history"></a>作業履歴

**作業履歴** タブに、すべての法人間での作業履歴が表示され、退職、有効、および保留中の従業員および契約社員に対して使用可能です。 アクセスできる法人に関する、すべての作業履歴を一度に表示できます。 さらに、データ コンテキストを変更することなく、各作業履歴入力の情報を編集することができるようになりました。 ページ上のすべての情報を直接更新することができます。 


## <a name="position-history"></a>職位履歴

作業者のメイン ページの **職位** タブに、過去、現在、および未来のすべての割り当てを含む、組織内のすべての職位が完全表示されます。 アクション ペインで作業者の職位履歴に直接移動することができます。
