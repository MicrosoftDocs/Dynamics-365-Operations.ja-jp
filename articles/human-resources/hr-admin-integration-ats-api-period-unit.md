# <a name="period-unit"></a>期間の単位

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

このトピックでは、Dynamics 365 Human Resources における期間の単位オプション セットについて説明します。

\[物理名\] : mshr_periodunit

このリストは、一定の期間の測定単位に対してオプションの値セットを提供します。

| 先頭値 | ラベル | 説明 |
| --- | --- | --- |
| 200000000 | 日数 | 日数です。 |
| 200000001 | 月数 | 月数です。 |
| 200000002 | 年 | 年です。 |
