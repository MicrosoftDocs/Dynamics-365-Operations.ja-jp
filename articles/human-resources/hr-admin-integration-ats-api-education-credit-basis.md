# <a name="education-credit-basis"></a>教育の履修証明基準


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

この記事では、Dynamics 365 Human Resources における教育の履修証明基準のオプションについて説明します。

物理名 : mshr_hcmeducationcreditbasis

このリストは、候補者の教育レコードの教育の履修証明基準となるオプション セットの値を提供します。

| 先頭値 | ラベル | 説明 |
| --- | --- | --- |
| 200000000 | 空白 | 値が選択されていません。 |
| 200000001 | 半期 | 学期です。 |
| 200000002 | 四半期 | 四半期です。 |
| 200000003 | 三半期 | 3学期制の1学期です。 |
| 200000004 | 期間 | 期間です。 |
| 200000005 | 外 | その他。 |
