# <a name="blank-yes-no"></a>空白の [はい] と [いいえ]


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

この記事では、Dynamics 365 Human Resources における空白の [はい] と [いいえ] のオプションについて説明します。

\[物理名\] : mshr_hrmblankyesno

このリストにより、空白を許可するプロパティを許可/なしを指定するオプション セットが提供されます。

| 先頭値 | ラベル | 説明 |
| --- | --- | --- |
| 200000000 | 空白 | 値が選択されていません。 |
| 200000001 | あり | はい。 |
| 200000002 | なし | No. |
