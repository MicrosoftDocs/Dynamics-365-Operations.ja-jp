# <a name="run-the-compensation-process"></a>報酬プロセスを実行する


[!INCLUDE [PEAP](../includes/peap-2.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

**払出**

顧客は Microsoft Dynamics 365 Human Resources で報酬プロセスを実行しようとしていて、成功していません。

**原因**

顧客にはドキュメントが必要です。

**解像度**

報酬のドキュメントを次に示します:

- 処理:

    - [報酬の処理](/dynamics365/unified-operations/talent/process-compensation)
