# <a name="request-time-off"></a>休暇の申請

Dynamics 365 Human Resources において、休暇申請の送信や、休暇残日数の表示、さらに申請中休暇のステータスを確認することができます。

## <a name="request-time-off"></a>休暇の申請

1. **従業員セルフ サービス** ワークスペースで、**休暇残日数**タイルの**休暇の申請**を選択します。

2. **休暇タイプ**、**理由コード**、**開始日**、および**終了日**に情報を入力します。

3. **日付**で、休暇申請の日付を選択します。

4. サポート ドキュメントを提出する必要がある場合、**添付ファイル**の中の**アップロード**を選択します。

5. 必要に応じて**コメント**に情報を入力します。

6. 申請を送信する準備ができたら、**送信**を選択します。 それ以外の場合は、**下書きの保存**を選択します。

## <a name="view-leave-balances"></a>休暇残日数の表示

1. **従業員セルフサービス** ワークスペースで、**休暇残日数**タイルの**詳細** (...) を選択します。

2. **残日数**を選択します。

## <a name="view-leave-request-status"></a>休暇申請のステータスの表示

1. **従業員セルフサービス** ワークスペースで、**休暇残日数**タイルの**詳細** (...) を選択します。

2. 承認された休暇申請を表示するには、**承認済休暇**を選択します。 保留中の休暇申請を表示するには、**休暇申請**を選択します。
