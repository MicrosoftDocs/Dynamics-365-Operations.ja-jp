# <a name="assign-workers-to-a-leave-plan"></a><span data-ttu-id="3ee9d-103">休暇計画に作業者を割り当てる</span><span class="sxs-lookup"><span data-stu-id="3ee9d-103">Assign workers to a leave plan</span></span>

   - <span data-ttu-id="3ee9d-109">**雇用開始日の登録**</span><span class="sxs-lookup"><span data-stu-id="3ee9d-109">**Enroll on employment start date**</span></span>

   - <span data-ttu-id="3ee9d-112">**層基準**</span><span class="sxs-lookup"><span data-stu-id="3ee9d-112">**Tier basis**</span></span>
