# <a name="assign-workers-to-a-leave-plan"></a><span data-ttu-id="502a6-103">休暇計画に作業者を割り当てる</span><span class="sxs-lookup"><span data-stu-id="502a6-103">Assign workers to a leave plan</span></span>

## <a name="assign-a-plan-to-a-group-of-employees"></a><span data-ttu-id="502a6-105">従業員のグループに計画を割り当てる</span><span class="sxs-lookup"><span data-stu-id="502a6-105">Assign a plan to a group of employees</span></span>

1. <span data-ttu-id="502a6-106">**休暇** ページで、割り当てたい休暇計画を選択します。</span><span class="sxs-lookup"><span data-stu-id="502a6-106">On the **Leave and absence** page, select the plan you want to assign.</span></span>

2. <span data-ttu-id="502a6-107">**従業員を割り当てる** を選択します。</span><span class="sxs-lookup"><span data-stu-id="502a6-107">Select **Assign to employees**.</span></span>

3. <span data-ttu-id="502a6-108">必要に応じて次の設定を調整します。</span><span class="sxs-lookup"><span data-stu-id="502a6-108">Adjust the following settings as appropriate:</span></span>

   - <span data-ttu-id="502a6-109">**雇用開始日の登録**</span><span class="sxs-lookup"><span data-stu-id="502a6-109">**Enroll on employment start date**</span></span>
   - <span data-ttu-id="502a6-110">**登録日**</span><span class="sxs-lookup"><span data-stu-id="502a6-110">**Enrollment date**</span></span>
   - <span data-ttu-id="502a6-111">**終了日**</span><span class="sxs-lookup"><span data-stu-id="502a6-111">**End date**</span></span>
   - <span data-ttu-id="502a6-112">**層基準**</span><span class="sxs-lookup"><span data-stu-id="502a6-112">**Tier basis**</span></span>

4. <span data-ttu-id="502a6-113">**登録する作業者** で、その休暇計画に割り当てるすべての作業者を選択します。</span><span class="sxs-lookup"><span data-stu-id="502a6-113">Under **Workers to enroll**, select all the works to assign to the plan.</span></span>

## <a name="assign-a-plan-to-an-employee"></a><span data-ttu-id="502a6-115">1 人の従業員に計画を割り当てる</span><span class="sxs-lookup"><span data-stu-id="502a6-115">Assign a plan to an employee</span></span>
