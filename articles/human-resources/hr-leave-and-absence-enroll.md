# <a name="assign-workers-to-a-leave-plan"></a><span data-ttu-id="11d6c-103">休暇計画に作業者を割り当てる</span><span class="sxs-lookup"><span data-stu-id="11d6c-103">Assign workers to a leave plan</span></span>

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

<span data-ttu-id="11d6c-104">Dynamics 365 Human Resources で休暇計画を作成したら、それを作業者に割り当てます。</span><span class="sxs-lookup"><span data-stu-id="11d6c-104">After you've created a leave and absence plan in Dynamics 365 Human Resources, assign it to workers.</span></span>

## <a name="assign-a-plan-to-a-group-of-employees"></a><span data-ttu-id="11d6c-105">従業員のグループに計画を割り当てる</span><span class="sxs-lookup"><span data-stu-id="11d6c-105">Assign a plan to a group of employees</span></span>

1. <span data-ttu-id="11d6c-106">**休暇** ページで、割り当てたい休暇計画を選択します。</span><span class="sxs-lookup"><span data-stu-id="11d6c-106">On the **Leave and absence** page, select the plan you want to assign.</span></span>

2. <span data-ttu-id="11d6c-107">**従業員を割り当てる** を選択します。</span><span class="sxs-lookup"><span data-stu-id="11d6c-107">Select **Assign to employees**.</span></span>

3. <span data-ttu-id="11d6c-108">必要に応じて次の設定を調整します。</span><span class="sxs-lookup"><span data-stu-id="11d6c-108">Adjust the following settings as appropriate:</span></span>

   - <span data-ttu-id="11d6c-109">**雇用開始日の登録**</span><span class="sxs-lookup"><span data-stu-id="11d6c-109">**Enroll on employment start date**</span></span>
   - <span data-ttu-id="11d6c-110">**登録日**</span><span class="sxs-lookup"><span data-stu-id="11d6c-110">**Enrollment date**</span></span>
   - <span data-ttu-id="11d6c-111">**終了日**</span><span class="sxs-lookup"><span data-stu-id="11d6c-111">**End date**</span></span>
   - <span data-ttu-id="11d6c-112">**層基準**</span><span class="sxs-lookup"><span data-stu-id="11d6c-112">**Tier basis**</span></span>

4. <span data-ttu-id="11d6c-113">**登録する作業者** で、その休暇計画に割り当てるすべての作業者を選択します。</span><span class="sxs-lookup"><span data-stu-id="11d6c-113">Under **Workers to enroll**, select all the works to assign to the plan.</span></span>

5. <span data-ttu-id="11d6c-114">ページの下部にある **割り当てる** を選択します。</span><span class="sxs-lookup"><span data-stu-id="11d6c-114">Select **Assign plan** at the bottom of the page.</span></span>

## <a name="assign-a-plan-to-an-employee"></a><span data-ttu-id="11d6c-115">1 人の従業員に計画を割り当てる</span><span class="sxs-lookup"><span data-stu-id="11d6c-115">Assign a plan to an employee</span></span>

1. <span data-ttu-id="11d6c-116">従業員のレコードで、**休暇** を選択します。</span><span class="sxs-lookup"><span data-stu-id="11d6c-116">On the employee's record, select **Leave**.</span></span>

2. <span data-ttu-id="11d6c-117">**休暇** を選択します。</span><span class="sxs-lookup"><span data-stu-id="11d6c-117">Select **Leave and absence**.</span></span>

3. <span data-ttu-id="11d6c-118">**新規** を選択します。</span><span class="sxs-lookup"><span data-stu-id="11d6c-118">Select **New**.</span></span>

4. <span data-ttu-id="11d6c-119">登録日を入力し、休暇計画を選択します。</span><span class="sxs-lookup"><span data-stu-id="11d6c-119">Enter an enrollment date and select a leave plan.</span></span>
