# <a name="assign-workers-to-a-leave-plan"></a>休暇計画に作業者を割り当てる

> [!Important]
> このトピックで説明した機能は、現在スタンドアロンの Dynamics 365 Human Resources の顧客が利用できます。 Finance のリリース 10.0.26 以降に、Finance インフラストラクチャの今後のリリースで、一部またはすべての機能を使用できます。


[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Dynamics 365 Human Resources で休暇計画を作成したら、それを作業者に割り当てます。

## <a name="assign-a-plan-to-a-group-of-employees"></a>従業員のグループに計画を割り当てる

1. **休暇** ページで、割り当てたい休暇計画を選択します。

2. **従業員を割り当てる** を選択します。

3. 必要に応じて次の設定を調整します。

   - **雇用開始日の登録**
   - **登録日**
   - **終了日**
   - **層基準**

4. **登録する作業者** で、その休暇計画に割り当てるすべての作業者を選択します。

5. ページの下部にある **割り当てる** を選択します。

## <a name="assign-a-plan-to-an-employee"></a>1 人の従業員に計画を割り当てる

1. 従業員のレコードで、**休暇** を選択します。

2. **休暇** を選択します。

3. **新規** を選択します。

4. 登録日を入力し、休暇計画を選択します。
