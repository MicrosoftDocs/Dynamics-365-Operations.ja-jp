# <a name="removed-or-deprecated-features-in-dynamics-365-human-resources"></a>Dynamics 365 Human resources の削除済みまたは非推奨の機能

この記事では、Microsoft Dynamics 365 Human resources から削除された、または削除される予定の機能について説明します。

- *削除された* 機能は製品では使用できません。
- *非推奨* の機能は現在開発中ではなく、将来の更新で削除される可能性があります。

このリストは、これらの削除および削除予定に対して、自身の計画を検討するために役立ちます。

## <a name="features-removed-or-deprecated-in-human-resources"></a>Human resources の削除済みまたは非推奨の機能 

### <a name="new-lcs-project-creation-in-human-resources-stand-alone"></a>Human resources での新しい LCS プロジェクトの作成 (スタンドアロン) 

[Human resources のプロビジョニング](/hr-admin-setup-provision)

| &nbsp; | &nbsp; |
|------------|--------------------|
| **廃止 / 削除の理由** | 財務と運用インフラストラクチャのプロジェクト作成に置き換えられます。 インフラストラクチャ マージの詳細については、[Dynamics 365 Human resource インフラストラクチャ マージに関するよく寄せられる質問](/hr-infrastructure-merge-faq) を参照してください。 |
| **別の機能で置き換えられているか?**   | 有効 |
| **影響を受ける製品領域**         | アプリケーション |
| **配置オプション**              | すべて |
| **状態**                         | 非推奨: 2022 年 7 月現在、Human resources での新しいプロジェクトの作成 (スタンドアロン) はサポートされません。 |

### <a name="production-environment-provisioning-on-human-resources-stand-alone"></a>Human resources (スタンドアロン) の実稼働環境プロビジョニング 

| &nbsp; | &nbsp; |
|------------|--------------------|
| **廃止 / 削除の理由** | 財務と運用インフラストラクチャの環境プロビジョニングに置き換えられます。 インフラストラクチャ マージの詳細については、[Dynamics 365 Human resource インフラストラクチャ マージに関するよく寄せられる質問](/hr-infrastructure-merge-faq.md) を参照してください。 |
| **別の機能で置き換えられているか?**   | 有効 |
| **影響を受ける製品領域**         | アプリケーション |
| **配置オプション**              | すべて |
| **状態**                         | 非推奨: 2022 年 7 月現在、Human resources (スタンドアロン) での HR 運用環境のプロビジョニングはサポートされなくなりました。 |
