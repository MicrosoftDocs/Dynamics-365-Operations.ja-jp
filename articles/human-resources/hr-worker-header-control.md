# <a name="worker-header-control"></a>作業者ヘッダー コントロール

Microsoft Dynamics 365 Human Resources は、**従業員セルフ サービス**、**People** ハブ、合理化された **作業者** ページで、カスタマイズ可能なヘッダー コントロールを提供します。 ヘッダーには、作業者に関する重要な情報と、メール、通話、メッセージングなどの 1 回のクリックによるアクションが含まれます。 ヘッダーを変更するには、フィールドを削除するか、カスタム フィールドを含むフィールドを追加して、追加情報を表示します。 新しいヘッダー コントロールを使用するには、**機能管理** に移動し、**作業者ヘッダー コントロール** 機能を有効にします。

## <a name="personalization"></a>個人用設定

作業者ヘッダー コントロールの主な利点の 1 つは、ヘッダーに表示される情報をカスタマイズする機能です。

ヘッダーの右上には、従業員のステータスに応じて異なるデータが表示される 3 つのフィールドのグループがあります。 このフィールドのグループは、ヘッダーのスポットライト部分と呼ばれます。 ヘッダーのスポットライト部分をカスタマイズするには、現在のフィールドを削除するか、フィールドを追加します。 ヘッダー コントロールのスポットライト部分に追加できるフィールドは、**従業員セルフ サービス**、**People** ハブ、および **作業者** ページで異なります。

## <a name="employee-self-service"></a>従業員セルフ サービス

**従業員セルフサービス** ページの作業者ヘッダーには、次の情報が含まれます:

- 作業者名
- 個人番号
- 職位のタイトル
- 部門
- 作業者タイプ
- 雇用の法人
- 勤続年数
- レポート先 (マネージャー)
- 職位タイプ

ヘッダーには、個人の個人情報を編集するための 1 回のクリックによるアクションも含まれます。 **個人的な詳細情報の編集** を選択して、**従業員セルフ サービス** の **個人情報** ページを開きます。

## <a name="people-hub"></a>People ハブ

**People** ハブ (**People ワークスペース** ページ) の作業者ヘッダーには、次の情報が含まれます:

- 作業者名
- 個人番号
- 職位のタイトル
- 部門
- 作業者タイプ
- 雇用の法人
- 勤続年数
- レポート先 (マネージャー)
- 職位タイプ

ヘッダーには、メール、通話、メッセージングなどの 1 回のクリックによるアクションも含まれます。 ユーザーが **People ワークスペース** ページで独自の情報を表示している場合、1 回のクリックによるアクション セクションには **個人的な詳細情報の編集** ボタンが含まれます。 ユーザーがこのボタンを選択して、**従業員セルフ サービス** の **個人情報** ページを開きます。

## <a name="streamlined-worker-page"></a>合理化された作業者ページ

合理化された **作業者** ページの作業者ヘッダーには、次の情報が含まれます:

- 作業者名
- 個人番号
- 職位のタイトル
- 部門
- 作業者タイプ
- 雇用の法人

従業員のステータスによっては、ヘッダーのデータが変わる場合があります。 たとえば、従業員が会社を退職した場合、ヘッダー コントロールには、**退職済** バッジ、雇用終了日、退職理由が含まれます。

次の表に、さまざまな従業員ステータスのヘッダーに表示されるデータを示します。

|-----------------|--------------------|------|
| 保留中 | <ul><li>Name</li><li>個人番号</li><li>職位のタイトル</li><li>部門</li><li>作業者タイプ</li><li>法人エンティティ</li><li>保留中バッジ</li><li>開始日</li><li>レポート先</li><li>職位タイプ</li></ul> | |
| 最近の採用 | <ul><li>Name</li><li>個人番号</li><li>職位のタイトル</li><li>部門</li><li>作業者タイプ</li><li>法人エンティティ</li><li>最近の採用バッジ</li><li>勤続年数</li><li>レポート先</li><li>職位タイプ</li></ul> | 既定では、最近 7 日間に採用された従業員に対して **最近の採用** バッジが表示されます。 この設定を変更するには、**人事管理パラメーター** ページの **全般** タブにある、**最近の採用日付範囲** セクションに概算時間を入力します。 たとえば、過去 14 日間に採用された従業員の **最近の採用** バッジを表示するには、**期間** フィールドを **14** に、**単位** フィールドを **日数** に設定します。 |
| 有効な従業員 | <ul><li>Name</li><li>個人番号</li><li>職位のタイトル</li><li>部門</li><li>作業者タイプ</li><li>法人エンティティ</li><li>開始日</li><li>レポート先</li><li>職位タイプ</li></ul> | |
| 退職 | <ul><li>Name</li><li>個人番号</li><li>職位のタイトル</li><li>部門</li><li>作業者タイプ</li><li>法人エンティティ</li><li>退職バッジ</li><li>勤続年数</li><li>レポート先</li><li>職位タイプ</li></ul> | 既定では、これから 7 日以内に退職する従業員に対して **退職** バッジが表示されます。 この設定を変更するには、**人事管理パラメーター** ページの **全般** タブにある、**退職予定の作業者の日付範囲** セクションに概算時間を入力します。 たとえば、これから 14 日以内に退職する従業員の **退職** バッジを表示するには、**期間** フィールドを **14** に、**単位** フィールドを **日数** に設定します。 |
