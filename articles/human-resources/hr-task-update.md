# <a name="set-up-tasks-in-task-management"></a>タスク管理でのタスクの設定

[!INCLUDE [PEAP](../includes/peap-1.md)]

バージョン 10.0.30 より前の Microsoft Dynamics 365 Human Resources のバージョンでは、タスクを編集するユーザーは、タスクを含む各チェックリストでそのタスクを個別に編集する必要がありました。 ただし、Human Resources バージョン 10.0.30 では、ユーザーは編集したタスクの処理方法を選択できます。 編集中のタスクがチェックリスト上にある場合は、**人事管理の共有パラメーター** ページの **タスク管理** タブで **タスク管理のアップグレードの有効化** オプションを選択し、編集したタスクを使用するためのチェックリストを有効にする必要があります。

[![人事管理の共有パラメーター ページで、タスク管理のアップグレード オプションを有効にします。](./media/task-update.png)](./media/task-update.png)

タスクの編集を、関連付けられているチェックリスト タスクすべてに適用する必要がある場合は、タスク ライブラリのタスクとチェックリスト上のタスクの間に既にリレーションシップが存在している必要があります。 ライブラリ タスクとチェックリスト タスク間の関係を作成するために、オプションが追加されました。

タスクを個別に作成した後、複数のチェックリストで再利用できます。 タスクを作成するには、**オンボード設定** ページの **タスク** タブで、**新規** を選択します。

## <a name="set-up-tasks"></a>タスクの設定

作成したタスクを複数のチェックリストに割り当てるには、タスクを選択し、メニューの **チェックリストに適用** を選択します。

または、チェックリストに直接タスクを追加することもできます。 タスクをチェックリストに追加するには、**オンボード設定** ページの **チェックリスト** タブで、タスクを追加する新しいチェックリストを作成するか、タスクを既存のチェックリストに追加します。

ライブラリ タスクを編集するには、タスク ライブラリ メニューで **編集** を選択します。 タスクがチェックリストに関連付けられている場合は、それらのチェックリストが **タスクの編集** ページに表示されます。 編集によってチェックリストのタスクを更新する場合は、**チェックリストに適用** セクションでそれらのチェックリストを選択します。

タスク ライブラリからタスクを削除するには、**削除** を選択します。 タスクがチェックリストに関連付けられている場合、このアクションはチェックリストからタスクを削除しません。 チェックリストからタスクを削除するには、別のアクションが必要です。

### <a name="set-up-checklists"></a>チェックリストの設定

チェックリストはタスクのグループです。 チェックリストは必要な数でも作成できます。また、同じタスクを複数のチェックリストに割り当てすることもできます。 チェックリストを作成する場合は、所有者とカレンダーを指定します。

チェックリストで新しいタスクを作成するには、タスク メニュー バーの **新規** を選択します。 新しいタスクを作成するときに、必要に応じてタスク ライブラリにタスクを追加して、複数のチェックリスト間で共有することができます。 タスクをタスク ライブラリに追加するには、**タスクをライブラリに適用** オプションを **はい** に設定する必要があります。 タスク ライブラリにタスクを追加することを選択した場合は、**チェックリストに適用** セクションでそれらのチェックリストを選択することで、同時に他のチェックリストにタスクを追加することもできます。 タスク ライブラリにタスクを追加しないことを選択した場合は、タスクは作成したチェックリストにのみ存在します。

チェックリストでタスクを編集するには、タスク メニューで **編集** を選択します。 タスクがチェックリストに関連付けられている場合は、それらのチェックリストが **タスクの編集** ページに表示されます。 編集で他のチェックリストのタスクを更新する場合は、**チェックリストに適用** セクションでそれらのチェックリストを選択します。

チェックリストからタスクを削除するには、**削除** を選択します。 このアクションは、チェックリストからタスクを削除します。 ただし、タスク ライブラリからタスクは削除されません。 タスク ライブラリからタスクを削除するには、**タスク ライブラリ** ページを開き **削除** を選択します。
