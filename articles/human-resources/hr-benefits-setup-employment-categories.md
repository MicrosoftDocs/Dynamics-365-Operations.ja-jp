# <a name="create-employment-categories"></a>雇用カテゴリの作成

Dynamics 365 Human Resources 雇用カテゴリを設定して、作業者と同じにすることができます。

1. **給付金管理** ワーク スペースの **設定** で、**雇用カテゴリ** を選択します。

2. **新規** を選択します。

3. **作業者タイプ**、**雇用カテゴリ**、および **説明** の値を指定します。

4. **保存** を選択します。
