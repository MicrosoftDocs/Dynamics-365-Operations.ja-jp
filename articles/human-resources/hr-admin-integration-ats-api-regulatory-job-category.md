# <a name="regulatory-job-category"></a>規制のジョブ カテゴリ


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

この記事では、Dynamics 365 Human Resources における規制のジョブ カテゴリのオプション セットについて説明します。

物理名 : mshr_hcmregulatoryjobcategory

この問題は、規制のジョブ カテゴリ オプション セットの値に対するオプション セットを指定します。 これは、既存の mshr_hcmregulatoryjobcategory オプション セットで提供されます。

| 先頭値 | ラベル | 説明 |
| --- | --- | --- |
| 200000000 | None | なし。 |
| 200000001 | 経営幹部 | 経営幹部/上級レベルの担当者およびマネージャーです。 |
| 200000002 | 管理者 | 第1/中間レベルの担当者およびマネージャーです。 |
| 200000003 | 職務 | 職務です。 |
| 200000004 | 技術者 | 技術者です。 |
| 200000005 | 営業社員 | 営業社員です。 |
| 200000006 | 管理 | 管理サポートの従業員です。 |
| 200000007 | 製作担当社員 | 製作担当社員です。 |
| 200000008 | 工員 | 工員です。 |
| 200000009 | 労働者 | 労働者/ヘルパーです。 |
| 200000010 | サービス作業員 | サービス作業員です。 |
