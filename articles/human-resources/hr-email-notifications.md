# <a name="benefits-email-notification"></a>給付金のメール通知

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]
[!include [banner](../includes/preview-banner.md)]

メール通知機能を使用すると、次のシナリオでメール通知とリマインダーを従業員に送信できます。

- 新規採用登録
- 登録を開く
- 特定ライフ イベント

複数のメール テンプレートを作成および設定し、**給付金管理** ワークスペースと **作業者の給付金** ページを使用して通知を送信することができます。 また、通知/リマインダーの履歴を追跡することもできます。

## <a name="set-up-human-resources-shared-parameters"></a>人事管理共有パラメーターの設定

**人事管理共有パラメーター** ページで、従業員または従業員グループに送信するさまざまなタイプのコミュニケーションで使用できる給付金メール テンプレートを作成できます。

## <a name="create-a-new-email-id-template"></a>新しいメール ID テンプレートの作成

新しいメール ID テンプレートを作成するには、次のステップに従います。

1. **システムのメール テンプレート** に移動します。
2. **新規** を選択します。
3. **メール ID** フィールドに、名前を入力します。
4. 必要に応じてその他のフィールドを設定します。
5. テンプレートをアップロードするには、**メール メッセージ** を選択します。
6. **人事管理共有パラメータ** ページで、**システム テンプレート** にある新しいテンプレートを選択し、そのテンプレートを **給付金用テンプレート** に移動します。

## <a name="send-email-to-employees"></a>メールを従業員に送信

まだ仕事を開始していない新規採用者にメールを送信するには、次のステップに従います。

1. **給付金管理** ワークスペースを開きます。
2. メールを送信したい従業員のグループ用のタイルを選択します。
3. **メールの送信** を選択します。
4. メールを送信したい従業員を選択します。
5. **メールの送信** を選択します。
6. 使用するテンプレートを選択します。
7. **OK** を選択してメールを送信します。

    従業員の **従業員の給付金** ページから、または **給付金管理** ワークスペースの **リンク** セクションにある **給付金メール履歴** を選択して、メール メッセージと状態を確認できます。 **作業者の給付金プラン** ページからメールを送信することもできます。

8. 給付金メールの履歴を表示するには、**リンク** セクションの **給付金管理** ワークスペースで、**給付金メールの履歴** を選択します。
9. 送信された給付金メールの完全なメール履歴を表示します。 メールの内容を確認するには、**メッセージの表示** を選択します。
10. **作業者** を選択します。
11. **従業員の給付金プラン** ページで、メールを送信し、給付金のメール履歴を表示できます。

**給付金管理** ワークスペースには、異なるタイルが含まれます。 関連するテンプレートを選択すると、メールを送信できます。 新規採用登録のメールが送信された場合は、**新規採用 (未開始)** タイルを選択してから **リマインダーを送信** リンクを選択します。 リマインダー テンプレートを選択し、通知のタイトルを 1 回目、2 回目、または 3 回目のリマインダーとして設定できます。
