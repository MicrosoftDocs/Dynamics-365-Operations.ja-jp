# <a name="gender"></a>性別


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

この記事では、Dynamics 365 Human Resources における性別オプションについて説明します。

物理名 : mshr_hcmpersongender

このリストは、候補者の性別に対するオプション セットを提供します。 このオプションは、mshr_hcmpersongender のオプションセットで使用できます。

| 先頭値 | ラベル | 説明 |
| --- | --- | --- |
| 200000000 | None | 性別が指定されていません。 |
| 200000001 | 男性 | 男性です。 |
| 200000002 | 女性 | 女性です。 |
| 200000003 | 不特定 | 特定の性別以外の選択です。 |
