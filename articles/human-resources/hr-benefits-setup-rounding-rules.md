# <a name="set-up-rounding-rules"></a>丸めルールの設定


[!INCLUDE [PEAP](../includes/peap-2.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Dynamics 365 Human Resources で丸めルールを設定して、組織のニーズを合わせることができます。

1. **給付金管理** ワーク スペースの **設定** で、**丸めルール** を選択します。

2. **新規** を選択します。

3. **最も近い値に丸める**、**シナリオ**、および **丸めフォーム** の値を指定します。

4. **保存** を選択します。
