# <a name="screening-frequency"></a>審査の頻度

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

このトピックでは、Dynamics 365 Human Resources における審査の頻度のオプション セットについて説明します。

物理名 : mshr_hcmfrequencyunit

このリストは、審査の頻度のオプション セットを提供します。 

| 先頭値 | ラベル | 説明 |
| --- | --- | --- |
| 200000000 一時のみ | 審査が必要となるのは一度のみです。 |
| 200000001 日次 | 審査頻度は日数で計算されます。 |
| 200000002 週次 | 審査頻度は週で計算されます。 |
| 200000003 月次 | 審査頻度は月で計算されます。 |
| 200000004 年次 | 審査頻度は年で計算されます。 |
