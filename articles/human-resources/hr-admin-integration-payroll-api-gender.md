# <a name="payroll---gender-option-set"></a>給与 - 性別オプション セット

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

このトピックでは、Dynamics 365 Human Resources における性別のオプションについて説明します。

## <a name="option-set-gender-option"></a>オプション セット: 性別オプション

このリストは、従業員の性別に対するオプション セットを提供します。 このオプションは、mshr_hcmpersongender のオプションセットで使用できます。

| 先頭値 | ラベル | 説明 |
| --- | --- | --- |
| 200000000 | None | 性別が指定されていません。 |
| 200000001 | 男性 | 男性です。 |
| 200000002 | 女性 | 女性です。 |
| 200000003 | 不特定 | 特定の性別以外の選択です。 |
