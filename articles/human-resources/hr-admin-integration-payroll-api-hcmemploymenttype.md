# <a name="payroll---employment-type-option-set"></a>給与 - 雇用タイプ オプション セット

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

このトピックでは、Dynamics 365 Human Resources における雇用タイプのオプション セットについて説明します。

## <a name="option-set-employment-type"></a>オプション セット: 雇用タイプ

このリストは、従業員の雇用タイプに対するオプション セットを提供します。 このオプションは、mshr_hcmemploymenttype のオプション セットで使用できます。

| 先頭値 | ラベル | 説明 |
| --- | --- | --- |
| 200000000 | 従業員 | 従業員 |
| 200000001 | 契約社員 | 契約社員 |
