# <a name="award-type"></a>報奨タイプ

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

このトピックでは、Dynamics 365 Human Resources における報奨タイプのオプションについて説明します。

## <a name="option-set-award-type"></a>オプション セット: 報奨タイプ

物理名: mshr_HrmCompVarAwardEmplType

このオプションは、Human Resources 仮想エンティティの変動報酬の報奨タイプに対するオプション セットを提供します。

| 先頭値 | ラベル | 説明 |
| --- | --- | --- |
| 200000000 | 一般報奨 | 一般報奨。 |
| 200000001 | 一括比例配分固定支払 | 一括比例配分固定支払。 |
