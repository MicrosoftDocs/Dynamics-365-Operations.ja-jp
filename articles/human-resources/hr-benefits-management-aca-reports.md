# <a name="generate-aca-reports-in-benefits-management"></a>福利厚生管理で ACA レポートを生成する

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

福利厚生管理は、Form1095-B および Form 1095-C でアフォーダブル ケア法 (ACA) の雇用主の命令に関して報告された情報を追跡します。 古い **福利厚生** ワークスペースの ACA レポート機能と同様に、この機能は米国の法人にのみ適用されるものです。

この機能を使用するには、最初に **高度な福利厚生管理** を有効にする必要があります。 福利厚生管理に関する重要な説明などの詳細については、[福利厚生管理の有効化または無効化](hr-admin-manage-features.md#enable-or-disable-benefits-management) を参照してください。

> [!IMPORTANT]
> ACA レポートは、**福利厚生管理** ワークスペース、または古い **福利厚生** ワークスペースのいずれかからのみ使用してください。この両方から使用することはできません。 たとえば、福利厚生管理に切り替えた場合、ACA レポートは、古い **福利厚生** ワークスペースからではなく、**福利厚生管理ワークスペース** からのみ使用できます。
>
> 加入契約をした年度の途中で福利厚生管理に切り替える場合、福利厚生管理で通年分の従業員データを正しく構成する必要があります。 これにより、年間を通して正確なレポート情報を確実に受け取ることができます。

## <a name="getting-started"></a>はじめに

Form 1095-B および Form 1095-C の情報を追跡するには、まずアフォーダブル ケア法の補償グループを 1 つ以上作成する必要があります。 これらのグループは、次の情報を示しています :

- 従業員に提供された保険のオファー
- コストが連邦政府の貧困ラインを超えている場合の、最低月額保険料の従業員負担額
- 雇用主が使用する免責事項 (該当する場合)

変更されたアフォーダブル ケア法のグループを使用すると、同じ条件を持つ複数の従業員記録に対して、この情報を管理するの場合に役立ちます。 1人または複数の従業員に簡単にグループを割り当てることができます。

### <a name="create-or-edit-an-affordable-care-coverage-group"></a>アフォーダブル ケア法のグループを作成・編集する

1. **福利厚生管理** ワークスペースで、**アフォーダブル ケア法のグループ** を選択します。

    ![アフォーダブル ケア法のグループを選択する。](./media/hr-benefits-management-aca-coverage-group.png)

2. アフォーダブル ケア法のグループを新規作成するには、**新規** を選択し、既存のグループを編集するには、**編集** を選択します。

    ![新規または編集を選択する。](./media/hr-benefits-management-aca-new.png)

3. 次のフィールドを設定します。

    | フィールド | 説明 |
    |---|---|
    | 氏名 | グループの名前を入力します。 |
    | 説明 | グループの説明を入力します。 |
    | 補償内容 | 従業員、その配偶者、パートナー、その扶養家族に提供される保険です。 |
    | 費用の従業員負担 | 従業員が負担する金額です。 |
    | 適用区域 4980H Safe Harbor | 4980 H 免責条項、または移行支援コードです。 |
    | 計画開始月 | 福利厚生プランの年度が始まる暦月を選択します。 |
    | グループの発効日 | このレコードが有効となる最初の日付です。 |
    | グループの有効期限 | このレコードが有効となる最後の日付です。 有効期限がない場合は、**なし** と入力します。 |

    ![補充グループを作成する。](./media/hr-benefits-management-aca-new-group.png)

4. **保存** を選択します。

### <a name="assign-multiple-employees-to-an-affordable-care-coverage-group"></a>アフォーダブル ケア法のグループに複数の従業員を割り当てる

1. **福利厚生管理** ワークスペースで、**アフォーダブル ケア法のグループ** を選択します。
2. 従業員を割り当てるグループを選択します。
3. **一括割り当て** を選択します。

    ![一括割り当てを選択する。](./media/hr-benefits-management-aca-mass-assignment.png)

4. 一覧で従業員を選択し、**割り当て** を選択します。

    ![選択した従業員をグループに割り当てる。](./media/hr-benefits-management-aca-assign-coverage-group.png)

## <a name="maintain-multiple-versions-of-coverage-options"></a>アフォーダブル ケア法の複数バージョンのオプションを管理する

任意のアフォーダブル ケア法のグループの複数のバージョンを維持することができます。 組織内で変更があった場合や、提供する福利厚生が変わった際に、新しいグループを作成して社員を再割り当てすることなく、グループの情報を常に最新の状態に保つことができます。

アフォーダブル ケア法のグループを作成した後は、従業員を大量に割り当てることができます。 従業員をグループに個別に割り当て、ACA の情報を追跡および報告するかどうかを指定することもできます。

従業員の ACA 情報を追跡・報告する必要がない場合は、**レポート補充** オプションを **いいえ** に 設定できます。 たとえば、ACA のレポートが不要なパートタイムの従業員がいるとします。

### <a name="override-default-values-for-an-employee"></a>従業員の既定値を上書きする

アフォーダブル ケア法のグループに割り当てられている従業員の場合、異なる値を必要とする月については、以下のオプションを変更することができます :

- 補償内容
- 費用の従業員負担
- 適用区域 4980H Safe Harbor

> [!NOTE]
> 2020 のACA レポートについては、Form 1095-C に仕事場と自宅の両方の郵便番号を報告刷る必要があります。 値は現在のアクティブな場所に基づいて自動的に入力されます。 いずれかの場所が年間のいずれかの期間で異なっていた場合は、値を上書きする必要があります。 1095-C レポートの **郵便番号** フィールド (17行目) は、 **補償の提供** コードに **1L** から **1Q** が含まれている場合にのみ入力されます。
>
> - **1L、1M、1N** : 基本住所の郵便番号です
> - **1O、1P、1Q**: 基本勤務場所の郵便番号です

アフォーダブル ケア法のグループの値の例外を入力するには、以下の手順に従います。

1. **人事管理** ワークスペースで、**リンク** を選択し、**作業者** を選択します。
2. リストで従業員を選択します。
3. **雇用** タブの **詳細** セクションで、**アフォーダブル ケア法** を選択します。

    ![1 人の従業員のオプションを変更する。](./media/hr-benefits-management-aca-change-single-employee.png)

4. **編集** を選択します。
5. 変更が必要な各月については、**既定を上書きする** チェックボックスを選択し、必要に応じて他の値を変更します。

    ![既定値を上書きする。](./media/hr-benefits-management-aca-override-default.png)

6. **保存** を選択します。

## <a name="report-health-care-coverage"></a>医療補償のレポート

フルタイム従業員およびパートタイムの従業員に対して、雇用主がスポンサーとなっている自己負担の医療保険の適用範囲を追跡する必要があります。 各従業員を扶養家族とともに、対象となった月の 1095-C レポートに記載してください。

福利厚生プランをレポートする必要がある場合は、次の手順に従います。

1. **給付金管理** ワーク スペースで、**福利厚生プラン** を選択します。
2. レポートする福利厚生プランを選択します。
3. **編集** を選択します。
4. **アフォーダブル ケア法に基づくレポート** のオプションを **はい** に設定します。

    ![医療保険のレポート。](./media/hr-benefits-management-aca-report-coverage.png)

5. **保存** を選択します。

従業員が被扶養者の医療保険を選択した場合、被扶養者の保険期間は、被扶養者が登録または削除された日によって決定されます。 扶養家族の保険適用日は、加入年度に扶養家族の資格があり、プランに加入していた期間に基づいて自動的に計算されます。

## <a name="generate-1095-b-and-1095-c-forms"></a>Form 1095-B および Form 1095-C

ACA の 1095-B と 1095-C を作成し、各従業員に配布することができます。 また、国税庁 (IRS) に送信するための Form1095-C と対応する Form 1094-C の送信ファイルを電子的に作成することもできます。

1. **福利厚生管理** ワークスペースで、**Form ACA 1095-B**、または **Form ACA 1095-C** を選択します。
2. 必要に応じてパラメータを変更し、 **OK** を選択します。

    > [!NOTE]
    > 500 人を超える従業員に From 1095-C を印刷する場合は、複数の PDF ファイルが作成されます。 **ドキュメント管理パラメーター** ページの **最大ファイル サイズ (MB)** フィールドの値を **150** にすることをお勧めします。 (このページをすぐに開くには、ナビゲーション バーの検索フィールドを使用します。)
    >
    > ![最大ファイル サイズを変更する。](./media/hr-benefits-management-aca-maximum-file-size.png)

3. レポートの状態を確認して表示するには、ナビゲーション バーの検索フィールドを使用して **電子レポート ジョブ** のページを開きます。

    ![電子レポート ジョブのページで検索する。](./media/hr-benefits-management-aca-search-electronic-reporting-jobs.png)

4. 表示するレポートを選択し、**ファイルを表示する** を選択します。

    ![ファイルを表示する。](./media/hr-benefits-management-aca-show-files.png)

5. **開く** を選択します。

    ![ファイルを開く。](./media/hr-benefits-management-aca-open-file.png)

6. ブラウザのウィンドウ下部に表示される通知バーから、zip ファイルを開き、レポートを選択します。 PDF ファイルを表示または印刷できます。

    ![1095-C フォームのサンプル。](./media/hr-benefits-management-aca-1095-c-form.png)

## <a name="view-aca-coverage-information"></a>ACA 補償情報の表示

**労働者のアフォーダブル ケア** ページには、次の情報が表示されます:

- 各補償グループに割り当てられている従業員
- レポートに含める必要ない従業員
- 未割り当ての従業員

この情報を表示するには、以下の手順に従います。

1. **福利厚生管理** ワークスペースで、**労働者のアフォーダブル ケア法** を選択します。
2. **グループ゜名** フィールドで、グループを選択します。

    ![ACA 補償の表示。](./media/hr-benefits-management-aca-view-coverage.png)

アフォーダブルケア法グループに由来する既定値のいずれかが上書きされた場合、変更された値の横にアスタリスクが表示されます。 12 か月間すべての値が同じ、または上書きされていない場合、**12 か月すべて** 列に値が表示されます。

ACA のレポート可能ではないとマークされ、Form 1095-C が必要ない従業員を表示できます。 **フィルタ方法** フィールドで、 **ACA レポート不可** を選択します 。

グループに割り当てられていない従業員、または期限切れのグループに割り当てられている従業員を表示できます。 **フィルタ方法** フィールドで 、**見割り当て、または期限切れのグループ** を選択します。

### <a name="export-to-excel"></a>Excel にエクスポート

リストを Microsoft Excel にエクスポートするには、次の手順に従います。

1. **Microsoft Office で開く** ボタンを選択します。
2. **内部利用 HCM 人事管理一時テーブル** を選択します。
3. **ダウンロード** を選択します。

### <a name="view-aca-reportable-dependents"></a>ACA レポート可能な従属データの表示

自己保険を提供している理由で、被保険者を報告する必要がある場合は、**ACA のレポート対象** としてマークされている福利厚生プランで被保険者となっている扶養家族を表示できます。 アクション ペインで、**扶養家族の補償を表示する** を選択します。

![扶養家族の補償を表示する。](./media/hr-benefits-management-aca-view-dependent-coverage.png)

従業員の扶養家族の補償情報が表示されます。

![被扶養者の補償。](./media/hr-benefits-management-aca-dependents.png)

> [!NOTE]
> このページには、**ACA レポ-ト可能** としてマークされている福利厚生プランだけが表示されます。
