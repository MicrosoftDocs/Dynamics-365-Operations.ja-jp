# <a name="marital-status"></a>配偶者の有無

このトピックでは、Dynamics 365 Human Resources における婚姻状態のオプション セットについて説明します。

物理名 : mshr_dirpersonmaritalstatus

このリストは、個人レコードに関する高人の状態のオプション セットを提供します。

| 先頭値 | ラベル | 説明 |
| --- | --- | --- |
| 200000000 | None | 値が選択されていません。
| 200000001 | 未婚 | 未婚であることを表わします。 |
| 200000002 | 既婚 | 既婚であることを表わします。 |
| 200000003 | 離婚 | 離婚であることを表わします。 |
| 200000004 | 死別 | 寡婦/寡夫であることを表わします。 |
