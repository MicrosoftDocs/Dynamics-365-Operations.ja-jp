# <a name="payroll---no-yes-option-set"></a>給与 - いいえ、はいオプション セット


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

この記事では、Dynamics 365 Human Resources における [いいえ] と [はい] のオプションについて説明します。

## <a name="option-set-no-yes"></a>オプション セット : いいえ、はい

\[物理名\] : mshr_noyes

このオプションは、Human Resources 仮想エンティティの一般的なブール値プロパティに対するオプション セットを提供します。

| 先頭値 | ラベル | 説明 |
| --- | --- | --- |
| 200000000 | なし | いいえ。 |
| 200000001 | あり | はい。 |
