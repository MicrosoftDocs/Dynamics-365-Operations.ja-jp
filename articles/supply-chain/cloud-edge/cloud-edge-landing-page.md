# <a name="scale-unit-capability-for-supply-chain-management-is-currently-paused-for-new-customers"></a>Supply Chain Management 向けのスケール ユニット機能は新規顧客に対して現在一時停止されています

[!include [banner](../includes/banner.md)]

> [!IMPORTANT]
> アーキテクチャの機能拡張が行っている間、Microsoft Dynamics 365 Supply Chain Managementのスケール ユニット機能が新しい顧客に対して一時停止されている状態について説明します。 現在、このソリューションの新規顧客の開拓は行っていませんが、サプライ チェーン事業に対して、構成可能で、柔軟で簡単に使いやすいアーキテクチャに積極的に取り組み、お客様に世界クラスのスケール ユニット ソリューションを提供する機能をさらに強化しています。
>
> 倉庫管理ような特定のモジュールを超えてさらなる構成可能性を許可するため、Dynamics 365 Supply Chain Management の強化に取り組んでいます。 今後も、顧客のニーズに応える新しいクラウドおよびエッジ スケール ユニットの提供を計画および開発します。 詳細については、[d365scmcneext@microsoft.com](mailto:d365scmcneext@microsoft.com) にお問合せください。
>
> Microsoft Dynamics 365 Commerce 向け Commerce Scale Unit (CSU) Core プラットフォームは、Dynamics 365 Supply Chain Management のスケール ユニット機能に関連していません。 CSU Core は、今回の発表の影響を受けず、引き続き Dynamics 365 Commerce アプリケーションで完全にサポートされます。 その機能の詳細については、[Commerce Scale Unit (CSU) Core の概要](../../commerce/dev-itpro/CSU-core.md) を参照してください。
