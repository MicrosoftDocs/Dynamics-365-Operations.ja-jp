# <a name="create-and-edit-sales-quotations"></a>販売見積の作成および編集

[!include [banner](../../includes/banner.md)]

この手順は、販売見積を作成および更新する方法を示します。 この手順は、独自のデータで、またはデモ データの会社 USMF のデータで実行できます。


## <a name="create-a-sales-quotation"></a>販売見積の作成
1. **ナビゲーション ウィンドウ > モジュール > 販売とマーケティング > 販売見積 > すべての見積** の順に移動します。
2. **新規** をクリックします。
3. **勘定タイプ** フィールドで、「見込顧客」を選択します。
4. **見込顧客** フィールドで、値を入力または選択します。
5. **一般** セクションを展開します。 販売とマーケティング領域から見積書を作成する選択をしたため、タイプは自動的に「販売見積」に設定されます。 プロジェクトの見積書を作成するには、**プロジェクト管理および会計** モジュールからアクセスする必要があります。
6. **OK** をクリックします。 見積明細行のフィールドおよびアクションは、販売注文明細行のフィールドおよびアクションに非常に似ています。   販売注文と同様に、見積は特定の品目に対して作成できます。品目番号がわかっていないか、見積の作成時に存在しない場合、見積は販売カテゴリに対して作成できます。     
7. **品目** フィールドで、値を入力または選択します。
8. **サイト** フィールドに値を入力します。
9. **数量** フィールドに、数値を入力します。 明細行で選択した品目の有効な売買契約が存在する場合、適用できる価格と割引は見積明細行に自動的にコピーされます。 [単価] フィールドに値が含まれていることを確認します。割引の値も入力できます。 
10. **保存** をクリックします。
11. **アクション ウィンドウ** で、**販売見積** をクリックします。
12. **合計** をクリックします。
13. **OK** をクリックします。
14. 販売見積明細行を選択します。
15. **アクション ウィンドウ** で、**見積書** をクリックします。
16. **価格シミュレーション** をクリックします。
    - **価格シミュレーションの実行** ページでは、目的の単価、割引金額、割引率、合計金額、利益または利益率に基づいての見積の予測売上または収益性の調整を実験できます。 目標数値に問題がなければ、見積明細行に提案を適用し、価格関連フィールドがそれに応じて更新されます。  
    - 思い通りの価格シミュレーションを作成できます。 **新規** をクリックすると、現在の見積明細行の価格条件がページにコピーされます。 いずれかの価格関連フィールドの値をターゲット値に変更できます。 1 つのフィールドの変更により、他のすべてのフィールドの再計算が発生します。 販売の利益幅と利益率をシステムが計算するためには、製品の単位コストが明らかになっている必要があります。 [シミュレーション価格] タブを使用して、見積合計上にある元の価格、提案された変更およびその効果の詳細表示をします。 一般に、新しい金額を設定するシミュレーションが見積明細行に適用されると、システムは [単価] フィールドに新しい値を再計算して入力します。 シミュレーションが新しい利益幅と新しい利益率に基づく場合は、[正味金額] フィールドのみが更新され、[単価] は空白になります。 どちらの場合も、シミュレーション前の見積明細行に含まれていた割引は削除されます。
17. **アクション ウィンドウ** で、**見積書** をクリックします。
18. **見積の送信** をクリックします。
19. **見積書の印刷** フィールドで、「はい」を選択します。
20. **OK** をクリックします。 レポートが生成するのに、数分かかることがあります。 完了するまでページを閉じないでください。

## <a name="update-a-sales-quotation"></a>販売見積の更新
1. **ナビゲーション ウィンドウ > モジュール > 販売とマーケティング > 販売見積 > すべての見積** の順に移動します。
2. **アクション ウィンドウ** で、**フォローアップ** をクリックします。
3. **顧客に変更** をクリックします。
4. **顧客口座** フィールドに値を入力します。
5. **小切手** をクリックします。 入力した勘定番号が使用可能であるというメッセージの表示を確認します。  
6. **OK** をクリックします。 システムは、見積の見込顧客用の新しい顧客 ID を作成しています。  
7. ページを閉じます。
8. **アクション ウィンドウ** で、**フォローアップ** をクリックします。
9. **確定** をクリックします。
10. **理由** フィールドで、値を入力または選択します。
11. **OK** をクリックします。
12. **アクション ウィンドウ** で、**一般** をクリックします。
13. **販売注文** をクリックします。
14. ページを閉じます。
