# <a name="change-or-delete-an-original-intercompany-sales-order"></a>オリジナルの会社間販売注文書の変更または削除

[!include [banner](../../includes/banner.md)]

販売注文を変更すると、その変更内容は関連する会社間発注書および会社間販売注文書と自動的に同期されます。

> [!NOTE]
> 外部の仕入先に対する発注書は、変更しても影響を受けません。また、外部の仕入先に対する発注書に接続されているオリジナルの販売注文明細行も影響を受けません。

次の表に、実行できる変更と期待される結果を示します。

| 操作 | 結果 |
|---|---|
| オリジナルの&nbsp;販売注文&nbsp;の&nbsp;&nbsp;削除。 | 関連付けられている会社間発注書と会社間販売注文書も削除されます。 注文の状態がプロセス内で **ピッキング リスト** 以降となっている場合は、販売注文明細行を削除できません。 |
| オリジナルの販売注文明細行の削除。 | 関連付けられている会社間発注書明細行と会社間販売注文書明細行も削除されます。 注文の状態がプロセス内で **ピッキング リスト** 以降となっている場合は、販売注文明細行を削除できません。 |
| オリジナルの販売注文への新しい販売注文明細行を追加します。 | 会社間発注書と会社間販売注文書の両方に、新しい販売注文明細行が作成されます。 |
| オリジナルの販売注文明細行の数量を変更します。 | 数量は、会社間発注書明細行および会社間販売注文書明細行の両方と同期されます。 すべての注文の正味金額が再計算されます。 |
| オリジナルの販売注文のダイレクトデリバリーを無効にします。 | 会社間販売注文書の行の最小ステータスが **ピッキング リスト**、 **出荷注文**、または **ピッキング リスト仕訳帳** の状態に達している場合は、オリジナルの販売注文の **ダイレクト デリバリー** フィールドを変更できません。 会社間発注書の配送先住所が、標準配送先住所 (発注書の標準配送先住所) に変更されます。 また、会社間発注書の明細行が、明細行の標準配送先住所に変更されます。 そしてその両方が、会社間販売注文書と明細行に同期されます。 オリジナルの販売注文のすべての明細行の配送タイプが **なし** に変更され、そのフィールドが会社間発注書明細行と同期されます。 外部仕入先の発注書や、外部仕入先の購買注文明細行に関連付けられているオリジナルの販売注文明細行は、いずれも影響を受けません。 会社間発注書と明細行の配送日と、会社間販売注文書および明細行の指定入荷/配送日が更新されます。 |
| オリジナルの販売注文のダイレクト デリバリーを有効化します。 | 会社間販売注文書の明細行が、**ピッキング リスト**、**出力オーダー**、**ピッキングリスト仕訳帳** のいずれかの最小の状態に達している場合、オリジナルの販売注文の **ダイレクト デリバリー**  フィールドを変更することはできません。 会社間発注書の配送先住所が、オリジナルの販売注文の配送先住所に変更され、会社間販売注文書と同期されます。 オリジナルの販売注文のすべての明細行の配送タイプが **ダイレクト デリバリー** に変更され、そのフィールドが会社間発注書明細行と同期されます。 オリジナルの販売注文の各明細行の配送先住所が、会社間発注書明細行および会社間販売注文書明細行の両方と同期されます。 会社間発注書と明細行の配送日と、会社間販売注文書および明細行の指定入荷/配送日が更新されます。 |
| オリジナルの販売注文のヘッダーと明細行の両方にメモを追加します。 | 会社間発注書と会社間販売注文書にメモがコピーされます。 |
| メモを変更、または削除します。 | メモを変更または削除すると、それに応じて会社間発注書と会社間販売注文書の対応するメモが変更または削除されます。 |
