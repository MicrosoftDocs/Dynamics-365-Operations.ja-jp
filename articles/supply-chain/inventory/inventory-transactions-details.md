# <a name="inventory-transaction-details"></a>在庫トランザクションの詳細

**トランザクションの詳細** ページを使用して、選択した在庫トランザクションの詳細を表示します。

## <a name="open-the-transaction-details-page"></a>トランザクションの詳細ページを開く

**トランザクションの詳細** ページを開くには、次の手順に従います。

1. **在庫管理 \> 照会およびレポート \> トランザクション** の順に移動します。
1. 検査するトランザクションを選択します。
1. アクション ペインで、**トランザクションの詳細** を選択します。

## <a name="fasttabs-overview"></a>FastTabs の概要

**トランザクションの詳細** ページは、複数の FastTabs で分割されています 次の表に、各 FastTab の目的を示します。

| クイックタブ | Description |
|---|---|
| 一般 | この FastTab には、選択した在庫トランザクションに関する基本情報が表示されます。 **在庫トランザクション** ページに表示されるフィールドに加えて、この記事で後述する追加フィールドが含まれます。 |
| 更新 | この FastTab には、選択した在庫トランザクションの現物、財務、および決済の各側面に関連する情報が表示されます。 トランザクションの現在のステータスによっては、一部のフィールドが空白になる場合があります。 ただし、これらのフィールドは、トランザクションが転記される際に自動的に設定されます。 **在庫トランザクション** ページに表示されるフィールドに加えて、この FastTab には、この記事で後述する追加フィールドが含まれます。 |
| 元帳転記 | この FastTab には、選択した在庫トランザクションに関連する現物更新、財務更新、現物売上、現物費用、財務収益、および財務費用に使用される転記タイプと勘定科目が表示されます。 |
| 参照 | この FastTab には、選択した在庫トランザクションを作成したソース トランザクションに関する追加情報が表示されます。 たとえば、この情報には発注書や販売注文からの詳細が含まれる場合があります。 |
| 関連情報 | この FastTab には、選択した在庫トランザクションに関する追加情報が表示されます。 調達カテゴリやプロジェクトなど、一部の機能を使用していない場合、これらのフィールドが設定されない場合があります。 |
| 財務分析コード – 現物 | この FastTab には、現物更新が転記された際に使用された財務分析コード値が表示されます。 現物更新が転記されていない場合、フィールドは空白になります。 |
| 財務分析コード – 財務 | この FastTab には、財務更新が転記された際に使用された財務分析コード値が表示されます。 財務更新が転記されていない場合、フィールドは空白になります。 |
| 在庫分析コード | この FastTab には、選択した在庫トランザクションに割り当てられている在庫分析コード値が表示されます。 これらの値には、サイト、倉庫、サイズ、色、構成、場所、バッチ番号、シリアル番号、在庫状態、ライセンス プレート、所有者が含まれます。 |

## <a name="fields-on-the-general-fasttab"></a>全般 FastTab のフィールド

次の表では、**在庫トランザクション** ページに表示されない **全般** FastTab のフィールドについて説明します。

| フィールド | Description |
|---|---|
| 関係者 | 選択した在庫トランザクションの関連当事者の名前。 たとえば、発注書トランザクションでは発注書に仕入先の名前が表示され、販売注文トランザクションには顧客の名前が表示されます。 このフィールドは、すべてのタイプの在庫トランザクションでは使用できるわけではありません。 |
| 在庫参照 | 選択した在庫トランザクションに別の在庫トランザクションが関連付けられている場合は、その他のトランザクションのタイプ。 たとえば、発注書が販売注文に対してマークされている場合、発注書のトランザクションの **在庫参照** フィールドは *販売注文* に設定されます。 直納注文および会社間注文に対して自動的にマーキングが行われます。 *標準原価* および *移動平均* 以外の原価計算方法でマーキングを使用すると、マークされた正確なトランザクションの決済および調整がシステムによって作成されます。 これにより、先入れ先出し (FIFO); 後入れ先出し (LIFO); または加重平均のロジックを上書きする "実績" 原価計算を実現できます。 |
| 在庫番号 | 選択した在庫トランザクションに関連する特定のトランザクション。 たとえば、発注書が販売注文に対してマークされている場合、発注書のトランザクションの **在庫番号** フィールドは販売注文番号に設定されます。 |
| プロジェクト ID | 選択した在庫トランザクションがプロジェクトに関連付されている場合、このフィールドはプロジェクト番号に設定されます。 |
| ロット ID | 選択した在庫トランザクションにシステムが割り当てた固有のロット ID。 |
| 参照ロット | 選択した在庫トランザクションに別の在庫トランザクションが関連付けられている場合は、その他のトランザクションのロット ID。 たとえば、発注書が販売注文に対してマークされている場合、発注書のトランザクションの **参照ロット** フィールドは、販売注文明細行の在庫トランザクションの **ロット ID** 値になります。 |
| 分析コード番号 | 選択した在庫トランザクションに割り当てられているすべての在庫分析コード値の組み合わせを表す固有 ID。 |
| 金額未計上 | 選択した在庫トランザクションが在庫決算プロセスによって決済されたかどうかを示す値。 このフィールドが *はい* に設定されている場合、トランザクションは決済されていません。 |
| 予定日 | 受入トランザクションの場合、在庫受入が発生する予定の日付。 たとえば、このフィールドには、在庫ブロック注文の受入予定日や発注書明細行の納期が表示されます。 |
| 在庫日 | このフィールドは、現物入庫が転記される前に入庫注文で登録トランザクションが行われたときに設定されます。 |
| 非財務転送 | 選択した在庫トランザクションが非財務転送であるかどうかを示す値。 非財務転送は、在庫分析コードの変更が **品目モデル グループ** ページで財務追跡されていない場合に発生します。 |
| ロット ID の移動 | 選択した在庫トランザクションが非財務転送である場合、このフィールドは、選択したトランザクションが決済される他の在庫トランザクションの **ロット ID** 値に設定されます。 |

## <a name="fields-on-the-updates-fasttab"></a>更新 FastTab のフィールド

次の表では、**在庫トランザクション** ページに表示されない **更新** FastTab のフィールドについて説明します。

| フィールド | Description |
|---|---|
| 現物伝票 | 選択した在庫トランザクションの現物転記が生成された一般会計の伝票番号。 |
| ルート | 選択した在庫トランザクションに関連するルート。 このフィールドは、部品表 (BOM) またはフォーミュラ明細行が特定の品目に関連付けらた生産ピッキング リスト トランザクションにのみ設定されます。 |
| 梱包明細 | 発注書製品の受入トランザクションに対して入力された梱包明細番号。 |
| 現物原価金額 | 現物更新のために一般会計に転記された金額。 標準原価製品の場合、この金額は標準原価を表します。 他の原価計算方法の場合、転記時の製品の加重平均を表します。 |
| 現物売上 | 現物更新のために一般会計に転記された繰延収益の金額。 |
| 貨物 ID | 現在のトランザクションが輸送管理の貨物に関連している場合、このフィールドには品目を含む貨物の固有の番号が表示されます。 |
| 現物転記済 | 選択した在庫トランザクションが現物転記済みかどうかを示す値。 現在のトランザクションが一般会計に転記されている場合は、現物伝票もあります。 |
| 財務転記済 | 選択した在庫トランザクションが財務転記済みかどうかを示す値。 現在のトランザクションが一般会計に転記されている場合は、会計伝票もあります。 |
| 転記済の物理的費用 | 選択した在庫トランザクションに関連する現物費用が転記されているかどうかを示す値。 |
| 転記済の金融費用 | 選択した在庫トランザクションに関連する財務費用が転記されているかどうかを示す値。 |
| 会計伝票 | 財務転記が生成された一般会計の伝票番号。 |
| 請求書 | 例えば、発注書や販売注文に対して生成された請求書番号。 |
| 調整 | 在庫決算および調整プロセスから、選択した在庫トランザクションで調整された金額。 |
| 損益勘定、転記金額 | 損益計算書に転記された、選択した在庫トランザクションの金額。 |
| 未転記の請求書 | **保留中の仕入先請求書** ページに表示される、関連する発注書の請求書番号。 |
| 財務上決算済 | トランザクションが完全に決済された日付。 |
| 対象となる CW 数量 | 決済の対象となる CW 数量。 |
| 決済数量 | 選択した在庫トランザクションに対して決済された数量。 |
| 決済金額 | 選択した在庫トランザクションに対して決済された値。 |
