<span data-ttu-id="9e976-104">この手順では、プロジェクト リソースを設定する方法を示します。</span><span class="sxs-lookup"><span data-stu-id="9e976-104">This procedure shows how to set up a project resource.</span></span> <span data-ttu-id="9e976-105">このタスクでは、USMF のデータ セットを使用します。</span><span class="sxs-lookup"><span data-stu-id="9e976-105">This task uses the USMF data set.</span></span>

1. <span data-ttu-id="9e976-106">[プロジェクト管理および会計] > [プロジェクト リソース] > [リソース リスト] の順にクリックします。</span><span class="sxs-lookup"><span data-stu-id="9e976-106">Go to Project management and accounting > Project resources > Resources list.</span></span>

7. <span data-ttu-id="9e976-112">[原価価格] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="9e976-112">Click Cost prices.</span></span>
8. <span data-ttu-id="9e976-113">[原価価格 (時間)] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="9e976-113">Click Cost price - hour.</span></span>

12. <span data-ttu-id="9e976-117">[保存] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="9e976-117">Click Save.</span></span>
13. <span data-ttu-id="9e976-118">ページを閉じます。</span><span class="sxs-lookup"><span data-stu-id="9e976-118">Close the page.</span></span>
14. <span data-ttu-id="9e976-119">[販売価格] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="9e976-119">Click Sales prices.</span></span>
15. <span data-ttu-id="9e976-120">[販売価格 (時間)] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="9e976-120">Click Sales price - hour.</span></span>
16. <span data-ttu-id="9e976-121">[新規] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="9e976-121">Click New.</span></span>
17. <span data-ttu-id="9e976-122">[有効日] フィールドに日付を入力します。</span><span class="sxs-lookup"><span data-stu-id="9e976-122">In the Effective date field, enter a date.</span></span>

20. <span data-ttu-id="9e976-125">[保存] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="9e976-125">Click Save.</span></span>
21. <span data-ttu-id="9e976-126">ページを閉じます。</span><span class="sxs-lookup"><span data-stu-id="9e976-126">Close the page.</span></span>
