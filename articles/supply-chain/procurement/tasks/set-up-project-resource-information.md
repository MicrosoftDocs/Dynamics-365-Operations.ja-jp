1. <span data-ttu-id="85a0a-106">[プロジェクト管理および会計] > [プロジェクト リソース] > [リソース リスト] の順にクリックします。</span><span class="sxs-lookup"><span data-stu-id="85a0a-106">Go to Project management and accounting > Project resources > Resources list.</span></span>

13. <span data-ttu-id="85a0a-118">ページを閉じます。</span><span class="sxs-lookup"><span data-stu-id="85a0a-118">Close the page.</span></span>
14. <span data-ttu-id="85a0a-119">[販売価格] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="85a0a-119">Click Sales prices.</span></span>
15. <span data-ttu-id="85a0a-120">[販売価格 (時間)] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="85a0a-120">Click Sales price - hour.</span></span>
16. <span data-ttu-id="85a0a-121">[新規] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="85a0a-121">Click New.</span></span>
17. <span data-ttu-id="85a0a-122">[有効日] フィールドに日付を入力します。</span><span class="sxs-lookup"><span data-stu-id="85a0a-122">In the Effective date field, enter a date.</span></span>

21. <span data-ttu-id="85a0a-126">ページを閉じます。</span><span class="sxs-lookup"><span data-stu-id="85a0a-126">Close the page.</span></span>
