# <a name="set-up-project-resource-information"></a>プロジェクト リソース情報の設定

[!include [banner](../../includes/banner.md)]

この手順では、プロジェクト リソースを設定する方法を示します。 このタスクでは、USMF のデータ セットを使用します。

1. [プロジェクト管理および会計] > [プロジェクト リソース] > [リソース リスト] の順にクリックします。
2. 新しいリソースをクリックします。
3. [作業者] フィールドで値を入力または選択します。
4. [カレンダー] フィールドで値を入力または選択します。
5. [期間コード] フィールドで、値を入力または選択します。
6. [作成] をクリックします。
7. [原価価格] をクリックします。
8. [原価価格 (時間)] をクリックします。
9. [新規] をクリックします。
10. [有効日] フィールドに日付を入力します。
11. [原価価格] フィールドに番号を入力します。
12. [保存] をクリックします。
13. ページを閉じます。
14. [販売価格] をクリックします。
15. [販売価格 (時間)] をクリックします。
16. [新規] をクリックします。
17. [有効日] フィールドに日付を入力します。
18. [価格決定] フィールドに、数を入力します。
19. [有効] フィールドで、オプションを選択します。
20. [保存] をクリックします。
21. ページを閉じます。
