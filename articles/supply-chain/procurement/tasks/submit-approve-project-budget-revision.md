# <a name="submit-and-approve-project-budget-revision"></a><span data-ttu-id="7e457-103">プロジェクト予算のリビジョンの送信および承認</span><span class="sxs-lookup"><span data-stu-id="7e457-103">Submit and approve project budget revision</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="7e457-104">この手順には、プロジェクト予算のリビジョンに関する情報が用意されています。</span><span class="sxs-lookup"><span data-stu-id="7e457-104">This procedure provides information about revising a project budget.</span></span> <span data-ttu-id="7e457-105">プロジェクト予算は必要に応じて何度でも改訂することができますが、予算が複数回改訂される場合、個々のリビジョンの量を表示することはできません。</span><span class="sxs-lookup"><span data-stu-id="7e457-105">You can revise a project budget as many times as necessary, however you can't see the amount of an individual revision if the budget has been revised more than one time.</span></span> 

<span data-ttu-id="7e457-106">このタスクでは、SSI データ セットを使用します。</span><span class="sxs-lookup"><span data-stu-id="7e457-106">This task uses the SSI data set.</span></span>

1. <span data-ttu-id="7e457-107">[プロジェクト管理および会計] > [プロジェクト] > [すべてのプロジェクト] の順に選択します。</span><span class="sxs-lookup"><span data-stu-id="7e457-107">Go to Project management and accounting > Projects > All projects.</span></span>
2. <span data-ttu-id="7e457-108">一覧で、目的のレコードを見つけ、選択します。</span><span class="sxs-lookup"><span data-stu-id="7e457-108">In the list, find and select the desired record.</span></span>
3. <span data-ttu-id="7e457-109">一覧で、選択された行のリンクをクリックします。</span><span class="sxs-lookup"><span data-stu-id="7e457-109">In the list, click the link in the selected row.</span></span>
4. <span data-ttu-id="7e457-110">アクション ウィンドウで、[計画] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="7e457-110">On the Action Pane, click Plan.</span></span>
5. <span data-ttu-id="7e457-111">[プロジェクト予算] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="7e457-111">Click Project budget.</span></span>
6. <span data-ttu-id="7e457-112">新しいリビジョンをクリックします。</span><span class="sxs-lookup"><span data-stu-id="7e457-112">Click New revision.</span></span>
7. <span data-ttu-id="7e457-113">[原価] セクションを展開します。</span><span class="sxs-lookup"><span data-stu-id="7e457-113">Expand the Costs section.</span></span>

9. <span data-ttu-id="7e457-115">[保存] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="7e457-115">Click Save.</span></span>
10. <span data-ttu-id="7e457-116">[ワークフロー] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="7e457-116">Click Workflow.</span></span>
11. <span data-ttu-id="7e457-117">[送信] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="7e457-117">Click Submit.</span></span>
12. <span data-ttu-id="7e457-118">[コメント] フィールドで、値を入力します。</span><span class="sxs-lookup"><span data-stu-id="7e457-118">In the Comment field, type a value.</span></span>
13. <span data-ttu-id="7e457-119">[送信] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="7e457-119">Click Submit.</span></span>
