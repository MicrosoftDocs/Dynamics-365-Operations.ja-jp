# <a name="submit-and-approve-project-budget-revisions"></a><span data-ttu-id="e1aab-103">プロジェクト予算のリビジョンの送信および承認</span><span class="sxs-lookup"><span data-stu-id="e1aab-103">Submit and approve project budget revisions</span></span>

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="e1aab-104">この手順には、プロジェクト予算のリビジョンに関する情報が用意されています。</span><span class="sxs-lookup"><span data-stu-id="e1aab-104">This procedure provides information about revising a project budget.</span></span> <span data-ttu-id="e1aab-105">プロジェクト予算は必要に応じて何度でも改訂することができますが、予算が複数回改訂される場合、個々のリビジョンの量を表示することはできません。</span><span class="sxs-lookup"><span data-stu-id="e1aab-105">You can revise a project budget as many times as necessary, however you can't see the amount of an individual revision if the budget has been revised more than one time.</span></span> 

<span data-ttu-id="e1aab-106">このタスクでは、SSI データ セットを使用します。</span><span class="sxs-lookup"><span data-stu-id="e1aab-106">This task uses the SSI data set.</span></span>

1. <span data-ttu-id="e1aab-107">[プロジェクト管理および会計] > [プロジェクト] > [すべてのプロジェクト] の順に選択します。</span><span class="sxs-lookup"><span data-stu-id="e1aab-107">Go to Project management and accounting > Projects > All projects.</span></span>
2. <span data-ttu-id="e1aab-108">一覧で、目的のレコードを見つけ、選択します。</span><span class="sxs-lookup"><span data-stu-id="e1aab-108">In the list, find and select the desired record.</span></span>
3. <span data-ttu-id="e1aab-109">一覧で、選択された行のリンクをクリックします。</span><span class="sxs-lookup"><span data-stu-id="e1aab-109">In the list, click the link in the selected row.</span></span>
4. <span data-ttu-id="e1aab-110">アクション ウィンドウで、[計画] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="e1aab-110">On the Action Pane, click Plan.</span></span>
5. <span data-ttu-id="e1aab-111">[プロジェクト予算] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="e1aab-111">Click Project budget.</span></span>
6. <span data-ttu-id="e1aab-112">新しいリビジョンをクリックします。</span><span class="sxs-lookup"><span data-stu-id="e1aab-112">Click New revision.</span></span>
7. <span data-ttu-id="e1aab-113">[原価] セクションを展開します。</span><span class="sxs-lookup"><span data-stu-id="e1aab-113">Expand the Costs section.</span></span>
8. <span data-ttu-id="e1aab-114">[リビジョン金額] フィールドに、数値を入力します。</span><span class="sxs-lookup"><span data-stu-id="e1aab-114">In the Revision amount field, enter a number.</span></span>
9. <span data-ttu-id="e1aab-115">[保存] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="e1aab-115">Click Save.</span></span>
10. <span data-ttu-id="e1aab-116">[ワークフロー] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="e1aab-116">Click Workflow.</span></span>
11. <span data-ttu-id="e1aab-117">[送信] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="e1aab-117">Click Submit.</span></span>
12. <span data-ttu-id="e1aab-118">[コメント] フィールドで、値を入力します。</span><span class="sxs-lookup"><span data-stu-id="e1aab-118">In the Comment field, type a value.</span></span>
13. <span data-ttu-id="e1aab-119">[送信] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="e1aab-119">Click Submit.</span></span>
