# <a name="partial-shipment-of-a-transport-load"></a><span data-ttu-id="1a021-103">輸送積荷の部分的出荷</span><span class="sxs-lookup"><span data-stu-id="1a021-103">Partial shipment of a transport load</span></span>

[!include[banner](../includes/banner.md)]

<span data-ttu-id="1a021-104">積荷の部分的出荷を設定することにより、すべての販売明細行が積荷に追加されるまで、能力を判別できない積荷を処理できます。</span><span class="sxs-lookup"><span data-stu-id="1a021-104">By setting up partial shipment of loads, you can handle loads where the capacity can't be determined until all the sales lines have been added to a load.</span></span> <span data-ttu-id="1a021-105">正確なパレット数がわかっている場合、プロセスを確定できます。</span><span class="sxs-lookup"><span data-stu-id="1a021-105">The process can then be finalized when the exact pallet count is known.</span></span> <span data-ttu-id="1a021-106">したがって、配送ステージ完了在庫から物理的に積載される時点まで、どのパレットがどの配送に割り当てられるかを決定する必要はありません。</span><span class="sxs-lookup"><span data-stu-id="1a021-106">Therefore, you don't have to decide which pallets will be assigned to which transport until the moment when a transport is being physically loaded out of the staged inventory.</span></span>

<span data-ttu-id="1a021-107">この機能は、ピッキング作業または積載作業が作成される前に、どのパレットがどの配送に割り当てられるかを判断する必要がある中で、より厳格な構造の実施への代替をオファーします。</span><span class="sxs-lookup"><span data-stu-id="1a021-107">This feature offers an alternative to the enforcement of a more rigid structure, where you must determine which pallets will be assigned to which transport before picking work or loading work can be created.</span></span> <span data-ttu-id="1a021-108">代わりに、ユーザーは部分的出荷確認の個々の積荷をコンフィギュレーションできます。</span><span class="sxs-lookup"><span data-stu-id="1a021-108">Instead, users can configure individual loads for a partial shipment confirmation.</span></span> <span data-ttu-id="1a021-109">それらの積荷の輸送積荷プロセスが発生します。</span><span class="sxs-lookup"><span data-stu-id="1a021-109">The transport loading processes for those loads can then occur.</span></span> <span data-ttu-id="1a021-110">したがって、個々の配送の能力を考慮せずに、輸送計画部門が積荷計画できます。</span><span class="sxs-lookup"><span data-stu-id="1a021-110">Therefore, the transportation planning department can plan loads without having to consider the capacity of individual transports.</span></span>

<span data-ttu-id="1a021-111">積載時に、作業者は、パレットが積載できる新しい輸送積荷を定義できます。</span><span class="sxs-lookup"><span data-stu-id="1a021-111">At the time of loading, workers can define a new transport load that a pallet can be loaded to.</span></span> <span data-ttu-id="1a021-112">または、既存の輸送積荷を識別することもできます。</span><span class="sxs-lookup"><span data-stu-id="1a021-112">Alternatively, they can identify an existing transport load.</span></span> <span data-ttu-id="1a021-113">モバイル デバイスを通して、このデータを記録できます。</span><span class="sxs-lookup"><span data-stu-id="1a021-113">This data can be recorded via a mobile device.</span></span> <span data-ttu-id="1a021-114">したがって、複数の倉庫作業者は、同じ積荷または同じトラックの異なる積荷から在庫を積載することができます。</span><span class="sxs-lookup"><span data-stu-id="1a021-114">Therefore, several warehouse workers can load inventory from the same loads or different loads onto the same truck.</span></span> <span data-ttu-id="1a021-115">積荷は、完全または部分的に出荷されます。</span><span class="sxs-lookup"><span data-stu-id="1a021-115">The loads can then be fully or partially shipped.</span></span>

> [!NOTE] 
> <span data-ttu-id="1a021-116">特定の輸送積荷から在庫を積載し、積荷を部分的出荷するためには、作業テンプレートで積載クラスを使用して作業を生成する必要があります。</span><span class="sxs-lookup"><span data-stu-id="1a021-116">In order to load inventory from a load to a specific transport load and partially ship the load, work must be generated by using a loading class in a work template.</span></span> <span data-ttu-id="1a021-117">通常の **ピッキング** タイプのピッキング作業では、トラックなどの輸送積荷を積載することができません。</span><span class="sxs-lookup"><span data-stu-id="1a021-117">Ordinary picking work of the **Picking** type can't be loaded to a transport load such as a truck.</span></span>

## <a name="set-up-transport-loads-for-partial-shipment"></a><span data-ttu-id="1a021-118">部分的出荷の輸送積荷を設定します。</span><span class="sxs-lookup"><span data-stu-id="1a021-118">Set up transport loads for partial shipment</span></span>

<span data-ttu-id="1a021-119">積荷の部分的出荷の設定は、次の 2 つの手順で構成されます。</span><span class="sxs-lookup"><span data-stu-id="1a021-119">The setup for partial shipment of loads consists of the following two procedures.</span></span>

### <a name="set-the-loading-strategy"></a><span data-ttu-id="1a021-120">積荷戦略の設定</span><span class="sxs-lookup"><span data-stu-id="1a021-120">Set the loading strategy</span></span>

<span data-ttu-id="1a021-121">積荷戦略を設定して、部分的積載を有効にする必要があります。</span><span class="sxs-lookup"><span data-stu-id="1a021-121">You must enable partial loading by setting the loading strategy.</span></span> <span data-ttu-id="1a021-122">積荷を作成した後、積荷戦略を設定できます。</span><span class="sxs-lookup"><span data-stu-id="1a021-122">You can set the loading strategy after you've created a load.</span></span>

1. <span data-ttu-id="1a021-123">**倉庫管理** \> **積荷** \> **すべての積荷** を選択します。</span><span class="sxs-lookup"><span data-stu-id="1a021-123">Select **Warehouse management** \> **Loads** \> **All loads**.</span></span>

3. <span data-ttu-id="1a021-125">**積荷戦略** フィールドで、**一部積荷出荷が許可されている** を選択します。</span><span class="sxs-lookup"><span data-stu-id="1a021-125">In the **Loading strategy** field, select **Partial load shipping allowed**.</span></span>

### <a name="create-a-menu-item-for-loading-of-transport-loads"></a><span data-ttu-id="1a021-126">輸送積荷の積載のメニュー項目を作成します。</span><span class="sxs-lookup"><span data-stu-id="1a021-126">Create a menu item for loading of transport loads</span></span>

<span data-ttu-id="1a021-127">積載されるべき輸送積荷を有効にする新しいメニュー項目を作成する必要があります。</span><span class="sxs-lookup"><span data-stu-id="1a021-127">You must create a new menu item that enables transport loads to be loaded.</span></span> <span data-ttu-id="1a021-128">輸送積荷により、1 つまたは複数の積載からの作業明細行をグループ化できます。</span><span class="sxs-lookup"><span data-stu-id="1a021-128">A transport load lets you group work lines from one load or multiple loads.</span></span> <span data-ttu-id="1a021-129">モバイル スキャナーを使用して、輸送積荷に追加されるすべてが出荷されます。</span><span class="sxs-lookup"><span data-stu-id="1a021-129">Everything that is added to the transport load can then be shipped by using a mobile scanner.</span></span>

1. <span data-ttu-id="1a021-130">**倉庫管理** \> **設定** \> **モバイル デバイス** \> **モバイル デバイスのメニュー品目** の順に選択します。</span><span class="sxs-lookup"><span data-stu-id="1a021-130">Select **Warehouse management** \> **Setup** \> **Mobile device** \> **Mobile device menu items**.</span></span>

3. <span data-ttu-id="1a021-132">**既存の作業を使用** オプションを **はい** に設定します。</span><span class="sxs-lookup"><span data-stu-id="1a021-132">Set the **Use existing work** option to **Yes**.</span></span>

2. <span data-ttu-id="1a021-138">**確定** グループの **出荷と入荷** タブのアクション ウィンドウで、**配送** を選択します。</span><span class="sxs-lookup"><span data-stu-id="1a021-138">On the Action Pane, on the **Ship and receive** tab, in the **Confirm** group, select **Transport**.</span></span>
