# <a name="global-mobile-device-parameters"></a>グローバル モバイル デバイス パラメーター

[!include [banner](../includes/banner.md)]

この記事では、システムのモバイル デバイスとのやり取りに影響を与えるグローバルな Warehouse Management パラメーターの設定方法について説明します。

倉庫作業者の設定方法の詳細については、[倉庫作業者の管理](manage-warehouse-workers.md)を参照してください。 モバイル デバイスにおけるライセンス プレートの取扱いについての詳細については、[Warehouse Management App を介したライセンス プレートの受領](warehousing-mobile-device-app-license-plate-receiving.md) を参照してください。 棚卸プロセスの詳細については、[循環棚卸](cycle-counting.md)および[循環棚卸のサンプル例](cycle-counting-scenarios.md)を参照してください。

## <a name="open-the-warehouse-management-parameters-page"></a>倉庫管理パラメーターのページを開く

**倉庫管理パラメーター** ページを開くには、**倉庫管理 \> 設定 \> 倉庫管理パラメーター** にアクセスします。 この記事の次のセクションで示されている方法で、モバイル デバイスの作業に関連するフィールドを設定します。

## <a name="mobile-device-fasttab-on-the-general-tab"></a>[全般] タブの [モバイル デバイス] クイック タブ

グローバル モバイル デバイス設定は、**倉庫管理パラメーター** ページの **全般** タブにある **モバイル デバイス** クイック タブにあります。 次のフィールドを使用できます:

| フィールド | 説明 |
|---|---|
| モバイル デバイス ノート タイプ | 販売注文のピッキング中に作業者に対して表示される情報のタイプを選択します。 |
| スキャン済数量の限度 | 作業者がそれぞれのセッションで、**作業作成プロセス** の値が *調整* となっているモバイル デバイスのメニュー項目を使用してスキャンできる最大の品目数を入力します。 このフィールドは、他のタイプのメニュー項目を使用して行われるスキャンには影響します。 |
| モバイル デバイス セッション ログを使用 | 作業者のサインイン履歴のログを保持するには、このオプションを *はい* に設定します。 ログを表示するには、**作業ユーザー セッション \> 照会とレポート \> モバイル デバイス ログ \> ユーザー セッション** に移動します。 |
| 保管されているエラーの数 | システムが保存するエラー レコードの最大数を入力してください。 エラー ログを表示するには、**作業ユーザー セッション \> 照会とレポート \> モバイル デバイス ログ \> ユーザー セッション** に移動します。 |
| 既定の倉庫振替仕訳帳 | 作業者がモバイル デバイスを使用して製品を倉庫間で移動する際に使用する仕訳帳を選択します。 |
| 外部レビュー時に購買注文明細行の登録を許可 | **承認状態** の値が *外部審査中* の発注書に対して、モバイル デバイスを使用して注文明細行を登録できるようにする場合は、このオプションを *はい* に設定します。 作業者が外部レビュー中の発注書に対する明細行の登録を防ぐには、このオプションを *いいえ* に設定します。 |
| RSAT サポートを有効にする | このフィールドは、Warehouse Management Mobile App のタスク検証コントロールを有効にするもので、アプリが実行する各ステップを記録し、検証します。 このプロセスによってシステムのパフォーマンスが大幅に低下する可能性があるため、検証コントロールを有効にできるのはテスト中のみです。 運用環境では、この機能を絶対に有効にしなでください。 |
