<span data-ttu-id="26111-162">場所のディレクティブによって品目のピッキング *場所* と配置 *場所* が決まります。</span><span class="sxs-lookup"><span data-stu-id="26111-162">Location directives determine *where* items should be picked and *where* they should be put.</span></span> <span data-ttu-id="26111-163">このシステムでは、各作業明細行に対して場所のディレクティブが評価され、作業明細行の詳細に基づいて場所が選択されます。</span><span class="sxs-lookup"><span data-stu-id="26111-163">The system evaluates a location directive against each work line and then selects a location, based on the work line details.</span></span> <span data-ttu-id="26111-164">システムによって、特定の作業行に一致するすべての場所のディレクティブが検索されます (たとえば、正しい倉庫のもので、クエリに一致するもの)。</span><span class="sxs-lookup"><span data-stu-id="26111-164">The system first finds all location directives that match a particular work line (for example, they are for the correct warehouse and match the query).</span></span> <span data-ttu-id="26111-165">次に、検出されたディレクティブを順番に評価します。</span><span class="sxs-lookup"><span data-stu-id="26111-165">It then sequentially evaluates the directives that it has found.</span></span>

> <span data-ttu-id="26111-166">ピック場所またはプット場所が事前に選択されている特殊なケースがあります。</span><span class="sxs-lookup"><span data-stu-id="26111-166">There are special cases where the pick location or put location is pre-selected.</span></span> <span data-ttu-id="26111-167">たとえば、_購買登録_ 時に、最初のピッキングは、登録が行われる場所から常に行われます。</span><span class="sxs-lookup"><span data-stu-id="26111-167">For example, during _purchase registration_, the first pick is always from the location where the registration occurs.</span></span> <span data-ttu-id="26111-168">別の例は、*テンプレートによる在庫移動* です。ここは、倉庫の作業者がピッキング場所を選択し、場所のディレクティブを使用してプット場所のみを検索すします。</span><span class="sxs-lookup"><span data-stu-id="26111-168">Another example is *inventory movement by template*, where the warehouse worker selects the pick location, and only the put locations are found through location directives.</span></span>
