# <a name="define-cycle-counting"></a>循環棚卸の定義 

[!include [banner](../../includes/banner.md)]

循環棚卸は、手持在庫品目を監査するために使用できる倉庫プロセスです。 このタスク記録では、次の作業を実施する方法を説明します。棚卸作業の既定の優先順位を設定する。モバイル デバイス メニュー項目でピッキングと棚卸の両方の工程を処理できるようにする。場所が空になったときに棚卸のしきい値をトリガーできるようにする。および特定の倉庫にある特定の品目の循環棚卸計画を有効にする。 通常、これらのタスクは、倉庫マネージャーが実行します。 USMF デモ データ会社または独自のデータを使用して、この手順を踏むことができます。


## <a name="set-the-priority-of-counting-work"></a>棚卸作業の優先順位を設定する
1. **ナビゲーション ウィンドウ** で、 **モジュール > 倉庫管理 > 設定 > 倉庫管理パラメーター** の順に移動します。
2. **循環棚卸** タブをクリックします。
3. **既定の循環棚卸作業優先順位** フィールドで、数値を入力します。 この手順では、倉庫内での他の種類の作業と比較して、循環棚卸作業の優先順位を変更します。 他の種類の作業よりも小さい番号を入力すると、循環棚卸作業の優先順位が上がります。  
4. **保存** をクリックします。
5. ページを閉じます。

## <a name="enable-the-mobile-device"></a>モバイル デバイスを有効にする
1. **ナビゲーション ウィンドウ** で、 **モジュール > 倉庫管理 > 設定 > モバイル デバイス > モバイル デバイスのメニュー品目** の順に移動します。
2. **新規** をクリックします。
3. **メニュー項目名** フィールドで、値を入力します。
4. **タイトル** フィールドに値を入力します。
5. **モード** フィールドで '作業' を選択します。
6. **既存の作業を使用** オプションをはいに設定します。 このオプション [はい] に設定する場合、モバイル デバイスのメニュー項目を使用すると、システムは既存の作業を検索します。  
7. **指示者** フィールドで 'システム主導' を選択します。 「システム主導」が選択される場合、定義された作業クラスにある作業を開くため、倉庫作業員が指示されます。 (次にこれらの作業クラスを作成します。)  
8. **作業クラス** クイック タブを展開します。 次に、このモバイル デバイスのメニュー項目で使用する 2 個の作業クラスを作成します。 このメニュー項目が使用されると、これらの作業クラスが照会されて、最も優先順位の高い作業がユーザーに表示されることになります。  
9. **新規** をクリックします。
10. **作業クラス ID** フィールドで、値を選択します。
11. **新規** をクリックします。
12. **作業クラス ID** フィールドで、値を選択します。
13. **アクション ウィンドウ** で、 **保存** をクリックします。
14. ページを閉じます。
15. **ナビゲーション ウィンドウ** で、 **モジュール > 倉庫管理 > 設定 > モバイル デバイス > モバイル デバイスのメニュー** の順に移動します。
16. 一覧で、目的のレコードを見つけ、選択します。
17. ツリーで、「作成したメニュー項目」を選択します。
18. **編集** をクリックします。
19. 矢印をクリックしてメニュー項目をメニューに追加します。
20. **保存** をクリックします。

## <a name="create-a-counting-threshold"></a>棚卸のしきい値を作成する
1. **ナビゲーション ウィンドウ** で、 **モジュール > 倉庫管理 > 設定 > 循環棚卸 > 循環棚卸のしきい値** の順に移動します。
2. **新規** をクリックします。
3. **循環棚卸のしきい値 ID** フィールドに、値を入力します。
4. **循環棚卸をすぐに処理する** オプションをはいに設定します。
5. **説明** フィールドで、値を入力します。
6. **保存** をクリックします。
7. **場所の選択** をクリックします。
8. 一覧で、選択された行をマークします。
9. **基準** フィールドで値を選択します。
10. **OK** をクリックします。
11. ページを閉じます。

## <a name="create-a-cycle-count-plan"></a>循環棚卸計画を作成する
1. **ナビゲーション ウィンドウ** で、 **モジュール > 倉庫管理 > 設定 > 循環棚卸 > 循環棚卸計画** の順に移動します。
2. **新規** をクリックします。
3. **循環棚卸計画 ID** フィールドに値を入力します。
4. **説明** フィールドで、値を入力します。
5. **循環棚卸の最大数** フィールドに、数値を入力します。
6. **保存** をクリックします。
7. **場所の選択** をクリックします。
8. 一覧で、選択された行をマークします。
9. **基準** フィールドで値を選択します。
10. **OK** をクリックします。
11. **次回の循環棚卸までの日数** フィールドに、数値を入力します。 たとえば、 **次回の循環棚卸までの日数** フィールドを 5 に設定すると、循環棚卸作業は 5 日ごとに作成されます。 ただし、循環棚卸作業が 3 日目に処理される場合は、次の循環棚卸作業は、最後の循環棚卸処理の 5 日後の 8 日目に作成されます。  
12. **保存** をクリックします。
13. **新規** をクリックします。
14. **順序番号** フィールドに番号を入力します。 並べ替えは、最小番号から最大番号です。 値は 0 (ゼロ) より大きい値に設定する必要があります。  
15. 一覧で、選択された行をマークします。
16. **説明** フィールドで、値を入力します。
17. **保存** をクリックします。
18. **製品の定義** クエリをクリックします。
19. 一覧で、選択された行をマークします。
20. **基準** フィールドで、値を入力または選択します。
21. **OK** をクリックします。
22. ページを閉じます。
