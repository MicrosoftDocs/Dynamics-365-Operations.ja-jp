# <a name="enable-license-plate-label-printing"></a>ライセンス プレート ラベル印刷の有効化

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

この手順は、販売ピッキングの作業プロセスで最後の品目が在庫からピッキングされたあとに、出荷コンテナ シリアル コード (SSCC) ラベルを自動で印刷するようにします。 デモ データの会社 USMF でこの手順を確認できます。 自分のデータを使用して実行している場合、ライセンス番号に使用する番号順序を設定する必要があります。 このタスクを開始する前にラベル プリンターを設定する必要があります。 [組織管理] > [設定] > [ネットワーク プリンター] の順に移動します。 アクション ペインで [オプション] をクリックし、[ドキュメント回覧エージェント インストーラーのダウンロード] ボタンをクリックします。 インストーラを実行し、[有効] に設定された使用可能なネットワーク プリンタがあることを確認してから、手順を進めます。


## <a name="set-up-the-gs1-company-prefix"></a>GS1 会社の接頭語の設定
1. [倉庫管理] > [設定] > [倉庫管理パラメーター] の順に移動します。
2. [GS1 会社の接頭語] フィールドに、GS1 会社の番号 7 つを入力します。
3. [保存] をクリックします。
4. ページを閉じます。

## <a name="setup-the-sscc-license-plate-number-sequence"></a>SSCC ライセンス プレート番号の番号順序の設定
1. [組織管理] > [番号順序] > [番号順序] の順に移動します。
2. [エリア] フィールドで、オプションを選択します。
3. [参照] フィールドで、オプションを選択します。
4. [会社] フィールドで、値を入力します。
5. 一覧で、選択された行をマークします。
6. 一覧で、選択された行のリンクをクリックします。
7. [区分] セクションを展開します。
8. [編集] をクリックします。
9. 区分テーブルで、先頭行を選択します。
10. [削除] をクリックします。
11. [削除] をクリックします。
12. [保存] をクリックします。
13. ページを閉じます。

## <a name="create-the-document-route-layout"></a>ドキュメント回覧レイアウトの作成
1. [倉庫管理] > [設定] > [ドキュメント回覧] > [ドキュメント回覧レイアウト] の順に移動します。
    * SSCC レイアウトを有効にします。  
2. [新規] をクリックします。
3. [レイアウト ID] フィールドで、値を入力します。
4. [説明] フィールドに値を入力します。
5. 一覧で、目的のレコードを見つけ、選択します。
6. [テキストの末尾に挿入] をクリックします。
7. [保存] をクリックします。
8. ページを閉じます。

## <a name="set-up-the-document-routing"></a>ドキュメント回覧の設定
1. [倉庫管理] > [設定] > [ドキュメント回覧] > [ドキュメント回覧] の順に移動します。
2. [ワーク オーダー タイプ] フィールドで、オプションを選択します。
3. [新規] をクリックします。
4. [倉庫] フィールドに値を入力します。
5. [名前] フィールドに値を入力します。
6. [新規] をクリックします。
7. [レイアウト ID] フィールドで、値を入力または選択します。
8. [名前] フィールドで、使用したいプリンター名を入力します。
9. [保存] をクリックします。
10. ページを閉じます。

## <a name="create-mobile-device-menu"></a>モバイル デバイスのメニューの作成
1. [倉庫管理] > [設定] > [モバイル デバイス] > [モバイル デバイスのメニュー品目] の順に移動します。
2. [新規] をクリックします。
3. [メニュー項目名] フィールドで、値を入力します。
4. [タイトル] フィールドに値を入力します。
5. [モード] フィールドで、オプションを選択します。
6. [既存の作業を使用] フィールドで [はい] を選択します。
7. [ライセンス プレートの生成] フィールドで、[はい] を選択します。
8. 作業クラスのセクションを展開します。
9. [新規] をクリックします。
10. [作業クラス ID] フィールドで、値を入力します。
11. [保存] をクリックします。
12. ページを閉じます。
13. [倉庫管理] > [設定] > [モバイル デバイス] > [モバイル デバイスのメニュー] の順に移動します。
14. 一覧で、目的のレコードを見つけ、選択します。
15. ツリーで、[ツリーで、以前に作成したメニュー項目を選択する] を選択します。
16. [編集] をクリックします。
17. 矢印をクリックしてメニュー項目をメニューに追加します。
18. [保存] をクリックします。
19. ページを閉じます。

## <a name="update-a-work-template"></a>作業テンプレートの更新
1. [倉庫管理] > [設定] > [作業] > [作業テンプレート] の順に移動します。
2. 一覧で、目的のレコードを見つけ、選択します。
3. [編集] をクリックします。
4. [新規] をクリックします。
5. 一覧で、選択された行をマークします。
6. [作業タイプ] フィールドで [印刷] を選択します。
7. [作業クラス ID] フィールドで、値を入力または選択します。
8. 一覧で、選択された行のリンクをクリックします。
9. [上へ移動] をクリックします。
10. [保存] をクリックします。
11. ページを閉じます。
