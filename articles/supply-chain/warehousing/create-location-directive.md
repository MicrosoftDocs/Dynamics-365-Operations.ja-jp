# <a name="work-with-location-directives"></a><span data-ttu-id="8d137-104">場所のディレクティブを使って作業する</span><span class="sxs-lookup"><span data-stu-id="8d137-104">Work with location directives</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="8d137-105">場所ディレクティブは、棚卸資産移動のピッキングとプットの場所を識別するのにに役立つルールです。</span><span class="sxs-lookup"><span data-stu-id="8d137-105">Location directives are rules that help identify pick and put locations for inventory movement.</span></span> <span data-ttu-id="8d137-106">たとえば、販売注文トランザクションでは、場所ディレクティブにより、品目がピッキングされる場所とピッキングされた品目が置かれる場所が決定されます。</span><span class="sxs-lookup"><span data-stu-id="8d137-106">For example, in a sales order transaction, a location directive determines where the items will be picked, and where the picked items will be put.</span></span> <span data-ttu-id="8d137-107">場所のディレクティブは、ヘッダーと関連付けられた明細行から構成されます。</span><span class="sxs-lookup"><span data-stu-id="8d137-107">Location directives consist of a header and associated lines.</span></span> <span data-ttu-id="8d137-108">これらは、特定の *作業指示タイプ* に対して作成されます。</span><span class="sxs-lookup"><span data-stu-id="8d137-108">They are created for specific *work order types*.</span></span>

> [!NOTE]
> <span data-ttu-id="8d137-109">このトピックは、**倉庫管理** モジュールの機能に適用されます。</span><span class="sxs-lookup"><span data-stu-id="8d137-109">This topic applies to features in the **Warehouse management** module.</span></span> <span data-ttu-id="8d137-110">[在庫管理](../inventory/inventory-home-page.md) モジュールの機能は対象外です。</span><span class="sxs-lookup"><span data-stu-id="8d137-110">It doesn't apply to features in the [Inventory management](../inventory/inventory-home-page.md) module.</span></span>

<span data-ttu-id="8d137-111">場所ディレクティブを使用して、次のタスクを実行できます:</span><span class="sxs-lookup"><span data-stu-id="8d137-111">You can use location directives to perform the following tasks:</span></span>

- <span data-ttu-id="8d137-112">受入品目のプット アウェイ。</span><span class="sxs-lookup"><span data-stu-id="8d137-112">Put away incoming items.</span></span>

- <span data-ttu-id="8d137-115">場所の補填。</span><span class="sxs-lookup"><span data-stu-id="8d137-115">Replenish locations.</span></span>
