# <a name="troubleshoot-warehouse-setup"></a>倉庫の設定のトラブルシューティング

[!include [banner](../includes/banner.md)]

このトピックでは、Microsoft Dynamics 365 Supply Chain Management で倉庫を設定する際に発生する可能性がある一般的な問題の修正方法について説明します。

## <a name="i-cant-use-any-role-except-administrator-to-access-the-mobile-device-app-emulator"></a>管理者以外の役割を使用して、モバイル デバイスのアプリ エミュレーターにアクセスすることはできません。

### <a name="issue-description"></a>問題の説明

管理者の役割以外の役割を使用して、モバイル デバイスのアプリ エミュレーターにアクセスすることはできません。

### <a name="issue-resolution"></a>問題の解決

モバイル デバイスのアプリ エミュレーターは、管理者アカウントでのみ機能するように設定されています。 すべてのテストとライブ プロセスの目的では、倉庫アプリ自体を使用することをお勧めします。
