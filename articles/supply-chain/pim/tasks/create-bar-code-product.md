1. <span data-ttu-id="19160-106">ホーム ページで、**リリース済製品のメンテナンス**を選択します。</span><span class="sxs-lookup"><span data-stu-id="19160-106">Select **Released product maintenance** on the homepage.</span></span>

3. <span data-ttu-id="19160-108">一覧で、目的のレコードを見つけ、選択します。</span><span class="sxs-lookup"><span data-stu-id="19160-108">In the list, find and select the desired record.</span></span> <span data-ttu-id="19160-109">この例では、品目番号 **M0001** を選択します。</span><span class="sxs-lookup"><span data-stu-id="19160-109">For this example, select item number **M0001**.</span></span>

5. <span data-ttu-id="19160-111">**バーコード**を選択します。</span><span class="sxs-lookup"><span data-stu-id="19160-111">Select **Bar codes**.</span></span>
6. <span data-ttu-id="19160-112">**新規** を選択します。</span><span class="sxs-lookup"><span data-stu-id="19160-112">Select **New**.</span></span>
7. <span data-ttu-id="19160-113">下のリストに作成された、選択した行をマークします。</span><span class="sxs-lookup"><span data-stu-id="19160-113">Mark the selected row that is created in the list below.</span></span>

9. <span data-ttu-id="19160-115">**バーコード** フィールドで、値を入力または選択します。</span><span class="sxs-lookup"><span data-stu-id="19160-115">In the **Bar code** field, enter or select a value.</span></span>
10. <span data-ttu-id="19160-116">**バーコード** フィールドに、値を入力します。</span><span class="sxs-lookup"><span data-stu-id="19160-116">In the **Bar code** field, type a value.</span></span>

12. <span data-ttu-id="19160-118">**保存** を選択します。</span><span class="sxs-lookup"><span data-stu-id="19160-118">Select **Save**.</span></span>
13. <span data-ttu-id="19160-119">ページを閉じます。</span><span class="sxs-lookup"><span data-stu-id="19160-119">Close the page.</span></span>
