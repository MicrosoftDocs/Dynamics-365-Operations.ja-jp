1. <span data-ttu-id="450e5-106">ホーム ページで、**リリース済製品のメンテナンス**を選択します。</span><span class="sxs-lookup"><span data-stu-id="450e5-106">Select **Released product maintenance** on the homepage.</span></span>

7. <span data-ttu-id="450e5-113">下のリストに作成された、選択した行をマークします。</span><span class="sxs-lookup"><span data-stu-id="450e5-113">Mark the selected row that is created in the list below.</span></span>

9. <span data-ttu-id="450e5-115">**バーコード** フィールドで、値を入力または選択します。</span><span class="sxs-lookup"><span data-stu-id="450e5-115">In the **Bar code** field, enter or select a value.</span></span>
10. <span data-ttu-id="450e5-116">**バーコード** フィールドに、値を入力します。</span><span class="sxs-lookup"><span data-stu-id="450e5-116">In the **Bar code** field, type a value.</span></span>

13. <span data-ttu-id="450e5-119">ページを閉じます。</span><span class="sxs-lookup"><span data-stu-id="450e5-119">Close the page.</span></span>
