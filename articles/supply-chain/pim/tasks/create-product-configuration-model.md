# <a name="create-a-product-configuration-model"></a>製品コンフィギュレーション モデルの作成

[!include[task guide banner](../../includes/task-guide-banner.md)]

この手順は、製品コンフィギュレーション モデルの作成方法ならびに属性およびサブコンポーネントなどの基本情報の入力方法を示します。 この手順の作成に使用するデモ データの会社は USMF です。


## <a name="create-a-product-model"></a>製品モデルの作成
1. [製品バリアント モデルの定義] をクリックします。
2. [製品コンフィギュレーション モデル] をクリックします。
3. [新規] をクリックします。
4. [名前] フィールドに値を入力します。
5. [説明] フィールドに値を入力します。
6. [ソルバー戦略] フィールドで、オプションを選択します。
    * ソルバー戦略によって、制約ベースの製品コンフィギュレーション モデルの制約がどのように処理されるかが決まります。 この選択は、製品コンフィギュレーション モデルのパフォーマンスに影響することがあります。  
7. [名前] フィールドに値を入力します。
    * ルート コンポーネントは、製品コンフィギュレーション モデルを表しますが、他の製品モデルでも使用できます。  
8. [OK] をクリックします。
9. [コンフィギュレーションの再利用] フィールドで、オプションを選択します。
    * [コンフィギュレーションの再利用] パラメータが [はい] に設定されている場合、システムは、各コンフィギュレーション セッションの後で同一のコンフィギュレーションがあるかどうかを確認し、完全に一致するコンフィギュレーションがあれば再利用します。  

## <a name="add-attributes"></a>属性の追加
1. [属性] セクションを展開します。
2. [追加] をクリックします。
3. 一覧で、選択された行をマークします。
4. [名前] フィールドに値を入力します。
5. [ソルバー名] フィールドに値を入力します。
    * ソルバー名は、製品コンフィギュレーターの制約ソルバーで使用されます。 スペースまたは _ (アンダースコア) 以外の特殊文字は含めないようにしてください。  
6. [説明] フィールドに値を入力します。
    * 説明テキストはコンフィギュレーション ユーザーに表示されるため、適切な属性値を選択するためのヘルプとして使用できます。  
7. [属性タイプ] フィールドで、値を入力または選択します。
    * 属性タイプによって、属性で使用できる値が決まります。  
8. [再利用に含める] チェック ボックスを選択します。
    * このオプションは、[コンフィギュレーションの再利用] オプションが選択されている場合にのみ使用できます。 [コンフィギュレーションの再利用] チェック ボックスに属性を含めると、システムが完全一致を検索する際にこの属性を考慮することになります。  

## <a name="add-subcomponents"></a>サブコンポーネントを追加
1. [サブコンポーネント] セクションを展開します。
2. [追加] をクリックします。
3. 一覧で、選択された行をマークします。
4. [名前] フィールドに値を入力します。
5. [ソルバー名] フィールドに値を入力します。
6. [説明] フィールドに値を入力します。
7. [コンポーネント] フィールドで、値を入力または選択します。
    * 各サブコンポーネントは、コンポーネント定義を参照する必要があります。 このデザインは再利用可能コンポーネントをサポートしており、1 回定義したコンポーネントを多くの製品モデルで使用できるようになります。  
8. [保存] をクリックします。
9. [BOM 明細行の詳細] をクリックします。
    * "BOM 明細行の詳細" フォームでは、サブコンポーネントで必要なプロパティをユーザーが選択できます。 各プロパティには、固定値を指定するか、属性にマップできます。 属性へのマッピングを行うと、コンフィギュレーションの選択に応じて、異なる値が BOM 明細行プロパティに割り当てられることになります。  
10. [品目番号] フィールドで、値を入力または選択します。
    * 各サブコンポーネントは、制約ベースのコンフィギュレーション テクノロジを使用したコンフィギュレーション可能な製品マスターを表します。 参照は、品目番号を使用して行われます。  
11. [設定] チェック ボックスを選択します。
12. [計算] フィールドで、[はい] を選択します。
    * 計算オプションを設定して、製品の原価計算を実行するときに製品が含まれるようにします。  
13. [設定] タブをクリックします。
14. [設定] チェック ボックスを選択します。
15. [数量] フィールドに数値を入力します。
    * 数量フィールドは、構成済製品にこの製品が消費される量を決定します。  
16. [設定] チェック ボックスを選択します。
17. [生産数] フィールドに数値を入力します。
18. [OK] をクリックします。
