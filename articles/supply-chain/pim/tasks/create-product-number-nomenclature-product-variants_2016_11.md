# <a name="create-a-product-number-for-configured-product-variants"></a>コンフィギュレーションされている製品バリアントの製品番号の作成

[!include [task guide banner](../../includes/task-guide-banner.md)]

この手順では、コンフィギュレーション済製品バリアントにおける製品番号の分類を設定する方法、およびそれをコンフィギュレーション可能な製品マスターに関連付ける方法を示します。 この手順では、製品コンフィギュレーション モデル コンポーネントのコンフィギュレーションに関する用語をどのように作成するかを示します。 この手順の作成に使用するデモ データの会社は USMF です。 新しい製品番号の分類は、「D0004」製品マスターに割り当てられます。 このタスクは、通常、製品デザイナーが行います。


## <a name="create-a-product-number-nomenclature"></a>[製品番号の分類] を作成します。
1. [製品バリアント モデルの定義] をクリックします。
2. [製品の分類] をクリックします。
3. [新規] をクリックします。
4. [名前] フィールドに値を入力します。
5. [説明] フィールドに値を入力します。
6. [追加] をクリックします。
7. [製品マスター番号] をクリックします。
8. [追加] をクリックします。
9. [テキスト定数] をクリックします。
10. 一覧で、選択された行をマークします。
11. [テキスト] フィールドに値を入力します。
12. [追加] をクリックします。
13. [コンフィギュレーション] をクリックします。
14. ページを閉じます。

## <a name="assign-the-product-number-nomenclature-to-a-product-master"></a>製品番号の分類を製品マスターに割り当てる
1. [製品マスター] をクリックします。
2. クイック フィルターを使用して、レコードを見つけます。 たとえば、「D」という値を含む [製品番号] フィールドをフィルターします。
3. 一覧で、選択された行のリンクをクリックします。
4. [編集] をクリックします。
5. [分類の使用] フィールドで、[はい] を選択します。
6. [製品バリアント番号の分類] フィールドに、値を入力または選択します。
7. ページを閉じます。
8. ページを閉じます。

15. [属性値] をクリックします。
16. 一覧で、選択された行をマークします。

24. 一覧で、選択された行をマークします。
25. [属性] フィールドに、値を入力または選択します。
26. [追加] をクリックします。
27. [テキスト定数] をクリックします。
28. 一覧で、選択された行をマークします。
