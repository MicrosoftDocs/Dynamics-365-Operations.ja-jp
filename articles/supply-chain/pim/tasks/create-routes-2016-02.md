# <a name="create-routes-february-2016"></a>工順の作成 (2016 年 2 月)

[!include [task guide banner](../../includes/task-guide-banner.md)]

このタスクでは、完成品と半完成品の生産工順の作成について説明します。 これは、BOM 計算シリーズの 5 番目のタスクです。 このタスクの作成に使用するデモ データの会社は USMF です。


## <a name="create-a-route-for-a-semi-finished-product"></a>半完成品の工順の作成
1. [製品情報管理] > [製品] > [リリースされた製品] の順に移動します。
2. 一覧で、選択された行のリンクをクリックします。
    * 品目番号 BOM_2 を選択します。  
3. アクション ペインで、[エンジニア] をクリックします。
4. [工順] をクリックします。
5. [新規] をクリックします。
6. [工順と工順バージョン] をクリックします。
7. [説明] フィールドに値を入力します。
    * たとえば、「ROUTE_2」を入力します。  
8. [サイト] フィールドで値を入力または選択します。
    * たとえば、サイト 1 を入力または選択します。  
9. [OK] をクリックします。
10. [新規] をクリックします。
11. [操作] フィールドで、値を入力または選択します。
    * この例では、[組み立て] を選択します。  
12. [実行時間] フィールドで、数値を入力します。
    * たとえば、「1」と入力します。 実行時間は、多くの場合、品目について計算される価格の一部になります。  
13. [工順グループ] フィールドで、値を入力または選択します。
    * この例では、「Std」を選択します。  
14. [設定] タブをクリックします。
15. [段取りカテゴリ] フィールドで、値を入力または選択します。
    * この例では、[組み立て] を選択します。  
16. [時間] タブをクリックします。
17. [段取り時間] フィールドに、数値を入力します。
    * この例では、「1」を入力します。 段取り時間は、多くの場合、品目について計算される価格の一部になります。  
18. [アクション ペイン] で、[工順バージョン] をクリックします。
19. [承認] をクリックします。
20. [工順も承認しますか?] で [はい] を選択します。] アクションを使用すると、このチェック ボックスは自動的にオンになります。
21. [OK] をクリックします。
22. [アクション ペイン] で、[工順バージョン] をクリックします。
23. [有効化] をクリックします。
24. ページを閉じます。
25. ページを閉じます。

## <a name="create-a-route-for-a-finished-product"></a>完成品の工順の作成
1. 一覧で、選択された行のリンクをクリックします。
    * 品目番号 BOM_1 を選択します。  
2. アクション ペインで、[エンジニア] をクリックします。
3. [工順] をクリックします。
4. [新規] をクリックします。
5. [工順と工順バージョン] をクリックします。
6. [説明] フィールドに値を入力します。
    * この例では、「ROUTE_1」を入力します。  
7. [サイト] フィールドで値を入力または選択します。
    * たとえば、サイト 1 を入力または選択します。  
8. [OK] をクリックします。
9. [新規] をクリックします。
10. [操作] フィールドで、値を入力または選択します。
    * この例では、[梱包] を選択します。  
11. [実行時間] フィールドで、数値を入力します。
    * この例では、「1」を入力します。  
12. [工順グループ] フィールドで、値を入力または選択します。
    * この例では、「Std」を選択します。  
13. [設定] タブをクリックします。
14. [段取りカテゴリ] フィールドで、値を入力または選択します。
    * この例では、[梱包] を選択します。  
15. [時間] タブをクリックします。
16. [段取り時間] フィールドに、数値を入力します。
    * この例では、「1」を入力します。  
17. [アクション ペイン] で、[工順バージョン] をクリックします。
18. [承認] をクリックします。
19. [OK] をクリックします。
20. [アクション ペイン] で、[工順バージョン] をクリックします。
21. [有効化] をクリックします。
22. ページを閉じます。
23. ページを閉じます。

## <a name="enable-automatic-consumption-of-setup-time"></a>段取り時間の自動消費の有効化
1. [生産管理] > [段取り] > [工順] > [工順グループ] に順に進みます。
2. 一覧で、目的のレコードを見つけ、選択します。
    * 一覧で「Std」を選択します。  
3. [編集] をクリックします。
4. [段取り時間] フィールドで [はい] を選択します。
    * 段取り時間は、多くの場合、品目について計算される価格の一部になります。  
5. [保存] をクリックします。
6. ページを閉じます。
