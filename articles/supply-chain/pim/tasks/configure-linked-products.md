# <a name="configure-linked-products"></a><span data-ttu-id="67067-103">リンクされた製品のコンフィギュレーション</span><span class="sxs-lookup"><span data-stu-id="67067-103">Configure linked products</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="67067-104">この手順では、リンクされた製品の構成方法を説明します。</span><span class="sxs-lookup"><span data-stu-id="67067-104">This procedure walks through configuring linked products.</span></span> <span data-ttu-id="67067-105">この手順は USRT のデモ データを使用し、両方の製品が既に作成され、リリースされていると想定しています。</span><span class="sxs-lookup"><span data-stu-id="67067-105">This procedure uses the USRT demo data, and assumes both products have already been created and released.</span></span>

1. <span data-ttu-id="67067-106">[製品情報管理] > [製品] > [リリースされた製品] の順に移動します。</span><span class="sxs-lookup"><span data-stu-id="67067-106">Go to Product information management > Products > Released products.</span></span>
2. <span data-ttu-id="67067-107">一覧で、目的のレコードを見つけ、選択します。</span><span class="sxs-lookup"><span data-stu-id="67067-107">In the list, find and select the desired record.</span></span>

4. <span data-ttu-id="67067-109">[リンクされた製品] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="67067-109">Click Linked products.</span></span>
5. <span data-ttu-id="67067-110">[編集] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="67067-110">Click Edit.</span></span>
6. <span data-ttu-id="67067-111">[リンクされた製品番号] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。</span><span class="sxs-lookup"><span data-stu-id="67067-111">In the Linked product number field, click the drop-down button to open the lookup.</span></span>
7. <span data-ttu-id="67067-112">一覧で、選択された行のリンクをクリックします。</span><span class="sxs-lookup"><span data-stu-id="67067-112">In the list, click the link in the selected row.</span></span>
8. <span data-ttu-id="67067-113">[単位] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。</span><span class="sxs-lookup"><span data-stu-id="67067-113">In the Unit field, click the drop-down button to open the lookup.</span></span>
9. <span data-ttu-id="67067-114">一覧で、選択された行のリンクをクリックします。</span><span class="sxs-lookup"><span data-stu-id="67067-114">In the list, click the link in the selected row.</span></span>
10. <span data-ttu-id="67067-115">[数量] フィールドに数値を入力します。</span><span class="sxs-lookup"><span data-stu-id="67067-115">In the Quantity field, enter a number.</span></span>
11. <span data-ttu-id="67067-116">[保存] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="67067-116">Click Save.</span></span>
