# <a name="enter-the-gtin-code-for-a-product"></a>製品の GTIN コードの入力

[!include [banner](../../includes/banner.md)]

GTIN コードは、GS1 組織によって割り当てられます。 この例では、デモ データの会社 USMF で製品番号「M0001」のコードを入力します。

1. [リリース済製品の保守] をクリックします。
2. [リリース済製品] をクリックします。
3. 製品画像の情報ボックスを展開します。
4. 一覧で、目的のレコードを見つけ、選択します。
5. [アクション] ウィンドウで [在庫の管理] をクリックします。
6. [GTIN コード] をクリックします。
7. [新規] をクリックします。
8. 一覧で、選択された行をマークします。
9. [単位] フィールドで、値を入力または選択します。
10. [GTIN コード] フィールドで、[オプション] を選択します。
11. [GTIN] フィールドに値を入力します。
12. ページを閉じます。
