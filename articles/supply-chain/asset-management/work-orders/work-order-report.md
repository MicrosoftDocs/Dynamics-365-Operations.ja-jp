# <a name="work-order-report"></a>作業指示書レポート

[!include [banner](../../includes/banner.md)]


選択した 1 つ以上の作業指示書に関する詳細情報を示す作業指示書レポートを生成できます。

1. **資産管理** > **レポート** > **作業指示書** > **作業指示書レポート** を選択します。

2. レポートに含める詳細を決定するために、必要に応じて切り替えボタンの「はい」/「いいえ」を選択します。  
    - **印刷設定** セクションでは、関連するメンテナンス作業タイプの設定から添付ファイルを含めるかどうかを選択できます。

3. クイックタブを **含めるレコード** では、**作業指示書** 別にレポートのコンテンツをフィルタリングできます。

4. **バックグラウンドで実行** クイック タブでは、必要に応して作業指示書レポートの生成をバッチ ジョブとして設定できます。

5. **OK** を選択してレポートを生成します。
