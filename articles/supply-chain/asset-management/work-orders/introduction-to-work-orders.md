[!include [banner](../../includes/banner.md)]



<span data-ttu-id="12b93-104">作業指示書は、メンテナンス作業の管理、それに必要となる情報の提供、そしてその消費の登録に使用されます。</span><span class="sxs-lookup"><span data-stu-id="12b93-104">Work orders are used to manage maintenance jobs, provide required information for them, and register consumption on them.</span></span> <span data-ttu-id="12b93-105">各作業指示書には 1 つ以上の作業指示書ジョブが含まれ、1 つ以上の資産を各作業指示書に関連付けることができます。</span><span class="sxs-lookup"><span data-stu-id="12b93-105">Each work order can contain one or more work order jobs, and one or more assets can be connected to each work order.</span></span> <span data-ttu-id="12b93-106">各作業指示書ジョブは、資産でスケジュールされているメンテナンス作業を定義します。</span><span class="sxs-lookup"><span data-stu-id="12b93-106">Each work order job defines a maintenance job that is scheduled on the asset.</span></span>

<span data-ttu-id="12b93-107">作業指示書は、次の方法で作成できます:</span><span class="sxs-lookup"><span data-stu-id="12b93-107">Work orders can be created in the following ways:</span></span>

- <span data-ttu-id="12b93-108">「自動作成」の設定が有効になっているカレンダーベースのメンテナンス計画では、 [メンテナンス計画のスケジュール](../preventive-and-reactive-maintenance/schedule-maintenance-plans.md) フォームを自動的に利用します。</span><span class="sxs-lookup"><span data-stu-id="12b93-108">For  calendar-based maintenance plans where the "Auto create" setting is turned on, automatically by using [Schedule maintenance plans](../preventive-and-reactive-maintenance/schedule-maintenance-plans.md).</span></span>

- <span data-ttu-id="12b93-109">「自動作成」の設定が有効になっているメンテナンス ラウンドでは、[メンテナンス ラウンドのスケジュール](../preventive-and-reactive-maintenance/maintenance-rounds.md) フォームを自動的に利用します。</span><span class="sxs-lookup"><span data-stu-id="12b93-109">For maintenance rounds where the "Auto create" setting is turned on, automatically by using [Schedule maintenance rounds](../preventive-and-reactive-maintenance/maintenance-rounds.md).</span></span>

- <span data-ttu-id="12b93-110">予防的メンテナンス作業またはメンテナンス要求は、[メンテナンス スケジュール](../preventive-and-reactive-maintenance/maintenance-schedule.md) から作成します。</span><span class="sxs-lookup"><span data-stu-id="12b93-110">For preventive maintenance jobs or maintenance requests, from [Maintenance schedule](../preventive-and-reactive-maintenance/maintenance-schedule.md).</span></span>

- <span data-ttu-id="12b93-111">手動</span><span class="sxs-lookup"><span data-stu-id="12b93-111">Manually</span></span>

- <span data-ttu-id="12b93-112">**すべてのメンテナンス要求**、**有効なメンテナンス要求**、または **個人用の機能の場所のメンテナンス要求** ページから。</span><span class="sxs-lookup"><span data-stu-id="12b93-112">From the **All maintenance requests**, **Active maintenance requests**, or **My functional location maintenance requests** page.</span></span>
