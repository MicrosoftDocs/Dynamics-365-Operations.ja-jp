# <a name="introduction-to-work-orders"></a><span data-ttu-id="6b305-103">作業指示書の概要</span><span class="sxs-lookup"><span data-stu-id="6b305-103">Introduction to work orders</span></span>

[!include [banner](../../includes/banner.md)]

[!include [banner](../../includes/preview-banner.md)]

<span data-ttu-id="6b305-104">作業指示書は、メンテナンス作業を管理し、必要な情報を提供し、および消費を登録するために使用されます。</span><span class="sxs-lookup"><span data-stu-id="6b305-104">Work orders are used to manage, provide required information for, and register consumption on maintenance jobs.</span></span> <span data-ttu-id="6b305-105">1 つの作業指示書に 1 つ以上の作業指示書ジョブが含まれ、1 つ以上の資産を作業指示書に関連付けることができます。</span><span class="sxs-lookup"><span data-stu-id="6b305-105">A work order may contain one or more work order jobs, and one or more assets can be connected to a work order.</span></span> <span data-ttu-id="6b305-106">各作業指示書明細行は、資産でスケジュールされているメンテナンス作業を定義します。</span><span class="sxs-lookup"><span data-stu-id="6b305-106">Each work order line defines a maintenance job scheduled on the asset.</span></span>

<span data-ttu-id="6b305-107">作業指示書は自動または手動で作成できます。</span><span class="sxs-lookup"><span data-stu-id="6b305-107">Work orders can be created automatically or manually:</span></span>

- <span data-ttu-id="6b305-108">「自動作成」に設定されたカレンダーベースのメンテナンス計画の場合、**メンテナンス計画のスケジュール** フォームを自動的に使用します。</span><span class="sxs-lookup"><span data-stu-id="6b305-108">Automatically using the **Schedule maintenance plans** form, for calendar-based maintenance plans set to "Auto create".</span></span>  

- <span data-ttu-id="6b305-109">「自動作成」に設定されたカレンダーベースのメンテナンス ラウンドの場合、**メンテナンス ラウンドのスケジュール** フォームを自動的に使用します。</span><span class="sxs-lookup"><span data-stu-id="6b305-109">Automatically using the **Schedule maintenance rounds** form, for maintenance rounds set to "Auto create".</span></span>  

- <span data-ttu-id="6b305-110">メンテナンス スケジュール (予防的メンテナンス作業またはメンテナンス要求) から作成します。</span><span class="sxs-lookup"><span data-stu-id="6b305-110">Create from maintenance schedule, which can be preventive maintenance jobs or maintenance requests.</span></span>  

- <span data-ttu-id="6b305-111">手動で作業指示書を作成します。</span><span class="sxs-lookup"><span data-stu-id="6b305-111">Create a work order manually.</span></span>  

- <span data-ttu-id="6b305-112">**すべてのメンテナンス要求**または**有効なメンテナンス要求**または**個人用の機能の場所のメンテナンス要求**を使用して、作業指示書を作成します。</span><span class="sxs-lookup"><span data-stu-id="6b305-112">Create a work order from the **All maintenance requests** or **Active maintenance requests** or **My functional location maintenance requests**.</span></span>

>[!NOTE]
><span data-ttu-id="6b305-113">同じ資産に関連する作業指示書ジョブは、同じプロジェクト ID に関連します。</span><span class="sxs-lookup"><span data-stu-id="6b305-113">Work order jobs related to the same asset are related to the same project ID.</span></span>
