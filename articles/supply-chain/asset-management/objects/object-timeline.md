# <a name="asset-event-history"></a>資産イベント履歴

[!include [banner](../../includes/banner.md)]

 

この記事では、資産管理で資産イベントの履歴にアクセスする方法について説明します。 **資産イベントの履歴** ページには、資産の有効期間中に作成された登録履歴が表示されます。 このページには、**全資産**、**有効な資産**、および **個人用の有効な資産** メニュー項目からアクセスできます。 資産を選択し、**イベントの履歴** を選択します。

**資産イベントの履歴** ページの **詳細** クイック タブで、利用可能なすべての情報を検索できます。 たとえば、資産イベントの履歴を使用して、次の情報を検索できます。

- ジョブ タイプが資産で最後に使用されたとき
- 特定の作業者がワーク オーダー ジョブで注釈を入力したとき

タイムラインは、ページが開かれるたびに更新されます。 次の情報が含まれています。

- 資産に加えられた変更: 資産場所、資産 ID、名前、および仕入先保証
- 資産の親
- 資産部品表 (BOM)
- 資産ライフサイクルの状態ログ
- 機能的な場所
- メンテナンス要求
- 転記されたアイテムとメモを含む、ワーク オーダー
- エラー
- 条件評価
