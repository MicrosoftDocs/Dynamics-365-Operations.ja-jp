# <a name="vendor-settings-added-for-landed-cost"></a>陸揚原価に仕入先の設定が追加されました

[!include [banner](../../includes/banner.md)]

**陸揚原価** モジュールを有効にすると、既存の **仕入先** ページにいくつかの新しいフィールドが追加されます。 これらのフィールドを使用して、陸揚原価機能と組み合わせて使用する仕入先を設定します。

関連するフィールドを設定するには、**調達 \> 仕入先 \> すべての仕入先** に移動します。 既存の仕入先を開くか、新しい仕入先を作成し、**その他の詳細** クイック タブを選択します。 **陸揚原価** モジュールが追加した新しいフィールドはすべて、**航海** ヘッダーの下に表示されます。 次の表でこれらのフィールドについて説明します。

| フィールド | 説明 |
|---|---|
| 出荷タイプ | <p>陸揚原価に関係する仕入先ロールを選択します。</p><ul><li>**なし** – 仕入先には、陸揚原価に関連する特定のロールはありません。 ほとんどの仕入先には特定のロールが割り当がないため、この値が既定の設定です。</li><li>**出荷会社** – 仕入先は出荷会社です。 この出荷タイプの仕入先は、**航海** ページの **出荷会社** フィールドで選択できます。</li><li>**関税ブローカー** – 仕入先は関税ブローカーです。 この出荷タイプの仕入先は、**フォリオ** ページの **関税ブローカー** フィールドで選択できます。</li><li>**エージェント** – 仕入先はエージェントです。 この出荷タイプの仕入先は、**仕入先** および **発注書** ページの **エージェント** フィールドで選択できます。</li></ul> |
| 原価タイプ グループ | [自動原価](auto-cost-setup.md) を選択するために、仕入先を原価タイプ グループに割り当てます。 |
| 出荷元港 | 航海の出港地を選択します。 |
| エージェント | 仕入先から購買を行う場合の既定のエージェント。 |
| 輸入原価仕入先 | <p>仕入先が陸揚原価仕入先であるかどうかを示します。</p><p>**ヒント:** このフィールドをレコード レベルのセキュリティと使用して、航海の作成時に表示される発注書を制限できます。</p> |
| 出荷会社 | 仕入先に対する発注書を作成するときに使用する、既定の出荷会社を選択します。 |
| サービス プロバイダー | 仕入先がサービス プロバイダーかどうかを示します。 |
| 超過/過少許容範囲グループ | 仕入先の既定の超過/過小許容範囲グループを選択します。 |
