# <a name="shipping-container-setup"></a>出荷コンテナーの設定

[!include [banner](../../includes/banner.md)]

この記事では、**陸揚原価** モジュールの出荷コンテナーを設定する方法について説明します。

## <a name="set-up-shipping-container-types"></a><a id="shipping-container-types"></a> コンテナー タイプを設定します

出荷コンテナのタイプは、出荷時および航海中に使用できる出荷コンテナーのタイプを定義します。

出荷コンテナー タイプを使用するには、**陸揚原価 \> コンテナ―設定 \> 出荷コンテナー タイプ** に移動します。 そこで、コンテナ タイプのレコードを表示、追加、および削除できます。 次の表は、各レコードに使用できるフィールドを説明します。

| フィールド | 説明 |
|---|---|
| 出荷コンテナー タイプ | 出荷コンテナー タイプの固有 ID 名/番号を入力します。 |
| 説明 | 出荷コンテナー タイプの説明を入力します。 |
| 量 | このタイプの出荷コンテナーの最大容積を入力します。 |
| 太さ | このタイプの出荷コンテナーの最大重量を入力します。 |
| 返品可能 | このタイプの出荷コンテナーは、航海中に使用された後で仕入先に返品できるかどうかを指定します。 このオプションが *はい* に設定されている場合、このタイプの出荷コンテナーを原産港に戻す場合に追加費用が適用される場合があります。 |

## <a name="set-up-shipping-containers"></a>出荷コンテナーの設定

出荷コンテナー レコードは、航海中に使用する各コンテナーを識別するために使用します。 航海を作成する場合、ここで定義した出荷コンテナー レコードの一覧で、特定のコンテナーを選択できます。 この機能は、会社が独自の出荷コンテナーを所有している場合に役立ちます。

1 回だけ使用される出荷コンテナーの出荷コンテナー番号を入力する必要はありません。 代わりに、航海を作成するときに出荷コンテナー番号を追加できます。

出荷コンテナー レコードは、陸揚原価でのみ使用されます。 標準の **輸送管理モジュール** (INS) では使用できません。

出荷コンテナーを使用するには、**陸揚原価 \> コンテナ―設定 \> 出荷コンテナー** に移動します。 そこで、出荷コンテナーのレコードを表示、追加、および削除できます。 次の表は、各レコードに使用できるフィールドを説明します。

| フィールド | 説明 |
|---|---|
| 出荷コンテナー | 出荷コンテナーの固有 ID 名/番号を入力します。 |
| 出荷コンテナー タイプ | 出荷コンテナーのタイプを選択します。 詳細については、この記事で前述した [出荷コンテナー タイプの設定](#shipping-container-types) を参照してください。 |

> [!NOTE]
> - 出荷コンテナーの設定はオプションです。 通常、このコンテナーは、会社が独自の出荷コンテナーを所有している場合、または多くの場合は同じ出荷コンテナーを再利用する場合にのみ使用します。
> - 出荷コンテナー番号のチェック ディジットは計算されません。

## <a name="set-up-unit-types"></a><a name="unit-types"></a> 単位タイプの設定

単位タイプにより、出荷コンテナーの追加のグループ化と識別方法が設定されます。 通常、単位タイプは、パレットまたはドラムなど、商品が梱包されているコンテナーのタイプを識別するために使用されます。 単位タイプは、**すべての出荷コンテナー** ページでコンテナーを設定するときに選択できます。

単位タイプは、陸揚原価でのみ使用されます。 TMS では使用できません。

単位タイプを使用するには、**陸揚原価 \> コンテナ―設定 \> 単位タイプ** に移動します。 そこで、単位タイプのレコードを表示、追加、および削除できます。 次の表は、各レコードに使用できるフィールドを説明します。

| フィールド | 説明 |
|---|---|
| 単位タイプ | 単位タイプの固有 ID 名/番号を入力します。 |
| 説明 | 単位タイプの説明を入力します。 |

## <a name="set-up-refrigeration-types"></a><a name="refrigeration-types"></a> 冷却タイプの設定

冷却タイプは、出荷コンテナーを追加のグループ化と、識別方法を確立します (通常、冷却コンテナーの出荷コンテナー)。 冷却タイプは、**すべての出荷コンテナー** ページでコンテナーを設定するときに選択できます。

冷却タイプを使用するには、**陸揚原価 \> コンテナ―設定 \> 冷却タイプ** に移動します。 そこで、冷却タイプのレコードを表示、追加、および削除できます。 次の表は、各レコードに使用できるフィールドを説明します。

| フィールド | 説明 |
|---|---|
| 冷却タイプ | 冷却タイプの固有 ID 名/番号を入力します。 |
| 説明 | 冷却タイプの説明を入力します。 |
