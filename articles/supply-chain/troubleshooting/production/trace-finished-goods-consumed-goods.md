# <a name="cant-trace-serial-numbers-of-finished-goods-against-those-of-consumed-goods"></a>消費された商品のシリアル番号に対する完成品のシリアル番号を追跡できない

## <a name="symptoms"></a>現象

完成品のシリアル番号を、製造オーダーが完成品を作成するために消費する材料のシリアル番号に対して追跡することはできません。

## <a name="workaround"></a>回避策

回避策として、数量 1 の製造オーダーを作成できます。
