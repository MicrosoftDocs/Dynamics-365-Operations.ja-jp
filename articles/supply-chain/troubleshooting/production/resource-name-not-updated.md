# <a name="resource-name-isnt-updated-after-changing-a-worker-name-in-the-address-book"></a>アドレス帳で作業者の名前を変更しても、リソース名が更新されない

## <a name="symptoms"></a>現象

グローバル アドレス帳の作業者の名前を変更した場合、リソース グループ マスター内の一致するリソース名は更新されません。

## <a name="resolution"></a>解決策

現在、一致するリソース名は自動的に更新されません。 問題を修正するには、リソース名を手動で更新する必要があります。
