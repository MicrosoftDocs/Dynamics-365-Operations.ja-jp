# <a name="picking-list-report-doesnt-show-new-notes-that-are-attached-to-a-journal"></a>仕訳帳に関連付けられた新しいメモがピッキング リスト レポートに表示されない

KB 番号: 4611660

## <a name="symptoms"></a>現象

ピッキング リスト仕訳帳に関連付けられた新しいメモがピッキング リスト レポートに表示されません。

## <a name="resolution"></a>解像度

ピッキング リスト レポートには、販売注文行と販売注文ヘッダーに関連付けられたメモだけが表示されます。 請求仕訳帳に追加された新しいメモは表示されません。
