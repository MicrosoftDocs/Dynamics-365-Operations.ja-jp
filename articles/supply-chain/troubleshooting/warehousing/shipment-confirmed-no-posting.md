# <a name="shipment-for-load-has-been-confirmed-but-no-lines-are-posted"></a>積荷の出荷は確認されたが、明細行は転記されていない

## <a name="symptoms"></a>現象

出荷倉庫の操作を行う場合、次のメッセージが表示される場合があります。

> 積荷 1% の出荷が確認されました。

ただし、それ以上の転記は行われませんでした。

## <a name="resolution"></a>解決策

これは仕様です。 出荷確認は転記に影響しません。 出荷と積荷のステータスが更新されるだけです。 転記は、別のプロセスで行われる必要があります。
