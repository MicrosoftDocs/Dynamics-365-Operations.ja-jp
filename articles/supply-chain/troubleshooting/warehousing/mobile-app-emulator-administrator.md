# <a name="can-only-use-administrator-role-to-access-the-mobile-device-app-emulator"></a>管理者の役割を使用してのみ、モバイル デバイスのアプリ エミュレーターにアクセスできます

## <a name="symptoms"></a>現象

Supply Chain Management で倉庫を設定する場合は、管理者以外の役割を使用して、モバイル デバイスのアプリ エミュレーターにアクセスすることはできません。

## <a name="resolution"></a>解決策

モバイル デバイスのアプリ エミュレーターは、管理者アカウントでのみ機能するように設定されています。 すべてのテストとライブ プロセスの目的では、倉庫管理モバイル アプリ自体を使用することをお勧めします。
