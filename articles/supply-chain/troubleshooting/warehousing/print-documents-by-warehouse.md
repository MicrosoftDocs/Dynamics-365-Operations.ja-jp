# <a name="print-a-delivery-note-or-packing-content-by-warehouse-or-site"></a>納品書または梱包コンテンツを倉庫またはサイトごとに印刷する

## <a name="symptoms"></a>現象

**作業監査テンプレート行の更新** ページで、倉庫またはサイト別に納品書または梱包コンテンツを印刷しようとしています。

## <a name="resolution"></a>解決策

印刷管理設定を使用してドキュメントを印刷する場合は、**作業監査テンプレート行の更新** ページの **印刷設定の編集** を選択するのではなく、印刷管理を使用して範囲 (サイト/倉庫) を制限します。
