# <a name="inventory-aging-report-discrepancies"></a>在庫エイジング レポートの差異

## <a name="symptoms"></a>現象

**在庫エイジング レポート** には、さまざまな保管分析コード (サイトや倉庫など) で表示した場合、異なる値が表示されます。

## <a name="resolution"></a>解決策

レポート ロジックの詳細については、[在庫エイジング レポートの例とロジック](/dynamics365/supply-chain/cost-management/inventory-aging-report.md)を参照してください。
