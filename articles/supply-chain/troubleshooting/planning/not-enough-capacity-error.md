# <a name="not-enough-capacity-could-be-found"></a>十分な領域が見つかりませんでした

## <a name="symptoms"></a>現象

スケジューリングを実行した際に、次のエラー メッセージが表示される場合があります。

> 製造オーダー %1 をスケジュールできませんでした。 十分な容量が見つかりませんでした。

スケジューリング エンジンが失敗してこのエラー メッセージが表示される理由は、いくつか考えられます。 このトピックでは、エラーの根本原因を見つけて軽減するための情報を提供します。

## <a name="resolution"></a>解決策

スケジューリング エンジンが失敗してエラー メッセージが表示される理由は、いくつか考えられます。 エラーの根本原因を見つけて軽減するためのガイドラインについては、[スケジューリング エンジン エラー "十分な容量が見つかりませんでした" を修正する](/dynamics365/supply-chain/master-planning/not-enough-capacity-error-resolution) を参照してください。
