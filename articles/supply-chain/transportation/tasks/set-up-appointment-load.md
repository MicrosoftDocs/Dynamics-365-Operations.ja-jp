# <a name="set-up-an-appointment-for-a-load"></a>積荷の予定の設定

[!include [banner](../../includes/banner.md)]

この手順では、積荷に対してドッキング予定を設定し、計画する方法を示します。 この設定は、通常、配送コーディネーターにより実行されます。 デモ データの会社 USMF でこの手順を使用できます。


## <a name="select-the-load"></a>積荷の選択
1. [配送管理] > [計画] > [積荷計画ワークベンチ] の順に移動します。
2. [出荷と入庫を非表示にする] チェック ボックスをオフにします。
3. リストで、[出荷済] のステータスがある積荷を選択します。
4. [輸送管理] をクリックします。
5. [予定のスケジューリング] をクリックします。

## <a name="create-an-appointment"></a>予定の作成
1. [新規] をクリックします。
2. [予定ルール] フィールドで値を入力または選択します。
3. [保存] をクリックします。
4. [ステータスの更新] をクリックします。
5. [確定] をクリックします。
6. [保存] をクリックします。
7. ページを閉じます。
