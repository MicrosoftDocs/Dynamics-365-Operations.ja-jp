# <a name="set-up-a-transportation-tender"></a>輸送業者の設定

[!include [banner](../../includes/banner.md)]

この手順では、輸送業者を設定する方法を示します。 この設定は、通常、配送コーディネーターにより実行されます。 デモ データの会社 USMF でこの手順を使用できます。


## <a name="select-a-route"></a>工順を選択します。
1. [配送管理] > [計画] > [積荷計画ワークベンチ] の順に移動します。
2. [出荷と入庫を非表示にする] チェック ボックスをオフにします。
3. 積荷 ID 00006 で行を選択します。
4. [評価とルート指定] をクリックします。
5. [工順] をクリックします。

## <a name="create-the-transportation-tender"></a>輸送業者の作成
1. [輸送業者] をクリックします。
2. [新規] をクリックします。
3. [一般] セクションを展開します。
4. [要求レート] フィールドに数を入力します。
5. [保存] をクリックします。
6. [ステータスの更新] をクリックします。
7. [送信] をクリックします。
8. 工順を選択します。
