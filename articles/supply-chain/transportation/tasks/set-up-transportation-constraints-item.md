## <a name="create-an-item-constraint"></a><span data-ttu-id="99bbe-107">品目制約の作成</span><span class="sxs-lookup"><span data-stu-id="99bbe-107">Create an item constraint</span></span>

6. <span data-ttu-id="99bbe-113">[倉庫] フィールドで、値を入力または選択します。</span><span class="sxs-lookup"><span data-stu-id="99bbe-113">In the Warehouse field, enter or select a value.</span></span>
