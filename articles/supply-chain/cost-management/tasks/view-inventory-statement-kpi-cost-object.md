# <a name="view-inventory-statement-and-kpi-by-cost-object"></a>原価オブジェクトごとの在庫明細書および KPI の表示

[!include [banner](../../includes/banner.md)]

この手順の作成に使用するデモ データの会社は USMF です。 この手順は、原価の管理者を対象としています。

1. **原価管理** を選択します。
2. **リリース済製品** を選択します。
3. クイック フィルターを使用して、レコードを見つけます。 たとえば、*d0005* の値を含む **品目番号** フィールドをフィルターします。
4. アクション ペインで **原価の管理** を選択します。
5. **原価オブジェクト** を選択します。
6. **在庫明細書および KPI** を選択します。
7. **開始日** フィールドに日付を入力します。
8. **終了日** フィールドに、日付を入力します。
