# <a name="view-cost-object-balance"></a>原価オブジェクト残高の表示

[!include [task guide banner](../../includes/task-guide-banner.md)]

この手順の作成に使用するデモ データの会社は USMF です。 この手順は、原価の管理者を対象としています。

1. [原価管理] をクリックします。
2. [リリース済製品] をクリックします。
3. [クイック フィルター] を使用して、「m0004」の値を含む [品目番号] をフィルターします。
4. [アクション] ペインで [原価の管理] をクリックします。
5. [原価オブジェクト] をクリックします。
6. [現物数量] をクリックします。
7. [分析コード表示] をクリックします。
8. [保管場所] チェック ボックスをクリアします。
9. [OK] をクリックします。
