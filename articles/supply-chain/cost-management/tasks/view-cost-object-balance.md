# <a name="view-cost-object-balance"></a>原価オブジェクト残高の表示

[!include [banner](../../includes/banner.md)]

この手順の作成に使用するデモ データの会社は USMF です。 この手順は、原価の管理者を対象としています。

1. **原価管理** を選択します。
2. **リリース済製品** を選択します。
3. クイック フィルターを使用して、*m0004* の値を含む **品目番号** フィールドをフィルターします。
4. アクション ペインで **原価の管理** を選択します。
5. **原価オブジェクト** を選択します。
6. **現物数量** を選択します。
7. **分析コード表示** を選択します。
8. **保存先** チェック ボックスをクリアします。
9. **OK** を選択します。
