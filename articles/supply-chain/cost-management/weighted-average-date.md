# <a name="weighted-average-date"></a><span data-ttu-id="60b28-103">加重平均日付</span><span class="sxs-lookup"><span data-stu-id="60b28-103">Weighted average date</span></span>

<span data-ttu-id="60b28-104">加重平均日は加重平均原則に基づく在庫モデルです。</span><span class="sxs-lookup"><span data-stu-id="60b28-104">Weighted average date is an inventory model that is based on the weighted average principle.</span></span> <span data-ttu-id="60b28-105">加重平均原則では、在庫からの出庫は、在庫原価計算期間に毎日受け取って入庫された品目の平均値で評価されます。</span><span class="sxs-lookup"><span data-stu-id="60b28-105">For the weighted average principle, issues from inventory are valued at the average value of the items that are received into inventory for each day in the inventory closing period.</span></span> 

<span data-ttu-id="60b28-106">加重平均日付を使用して在庫原価計算を実行すると、1 日のすべての入庫が、仮想出庫に対して決済されます。</span><span class="sxs-lookup"><span data-stu-id="60b28-106">When you run an inventory closing by using weighted average date, all daily receipts are settled against a virtual issue.</span></span> <span data-ttu-id="60b28-107">この仮想出庫は、その日の合計入庫数量と金額を保持します。</span><span class="sxs-lookup"><span data-stu-id="60b28-107">This virtual issue holds the total received quantity and value for that day.</span></span> <span data-ttu-id="60b28-108">仮想出庫には対応する仮想入庫があり、この仮想入庫に対して出庫が決済されます。</span><span class="sxs-lookup"><span data-stu-id="60b28-108">The virtual issue has a corresponding virtual receipt that the issues will be settled against.</span></span> <span data-ttu-id="60b28-109">そのため、すべての出庫が同じ平均原価を取得します。</span><span class="sxs-lookup"><span data-stu-id="60b28-109">Therefore, all issues receive the same average cost.</span></span> <span data-ttu-id="60b28-110">仮想の払出と入庫は仮想の転送と見なすことができ、*加重平均在庫原価計算転送*と呼ばれます。</span><span class="sxs-lookup"><span data-stu-id="60b28-110">The virtual issue and virtual receipt can be considered a virtual transfer that is known as the *weighted average inventory closing transfer*.</span></span> 

<span data-ttu-id="60b28-111">その日付以前に発生した入庫が 1 つしかない場合、平均を求める必要はありません。</span><span class="sxs-lookup"><span data-stu-id="60b28-111">If only one receipt has occurred on or before the date, you don't have to value the average.</span></span> <span data-ttu-id="60b28-112">すべての出庫がその入庫から決済されるため、仮想移動は作成されません。</span><span class="sxs-lookup"><span data-stu-id="60b28-112">Because all issues are settled from that receipt, the virtual transfer won't be created.</span></span> <span data-ttu-id="60b28-113">同様に、その日付に出庫しか発生しない場合、平均を求めるための入庫がなく、仮想移動は作成されません。</span><span class="sxs-lookup"><span data-stu-id="60b28-113">Likewise, if only issues occur on the date, there are no receipts to value the average from, and the virtual transfer won't be created.</span></span> <span data-ttu-id="60b28-114">加重平均日付を使用する場合、特定の品目の受入が特定の払出を対象に決済されるように、在庫トランザクションをマークすることもできます。</span><span class="sxs-lookup"><span data-stu-id="60b28-114">When you use weighted average date, you can mark inventory transactions so that a specific item receipt is settled against a specific issue.</span></span> <span data-ttu-id="60b28-115">この場合、加重平均日付のルールを使用しません。</span><span class="sxs-lookup"><span data-stu-id="60b28-115">In this case, you don't use the weighted average date rule.</span></span> <span data-ttu-id="60b28-116">加重平均日付の在庫モデルを使用する場合は、在庫原価計算を毎月行うことをお勧めします。</span><span class="sxs-lookup"><span data-stu-id="60b28-116">We recommend a monthly inventory closing when you use the weighted average date inventory model.</span></span>
