# <a name="weighted-average-with-physical-value-and-marking"></a><span data-ttu-id="9df59-103">現物価格とマーキングを使用した加重平均</span><span class="sxs-lookup"><span data-stu-id="9df59-103">Weighted average with physical value and marking</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="9df59-104">加重平均は加重平均原則に基づく在庫モデルであり、在庫原価計算期間中に受け入れて在庫となった品目の平均価値に、前の期間からの手持在庫を加えて、在庫からの払出の価値を決めます。</span><span class="sxs-lookup"><span data-stu-id="9df59-104">Weighted average is an inventory model based on the weighted average principle, where issues from inventory are valued at the average value of the items that are received into inventory during the inventory closing period, plus any on-hand inventory from the previous period.</span></span>

<span data-ttu-id="9df59-105">在庫原価計算を実行するときは、すべての受入が仮想払出に対して決済され、仮想払出は受け入れた合計の数量と価値を保持します。</span><span class="sxs-lookup"><span data-stu-id="9df59-105">When you run an inventory closing, all receipts are settled against a virtual issue, which holds the total received quantity and value.</span></span> <span data-ttu-id="9df59-106">この仮想払出には、対応する仮想受入があり、そこから払出を決済します。</span><span class="sxs-lookup"><span data-stu-id="9df59-106">This virtual issue has a corresponding virtual receipt from which the issues are settled.</span></span> <span data-ttu-id="9df59-107">このようにして、すべての出庫が同じ平均原価を取得します。</span><span class="sxs-lookup"><span data-stu-id="9df59-107">In this manner, all issues get the same average cost.</span></span> <span data-ttu-id="9df59-108">仮想の払出と入庫は仮想の転送と見なすことができ、加重平均在庫原価計算転送と呼ばれます。</span><span class="sxs-lookup"><span data-stu-id="9df59-108">The virtual issue and receipt can be seen as a virtual transfer, which is named the weighted average inventory closing transfer.</span></span>

<span data-ttu-id="9df59-109">入庫が 1 つしかない場合、すべての出庫をその入庫から決済することでき、仮想移動は作成されません。</span><span class="sxs-lookup"><span data-stu-id="9df59-109">If there is only one receipt, all issues can be settled from it and the virtual transfer will not be created.</span></span> 

<span data-ttu-id="9df59-110">加重平均を使用する場合、加重平均ルールを使用する代わりに、特定の品目入庫が特定の出庫を対象に決済されるように、在庫トランザクションをマークすることができます。</span><span class="sxs-lookup"><span data-stu-id="9df59-110">When using weighted average, you can mark inventory transactions so that a specific item receipt is settled against a specific issue, instead of using the weighted average rule.</span></span> 

<span data-ttu-id="9df59-111">加重平均在庫モデルを使用する場合は、在庫原価計算を毎月行うことをお勧めします。</span><span class="sxs-lookup"><span data-stu-id="9df59-111">We recommend a monthly inventory closing when you use the weighted average inventory model.</span></span> 

<span data-ttu-id="9df59-112">加重平均在庫原価計算方法の計算式は次のとおりです。</span><span class="sxs-lookup"><span data-stu-id="9df59-112">The weighted average inventory costing method is calculated by the following formula:</span></span>
-   <span data-ttu-id="9df59-113">加重平均 = (Q1\*P1 + Q2\*P2 + Qn\*Pn) / (Q1 + Q2 + Qn)</span><span class="sxs-lookup"><span data-stu-id="9df59-113">Weighted average = (Q1\*P1 + Q2\*P2 + Qn\*Pn) / (Q1 + Q2 + Qn)</span></span>
