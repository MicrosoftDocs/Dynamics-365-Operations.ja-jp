# <a name="costing-sheets"></a><span data-ttu-id="50328-108">原価計算表</span><span class="sxs-lookup"><span data-stu-id="50328-108">Costing sheets</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="50328-109">原価計算表の設定には、2 つの目的があります。</span><span class="sxs-lookup"><span data-stu-id="50328-109">Setting up the costing sheet involves two objectives.</span></span> <span data-ttu-id="50328-110">最初の目的は、製品または製造オーダーに関する売却済商品の原価情報の表示形式を定義することです。</span><span class="sxs-lookup"><span data-stu-id="50328-110">As the first objective, you define the format for displaying cost of goods sold information about a manufactured item or production order.</span></span> <span data-ttu-id="50328-111">書式設定された表示は、原価計算表と呼ばれます。</span><span class="sxs-lookup"><span data-stu-id="50328-111">The formatted display is termed a costing sheet.</span></span> <span data-ttu-id="50328-112">2 つ目の目的は、間接原価の計算の基準を定義することです。</span><span class="sxs-lookup"><span data-stu-id="50328-112">As the second objective, you define the basis for calculating indirect costs.</span></span> <span data-ttu-id="50328-113">原価計算表設定は、情報表示用および間接原価計算式用の原価グループ機能の上に構築されます。</span><span class="sxs-lookup"><span data-stu-id="50328-113">The costing sheet setup builds on the cost group feature for displaying information and for the indirect cost calculation formulas.</span></span> <span data-ttu-id="50328-114">原価計算表設定の 2 つの目的はこの項目で説明します。</span><span class="sxs-lookup"><span data-stu-id="50328-114">The two objectives of costing sheet setup are described in this article.</span></span> 

<span data-ttu-id="50328-115">原価計算表は、製造品目や製造オーダーでの販売商品の原価についての書式設定された情報表示です。</span><span class="sxs-lookup"><span data-stu-id="50328-115">A costing sheet is the formatted display of information about the cost of goods that are sold for a manufactured item or a production order.</span></span> <span data-ttu-id="50328-116">原価計算表を設定すると、情報の書式定義や、間接原価の計算基準の定義を行うことができます。</span><span class="sxs-lookup"><span data-stu-id="50328-116">When you set up a costing sheet, you define the format for the information and also define the basis for calculating indirect costs.</span></span> <span data-ttu-id="50328-117">原価計算表設定は、情報表示用および間接原価計算式用に原価グループ機能の上に構築されます。</span><span class="sxs-lookup"><span data-stu-id="50328-117">The costing sheet setup builds on the cost group features for displaying information and for the formulas that are used to calculated indirect cost.</span></span> <span data-ttu-id="50328-118">原価計算表の設定の 2 つの目的について、詳細情報を次に示します。</span><span class="sxs-lookup"><span data-stu-id="50328-118">Here is more information about the two objectives of costing sheet setup:</span></span>
-   <span data-ttu-id="50328-119">**原価計算表の形式を定義します。**</span><span class="sxs-lookup"><span data-stu-id="50328-119">**Define the format for the costing sheet.**</span></span> <span data-ttu-id="50328-120">原価計算表のユーザー定義形式は、製造された品目の売却済商品原価を含む原価区分を識別します。</span><span class="sxs-lookup"><span data-stu-id="50328-120">The user-defined format for a costing sheet identifies the segmentation of costs that contain a manufactured item’s cost of goods sold.</span></span> <span data-ttu-id="50328-121">たとえば、品目の売却済商品の原価情報は、原価グループに基づいて、材料、労務、および間接費に分割できます。</span><span class="sxs-lookup"><span data-stu-id="50328-121">For example, the information about an item’s cost of goods sold can be segmented into material, labor, and overhead, based on cost groups.</span></span> <span data-ttu-id="50328-122">これらの原価グループは、品目、工順工程用の原価カテゴリ、および間接原価計算式に割り当てられます。</span><span class="sxs-lookup"><span data-stu-id="50328-122">These cost groups are assigned to items, cost categories for routing operations, and indirect cost calculation formulas.</span></span> <span data-ttu-id="50328-123">原価計算表の形式では、複数の原価グループが定義されている場合には中間合計が必要となるのが普通です。</span><span class="sxs-lookup"><span data-stu-id="50328-123">The format for the costing sheet typically requires intermediate totals when multiple cost groups have been defined.</span></span> <span data-ttu-id="50328-124">たとえば、原材料に関連付けられる複数の原価グループは、集計できます。</span><span class="sxs-lookup"><span data-stu-id="50328-124">For example, multiple cost groups that are related to material can be aggregated.</span></span> <span data-ttu-id="50328-125">原価計算表の形式の定義はオプションですが、間接原価が計算される場合は、原価計算表の形式を定義する必要があります。</span><span class="sxs-lookup"><span data-stu-id="50328-125">The definition of a costing sheet format is optional, but a costing sheet format must be defined if indirect costs will be calculated.</span></span>
-   <span data-ttu-id="50328-126">**間接原価の計算の基準を定義します。**</span><span class="sxs-lookup"><span data-stu-id="50328-126">**Define the basis for calculating indirect costs.**</span></span> <span data-ttu-id="50328-127">間接原価は、製品の生産に関連付けられている製造間接費を反映します。</span><span class="sxs-lookup"><span data-stu-id="50328-127">Indirect costs reflect manufacturing overhead that is associated with the production of a manufactured item.</span></span> <span data-ttu-id="50328-128">間接原価計算式は、割増金またはレートとして表現できます。</span><span class="sxs-lookup"><span data-stu-id="50328-128">An indirect cost calculation formula can be expressed as either a surcharge or a rate.</span></span> <span data-ttu-id="50328-129">割増金は値のパーセンテージを表し、レートは工順工程の 1 時間当たりの金額を表します。</span><span class="sxs-lookup"><span data-stu-id="50328-129">A surcharge represents a percentage of value, whereas a rate represents an amount per hour for a routing operation.</span></span> <span data-ttu-id="50328-130">原価グループは、労務費グループの 100% 割増金や機械原価グループの USD 50.00 の時間レートなど、計算式の基準を定義します。</span><span class="sxs-lookup"><span data-stu-id="50328-130">A cost group defines the basis for the calculation formula, such as a 100-percent surcharge for a labor cost group or a USD 50.00 hourly rate for a machine cost group.</span></span> <span data-ttu-id="50328-131">計算式とその原価グループの基準を定義する場合は、原価計算表設定で、間接費を代表する原価グループを識別し、割増金またはレートのどちらの方法を使用するか選択する必要があります。</span><span class="sxs-lookup"><span data-stu-id="50328-131">If you want to define a calculation formula and its cost group basis, the costing sheet setup requires that you identify the cost group that represents the overhead, and select whether a surcharge or rate approach is used.</span></span>

<span data-ttu-id="50328-132">各計算式は、原価レコードとして入力する必要があります。</span><span class="sxs-lookup"><span data-stu-id="50328-132">Each calculation formula must be entered as a cost record.</span></span> <span data-ttu-id="50328-133">原価レコードは、指定された原価バージョン、割増金パーセンテージまたはレート金額、原価グループ基準、ステータス、および有効日から構成されます。</span><span class="sxs-lookup"><span data-stu-id="50328-133">The cost record consists of a specified costing version, a surcharge percentage or a rate amount, the cost group basis, a status, and an effective date.</span></span> <span data-ttu-id="50328-134">原価レコードを初期入力すると、ステータスは**保留中**であり、発効日が設定されています。</span><span class="sxs-lookup"><span data-stu-id="50328-134">When a cost record is first entered, it has **Pending** status and an effective date.</span></span> <span data-ttu-id="50328-135">原価レコードを有効にすると、レコードが現在の有効なレコードになるようにステータスが更新され、発効日は有効化した日に更新されます。</span><span class="sxs-lookup"><span data-stu-id="50328-135">When you activate the cost record, the status is updated to so that the record is the current active record, and the effective date is updated to the activation date.</span></span> <span data-ttu-id="50328-136">原価レコードでは、サイト固有の計算式を使用するサイトを指定できます。</span><span class="sxs-lookup"><span data-stu-id="50328-136">The cost record can also specify a site for a site-specific calculation formula.</span></span> <span data-ttu-id="50328-137">また、**サイト** フィールドを空白にして、会社全体で使用する計算式であることを示すことができます。</span><span class="sxs-lookup"><span data-stu-id="50328-137">Alternatively, you can leave the **Site** field blank to indicate that the calculation formula is a company-wide formula.</span></span> <span data-ttu-id="50328-138">計算式が品目ごとの式としてマークされている場合は、オプションで、指定した品目または品目グループから成る原価レコードを使用できます。</span><span class="sxs-lookup"><span data-stu-id="50328-138">The cost record can optionally consist of a specified item or item group when the calculation formula has been marked as a per-item formula.</span></span> 

<span data-ttu-id="50328-139">間接原価計算式に対して現在有効な原価レコードは、製造オーダー原価を見積もるために使用されます。</span><span class="sxs-lookup"><span data-stu-id="50328-139">The current active cost records for indirect cost calculation formulas are used to estimate production order costs.</span></span> <span data-ttu-id="50328-140">また、時間および材料の実際の消費に関連する実際原価の計算にも使用されます。</span><span class="sxs-lookup"><span data-stu-id="50328-140">They are also used to calculate actual costs that are related to actual consumption of time and material.</span></span> <span data-ttu-id="50328-141">保留原価レコードは、未来の日付の部品表 (BOM) 計算で使用されます。</span><span class="sxs-lookup"><span data-stu-id="50328-141">Pending cost records are used in bill of materials (BOM) calculations for a future date.</span></span> 

<span data-ttu-id="50328-142">原価バージョンに対する 2 つのブロック ポリシーで、保留原価を保持するかどうか、および保留原価を開始できるかどうかが決定されます。</span><span class="sxs-lookup"><span data-stu-id="50328-142">Two blocking policies for a costing version determine whether pending costs can be maintained, and whether the pending cost can be started.</span></span> <span data-ttu-id="50328-143">ブロック ポリシーを使用すると、データ保持を許可し、その後、原価計算バージョン内に原価データが保持されないようにすることができます。</span><span class="sxs-lookup"><span data-stu-id="50328-143">Use the blocking policies to permit data maintenance, and then to prevent data maintenance for the cost data in a costing version.</span></span> 

<span data-ttu-id="50328-144">原価計算表の形式と間接原価の計算を定義した後で、別の手順を実行して、情報を検証および保存する必要があります。</span><span class="sxs-lookup"><span data-stu-id="50328-144">After you define the costing sheet format and calculations for indirect costs, you must perform a separate step to validate and save the information.</span></span> <span data-ttu-id="50328-145">原価計算表は、売却済商品の原価情報を一貫して表示するための会社全体の形式を表します。</span><span class="sxs-lookup"><span data-stu-id="50328-145">The costing sheet represents a company-wide format for consistently displaying information about the costs of goods sold.</span></span> 

<span data-ttu-id="50328-146">原価計算表は、**品目原価の計算**ページの一部として表示されます。</span><span class="sxs-lookup"><span data-stu-id="50328-146">The costing sheet is displayed as part of the **Calculate item cost** page.</span></span> <span data-ttu-id="50328-147">原価計算表は、**品目価格**ページの製品の計算済原価レコードに対して、または **BOM 計算の結果**ページの注文固有計算レコードに対して表示できます。</span><span class="sxs-lookup"><span data-stu-id="50328-147">The costing sheet can be displayed for a manufactured item’s calculated cost record on the **Item price** page or for an order-specific calculation record on the **BOM calculation results** page.</span></span> <span data-ttu-id="50328-148">また、製造オーダーの**価格計算**ページの一部として表示されます。</span><span class="sxs-lookup"><span data-stu-id="50328-148">It can also be displayed as part of the **Price calculation** page for a production order.</span></span>
