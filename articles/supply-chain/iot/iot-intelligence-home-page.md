# <a name="iot-intelligence-now-replaced-by-sensor-data-intelligence"></a>IoT インテリジェンスは Sensor Data Intelligence に置き換えられました

[!include [banner](../../includes/banner.md)]

IoT インテリジェンス機能はバージョン 2.0 に更新され、同時に名前が *Sensor Data Intelligence* に変更されました。 この新しいバージョンは現在、Supply Chain Management のバージョン 10.0.30 以降でプレビュー機能として提供されています。

Sensor Data Intelligence では、必要な Azure コンポーネントが Microsoft に管理されるのではなく、独自の Azure サブスクリプションに展開されるという新しい展開モデルを導入しています。 この新しいモデルによって、機能のカスタマイズやサード パーティ製システムとの統合の機能が強化されます。 また、この新しいバージョンでは資産管理の 2 つの新しいモノのインターネット (IoT) シナリオへのサポートが導入されます。

このような変更が控えているため、既存の IoT インテリジェンス機能に基づく新たなプロジェクトの開始はお勧めしません。 システムに IoT 機能を追加することに関心をお持ちの場合は、代わりにまずプレビュー バージョンの Sensor Data Intelligence を使用することをお勧めします。 Sensor Data Intelligence の詳細と使用方法については、[Sensor Data Intelligence のホーム ページ](../sensor-data-intelligence/sdi-home-page.md)をご覧ください。
