# <a name="master-plans"></a><span data-ttu-id="20287-103">マスター プラン</span><span class="sxs-lookup"><span data-stu-id="20287-103">Master plans</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="20287-104">会社の日常業務のサポート、監視を要する多様な計画戦略のシミュレーション、社内の業績や顧客満足に関するポリシーの実装など、さまざまな目的のマスター プランを使用します。</span><span class="sxs-lookup"><span data-stu-id="20287-104">Use various master plans to support your company's daily working operations, simulate different planning strategies that you want to monitor, and implement a company policy, such as a policy about internal performance or customer satisfaction.</span></span> 

<span data-ttu-id="20287-105">**マスター プラン**ページでマスター プランを構成することができます。</span><span class="sxs-lookup"><span data-stu-id="20287-105">You can configure master plans on the **Master plans** page.</span></span>

<span data-ttu-id="20287-106">プランには次の 2 つのタイプがあります。</span><span class="sxs-lookup"><span data-stu-id="20287-106">There are two types of plans:</span></span>
-   <span data-ttu-id="20287-107">**静的マスター プラン**: マスター プランの計算では、現在のデータを使用して正味必要量プランが生成されます。</span><span class="sxs-lookup"><span data-stu-id="20287-107">**Static plan** – The master planning calculation uses the current data to generate a net requirements plan.</span></span> <span data-ttu-id="20287-108">このプランは、次回のマスター プランの実行まで変わりません。</span><span class="sxs-lookup"><span data-stu-id="20287-108">This plan remains unchanged until the next time that you run master planning.</span></span> <span data-ttu-id="20287-109">これは、購買担当者や生産計画担当者など、さまざまな従業員が意思決定の基準として使用し、日常業務を遂行するための業務プランです。</span><span class="sxs-lookup"><span data-stu-id="20287-109">It is an operating plan that various company personnel, such as a purchaser or production planner, can use to base their decisions on and perform their daily tasks and activities.</span></span>
-   <span data-ttu-id="20287-110">**動的マスタ プラン**: このプランは、マスター プランによって生成された同じ正味必要量プランで開始されます。</span><span class="sxs-lookup"><span data-stu-id="20287-110">**Dynamic plan** – This plan starts with the same net requirements plan that was generated by master planning.</span></span> <span data-ttu-id="20287-111">ただし、マスター データが変更されるたびに動的プランを更新できます。</span><span class="sxs-lookup"><span data-stu-id="20287-111">However, you can update the dynamic plan every time that the master data changes.</span></span> <span data-ttu-id="20287-112">これは、新しい販売注文を作成した場合などです。</span><span class="sxs-lookup"><span data-stu-id="20287-112">This could be when you create a new sales order, for example.</span></span> <span data-ttu-id="20287-113">これにより、他のユーザーがそれぞれの作業過程で使用している静的マスタ プランには影響を与えずに、注文のネットワークの変化や品目の在庫状態を監視することができます。</span><span class="sxs-lookup"><span data-stu-id="20287-113">This enables you to monitor the changing order network and item availability without disturbing the static plan that others are using for their work processes.</span></span>

<span data-ttu-id="20287-114">会社で動的マスター プランのみを使用する場合もあれば、静的マスター プランと動的マスター プランの両方を使用する場合もあります。</span><span class="sxs-lookup"><span data-stu-id="20287-114">A company may choose to work with just a dynamic plan or it may use both static and dynamic plans.</span></span> <span data-ttu-id="20287-115">また、特定の戦略の反映や特定の問題への対処を目的として任意のマスタ プランをコンフィギュレーションすることもできます。</span><span class="sxs-lookup"><span data-stu-id="20287-115">In addition, you can configure any master plan to reflect a particular strategy or address an issue.</span></span> <span data-ttu-id="20287-116">例は次のとおりです。</span><span class="sxs-lookup"><span data-stu-id="20287-116">Examples are as follows:</span></span>
-   <span data-ttu-id="20287-117">欠品を防ぐために在庫レベルの設定を高くする。</span><span class="sxs-lookup"><span data-stu-id="20287-117">Set higher inventory levels to guarantee against stockouts.</span></span>
-   <span data-ttu-id="20287-118">信頼できない仕入先への対策として安全マージンを長めに設定する。</span><span class="sxs-lookup"><span data-stu-id="20287-118">Set longer safety margins to protect against unreliable vendors.</span></span>

<span data-ttu-id="20287-119">このほか、マスター プランを実行するたびに新しい必要量プランを使用して更新されるように、初期動的マスター プランを設定することもできます。</span><span class="sxs-lookup"><span data-stu-id="20287-119">You can also set up the starting dynamic plan so that it is updated with the new requirements plan every time that you run master planning.</span></span> <span data-ttu-id="20287-120">**マスター プラン パラメータ**ページでこれらの設定を指定できます。</span><span class="sxs-lookup"><span data-stu-id="20287-120">You can specify these settings on the **Master planning parameters** page.</span></span>
