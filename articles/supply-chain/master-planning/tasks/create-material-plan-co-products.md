# <a name="create-a-material-plan-for-co-products"></a>連産品の材料計画の作成

[!include [task guide banner](../../includes/task-guide-banner.md)]

生産計画担当者は、フォーミュラ連産品である品目の材料要求を計画します。 この手順の作成に使用するデモ データの会社は USP2 です。


## <a name="create-requirement-for-a-co-product"></a>連産品要求の作成
1. [既定のダッシュボード] に移動します。
2. [販売注文の処理と照会] をクリックします。
3. [新規] をクリックします。
4. [販売注文] をクリックします。
5. [顧客口座] フィールドに値を入力します。
    * 例: US-001  
6. [OK] をクリックします。
7. [品目番号] フィールドに値を入力します。
    * 例: P6003  
8. [数量] フィールドに数値を入力します。
    * 例: 50000  
9. [保存] をクリックします。

## <a name="create-a-material-plan-for-co-products"></a>連産品の材料計画の作成
1. ページを閉じます。
2. ページを閉じます。
3. [マスター プラン] をクリックします。
4. [計画] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
5. 一覧で、選択された行のリンクをクリックします。
    * 例: MasterPlan  
6. [実行] をクリックします。
7. [対象に含めるレコード] セクションを展開または折りたたみます。
8. [フィルター] をクリックします。
9. リストで、フィールド = 品目番号の行を選択します。
10. [基準] フィールドに値を入力します。
    * 例: P6003  
11. [OK] をクリックします。
12. [OK] をクリックします。
13. [計画オーダー] をクリックします。
14. クイック フィルターを使用して、レコードを見つけます。 たとえば、「P6000」という値を含む [品目番号] フィールドをフィルター処理します。
    * 販売注文を作成した品目の連産品としてのフォーミュラ品目でフィルター抽出します。  
15. 一覧で、選択された行をマークします。
    * フィルターによって返された行を選択します。  
16. 一覧で、選択された行のリンクをクリックします。
17. [ペギング] セクションを展開または折りたたみます。
18. 一覧で、選択された行のリンクをクリックします。
    * 計画オーダーは、連産品の販売注文にペギングされます。  
19. ページを閉じます。
