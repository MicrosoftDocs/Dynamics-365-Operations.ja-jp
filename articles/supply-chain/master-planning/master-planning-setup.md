<span data-ttu-id="f791d-112">静的プランは、供給と需要の変化にかかわらず、マスター プランが次回実行されるまで変更されません。</span><span class="sxs-lookup"><span data-stu-id="f791d-112">The static plan remains unchanged, regardless of any changes in the supply and demand, until the next time that master planning is run.</span></span>

<span data-ttu-id="f791d-118">動的プランは、通常は静的プランのコピーとして開始されますが、マスター データが変更されるたびに更新できます。</span><span class="sxs-lookup"><span data-stu-id="f791d-118">The dynamic plan usually starts as a copy of the static plan, but it can be updated every time that master data changes.</span></span> <span data-ttu-id="f791d-119">たとえば、データが変更されると、新しい販売注文が作成されます。</span><span class="sxs-lookup"><span data-stu-id="f791d-119">For example, if the data changes, a new sales order is created.</span></span>

<span data-ttu-id="f791d-124">動的プランでは、差分変更が使用されます。</span><span class="sxs-lookup"><span data-stu-id="f791d-124">The dynamic plan uses net change.</span></span> <span data-ttu-id="f791d-125">したがって、実行時間を短縮できます。</span><span class="sxs-lookup"><span data-stu-id="f791d-125">Therefore, it helps guarantee a faster running time.</span></span>
