# <a name="indexed-sales-prices"></a><span data-ttu-id="2ea7a-103">指数化された販売価格</span><span class="sxs-lookup"><span data-stu-id="2ea7a-103">Indexed sales prices</span></span>  

[!include [banner](../includes/banner.md)]


<span data-ttu-id="2ea7a-104">定期売買手数料を作成する場合は、定期売買の販売価格の指数を設定します。</span><span class="sxs-lookup"><span data-stu-id="2ea7a-104">You set up the index for a subscription sales price when you create a subscription fee.</span></span>

<span data-ttu-id="2ea7a-105">**定期売買手数料の作成**フォームで、**価格の取得元**フィールドを**指標化された基準価格**に設定し、基準価格に**価格変更率**フィールドの割合を乗算して、定期売買トランザクションの販売価格を取得します。</span><span class="sxs-lookup"><span data-stu-id="2ea7a-105">In the **Create subscription fee** form, set the **Get pricing from** field to **Indexed base price**, and then multiply the base price by the percentage in the **Percent price change** field to get the sales price of the subscription transaction.</span></span>

<span data-ttu-id="2ea7a-106">たとえば、基準価格が 1,000 EUR で、指数が 110 の場合、販売価格は 1,100 EUR になります。</span><span class="sxs-lookup"><span data-stu-id="2ea7a-106">For example, if the base price is EUR 1,000, and the index is 110, the sales price is EUR 1,100.</span></span>
