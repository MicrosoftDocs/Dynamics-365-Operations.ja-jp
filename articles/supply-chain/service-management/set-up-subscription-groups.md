# <a name="set-up-subscription-groups"></a>定期売買グループを設定します 

[!include [banner](../includes/banner.md)]


1.  **サービス管理** \> **設定** \> **サービスの定期売買** \> **定期売買グループ** に移動します。

2.  **新規作成** を選択して、新しい定期売買グループを作成します。

3.  **定期売買グループ** フィールドに、新しい定期売買グループの名前を入力します。

4.  **請求** フィールドで、定期売買の請求頻度を選択します。 請求頻度は、**期間タイプ** フォームで設定する期間コードによって示されます。

5.  定期売買トランザクションの収益を見越計上するには、**未収収益** チェックボックスをオンにし、**見越計上** フィールドで、頻度を選択します。 見越計上の頻度は **期間** フォームで設定する期間コードに基づいています。
