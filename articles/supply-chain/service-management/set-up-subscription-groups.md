[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="5f3d6-104">**サービス管理** \> **設定** \> **サービスの定期売買** \> **定期売買グループ**の順にクリックします。</span><span class="sxs-lookup"><span data-stu-id="5f3d6-104">Click **Service management** \> **Setup** \> **Service subscriptions** \> **Subscription groups**.</span></span>

5.  <span data-ttu-id="5f3d6-109">定期売買トランザクションの収益を見越計上するには、**未収収益**チェックボックスをオンにし、**見越計上**フィールドで、頻度を選択します。</span><span class="sxs-lookup"><span data-stu-id="5f3d6-109">To accrue revenue for subscription transactions, select the **Accrue revenue** check box, and then in the **Accrual** field, select a frequency.</span></span> <span data-ttu-id="5f3d6-110">見越計上の頻度は**期間**フォームで設定する期間コードに基づいています。</span><span class="sxs-lookup"><span data-stu-id="5f3d6-110">The accrual frequency is based on a period code that is set up in the **Periods** form.</span></span>
