# <a name="set-up-subscription-groups"></a><span data-ttu-id="1a326-103">定期売買グループを設定します</span><span class="sxs-lookup"><span data-stu-id="1a326-103">Set up subscription groups</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="1a326-104">**サービス管理** \> **設定** \> **サービスの定期売買** \> **定期売買グループ** の順にクリックします。</span><span class="sxs-lookup"><span data-stu-id="1a326-104">Click **Service management** \> **Setup** \> **Service subscriptions** \> **Subscription groups**.</span></span>

2.  <span data-ttu-id="1a326-105">Ctrl + N キーを押すか、**新規** をクリックして、新しい定期売買グループを作成します。</span><span class="sxs-lookup"><span data-stu-id="1a326-105">Press Ctrl+N or click **New** to create a new subscription group.</span></span>

3.  <span data-ttu-id="1a326-106">**定期売買グループ** フィールドに、新しい定期売買グループの名前を入力します。</span><span class="sxs-lookup"><span data-stu-id="1a326-106">In the **Subscription group** field, enter a name for the new subscription group.</span></span>

4.  <span data-ttu-id="1a326-107">**請求** フィールドで、定期売買の請求頻度を選択します。</span><span class="sxs-lookup"><span data-stu-id="1a326-107">In the **Invoicing** field, select an invoicing frequency for the subscription.</span></span> <span data-ttu-id="1a326-108">請求頻度は、**期間タイプ** フォームで設定する期間コードによって示されます。</span><span class="sxs-lookup"><span data-stu-id="1a326-108">The invoicing frequency is indicated by a period code that is set up in the **Period types** form.</span></span>

5.  <span data-ttu-id="1a326-109">定期売買トランザクションの収益を見越計上するには、**未収収益** チェックボックスをオンにし、**見越計上** フィールドで、頻度を選択します。</span><span class="sxs-lookup"><span data-stu-id="1a326-109">To accrue revenue for subscription transactions, select the **Accrue revenue** check box, and then in the **Accrual** field, select a frequency.</span></span> <span data-ttu-id="1a326-110">見越計上の頻度は **期間** フォームで設定する期間コードに基づいています。</span><span class="sxs-lookup"><span data-stu-id="1a326-110">The accrual frequency is based on a period code that is set up in the **Periods** form.</span></span>
