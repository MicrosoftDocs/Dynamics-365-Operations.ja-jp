# <a name="move-service-order-lines"></a><span data-ttu-id="2e3f9-103">サービス注文明細行の移動</span><span class="sxs-lookup"><span data-stu-id="2e3f9-103">Move service order lines</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="2e3f9-104">**サービス管理** \> **共通** \> **サービス注文** \> **サービス注文** の順にクリックします。</span><span class="sxs-lookup"><span data-stu-id="2e3f9-104">Click **Service management** \> **Common** \> **Service orders** \> **Service orders**.</span></span>

2.  <span data-ttu-id="2e3f9-105">別のサービス注文に移動する明細行が含まれているサービス注文をダブルクリックします。</span><span class="sxs-lookup"><span data-stu-id="2e3f9-105">Double-click the service order that contains the lines that you want to move to another service order.</span></span>

3.  <span data-ttu-id="2e3f9-106">アクション ウィンドウの、**サービス注文** フォームで、**行の移動** ボタンをクリックします。</span><span class="sxs-lookup"><span data-stu-id="2e3f9-106">In the **Service orders** form, on the Action Pane, click the **Move lines** button.</span></span>

4.  <span data-ttu-id="2e3f9-107">**サービス注文** フィールドの、**サービス注文明細行の移動** フォームで、行の移動先となるサービス注文を選択します。</span><span class="sxs-lookup"><span data-stu-id="2e3f9-107">In the **Move service order lines** form, in the **Service order** field, select the service order that you want to move the lines to.</span></span>

5.  <span data-ttu-id="2e3f9-108">サービス注文明細行の移動元のサービス注文を削除するには、**空のサービス注文を削除** チェック ボックスをオンにします。</span><span class="sxs-lookup"><span data-stu-id="2e3f9-108">Select the **Delete empty service order** check box to delete the service order that you moved the service order lines from.</span></span>

6.  <span data-ttu-id="2e3f9-109">**OK** をクリックします。</span><span class="sxs-lookup"><span data-stu-id="2e3f9-109">Click **OK**.</span></span>


> [!NOTE]
> <P><span data-ttu-id="2e3f9-110">サービス注文明細行は、同じサービス契約に関連付けられているサービス注文間でのみ移動することができます。</span><span class="sxs-lookup"><span data-stu-id="2e3f9-110">You can move service order lines only between service orders that are attached to the same service agreement.</span></span></P>
