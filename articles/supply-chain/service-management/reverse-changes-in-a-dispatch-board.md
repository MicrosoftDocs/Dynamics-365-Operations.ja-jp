<span data-ttu-id="1656b-104">このトピックでは、派遣表で行った未保存の変更を取り消す方法について説明します。</span><span class="sxs-lookup"><span data-stu-id="1656b-104">This topic describes how to reverse unsaved modifications that you make in a dispatch board.</span></span> <span data-ttu-id="1656b-105">たとえば、作業者をサービス活動に割り当てて情報を保存した後、別のサービス活動に作業者を割り当てることができます。</span><span class="sxs-lookup"><span data-stu-id="1656b-105">For example, you assign a worker to a service activity, save the information, and then later decide to assign a different worker to the service activity.</span></span> <span data-ttu-id="1656b-106">派遣表の作業者を変更してから、変更を保存する前に、割り当てられた作業者がないことを知ることができます。</span><span class="sxs-lookup"><span data-stu-id="1656b-106">You modify the worker in the dispatch board, and then, before saving the change, learn that the worker just assigned is not available.</span></span> <span data-ttu-id="1656b-107">元の作業者をサービス注文に再割り当てされるように未保存の変更を取り消すことができます。</span><span class="sxs-lookup"><span data-stu-id="1656b-107">You can reverse the unsaved modification so that the original worker is reassigned to the service order.</span></span>

<span data-ttu-id="1656b-108">派遣表の未保存の変更を取り消すには、次の手順を使用します。</span><span class="sxs-lookup"><span data-stu-id="1656b-108">Use the following steps to reverse unsaved changes in a dispatch board:</span></span>
