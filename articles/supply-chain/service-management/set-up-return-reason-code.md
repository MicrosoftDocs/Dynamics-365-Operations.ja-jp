## <a name="set-up-return-reason-code-groups"></a><span data-ttu-id="c2441-104">返品理由コード グループの設定</span><span class="sxs-lookup"><span data-stu-id="c2441-104">Set up return reason code groups</span></span> 

1.  <span data-ttu-id="c2441-105">**販売とマーケティング** \> **設定** \> **販売注文** \> **返品** \> **返品理由コード グループ** に移動します。</span><span class="sxs-lookup"><span data-stu-id="c2441-105">Go to **Sales and marketing** \> **Setup** \> **Sales orders** \> **Returns** \> **Return reason code groups**.</span></span>

## <a name="set-up-return-reason-codes"></a><span data-ttu-id="c2441-109">返品理由コードの設定</span><span class="sxs-lookup"><span data-stu-id="c2441-109">Set up return reason codes</span></span> 

<span data-ttu-id="c2441-110">この手順では、顧客が製品を返品した理由を示すために使用できる、返品理由コードを設定できます。</span><span class="sxs-lookup"><span data-stu-id="c2441-110">This procedure helps you set up return reason codes that you can use to indicate why a product was returned by the customer.</span></span>

1.  <span data-ttu-id="c2441-111">**販売とマーケティング** \> **設定** \> **販売注文** \> **返品** \> **返品理由コード** に移動します。</span><span class="sxs-lookup"><span data-stu-id="c2441-111">Go to **Sales and marketing** \> **Setup** \> **Sales orders** \> **Returns** \> **Return reason codes**.</span></span>

2.  <span data-ttu-id="c2441-112">Ctrl + N キーを押して、新しい返品理由コード行を作成します。</span><span class="sxs-lookup"><span data-stu-id="c2441-112">Press CTRL + N to create a new return reason code line.</span></span>

3.  <span data-ttu-id="c2441-113">**返品理由コード** フィールドに、このコードを識別するテキストを入力します。</span><span class="sxs-lookup"><span data-stu-id="c2441-113">In the **Return reason code** field, enter text to identify this code.</span></span>

4.  <span data-ttu-id="c2441-114">**説明** フィールドに、返品理由コードを説明するテキストを入力します。</span><span class="sxs-lookup"><span data-stu-id="c2441-114">In the **Description** field, enter text to describe the return reason code.</span></span>

5.  <span data-ttu-id="c2441-115">**返品理由コード グループ** ドロップダウン リストを使用して、コードを返品理由コード グループに追加します。</span><span class="sxs-lookup"><span data-stu-id="c2441-115">Use the **Return reason code group** drop-down list to add the code to a return reason code group.</span></span>
