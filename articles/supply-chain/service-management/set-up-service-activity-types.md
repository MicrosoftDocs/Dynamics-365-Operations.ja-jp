# <a name="set-up-service-activity-types"></a>サービス活動の種類の設定 

[!include [banner](../includes/banner.md)]


1.  **販売およびマーケティング** \> **設定** \> **活動** \> **活動タイプ** の順にクリックします。
    
    - または -
    
    **組織管理** \> **設定** \> **活動** \> **活動タイプ** の順にクリックします。

2.  **活動タイプ** フォームで、Ctrl + N キーを押して新しい活動タイプ明細行を追加します。

3.  **タイプ** フィールドに、活動タイプの名前を入力します。

4.  **説明** フィールドに、活動タイプの説明を入力します。

5.  フォームを閉じて、変更を保存します。
