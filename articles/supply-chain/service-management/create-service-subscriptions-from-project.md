# <a name="create-service-subscriptions-from-a-project"></a>プロジェクトからサービスの定期売買を作成する方法    

[!include [banner](../includes/banner.md)]


**プロジェクト管理と会計** のプロジェクトからサービス定期売買を作成することができます。

1.  **プロジェクト管理および会計** \> **共通** \> **プロジェクト** \> **すべてのプロジェクト** の順にクリックします。 定期売買を作成するプロジェクトを開きます。

2.  **管理** タブの、**アクション ウィンドウ** における、**プロジェクト** フォームで、**サービス** \> **サービスの定期売買** の順にクリックします。

3.  **サービスの定期売買** フィールドの、**定期売買** フォームで、固有 ID 番号を入力します。 プロジェクト番号が選択したプロジェクトの定期売買に自動的に入力されます。

4.  フォームに残りの情報を入力します。

これで、定期売買の定期売買トランザクションを作成できます。


> [!NOTE]
> <P>定期売買グループの番号順序を設定している場合は、定期売買グループを指定すると、定期売買の定期売買 ID が自動的に作成されます。</P>
