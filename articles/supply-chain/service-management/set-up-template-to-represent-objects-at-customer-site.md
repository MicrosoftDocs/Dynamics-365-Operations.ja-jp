# <a name="set-up-template-to-represent-objects-at-customer-site"></a><span data-ttu-id="d6198-103">顧客のサイトのオブジェクトを表すためのテンプレートの設定</span><span class="sxs-lookup"><span data-stu-id="d6198-103">Set up template to represent objects at customer site</span></span> 

[!include [banner](../includes/banner.md)]


<span data-ttu-id="d6198-104">顧客サイトのオブジェクトを表すテンプレートを設定するには、この手順を使用します。</span><span class="sxs-lookup"><span data-stu-id="d6198-104">Use this procedure to set up a template that represents objects at a customer site.</span></span>

1.  <span data-ttu-id="d6198-105">**サービス管理** \> **共通** \> **サービス契約** \> **サービス契約**の順にクリックします。</span><span class="sxs-lookup"><span data-stu-id="d6198-105">Click **Service management** \> **Common** \> **Service agreements** \> **Service agreements**.</span></span>

2.  <span data-ttu-id="d6198-106">サービス契約をダブルクリックするか、**サービス契約**をクリックして新しいサービス契約を作成します。</span><span class="sxs-lookup"><span data-stu-id="d6198-106">Double-click a service agreement, or click **Service agreement** to create a new service agreement.</span></span>

3.  <span data-ttu-id="d6198-107">**設定** タブをクリックします。</span><span class="sxs-lookup"><span data-stu-id="d6198-107">Click the **Setup** tab.</span></span>

4.  <span data-ttu-id="d6198-108">**サービス対象**をクリックし、テンプレートの部品表 (BOM) をサービス契約に関連付けます。</span><span class="sxs-lookup"><span data-stu-id="d6198-108">Click **Service objects** to attach a template bill of materials (BOM) to the service agreement.</span></span>

5.  <span data-ttu-id="d6198-109">**サービス対象**フォームで、サービス対象を選択し、**機能** \> **テンプレート BOM の添付**をクリックします。</span><span class="sxs-lookup"><span data-stu-id="d6198-109">In the **Service objects** form, select a service object, and then click **Functions** \> **Attach template BOM**.</span></span>

6.  <span data-ttu-id="d6198-110">**テンプレート BOM の選択**フォームの、**テンプレート BOM**フィールドでテンプレート BOM を選択し、**OK** をクリックします。</span><span class="sxs-lookup"><span data-stu-id="d6198-110">In the **Select template BOM** form, select a template BOM in the **Template BOM** field, and then click **OK**.</span></span>

7.  <span data-ttu-id="d6198-111">**サービス対象**フォームで**デザイナ**をクリックして、BOM が目的の対象を表すようにテンプレート BOM の明細行を変更します。</span><span class="sxs-lookup"><span data-stu-id="d6198-111">In the **Service objects** form, click **Designer** to modify the template BOM so that the BOM represents the objects that you want.</span></span>

8.  <span data-ttu-id="d6198-112">フォームを閉じて、変更を保存します。</span><span class="sxs-lookup"><span data-stu-id="d6198-112">Close the forms to save your changes.</span></span>
