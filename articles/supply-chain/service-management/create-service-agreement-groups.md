# <a name="create-service-agreement-groups"></a><span data-ttu-id="a5988-103">サービス契約グループの作成</span><span class="sxs-lookup"><span data-stu-id="a5988-103">Create service agreement groups</span></span> 

[!include [banner](../includes/banner.md)]

<span data-ttu-id="a5988-104">サービス契約グループは、サービス契約の並べ替えやフィルタ処理に便利です。</span><span class="sxs-lookup"><span data-stu-id="a5988-104">Service agreement groups are useful for sorting and filtering service agreements.</span></span>

1. <span data-ttu-id="a5988-105">**サービス管理** \> **設定** \> **サービス契約** \> **サービス契約グループ** の順にクリックします。</span><span class="sxs-lookup"><span data-stu-id="a5988-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Service agreement groups**.</span></span>
2. <span data-ttu-id="a5988-106">**Alt + N** キーを押して、新しいサービス契約グループを作成します。</span><span class="sxs-lookup"><span data-stu-id="a5988-106">Press **Alt+N** to create a new service agreement group.</span></span>
3. <span data-ttu-id="a5988-107">**サービス契約グループ** フィールドで、サービス契約グループの ID を入力します。</span><span class="sxs-lookup"><span data-stu-id="a5988-107">In the **Service agreement group** field, enter an ID for the service agreement group.</span></span>
4. <span data-ttu-id="a5988-108">**説明** フィールドに、サービス契約グループの説明を入力します。</span><span class="sxs-lookup"><span data-stu-id="a5988-108">In the **Description** field, enter a description for the service agreement group.</span></span>
5. <span data-ttu-id="a5988-109">ページを閉じて、変更を保存します。</span><span class="sxs-lookup"><span data-stu-id="a5988-109">Close the page to save your changes.</span></span>

> [!NOTE]
> <span data-ttu-id="a5988-110">**サービス管理パラメーター** ページの **サービス契約グループ** フィールドでサービス契約グループを定義した場合、新しいサービス契約を作成するときは常に、このグループが既定として表示されます。</span><span class="sxs-lookup"><span data-stu-id="a5988-110">If a service agreement group is defined in the **Service agreement group** field of the **Service management parameters** page, the program suggests this group for all new service agreements that you create.</span></span>
