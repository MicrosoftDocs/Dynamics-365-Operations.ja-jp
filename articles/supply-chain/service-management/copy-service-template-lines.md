# <a name="copy-service-templates-lines"></a><span data-ttu-id="8e6b5-103">サービス テンプレート行のコピー</span><span class="sxs-lookup"><span data-stu-id="8e6b5-103">Copy service templates lines</span></span> 

[!include [banner](../includes/banner.md)]

<span data-ttu-id="8e6b5-104">テンプレート情報は、サービス合意またはサービス注文にコピーできます。</span><span class="sxs-lookup"><span data-stu-id="8e6b5-104">Template information can be copied to either a service agreement or a service order.</span></span>

## <a name="copy-service-template-lines-into-a-service-agreement"></a><span data-ttu-id="8e6b5-105">サービス テンプレート行をサービス合意へコピー</span><span class="sxs-lookup"><span data-stu-id="8e6b5-105">Copy service template lines into a service agreement</span></span>

1. <span data-ttu-id="8e6b5-106">**サービス合意**をクリックします。</span><span class="sxs-lookup"><span data-stu-id="8e6b5-106">Click **Service agreements**.</span></span>
2. <span data-ttu-id="8e6b5-107">**サービス合意** ページを開くには、サービス合意をダブルクリックします。</span><span class="sxs-lookup"><span data-stu-id="8e6b5-107">Double-click a service agreement to open the **Service agreements** page.</span></span>
3. <span data-ttu-id="8e6b5-108">**テンプレートの行** をクリックして **サービス テンプレート行のコピー** ページを開きます。</span><span class="sxs-lookup"><span data-stu-id="8e6b5-108">Click **Template lines** to open the **Copy service template lines** page.</span></span>
4. <span data-ttu-id="8e6b5-109">**ツリー コントロール** タブで、ツリーを展開し、サービス合意にコピーする行を含むテンプレートを探して、サービス テンプレートを選択します。</span><span class="sxs-lookup"><span data-stu-id="8e6b5-109">On the **Tree control** tab, expand the tree, find the template that contains the lines that you want to copy to the service agreement, and then select the service template.</span></span>
5. <span data-ttu-id="8e6b5-110">**明細行** タブで、コピーするサービス テンプレート行の **Select** チェック ボックスをオンにします。</span><span class="sxs-lookup"><span data-stu-id="8e6b5-110">On the **Lines** tab, select the **Select** check box for the service template lines that you want to copy.</span></span>
6. <span data-ttu-id="8e6b5-111">**OK** をクリックします。</span><span class="sxs-lookup"><span data-stu-id="8e6b5-111">Click **OK**.</span></span>
7. <span data-ttu-id="8e6b5-112">ページを閉じて、変更を保存します。</span><span class="sxs-lookup"><span data-stu-id="8e6b5-112">Close the pages to save your changes.</span></span>

<span data-ttu-id="8e6b5-113">サービス テンプレート行が、関連付けられているサービス対象とサービス作業の関係と共に、サービス合意にコピーされます。</span><span class="sxs-lookup"><span data-stu-id="8e6b5-113">The service template lines are now copied to the service agreement, together with the attached service object and service task relations.</span></span>
