# <a name="create-subscription-fee-transactions"></a><span data-ttu-id="01e1f-103">定期売買手数料トランザクションの作成</span><span class="sxs-lookup"><span data-stu-id="01e1f-103">Create subscription fee transactions</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="01e1f-104">**サービス管理** \> **共通** \> **サービスの定期売買** \> **すべてのサービス定期売買** の順にクリックします。</span><span class="sxs-lookup"><span data-stu-id="01e1f-104">Click **Service management** \> **Common** \> **Service subscriptions** \> **All service subscriptions**.</span></span>

2.  <span data-ttu-id="01e1f-105">**すべてのサービス定期売買** リスト ページで、定期売買手数料トランザクションを作成する定期売買を選択します。</span><span class="sxs-lookup"><span data-stu-id="01e1f-105">On the **All service subscriptions** list page, select the subscription for which you want to create a subscription fee transaction.</span></span>

3.  <span data-ttu-id="01e1f-106">**アクション ウィンドウ** の、**定期売買** タブで、**定期売買手数料** をクリックします。</span><span class="sxs-lookup"><span data-stu-id="01e1f-106">On the **Action Pane**, on the **Subscription** tab, click **Subscription fees**.</span></span>

4.  <span data-ttu-id="01e1f-107">**定期売買手数料の作成** フォームで、フィールドに適切な情報を入力します。</span><span class="sxs-lookup"><span data-stu-id="01e1f-107">On the **Create subscription fee** form, enter the appropriate information in the fields.</span></span>

5.  <span data-ttu-id="01e1f-108">**OK** をクリックします。</span><span class="sxs-lookup"><span data-stu-id="01e1f-108">Click **OK**.</span></span>
