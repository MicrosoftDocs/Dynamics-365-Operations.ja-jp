# <a name="allocate-time-to-jobs-in-a-job-bundle"></a><span data-ttu-id="187c1-104">ジョブ バンドル内のジョブへの時間割り当て</span><span class="sxs-lookup"><span data-stu-id="187c1-104">Allocate time to jobs in a job bundle</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="187c1-105">[製造実行] で、ジョブをバンドルできます。</span><span class="sxs-lookup"><span data-stu-id="187c1-105">In Manufacturing execution, you can bundle jobs.</span></span> <span data-ttu-id="187c1-106">これにより、[ジョブ リスト] ページで複数のジョブを同時に開始できます。</span><span class="sxs-lookup"><span data-stu-id="187c1-106">You can then start multiple jobs at the same time on the Job list page.</span></span>

<span data-ttu-id="187c1-107">ジョブをバンドルする場合は、すべてのジョブに登録された合計時間を各ジョブに割り当てる方法を定義する必要があります。</span><span class="sxs-lookup"><span data-stu-id="187c1-107">If you bundle jobs, you must define how the total registered time for all the jobs should be allocated to each job.</span></span> <span data-ttu-id="187c1-108">**配賦キー** ページの **バンドル タイプ** フィールドで、次のいずれかのオプションを選択して割り当てを定義します。</span><span class="sxs-lookup"><span data-stu-id="187c1-108">You define the allocation by selecting one of the following options in the **Bundle type** field on the **Allocation keys** page:</span></span>

-   <span data-ttu-id="187c1-109">**見積** – 時間が、ジョブの見積時間に基づいて、ジョブ間で分割されます。</span><span class="sxs-lookup"><span data-stu-id="187c1-109">**Estimation** – Time is divided among the jobs, based on the estimated time for the jobs.</span></span>
-   <span data-ttu-id="187c1-110">**ジョブ** – 時間が、バンドルされたジョブの合計数と、すべてのジョブを完了するためにかかった時間に従って分割されます。</span><span class="sxs-lookup"><span data-stu-id="187c1-110">**Jobs** – Time is divided according to total jobs that are bundled and how much time was spent finishing all the jobs.</span></span>
-   <span data-ttu-id="187c1-111">**正味時間** – 時間が、バンドル内のジョブ間で常に均等に分割されます。</span><span class="sxs-lookup"><span data-stu-id="187c1-111">**Net time** – Time is divided equally among the jobs that are in the bundle at any time.</span></span>
-   <span data-ttu-id="187c1-112">**リアル タイム** – 実際のジョブ時間が割り当てられます。</span><span class="sxs-lookup"><span data-stu-id="187c1-112">**Real time** – Actual job time is allocated.</span></span> <span data-ttu-id="187c1-113">原価は実際の人件費に基づいて計算されます。</span><span class="sxs-lookup"><span data-stu-id="187c1-113">The cost can be calculated based on the actual payroll cost.</span></span> <span data-ttu-id="187c1-114">**注記:** 会社が [時刻と出勤] で給与機能を使用する場合のみ、**リアル タイム** 配賦キーを使用できます。</span><span class="sxs-lookup"><span data-stu-id="187c1-114">**Note:** The **Real time** allocation key is available only if your company uses the payroll functionality in Time and attendance.</span></span>
