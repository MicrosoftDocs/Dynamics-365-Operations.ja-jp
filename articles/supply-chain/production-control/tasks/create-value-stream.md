# <a name="create-a-value-stream"></a>バリュー ストリームの作成

[!include [task guide banner](../../includes/task-guide-banner.md)]

バリュー ストリームは、作業単位であり、そのグループが 1 つまたは複数の生産フローです。 生産フローを定義するには、まずバリュー ストリームを定義する必要があります。


## <a name="open-the-form"></a>フォームの表示
1. [生産管理] > [設定] > [リーン生産フロー] > [バリュー ストリーム] に移動します。

## <a name="create-a-new-value-stream"></a>新しいバリュー ストリームの作成
1. [新規] をクリックすると、ドロップ ダイアログが開きます。
2. [名前] フィールドに値を入力します。

## <a name="select-a-value-stream-manager"></a>バリュー ストリーム マネージャーの選択
1. [マネージャー] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
    * バリュー ストリームのマネージャーは組織をドキュメント化するために定義されます。  
2. 一覧で、目的のレコードを見つけ、選択します。
3. 一覧で、選択された行のリンクをクリックします。

## <a name="save-and-close"></a>保存して閉じる
1. [保存] をクリックします。
2. ページを閉じます。
3. ページを更新します。
