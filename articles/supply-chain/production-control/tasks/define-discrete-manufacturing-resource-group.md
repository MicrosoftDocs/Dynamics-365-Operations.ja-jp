# <a name="define-discrete-manufacturing-resource-group"></a>個別の製造リソース グループの定義

[!include [banner](../../includes/banner.md)]

リソース グループは、通常、作業セルの物理的な構成に対応する、作業現場フロアで黄色のラインで示される一連の運営リソースです。 この手順では、個別生産で使用するリソース グループを定義する方法を示します。 デモ データ会社 USMF または独自のデータを使用して、この手順の説明を確認することができます。

1. [リソース グループ] に移動します。
2. [新規] をクリックします。
3. [リソース グループ] フィールドに値を入力します。
4. [説明] フィールドに値を入力します。
5. [サイト] フィールドで値を入力または選択します。
6. [生産単位] フィールドで値を入力または選択します。

## <a name="define-default-operational-parameters"></a>既定の工程パラメーターの定義
1. [操作] セクションを展開します。
2. [仕損率] フィールドに数値を入力します。
3. [段取りカテゴリ] フィールドで、値を入力または選択します。
4. [実行時間カテゴリ] フィールドで、値を入力または選択します。
5. [工程のスケジューリング率] フィールドに数値を入力します。

## <a name="define-operating-hours"></a>工程時間の定義
1. [カレンダー] セクションを展開します。
2. [追加] をクリックします。
3. [カレンダー] フィールドで値を入力または選択します。

## <a name="add-operations-resources"></a>運営リソースの追加
1. [リソース] セクションを展開します。
2. [追加] をクリックします。
3. [リソース] フィールドで値を入力または選択します。
4. [追加] をクリックします。
5. [リソース] フィールドで値を入力または選択します。
6. 一覧で、目的のレコードを見つけ、選択します。
7. 一覧で、選択された行のリンクをクリックします。
