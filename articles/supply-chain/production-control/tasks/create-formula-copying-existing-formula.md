# <a name="create-a-formula-by-copying-from-an-existing-formula"></a>既存のフォーミュラをコピーしてフォーミュラを作成

[!include [banner](../../includes/banner.md)]

この手順は、既存のフォーミュラからコピーして新しいフォーミュラをすばやく作成する方法を示します。 この手順を作成するのに、デモ データの会社 USMF、USP2、または USPI を使用できます。


## <a name="create-a-new-formula"></a>新しいフォーミュラのを作成
1. [フォーミュラ] に移動する。
2. [新規] をクリックします。
3. [フォーミュラ] フィールドに値を入力します。
4. [名前] フィールドに値を入力します。
5. [サイト] フィールドで値を入力または選択します。
6. [品目グループ] フィールドで、値を入力または選択します。
7. [保存] をクリックします。

## <a name="copy-from-an-existing-formula"></a>既存のフォーミュラからコピー
1. [コピー] をクリックします。
2. [品目番号] フィールドで、値を入力または選択します。
3. [フォーミュラ バージョン] フィールドで、値を入力または選択します。
4. [OK] をクリックします。

## <a name="revise-the-copied-formula"></a>コピーしたフォーミュラを改訂
1. 一覧で、目的のレコードを見つけ、選択します。
2. [削除] をクリックします。
3. [はい] をクリックします。

## <a name="approve-formula"></a>フォーミュラを承認
1. [フォーミュラを承認] をクリックします。
2. [承認者] フィールドで、値を入力または選択します。
3. [OK] をクリックします。
4. ページを閉じます。
