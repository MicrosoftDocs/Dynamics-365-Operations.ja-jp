# <a name="create-a-subcontracted-work-cell-for-lean-manufacturing"></a>リーン生産の外注作業セルの作成

[!include [banner](../../includes/banner.md)]

リーン生産の外注作業をモデル化するには、作業を提供する仕入先に関連付けられている作業セルを作成する必要があります。 外注作業セルは、仕入先タイプのリソースの関連付けを通じて仕入先にリンクされます。 USMF デモ会社でこの記録を行う場合、仕入先 ID 1002 およびサイト 1 を選択できます。


## <a name="create-a-vendor-resource"></a>仕入先リソースの作成
1. [リソース] に移動します。
2. [新規] をクリックします。
3. [リソース] フィールドに値を入力します。
4. [説明] フィールドに値を入力します。
5. [タイプ] フィールドで [仕入先] を選択します。
6. [仕入先] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。

## <a name="create-the-resource-group"></a>リソース グループを作成する
1. [リソース グループ] に移動します。
2. [新規] をクリックします。
3. [リソース グループ] フィールドに値を入力します。
4. [説明] フィールドに値を入力します。
5. [サイト] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
    * 作業セルが割り当てられるべきサイトを選択します。 理論上、サイトは仕入先が運用している 1 つのサイトを表すことができます。 ただし、ほとんどの場合、外注されたリソースは外注業務を発注するサイトに割り当てられます。 外注業務セルの入荷および出荷の倉庫が同じサイトである必要があることに注意してください。  
6. [サイト] フィールドに値を入力します。
7. @SysTaskRecorder:_RequestClose
8. [作業セル] チェック ボックスをオンまたはオフにします。
9. [入庫倉庫] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
    * 仕入先管理作業セルの材料を示すために使用される倉庫および場所を選択します。 多くの場合、仕入先ごとに 1 つの別の倉庫および作業のセルごとに 1 つの場所を使用して倉庫や場所がモデル化されます。  
10. [入庫場所] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
11. [出荷倉庫] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
    * 作業セルの外注活動が転記されるとき、材料が転記される倉庫および場所を定義します。 倉庫および場所は、仕入先がかんばん作業を報告する場合、仕入先サイトとなります。 または、倉庫および場所は、生産フローの次の手順に関連付けられている受入場所にもなります。  
12. [出荷場所] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
13. [カレンダー] セクションを展開または折りたたみます。
14. [追加] をクリックします。
15. [カレンダー] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
    * 作業セルの作業時間カレンダーをリソース グループに関連付けます。 重要なリソースについては、正確な作業時間、および作業セルまたは仕入先サイトに関連する能力を表す特定のカレンダーを定義することをお勧めします。  
16. [リソース] セクションを展開または折りたたみます。
17. [追加] をクリックします。
    * 外注リソース グループには、仕入先にリソース グループをリンクする仕入先タイプの関連付けられたリソースが必要です。  
18. [リソース] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
    * 以前のサブタスクで作成した仕入先リソースを選択または入力します。  
19. [作業セルの能力] セクションを展開、または折りたたみます。
20. [追加] をクリックします。
    * 作業セルには、定義済能力が必要です。 この例では、標準的な作業日あたり 100 個の処理能力を作成します。  
21. [生産フロー モデル] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
22. [能力期間] フィールドで、オプションを選択します。
23. [平均スループット数量] フィールドに数値を入力します。
24. [単位] フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
25. [単位の変更] を解決します。
