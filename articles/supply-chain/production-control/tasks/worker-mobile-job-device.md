# <a name="configure-a-worker-using-the-mobile-job-device"></a>モバイル ジョブ デバイスを使用した作業者のコンフィギュレーション

[!include[task guide banner](../../includes/task-guide-banner.md)]

この手順では、作業者のユーザー アカウントに適切なロールを割り当てる方法を表示します。それによって、作業者が作業現場の登録を行うことをできるようになります。


## <a name="assign-roles-to-user-account"></a>ユーザー アカウントへのロールの割り当て
1. [システム管理] > [ユーザー] > [ユーザー] の順に移動します。
2. ユーザー アカウントが機械オペレータのロールに関連付けられる場合に、[クイック フィルター] を使用して、作業者の名前をフィルターします。 サンプル データでは、名前はシャノンです。
3. ユーザー アカウントのレコードを強調表示します。
4. 一覧から、選択した行で「名前」のリンクをクリックして、ユーザー アカウントの詳細を表示します。
5. ツリーで、「Roles\Machine operator」を選択します。
6. ユーザー アカウントの詳細ページを閉じます。
7. ページを閉じます。

## <a name="configure-worker-account"></a>作業者のアカウントを構成します。
1. [人事管理] > [作業者] > [作業者] の順に移動します。
2. ユーザー アカウントが機械オペレータのロールに関連付けられる場合に、[クイック フィルター] を使用して、作業者の名前をフィルターします。 サンプル データでは、名前はシャノンです。
3. ユーザー アカウントのレコードを強調表示します。
4. 一覧から、選択した行で「名前」のリンクをクリックして、ユーザー アカウントの詳細を表示します。
5. [雇用] タブをクリックします。
6. [時間登録のクイックタブ] を展開し、[登録ターミナルで有効化] をクリックします。
7. [登録ターミナルで有効化] をクリックします。
8. [計算グループ] フィールドで、値を入力または選択します。
9. [既定の計算グループ] フィールドで、値を入力または選択します。
10. [承認グループ] フィールドで、値を入力または選択します。
11. [標準プロファイル] フィールドで、値を入力または選択します。
12. [プロファイル グループ] フィールドで、値を入力または選択します。
13. [OK] をクリックします。
14. [編集] をクリックして、新しい時間登録作業者のバッジ番号を入力します。
15. [バッジ ID] フィールドで、値を入力します。
16. [保存] をクリックします。
17. [ショートカットの記録保存] を使用します。
18. 作業者の詳細ページを閉じます。
19. ページを閉じます。

## <a name="assign-worker-to-device-group"></a>デバイス グループに作業者を割り当てます。
1. [生産管理] > [設定] > [製造実行] > [デバイスのジョブ カードのコンフィギュレーション] の順に移動します。
2. [追加] をクリックします。
3. 一覧で、選択された行をマークします。
4. [OK] をクリックします。
5. [編集] をクリックします。
6. [生産単位] フィールドで、作業者の既定のフィルターを設定できます。 これにより作業者がデバイスにログオンするときに、選択した生産単位の生産ジョブのみが表示されます。
7. ページを閉じます。
