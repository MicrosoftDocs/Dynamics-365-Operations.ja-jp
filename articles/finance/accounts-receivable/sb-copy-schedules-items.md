# <a name="copy-a-billing-schedule"></a>請求スケジュールのコピー

[!include [banner](../includes/banner.md)]

**スケジュールをコピー** ページを使用して、既存の請求スケジュールから新規の請求スケジュールを作成できます。 新規の請求スケジュールは、元の請求スケジュールと同じ設定になります。 また、ページを使用して、1 つの請求スケジュールから別の請求スケジュールに品目をコピーすることもできます。

## <a name="copy-a-billing-schedule"></a>請求スケジュールのコピー

請求スケジュールをコピーするには、次の手順に従います。

1. **すべての請求スケジュール** または **有効な請求スケジュール** リスト ページで、既存の請求スケジュールを選択します。
2. **請求スケジュール** タブで **スケジュールをコピー** を選択します。
3. **顧客アカウント** フィールドで顧客アカウントを選択します。
4. 必要に応じて請求の開始日と終了日を更新します。
5. 必要に応じて、次のオプション フィールドの値を更新します: **エンド ユーザー アカウント**、**パートナー アカウント**、**顧客の参照**、**顧客の要求**、**繰延開始日**。
6. **プレビューの表示** を選択して、コピーする請求スケジュール行を表示および選択します。
7. **コピー** を選択します。

コピー処理が完了すると、新規の請求スケジュールが、**すべての請求スケジュール** または **有効な請求スケジュール** のリスト ページの下部と **請求スケジュール** ページに表示されます。

## <a name="copy-items-from-one-billing-schedule-to-another"></a>ある請求スケジュールから別の請求スケジュールへの品目のコピー

1. **すべての請求スケジュール** または **有効な請求スケジュール** リスト ページで、既存の請求スケジュールを選択します。
2. **請求スケジュール** タブで **スケジュールをコピー** を選択します。
3. **顧客アカウント** フィールドで顧客アカウントを選択します。
4. **請求スケジュール番号** フィールド で、請求スケジュール番号を選択します。
4. 請求の開始日と終了日を更新します。
5. 必要に応じて、次のフィールドの値を更新します: **エンド ユーザー アカウント**、**パートナー アカウント**、**顧客の参照**、**顧客の要求**、**繰延開始日**。
6. **プレビューの表示** を選択して、既存の請求スケジュールにコピーする請求スケジュール行を表示および選択します。
7. **コピー** を選択します。

コピー処理が完了したら、品目のコピー先の請求スケジュールを確認し、期待どおりに品目が追加されたことを確認します。
