# <a name="deferral-posting-methods"></a>繰延転記方法

この記事では、サブスクリプション請求管理おける収益および経費の繰延の繰り上げ転記方法の違いについて説明します。

**収益および経費繰延パラメータ** ページで、繰延転記方法のオプションは **貸借対照表** と **損益** です。 この記事の例は、2 つの方法の違い、および 1 つの方法を使用する理由、または他の方法を使用する理由について説明するのに役立ちます。

**貸借対照表** 方法では、2 つの勘定のみを使用します。 したがって、必要な設定の数は少ない。 **損益** 方法には、トランザクション タイプに応じて、収益、割引、および原価に対する **初期認識** と **認識相殺** の 2 つの追加勘定があります。 これらの勘定は、**繰延の既定値** ページ (**サブスクリプション請求管理 \> 収益および経費の延期 \> 設定**) で設定されます。 この例では繰延収益に注目しています。繰延収益と繰延原価には同じ概念を適用できます。

## <a name="example"></a>例

売上請求書 1 には合計 $3,000 と繰延収益があります。 次の表に、この売上請求書が転記された場合の配分を示します。

**貸借対照表の方法**

| 種類 | 取引先企業 | 借方 | 貸方|
|---|---|---|---|
| 借方 | 売掛金管理 | 3,000.00 | |
| 貸方 | 繰延収益 | | 3,000.00 |

**損益勘定法**

| 種類 | 取引先企業 | 借方 | 貸方 |
|---|---|---|---|
| 借方 | 売掛金管理 | 3,000.00 | |
| 借方 | 収益認識のオフセット | 3,000.00 | |
| 貸方 | 繰延収益 | | 3,000.00 |
| 貸方 | 初期収益の認識 | | 3,000.00 |

売上請求書 2 には合計 $2,000 があり、繰延収益はありません。

| 種類 | 取引先企業 | 数量 |
|---|---|---|
| 借方 | 売掛金管理 | 3,000.00 |
| 貸方 | 収益 | 3,000.00 |

**売上請求書 1 と売上請求書 2 の合計を合計した貸借対照表方法**

| 取引先企業 | 借方 | 貸方 |
|---|---|---|
| 売掛金管理 | 5,000.00 | |
| 収益 | | 2,000.00 |
| 繰延収益 | | 3,000.00 |

**貸借対照表** 法を使用する場合、ある期間の総収益は簡単に確認できません。 一部の収益は、**繰延収益** 勘定に転記されます。 収益が期間ごとに認識される場合、**繰延収益** 勘定には複数の借方と貸方があります。 特定の期間の総収益が、収益認識のイン/アウトと混在します。

**売上請求書 1 と売上請求書 2 の合計を合計した損益方法**

| 取引先企業 | 借方 | 貸方 |
|---|---|---|
| 売掛金管理 | 5,000.00 | |
| 収益 | | 5,000.00 |
| 収益オフセット | 3,000.00 | |
| 繰延収益 | | 3,000.00 |

すべての収益は損益 **収益** 勘定にいきます。 その後、繰延収益が損益計算書から貸借対照表に移動します。 すべてが **収益** 勘定に転記されるので、総収益を簡単に確認できます。 ただし、一部の総収益は繰り上げされます。 この勘定は **繰延収益** 勘定に移動し、後で認識されます。

**損益** 法では、**利益** 勘定と **利益オフセット** 勘定を参照して、$5,000 の総収益と予測収益の $2,000 の正味収益 (繰延) を見ることができます。 **損益** 法を使用するとレポートが簡単になりますが、設定する勘定の数は増えやすくなります。
