# <a name="allocation-terms"></a><span data-ttu-id="5772d-103">配賦条件</span><span class="sxs-lookup"><span data-stu-id="5772d-103">Allocation terms</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="5772d-104">このトピックでは、主勘定での配賦条件の使用に関する情報を提供します。</span><span class="sxs-lookup"><span data-stu-id="5772d-104">This topic provides information about using allocation terms on a main account.</span></span> <span data-ttu-id="5772d-105">配賦条件は、複数の勘定科目の組み合わせ全体で金額を配分するために使用されます。</span><span class="sxs-lookup"><span data-stu-id="5772d-105">Allocation terms are used to distribute amounts across multiple ledger account combinations.</span></span> <span data-ttu-id="5772d-106">経費、または収益が正しい会計のオブジェクトに付けられることを保証します。</span><span class="sxs-lookup"><span data-stu-id="5772d-106">They help ensure that expenses or revenues are charged to the correct object in accounting.</span></span>

<span data-ttu-id="5772d-107">主勘定で作成する各配賦条件では、単一ソースの主勘定と財務分析コードの組み合わせから配賦される伝票の割合を定義します。</span><span class="sxs-lookup"><span data-stu-id="5772d-107">Each allocation term that you create on a main account defines the percentage of a voucher to be allocated from a single-source main account and a combination of financial dimensions.</span></span> <span data-ttu-id="5772d-108">さらに、金額を配賦する移動先の主勘定と財務分析コードを定義します。</span><span class="sxs-lookup"><span data-stu-id="5772d-108">In addition, you define the destination main account and financial dimensions where the amount will be allocated.</span></span> 

<span data-ttu-id="5772d-109">配賦のソース財務分析コードを定義する場合は、それぞれの分析コードが限定かどうかを選択できます。</span><span class="sxs-lookup"><span data-stu-id="5772d-109">When you define the source financial dimension for the allocation, you can select if each dimension is specific or unspecific.</span></span> <span data-ttu-id="5772d-110">限定は、ソース トランザクションの財務分析コードが選択した分析コードと一致する必要があることを示します。</span><span class="sxs-lookup"><span data-stu-id="5772d-110">Specific indicates that the financial dimension for the source transaction must match the selected dimension.</span></span> <span data-ttu-id="5772d-111">非限定を選択すると、財務分析コードの任意の値が一致に使用できることを示します。</span><span class="sxs-lookup"><span data-stu-id="5772d-111">When you select unspecific, it indicates that any value for the financial dimension can contribute to a match.</span></span>

<span data-ttu-id="5772d-112">配賦条件に対して相手方の勘定科目を定義する場合は、金額を配賦する主勘定を指定する必要があります。</span><span class="sxs-lookup"><span data-stu-id="5772d-112">When you define the destination ledger account for an allocation term, you must specify the main account that you are allocating the amount to.</span></span> <span data-ttu-id="5772d-113">ソース トランザクションが転記された同じ主勘定、または別の主勘定を使用できます。</span><span class="sxs-lookup"><span data-stu-id="5772d-113">You can use the same main account where the source transaction is posted to, or a different main account.</span></span> <span data-ttu-id="5772d-114">同じ主勘定を使用すると、レコードを保存するときに警告が表示されます。</span><span class="sxs-lookup"><span data-stu-id="5772d-114">If the same main account is used, a warning displays when saving the record.</span></span> <span data-ttu-id="5772d-115">主勘定を指定するだけでなく、相手方の分析コードも指定する必要があります。</span><span class="sxs-lookup"><span data-stu-id="5772d-115">In addition to specifying the main account, you must also specify the destination dimensions.</span></span> <span data-ttu-id="5772d-116">分析コードごとに、ソースの財務分析コード値を保持するか、または変更するかを指定できます。</span><span class="sxs-lookup"><span data-stu-id="5772d-116">For each dimension, you can specify if you want to keep the source financial dimension value or change it.</span></span> <span data-ttu-id="5772d-117">[いいえ] を選択した場合は、財務分析コードに対して新しい値を選択する必要があります。</span><span class="sxs-lookup"><span data-stu-id="5772d-117">If you select no, then you must select a new value for the financial dimension.</span></span> <span data-ttu-id="5772d-118">[はい] を選択すると、追加情報は指定されず、転記時に元の財務分析コードがシステムによって管理されます。</span><span class="sxs-lookup"><span data-stu-id="5772d-118">If you select yes, no additional information is specified, and the system will maintain the original financial dimensions when you post.</span></span>

<span data-ttu-id="5772d-119">主勘定に追加できる配賦条件の数に制限はありませんが、配賦条件で配賦する割合の合計が 100 を超えることはできません。</span><span class="sxs-lookup"><span data-stu-id="5772d-119">There is no limit to the number of allocations terms that you can add to a main account; however, the sum of the percent to allocate on an allocation term cannot exceed 100.</span></span> <span data-ttu-id="5772d-120">元の伝票金額の一部をソースの財務分析コードに残す必要がある場合は、100% 未満の配賦を作成できます。</span><span class="sxs-lookup"><span data-stu-id="5772d-120">You can create allocations for less than 100 percent if a portion of the original voucher amount should remain in the source financial dimensions.</span></span> <span data-ttu-id="5772d-121">配賦条件は、法人の上書きとして主勘定に追加できます。</span><span class="sxs-lookup"><span data-stu-id="5772d-121">Allocations terms can be added to a main account as a legal entity override.</span></span> <span data-ttu-id="5772d-122">勘定の共有チャートを使用している場合は、配賦条件を法人ごとに定義する必要があります。</span><span class="sxs-lookup"><span data-stu-id="5772d-122">If you are using a shared chart of accounts, allocation terms must be defined for each legal entity.</span></span> <span data-ttu-id="5772d-123">**配賦条件** ボタンにアクセスするには、最初に法人の上書きの **配賦** チェック ボックスを選択する必要があります。</span><span class="sxs-lookup"><span data-stu-id="5772d-123">To access the **Allocation terms** button, you must first select the **Allocation** check box on the Legal entity override.</span></span>

## <a name="allocation-term-example"></a><span data-ttu-id="5772d-124">配賦条件の例</span><span class="sxs-lookup"><span data-stu-id="5772d-124">Allocation term example</span></span>
<span data-ttu-id="5772d-125">000 という番号の CORPORATE と呼ばれる組織のコスト センターを定義しました。</span><span class="sxs-lookup"><span data-stu-id="5772d-125">You have defined a cost center for your organization called CORPORATE that is numbered 000.</span></span> <span data-ttu-id="5772d-126">公共料金の請求書が転記されると、この CORPORATE コスト センターにコード化されます。</span><span class="sxs-lookup"><span data-stu-id="5772d-126">When invoices are posted for utility expenses, they are coded to this CORPORATE cost center.</span></span> <span data-ttu-id="5772d-127">会社では、すべての公共料金を個々のコスト センターに割合で割り当てるルールを定義しています。</span><span class="sxs-lookup"><span data-stu-id="5772d-127">Your company has defined a rule that all utility expenses should be allocated by a percentage to each of the individual cost centers.</span></span> <span data-ttu-id="5772d-128">部署と事業単位の他の財務分析コードは同じままになります。</span><span class="sxs-lookup"><span data-stu-id="5772d-128">The other financial dimensions for department and business unit will stay the same.</span></span>

<span data-ttu-id="5772d-129">この例では、新しい配賦条件が公共料金の主勘定に対して作成されます。</span><span class="sxs-lookup"><span data-stu-id="5772d-129">With this example, a new allocation term would be created for the utility expense main account.</span></span> <span data-ttu-id="5772d-130">割り当てられた割合で、各コスト センターに対して 1 つの行が作成されます。</span><span class="sxs-lookup"><span data-stu-id="5772d-130">One row would be created for each cost center with the percentage to be allocated.</span></span> <span data-ttu-id="5772d-131">各行のソース財務分析コードの **選択基準** は、**コスト センター** に **限定** であり、値は 000: CORPORATE に設定されます。</span><span class="sxs-lookup"><span data-stu-id="5772d-131">The **Selection criteria** for the source financial dimensions for each row would be **Specific** for the **Cost center** and the value would be set to 000: CORPORATE.</span></span> <span data-ttu-id="5772d-132">部署および事業単位については、**選択基準** が **非限定** になります。</span><span class="sxs-lookup"><span data-stu-id="5772d-132">For department and business unit, the **Selection criteria** would be **Unspecific**.</span></span>

<span data-ttu-id="5772d-133">**相手方の勘定科目** クイック タブで、主勘定が同じ公共料金の勘定になり、**ソースの財務分析コードの保持** は **部署** と **事業単位** に対して **はい** に設定されます。</span><span class="sxs-lookup"><span data-stu-id="5772d-133">On the **Destination ledger account** FastTab, the main account will be the same utility expense account, and the **Keep source financial dimensions** will be set to **Yes** for **Business unit** and **Department.**</span></span> <span data-ttu-id="5772d-134">**ソースの財務分析コードの保持** は **コスト センター** に対して **いいえ** に設定され、選択された値は、配賦先であるそれぞれのコスト センターになります。</span><span class="sxs-lookup"><span data-stu-id="5772d-134">The **Keep source financial dimensions** will be set to **No** for **Cost center**, and the value selected will be each respective cost center that you are allocating to.</span></span> <span data-ttu-id="5772d-135">配賦先のコスト センターが 3 つある場合は、3 つの配賦条件レコードが作成されます。</span><span class="sxs-lookup"><span data-stu-id="5772d-135">If there are three cost centers to allocate to, then three allocation terms records would be created.</span></span> <span data-ttu-id="5772d-136">各配賦条件の唯一の違いは、相手方の勘定科目で指定されたコスト センターです。</span><span class="sxs-lookup"><span data-stu-id="5772d-136">The only difference between each allocation term is the cost center that is specified on the destination ledger account.</span></span>

## <a name="create-an-allocation-term-on-a-main-account"></a><span data-ttu-id="5772d-137">主勘定に配賦条件を作成する</span><span class="sxs-lookup"><span data-stu-id="5772d-137">Create an allocation term on a main account</span></span>

1. <span data-ttu-id="5772d-138">**ナビゲーション ペイン** で、**モジュール > 一般会計 > 勘定科目表 > 勘定 > 主勘定** の順に移動します。</span><span class="sxs-lookup"><span data-stu-id="5772d-138">In the **Navigation pane**, go to **Modules > General ledger > Chart of accounts > Accounts > Main accounts**.</span></span>
2. <span data-ttu-id="5772d-139">一覧で、目的のレコードを見つけ、選択します。</span><span class="sxs-lookup"><span data-stu-id="5772d-139">In the list, find and select the desired record.</span></span>
3. <span data-ttu-id="5772d-140">**法人の上書き** クイック タブで、**追加** を選択します。</span><span class="sxs-lookup"><span data-stu-id="5772d-140">On the **Legal entity overrides** FastTab, select **Add**.</span></span>
4. <span data-ttu-id="5772d-141">**会社** を選択し、**追加** を選択します。</span><span class="sxs-lookup"><span data-stu-id="5772d-141">Select the **Company** and then select **Add**.</span></span>
5. <span data-ttu-id="5772d-142">**配賦** チェック ボックスを選択します。</span><span class="sxs-lookup"><span data-stu-id="5772d-142">Select the **Allocation** check box.</span></span>
6. <span data-ttu-id="5772d-143">**配賦条件** を選択します。</span><span class="sxs-lookup"><span data-stu-id="5772d-143">Select **Allocation terms** .</span></span>
7. <span data-ttu-id="5772d-144">**編集** を選択して既定のレコードを変更するか、**新規** を選択してレコードを追加します。</span><span class="sxs-lookup"><span data-stu-id="5772d-144">Select **Edit** to modify the default record, or select **New** to add a record.</span></span>
8. <span data-ttu-id="5772d-145">**割合** フィールドに、配賦する伝票トランザクションの割合を入力します。</span><span class="sxs-lookup"><span data-stu-id="5772d-145">In the **Percent** field, enter the percentage of voucher transactions that you want to allocate.</span></span>
9. <span data-ttu-id="5772d-146">**ソースの財務分析コード** クイック タブの各財務分析コードの **選択基準** で、オプションを選択します。</span><span class="sxs-lookup"><span data-stu-id="5772d-146">On the **Source financial dimension** FastTab, in the **Selection criteria** for each financial dimension, select an option.</span></span>
    - <span data-ttu-id="5772d-147">**限定** を選択した場合は、右側のドロップダウン ボックスで財務分析コード値を選択します。</span><span class="sxs-lookup"><span data-stu-id="5772d-147">If you select **Specific**, then select the financial dimension value in the drop-down box on the right.</span></span>
    - <span data-ttu-id="5772d-148">**非限定** を選択した場合は、財務分析コードについての追加情報は必要ありません。</span><span class="sxs-lookup"><span data-stu-id="5772d-148">If you select **Unspecific**, no additional information is required for the financial dimension.</span></span>
10. <span data-ttu-id="5772d-149">**相手方の勘定** 科目のクイック タブの **配賦先勘定** フィールドで、伝票トランザクションの割合を配賦する主勘定を選択します。</span><span class="sxs-lookup"><span data-stu-id="5772d-149">On the **Destination ledger** account FastTab in the **To account** field, select the main account where you want to allocate the percentage of the voucher transaction.</span></span>
11. <span data-ttu-id="5772d-150">各財務分析コードの **ソースの財務分析コードの保持** で、オプションを選択します。</span><span class="sxs-lookup"><span data-stu-id="5772d-150">In **Keep source financial dimensions** for each financial dimension, select an option.</span></span>
    - <span data-ttu-id="5772d-151">**いいえ** を選択した場合は、伝票トランザクションを割り当てる右側のドロップダウン ボックスで財務分析コード値を選択します。</span><span class="sxs-lookup"><span data-stu-id="5772d-151">If you select **No**, then select the financial dimension value in the drop-down box on the right where you want the voucher transaction to be allocated to.</span></span>
    - <span data-ttu-id="5772d-152">**はい** を選択した場合は、追加の情報は必要ありません。</span><span class="sxs-lookup"><span data-stu-id="5772d-152">If you select **Yes**, then no additional information is required.</span></span> <span data-ttu-id="5772d-153">システムでは、配賦の転記時に元の伝票の値が保持されます。</span><span class="sxs-lookup"><span data-stu-id="5772d-153">The system will keep the value on the original voucher when posting the allocation.</span></span>
12. <span data-ttu-id="5772d-154">追加の配賦ごとに手順 7-11 を繰り返します。</span><span class="sxs-lookup"><span data-stu-id="5772d-154">Repeat steps 7-11 for each additional allocation.</span></span> <span data-ttu-id="5772d-155">すべての配賦の合計は、**合計割合** フィールドに示されます。</span><span class="sxs-lookup"><span data-stu-id="5772d-155">The sum of all allocations is indicated in the **Total percent** field.</span></span> <span data-ttu-id="5772d-156">この値は、100 を超えることはできません。</span><span class="sxs-lookup"><span data-stu-id="5772d-156">This amount cannot exceed 100.</span></span>
13. <span data-ttu-id="5772d-157">すべてのページを閉じます。</span><span class="sxs-lookup"><span data-stu-id="5772d-157">Close all of the pages.</span></span>

>[!NOTE] 
> <span data-ttu-id="5772d-158">必要に応じて **コピー** ボタンを使用して、選択した配賦を複製できます。</span><span class="sxs-lookup"><span data-stu-id="5772d-158">You can optionally use the **Copy** button to duplicate the selected allocation.</span></span>
