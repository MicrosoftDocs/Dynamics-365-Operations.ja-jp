# <a name="set-up-global-withholding-tax"></a>グローバル源泉徴収税の設定

この記事では、販売や購入に向けたグローバル源泉徴収税を設定する手順を説明します。 

1. **源泉徴収税の関係機関** ページで 、源泉徴収税の関係機関を設定します。

2. 源泉徴収税精算期間を **源泉徴収税の精算期間** ページで設定 します。

3. **源泉徴収税 > 元帳転記グループ** ページで、源泉徴収元帳転記グループを設定 します。

   > [!Note] 
   >
   > **源泉徴収税** の勘定は、**転記タイプ** が源泉徴収税の主要勘定に割り当てられます。 **源泉徴収税** オフセットの勘定は、**転記タイプ** が源泉徴収税オフセットの主要勘定に割り当てられます。 **総勘定元帳 > 勘定科目表 > 勘定科目 > 主要勘定** に移動し、主要勘定の **転記の検証** サブフォームに **転記タイプ** を設定します。

4. **源泉徴収税コード** ページで 、源泉徴収税のコードを設定します。

5. **源泉徴収税グループ** ページで 、源泉徴収税のグループを設定します。

6. **源泉徴収税の収益** の **タイプ** ページで、源泉徴収税の収益タイプ を設定します。

7. 品目、またはセービスの **品目の源泉徴収税グループ** ページで 、源泉徴収税のグループを設定します。

8. **総勘定元帳のパラメータ > 源泉徴収税** ページで、**請求金額の最小額** を設定します。
