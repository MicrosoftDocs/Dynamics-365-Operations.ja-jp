# <a name="view-posted-sales-tax-transactions"></a>転記された消費税トランザクションの表示

[!include [banner](../../includes/banner.md)]

この記事では、転記された売上税トランザクションを表示するいくつかの方法について説明します。 必要な内容に応じて、さまざまなページから転記済売上税トランザクションを表示できます。 このタスクでは、USMF というデモ会社を使用します。

## <a name="posted-sales-tax-page"></a>転記済売上税ページ

1. ナビゲーション ウィンドウで、**モジュール > 税 > 照会およびレポート > 売上税の照会 > 転記済売上税** の順に移動します。
2. ビューを特定のトランザクションに絞り込むには、**フィルターの表示** を選択します。
3. ページを閉じます。

## <a name="sales-tax-settlement-periods-page"></a>売上税決済期間ページ

1. ナビゲーション ウィンドウで、**モジュール > 税 > 間接税 > 売上税 > 売上税決済期間** の順に移動します。
2. **間隔** セクションを展開します。
3. 必要な間隔を選択します。
4. さらに、転記済売上税トランザクションの一覧をフィルター処理できます。
5. ページを閉じます。

## <a name="sales-tax-codes-page"></a>売上税コードのページ

1. ナビゲーション ウィンドウで、**モジュール > 税 > 間接税 > 売上税 > 売上税コード** の順に移動します。
2. アクション ウィンドウで、**売上税コード** を選択します。
3. **転記済売上税** を選択します。
