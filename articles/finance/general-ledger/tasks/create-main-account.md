# <a name="create-a-main-account"></a>主勘定の作成

[!include [banner](../../includes/banner.md)]

このタスクのガイドは、既存の勘定科目表への主勘定の追加方法について説明します。 このレコードでは、USMF デモ会社を使用します。  

1. **ナビゲーション ウィンドウ > モジュール > 総勘定元帳 > 勘定科目表 > 勘定 > 主勘定** の順に移動します。
2. **新規** をクリックします。
3. **主勘定** フィールドで、値を入力します。
4. **名前** フィールドに値を入力します。
5. **主勘定タイプ** フィールドで、財務諸表で勘定残高および場所を最も適切に表すタイプを選択します。
6. 一覧から主勘定が属する勘定カテゴリを選択します。 勘定カテゴリは既定の財務報告および Power BI ダッシュボードの内容に使用されます。  
7. 一覧で、選択された行のリンクをクリックします。 既定の借方または貸方残高を変更します。  
8. **既定通貨** フィールドで、通貨一覧から値を選択します。
9. 一覧で、目的のレコードを見つけ、選択します。
10. 一覧で、選択された行のリンクをクリックします。
11. **法人の上書き** セクションの展開を切り替えます。
12. **追加** をクリックして法人を選択します。
13. 一覧から [法人] を選択します。
14. **追加** をクリックします。
15. 一覧で、選択された行をマークします。
16. **中断** チェック ボックスをオンまたはオフにします。
17. **法定レポート** セクションを展開します。
18. **為替レート タイプ** フィールドで、ドロップダウン ボタンをクリックしてルックアップを開きます。
19. 一覧から、**口座の為替レート タイプ** を選択します。
20. 一覧で、選択された行のリンクをクリックします。
21. **為替換算タイプ** フィールドで、口座の為替レートの計算方法を選択します。
22. ページを閉じます。
