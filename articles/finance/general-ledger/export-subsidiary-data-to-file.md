# <a name="export-subsidiary-data-to-files"></a>関連会社のデータをファイルへエクスポートする

[!include [banner](../includes/banner.md)]

**エクスポート** ページ (**システム管理 \> ワークスペース \> インポート/エクスポート**) を使用して、連結法人にインポート可能なファイルに子会社のデータをエクスポートする準備します。 インポートとエクスポートのプロセスについての詳細情報は、[データのインポートとジョブのエクスポートの概要](../../fin-ops-core/dev-itpro/data-entities/data-import-export-job.md) を参照してください。

1. 連結プロセスで使用する法人を作成する。 法人の作成方法の詳細については、[法人の作成](../../fin-ops-core/fin-ops/organization-administration/tasks/create-legal-entity.md)を参照してください。 詳細については、[連結プロセスに使用する法人の準備](prepare-company-for-consolidation.md) と [連結用の子会社の設定](set-up-subsidiary-company-for-consolidation.md)を参照してください 。 

2. **連結 \> エクスポート 会社の 残高** に移動します。 **会社の残高のエクスポート** ページの **基準** タブで、次のフィールドを設定して連結の詳細を指定します。

    | フィールド                             | 説明 |
    |-----------------------------------|-------|
    | 主勘定                      | 連結する勘定を指定します。 すべての勘定を含めるには、このフィールドを空白のままにします。 |
    | 連結勘定を使用する         | 連結勘定を指定した場合は、このオプションを **はい** に設定します。 |
    | 連結勘定の選択元 | **主要勘定** または **連結勘定グループ** を選択します。 |
    | 連結勘定グループ       | 選択した連結勘定デ使用する連結勘定グループを選択します。 |
    | 連結期間              | 連結の "開始日" と "終了日" を指定します。 |
    | 実績金額を含む            | 実際の金額を含める場合は、このオプションを **はい** に設定します。 |
    | 予算金額を含む            | このオプションを **はい** に設定し、連結に予算額を含めます。 |
    | 予算モデル                     | 含める予算モデルを指定します。 |

3. **財務分析コード** タブで、連結の詳細を指定します。

    - 子会社勘定の取引から連結法人の取引に移行する財務分析コード情報を指定します。
    - リストの財務分析コードを選択します。
    - 連結される各財務分析コードに対して正しい詳細を識別します。 使用できるオプションには、**分析コード**、**グループ分析コード**、**会社の勘定**、**勘定** があります。

        > [!NOTE]
        > **グループ分析コード** オプションを使用すると、連結対象の会社グループで使用される分析コードの値を定義できます。

    - 連結するセグメントの順番を指定します。

4. **法人** タブで 、次の手順に従って、エクスポートする法人を指定します。

    1. **新規** を選択します。
    2. **ソースの法人** フィールドに、法人を入力します。

        同じデータベース内の複数の関連会社に同じ基準を適用する場合は、それらの関連会社から別のエクスポート ファイルへ一括してデータを転送できます。

        1. ファイルにエクスポートする勘定科目の子会社法人ごとに行を作成します。 これらのファイルは、後に連結法人にインポートされます。
        2. 関連会社ごとに、その会社名とエクスポート ジョブで作成されるエクスポート ファイルの名前を入力します。

    3. **換算差額の勘定タイプ** フィールドで 、**損益** または **貸借対照表** を選択します。
    4. 作成されるエクスポート ファイルのファイル名を入力します。

5. **OK** を選択してエクスポートを実行します。

エクスポートが完了すると、各ファイルに保存されたレコード数を示すメッセージが表示されます。 その後、連結法人にファイルをインポートできます。
