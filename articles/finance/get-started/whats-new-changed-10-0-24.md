# <a name="preview-of-dynamics-365-finance-10024-february-2022"></a>Dynamics 365 Finance 10.0.24 (2022 年 2 月) のプレビュー

[!include [banner](../includes/banner.md)]

[!include [banner](../includes/preview-banner.md)]

この記事では、Microsoft Dynamics 365 Finance バージョン 10.0.24 の新機能または変更された機能について説明します。 このバージョンには 10.0.1084 のビルド番号が含まれており、次のように使用できます。

- **リリースのプレビュー**: 2021 年 12 月
- **リリースの一般提供 (手動更新)**: 2022 年 1 月
- **リリースの一般提供 (自動更新)**: 2022 年 2 月

## <a name="features-included-in-this-release"></a>このリリースに含まれる機能

次の表に、このリリースに含まれる機能の一覧を示します。 この記事が最初に公開された後に、ビルドに加えた機能を含めるために、この記事を更新することがあります。

| 機能領域 | フィーチャー | 詳細 |  に  によって有効化  |
|----|----|----|----|
| 税の計算   | 現金割引の計算設定で税務署パラメーターを使用する    | 単一の法人が異なる国または地域で複数の付加価値税 (VAT) 登録をしている場合、現金割引や消費税の計算の要件が国や地域によって異なる場合があります。 この機能は、単一法人で異なるパラメーターを持つ現金割引と消費税を計算します。 詳細については、[現金割引の計算設定の地方税務署パラメーター](../localizations/global-tax-jurisdiction-cash-discount-setup.md)を参照してください。    | 機能管理   |
| 税の計算   | グローバリゼーション機能での税コードのインポート    | この機能は、税コードをグローバリゼーション機能にインポートするのに役立ちます。 詳細については、[税計算機能のエクスポートとインポート](https://go.microsoft.com/fwlink/?linkid=2182833)を参照してください。    | 既定で有効   |
| 税決済   | 消費税支払レポートの生成を消費税決済から分離する    | この機能により、消費税支払レポートの生成を消費税決済から分離できます。 この機能を有効にすると、消費税決済プログラムの完了後には消費税決済レポートは自動的に実行されません。    | 機能管理   |



## <a name="feature-enhancements-included-in-this-release"></a>このリリースに含まれる機能拡張

次の表に、このリリースに含まれる機能拡張の一覧を示します。 これらの機能拡張は、それぞれ既存の機能を段階的に改善します。 これらは拡張機能にすぎないため、[リリース計画](/dynamics365-release-plan/2021wave2/finance-operations/dynamics365-finance) には記載されません。 これらの機能を使用または無効にする場合は、[機能管理](../../fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview.md)で実行する必要があります。

| 機能領域 | 機能管理の機能名 | 詳細 |  
|---|---|---|
| 買掛金  | 2021 年用の 1099 フォームに変更を適用します   | この機能により、2021 1099-DIV、1099-NEC、1099-MISC フォームに新しい 1099 フィールドが追加されます。 この機能を有効にすると、2021 IRS 要件への準拠が保証されます。 これらの変更は必須で、既定で有効になっています。  |  
| 買掛金 | 仕入先請求書の自動化  |  仕入先請求書の自動化機能は、複数のリリースに対していくつかの機能を提供します。 重要なコンポーネントには、インポートされた請求書をワークフローに自動的に送信する機能、転記をシミュレートして業務プロセスを中断する前にエラーを検出して修正する機能、および仕入先請求書の自動化処理を評価および改善するための一連の Power BI メトリックスが含まれます。 新しい機能の各セットがそれぞれ使用可能になると、**機能管理** ページの新しい項目では、その項目に含まれる機能を有効にするオプションが表示されます。  | 
| 売掛金管理  | (イタリア) 受取手形の受取拒否処理 | 電子ファイルから受取手形の受取拒否情報をインポートできます。 |
| 資産リース  | 資産リース  |  資産リースを使用して、リースの初期認識、関連する支払いスケジュール、償却、調整、再分類を管理できます。  |
| 資産リース | 資産リースのリース規則  | リース開始日と組み合わせてリース規則を選択すると、リースの適切な開始日を決定できます。   |
| 資産リース  | 資産リースのリース調整ウィザード   |  この機能は、ウィザードを通じて親リースや関連付けられた帳簿を調整するのに役立ちます。 リース調整を確定する前に、使用権資産の残高、リース負債、および調整仕訳入力をプレビューできます。 この機能は親リースや関連付けられた帳簿のみの調整に使用されるため、調整帳簿機能はこれまでと同じです。  |
| 資産リース | リース減損のプレビュー付き転記 | この機能は、リース減損プロセスからの新しい資産残高および財務エントリを表示します。 この機能を使用すると、減損トランザクションを転記する前に、新しい資産残高を計算し、提供された減損データに基づいて結果および財務エントリの影響を表示できます。 |
| 現金および銀行管理  | すべての口座取引明細書とトランザクションの照合を解除  | 有効にすると、**すべて照合解除** ボタンが **口座調整ワークシート** ページに表示されます。 このボタンを使用すると、すべての明細書およびトランザクションを一度に照合解除できます。 |
| 現金および銀行管理  | 財務インサイト コンフィギュレーションの検証 | この機能は、Finance Insights のコンフィギュレーションとプロセスを検証し、一般的な問題が特定された場合に通知を行います。 |
| 一般会計 | 大規模な連結のためのパフォーマンスの向上  | 各バッチで効率的に実行できるように、総勘定元帳の連結にパフォーマンスの向上が追加されました。 拡張機能は総勘定元帳のデータを並列処理することで機能し、カスタマイズする必要がある場合のために拡張ポイントが追加されました。 |
| 一般会計 | 保留タイプ トランザクションを含む試算表の生成  | この機能を使用すると、**試算表の詳細** レポートに含める特定の保留タイプのトランザクションを選択できます。 |
| 一般会計 | ノルウェー向け OF-T の 'MasterFiles' レポート セクションの分析コード属性値コレクションの最適化  | この機能により、SAF-T レポートの **MasterFiles** レポート セクションの分析コード属性値のセットベース収集が可能になり、パフォーマンスが向上し、**MasterFiles** セクションからの **分析** フィールドと **GeneralLedgerEntries** セクションからの **分析** フィールドがより一貫するようになります。 |
| 一般会計 | 総勘定元帳への補助元帳転送のパフォーマンスの最適化 | この機能により、補助元帳から総勘定元帳へのデータの転送が改善されます。 これにより処理がより効率的になり、小さなトランザクションのグループ セットを一緒に転送して、バッチ サーバーをより効率的に使用できるようになります。 |
| 税の計算 | 税計算サービス | この機能を使用すると、顧客/仕入先マスター データ レベルで **消費税の上書き** パラメーターを定義できます。このパラメーター値は、選択した顧客/仕入先を使用する場合の販売/購買トランザクションの既定になります。 |
| 税の計算 | 税計算サービス | この機能は、メキシコでの法人の基本住所をサポートしています。 詳細については、[税計算の概要](../localizations/global-tax-calcuation-service-overview.md)の、**サポートされる国/地域** セクションを参照してください。 |
| 税の計算 | 税計算サービス | この機能は、グローバリゼーション機能の適用ルール設定で、最大 20 の分析コードをサポートしています。 |

### <a name="removed-and-deprecated-features"></a>削除済みおよび非推奨の機能

[Dynamics 365 Finance の削除済みまたは非推奨の機能](removed-deprecated-features-finance.md)の記事では、Dynamics 365 Finance の削除または非推奨の機能について説明します。

- *削除された* 機能は製品では使用できません。
- *削除予定* の機能は現在開発中ではなく、将来の更新で削除される可能性があります。

製品から機能が削除される前に、非推奨の通知が削除の 12 ヶ月前に [Dynamics 365 Finance の削除済みまたは非推奨の機能](removed-deprecated-features-finance.md)の記事に発表されます。

コンパイル時に影響する重大な変更が、サンドボックス環境および運用環境と互換性のあるバイナリの場合、廃止時間は 12 か月以内になります。 通常、これらはコンパイラに加える必要がある機能の更新です。
