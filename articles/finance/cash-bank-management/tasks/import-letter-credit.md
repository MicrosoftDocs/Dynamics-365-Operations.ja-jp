# <a name="import-letter-of-credit"></a>輸入信用状

[!include [banner](../../includes/banner.md)]

この手順は、信用状のインポートのプロセスを説明します。 この手順を実行する前に次の事柄が設定されている必要があります : 銀行融資、転記プロファイル、銀行融資契約と仕入先銀行の詳細。

この手順では、USMF というデモ会社を使用します。


## <a name="create-a-purchase-order-with-letter-of-credit"></a>信用状と共に [発注書] を作成します。
1. [買掛金勘定] > [発注書] > [すべての発注書] に移動します。
2. [新規] をクリックします。
3. [仕入先] フィールドで、値を入力または選択します。
4. 一覧で、目的のレコードを見つけ、選択します。
5. 一覧で、選択された行のリンクをクリックします。
6. [一般] セクションを展開します。
7. [サイト] フィールドで値を入力または選択します。
8. 一覧で、選択された行のリンクをクリックします。
9. [倉庫] フィールドで、値を入力または選択します。
10. 一覧で、選択された行のリンクをクリックします。
11. [会計日] フィールドに日付を入力します。
12. [配送日] フィールドに日付を入力します。
    * 注記 : 「銀行ドキュメント タイプ」フィールドは「信用状」の値と共に選択されているべきです。  
13. [OK] をクリックします。
14. [品目番号] フィールドで、値を入力または選択します。
15. 一覧で、目的のレコードを見つけ、選択します。
16. 一覧で、選択された行のリンクをクリックします。
17. [行の詳細] セクションを展開します。
18. [出荷] タブをクリックします。
19. [配送日] フィールドに日付を入力します。
20. [確認済配送日] フィールドに日付を入力します。
21. [単価] フィールドに数値を入力します。
    * [信用状の詳細] を定義します。  
22. [アクション] ウィンドウで、[管理] をクリックします。
23. [信用状 / 輸入取立] をクリックします。
24. [申請日] フィールドに日時を入力します。
    * 「銀行口座」フィールドに申請日に基づく既定の有効な [銀行口座] があることを確認します。  
25. [銀行ドキュメント番号] フィールドに値を入力します。
26. [受取日] フィールドに日時を入力します。
27. [銀行ドキュメント] セクションを展開します。
28. [有効期限] フィールドに日時を入力します。
29. [銀行詳細] セクションを展開します。
30. [通知銀行] フィールドで、値を入力または選択します。
31. 一覧で、選択された行のリンクをクリックします。
32. [保存] をクリックします。
33. [発注書の出荷のフェッチ] をクリックします。
34. ページを閉じます。
35. アクション ウィンドウで、[購買] をクリックします。
36. [確認] をクリックします。
37. [アクション] ウィンドウで、[管理] をクリックします。
38. [信用状 / 輸入取立] をクリックします。
39. [プロセス] をクリックします。
40. [確認] をクリックします。
    * [融資残高] により発注書の金額が減少することを確認します。  この例では、[購買金額] = 500.00、[融資限度] = 10000.00 なので、[融資残高] = 9500.00 となります。  
41. ページを閉じます。
42. [単価] フィールドに数値を入力します。
43. [保存] をクリックします。
44. アクション ウィンドウで、[購買] をクリックします。
45. [確認] をクリックします。
    * 単価の変更に応じて [信用状] を修正します。  
46. [アクション] ウィンドウで、[管理] をクリックします。
47. [信用状 / 輸入取立] をクリックします。
    * [購買単価] の変更に応じて信用状を修正します。  
48. [プロセス] をクリックします。
49. [修正] をクリックします。
50. [削除] をクリックします。
51. [はい] をクリックします。
52. [発注書の出荷のフェッチ] をクリックします。
53. [プロセス] をクリックします。
54. [確認] をクリックします。
    * [融資残高] により発注書の金額が減少することを確認します。  この例では、変更された [購買金額] = 600.00、[融資限度] = 10000.00 なので、[融資残高] = 9400.00 となります。  
55. ページを閉じます。

## <a name="post-packing-slip"></a>[梱包明細] の転記
1. アクション ウィンドウで、[入庫] をクリックします。
2. [製品受領書] をクリックします。
3. PurchParmTable_Num フィールドに値を入力します。
    * [信用状] の照会と一緒に作成された [出荷番号] を選択します。  
4. 一覧で、選択された行のリンクをクリックします。
5. [製品受領書の日付] フィールドに日付を入力します。
6. [OK] をクリックします。
7. ページを閉じます。
8. ページを閉じます。

## <a name="verify-import-letter-of-credit-status"></a>[輸入信用状] の状態の確認
1. [現金および銀行管理] > [信用状] > [輸入信用状] および [輸入取立] の順に移動します。
2. 一覧で、目的のレコードを見つけ、選択します。
3. 一覧で、選択された行のリンクをクリックします。
    * [輸入信用状] の状態を確認します。     
4. ページを閉じます。
5. ページを閉じます。

## <a name="post-purchase-invoice"></a>仕入請求書の転記
1. [買掛金勘定] > [発注書] > [すべての発注書] に移動します。
    * 信用状と一緒に作成された [発注書] を選択します。  
2. 一覧で、目的のレコードを見つけ、選択します。
3. 一覧で、選択された行のリンクをクリックします。
4. アクション ウィンドウで、[請求書] をクリックします。
5. [請求書] をクリックします。
6. [数] フィールドに値を入力します。
7. [出荷番号] フィールドで、値を入力または選択します。
8. 一覧で、選択された行のリンクをクリックします。
9. [請求日] フィールドに日付を入力します。
10. [照合状態の更新] をクリックします。
11. [転記] をクリックします。
12. ページを閉じます。
13. ページを閉じます。

## <a name="verify-import-letter-of-credit-status"></a>[輸入信用状] の状態の確認
1. [現金および銀行管理] > [信用状] > [輸入信用状] および [輸入取立] の順に移動します。
2. 一覧で、目的のレコードを見つけ、選択します。
3. 一覧で、選択された行のリンクをクリックします。
    * [輸入信用状 2] を確認します。  
    * 検証 : [出荷ステータス] = [請求済銀行融資の詳細]  
4. [表示] をクリックします。
5. [申請の印刷] をクリックします。
6. [OK] をクリックします。
    * [信用状の申請書] が印刷されていることを確認します。  
7. ページを閉じます。
8. ページを閉じます。
9. ページを閉じます。

## <a name="post-vendor-payment-journal-for-the-created-purchase-invoice-with-letter-of-credit"></a>信用状と共に作成済の仕入請求書の仕入先支払仕訳帳を転記
1. [買掛金勘定] > [支払] > [支払仕訳帳] の順に移動します。
2. [新規] をクリックします。
3. [名前] フィールドで値を入力または選択します。
4. 一覧で、選択された行のリンクをクリックします。
5. [行] をクリックします。
6. [日付] フィールドに日付を入力します。
7. [勘定] フィールドで、任意の値を指定します。
8. [トランザクションの決済] をクリックします。
9. [合計] セクションを展開します。
10. [表示] フィールドで、オプションを選択します。
    * [銀行ドキュメント番号] と [出荷番号] フィールドが更新済であることを確認します。  
11. [マーク] のチェック ボックスをオンにします。
12. [OK] をクリックします。
13. [支払] タブをクリックします。
    * [銀行ドキュメント番号] と [出荷番号] フィールドが更新済であることを確認します。  
14. [転記] をクリックします。
15. ページを閉じます。
16. ページを閉じます。

## <a name="verify-import-letter-of-credit-status-after-invoice-paid"></a>請求書が支払われた後の [輸入信用状] の状態の確認
1. [現金および銀行管理] > [信用状] > [輸入信用状] および [輸入取立] の順に移動します。
2. 一覧で、目的のレコードを見つけ、選択します。
3. 一覧で、選択された行のリンクをクリックします。
    * [輸入信用状] の状態を確認します。   
4. ページを閉じます。

## <a name="verify-the-bank-facility-limit-and-utilization-report"></a>[銀行融資限度] および [使用率レポート] の確認
1. [現金および銀行管理] > [照会およびレポート] > [信用状または信用保証状] > [銀行融資] と [使用率レポート] の順に移動します。
2. [対象に含めるレコード] セクションを展開します。
3. [フィルター] をクリックします。
    * 必須銀行口座により、[基準] フィールドを定義します。  
4. [基準] フィールドで、値を入力または選択します。
5. 一覧で、選択された行のリンクをクリックします。
6. [OK] をクリックします。
7. [OK] をクリックします。
    * トランザクションを一覧するレポートを確認します。  
    * [銀行ドキュメント番号]、[融資限度]、[使用金額] および [融資残高] のトランザクションのレポートを確認します。  
8. ページを閉じます。
