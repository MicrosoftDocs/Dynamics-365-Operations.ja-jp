# <a name="currency-revaluation-for-accounts-payable-and-accounts-receivable"></a><span data-ttu-id="ad328-103">買掛金勘定と売掛金勘定の通貨再評価</span><span class="sxs-lookup"><span data-stu-id="ad328-103">Currency revaluation for Accounts payable and Accounts receivable</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="ad328-104">為替レートの変動は、外貨での未処理トランザクションの計算上の金額 (簿価額) が時間の経過と共に変化する原因になります。</span><span class="sxs-lookup"><span data-stu-id="ad328-104">Fluctuations in exchange rates cause the theoretical value (book value) of open transactions in foreign currencies to vary over time.</span></span> <span data-ttu-id="ad328-105">このトピックは、[買掛金勘定] および [売掛金勘定] の未処理トランザクションの値を更新するために実行する外貨再評価プロセスに関する情報を提供します。</span><span class="sxs-lookup"><span data-stu-id="ad328-105">This topic provides information about the foreign currency revaluation process that you run to update the value of open transactions in Accounts payable and Accounts receivable.</span></span> 

<span data-ttu-id="ad328-106">外貨での未処理トランザクションの計算上の金額、つまり簿価額は、為替レートの変動によって時間の経過と共に変化します。</span><span class="sxs-lookup"><span data-stu-id="ad328-106">The theoretical value, or book value, of open transactions in foreign currencies varies over time because of fluctuations in exchange rates.</span></span> <span data-ttu-id="ad328-107">[買掛金勘定] および [売掛金勘定] の未処理トランザクションの値を更新するには、外貨再評価プロセスを実行します。</span><span class="sxs-lookup"><span data-stu-id="ad328-107">To update the value of open transactions in Accounts payable and Accounts receivable, run the foreign currency revaluation process.</span></span> <span data-ttu-id="ad328-108">外貨再評価は、[買掛金勘定] および [売掛金勘定] の両方に実行できます。</span><span class="sxs-lookup"><span data-stu-id="ad328-108">Foreign currency revaluation can be run for both Accounts payable and Accounts receivable.</span></span> <span data-ttu-id="ad328-109">プロセスでは、新しい為替レートを使用して、指定した日付にオープン、つまり未決済だった金額を再評価します。</span><span class="sxs-lookup"><span data-stu-id="ad328-109">The process uses a new exchange rate to revalue the open amounts, or not settled amounts, on a specified date.</span></span> <span data-ttu-id="ad328-110">転記された元の金額と再評価された金額との差異によって、未処理トランザクションに対する未実現差益または差損が生じます。</span><span class="sxs-lookup"><span data-stu-id="ad328-110">The differences between the original posted amounts and the revalued amounts will cause an unrealized gain or loss for each open transaction.</span></span> <span data-ttu-id="ad328-111">[買掛金勘定] および [売掛金勘定] の補助元帳を更新して、未実現差益または差損を反映すると、簿記入力が [総勘定元帳] に転記されます。</span><span class="sxs-lookup"><span data-stu-id="ad328-111">The Accounts payable and Accounts receivable subledgers are then updated to reflect the unrealized gain or loss, and an accounting entry is posted to General ledger.</span></span>
