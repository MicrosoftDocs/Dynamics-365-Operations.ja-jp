# <a name="view-an-expense-report"></a><span data-ttu-id="294fb-103">経費精算書の表示</span><span class="sxs-lookup"><span data-stu-id="294fb-103">View an expense report</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="294fb-104">総勘定元帳または調達でトランザクションを表示するクエリを実行すると、クエリが大量の情報を返す可能性があります。</span><span class="sxs-lookup"><span data-stu-id="294fb-104">When you run a query to view transactions in General ledger or Procurement and sourcing, the query might return too much information.</span></span> <span data-ttu-id="294fb-105">したがって、検索を絞り込むこともできます。</span><span class="sxs-lookup"><span data-stu-id="294fb-105">Therefore, you might want to narrow your search.</span></span> <span data-ttu-id="294fb-106">必要な情報を取得するには、表示されている経費精算書を確認するトランザクションにドリルダウンできます。</span><span class="sxs-lookup"><span data-stu-id="294fb-106">To get the information that you require, you can drill down into a transaction to see the expense report that it appeared on.</span></span>

<span data-ttu-id="294fb-107">経費精算書に関して、トランザクションから元の清算書 (経費管理に保存されている) にドリルダウンできます。</span><span class="sxs-lookup"><span data-stu-id="294fb-107">For expense reports, you can drill down from a transaction to the original report that is stored in Expense management.</span></span> <span data-ttu-id="294fb-108">経費の合計金額に加えて、明細化されたトランザクション、付加価値税 (VAT) の復旧に適用できるトランザクション、および経費に関連付けられている受取も表示できます。</span><span class="sxs-lookup"><span data-stu-id="294fb-108">In addition to the total amount of expenses, you can view the itemized transactions, the transactions that are eligible for recovery of value-added tax (VAT), and the receipts that are attached to the expenses.</span></span>

<span data-ttu-id="294fb-109">トランザクションが表示された元の経費精算書にドリルダウンするには、トランザクションを選択し、次に**元の文書**を選択します。</span><span class="sxs-lookup"><span data-stu-id="294fb-109">To drill down to the original expense report that a transaction appeared on, select the transaction, and then select **Original document**.</span></span>

<span data-ttu-id="294fb-110">個別のトランザクションを表示すると、トランザクションに関連する追加データを表示できます。</span><span class="sxs-lookup"><span data-stu-id="294fb-110">When you view individual transactions, you can also view any additional data that is attached to a transaction.</span></span> <span data-ttu-id="294fb-111">たとえば、明細化されるホテル経費に対して、インターネット アクセス、通話、ルーム サービスなどの経費の詳細の一覧を表示できます。</span><span class="sxs-lookup"><span data-stu-id="294fb-111">For example, for an itemized hotel expense, you can view a detailed list of expenses such as internet access, telephone calls, and room service.</span></span>
