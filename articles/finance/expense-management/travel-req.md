# <a name="travel-requisitions"></a><span data-ttu-id="a52fe-104">出張費要求</span><span class="sxs-lookup"><span data-stu-id="a52fe-104">Travel requisitions</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="a52fe-105">*出張費要求* は、出張のために発生する経費を一覧にする元伝票です。</span><span class="sxs-lookup"><span data-stu-id="a52fe-105">A *travel requisition* is a source document that lists the expenses that will be incurred for the purpose of travel.</span></span> <span data-ttu-id="a52fe-106">出張費要求は、レビュー用に送信され、次に経費を承認するために使用できます。</span><span class="sxs-lookup"><span data-stu-id="a52fe-106">A travel requisition is submitted for review and can then be used to authorize expenses.</span></span>

<span data-ttu-id="a52fe-107">組織では、従業員が組織に請求される経費が発生する前に、出張費要求を送信する必要がある場合があります。</span><span class="sxs-lookup"><span data-stu-id="a52fe-107">Your organization might require that a travel requisition be submitted before employees incur any expense that is charged to the organization.</span></span> <span data-ttu-id="a52fe-108">この要求は、従業員が会社のクレジット カードに経費を付けるか、現金前貸しから受けた現金を使用するか、または組織で払戻される現金支払経費を負うかどうかにかかわりなく、適用されます。</span><span class="sxs-lookup"><span data-stu-id="a52fe-108">This requirement applies, regardless of whether employees charge expenses to a corporate credit card, spend cash that they received from a cash advance, or incur out-of-pocket expenses that will be reimbursed by the organization.</span></span>

<span data-ttu-id="a52fe-109">出張費要求およびポリシーは、予算管理で役立てるために使用できます。</span><span class="sxs-lookup"><span data-stu-id="a52fe-109">Travel requisitions and policies can be used to help with budget control.</span></span> <span data-ttu-id="a52fe-110">たとえば、組織が、出張が必要な固定価格プロジェクトで作業している場合、プロジェクト チームのメンバの旅行経費は、プロジェクトの予算内に合わせる必要があります。</span><span class="sxs-lookup"><span data-stu-id="a52fe-110">For example, if your organization is working on a fixed-price project that requires travel, the travel expenses of the project's team members must fit within the budget for the project.</span></span> <span data-ttu-id="a52fe-111">旅行経費が発生する前に承認するように要求すると、組織はプロジェクトが予算内であることを確認することが可能になります。</span><span class="sxs-lookup"><span data-stu-id="a52fe-111">By requiring that travel expenses be approved before they are incurred, the organization can help make sure that the project remains within budget.</span></span>

<span data-ttu-id="a52fe-112">**出張費要求**ぺージで、出張費要求のポリシーを設定できます。</span><span class="sxs-lookup"><span data-stu-id="a52fe-112">You can set up policies for travel requisitions from the **Travel requisition** page.</span></span>
