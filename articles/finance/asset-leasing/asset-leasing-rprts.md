# <a name="asset-leasing-reports"></a>資産リース レポート

[!include [banner](../includes/banner.md)]

このトピックでは、資産リースで利用可能なレポートを一覧表示し、その内容について簡単な説明をしていきます。 ほとんどのレポートは、以下の手順、または似通った手順を完了することで表示されます。 

- ほとんどの資産リースのレポートを表示するには、**資産リース  > 照会とレポート > リースレポート** に移動し て、表示するレポートを選択します。 異なる選択パスが必要となるレポートの場合は、そのレポートの説明にレポートを開く手順が含まれています。 
- 印刷するレポートを選択すると、パラメーター ページが開き、レポートに含まれる情報をフィルター処理できます。 フィルター条件を入力し、**OK** をクリックしてレポートを生成します。 生成されるレポートには、指定したフィルター内の情報が表示されます。

## <a name="asset-movement"></a>資産の移動
資産の移動レポートは、各リースの使用権資産の資産残高のロールフォワードレポートとして機能します。 このレポートでは、指定された期間のリースの資産取引を確認することができます。 このレポートには、次のフィールドが含まれています。 

|     レポート フィールド                  |     説明                                                                |
|------------------------------------|--------------------------------------------------------------------------------|
|     開始日              |     リースの最初のバージョンの開始日。                     |   
|     リース期間                     |     リース期間の現在のバージョン。                            |
|     短期リース               |     リースが短期リースに分類される場合は、**はい** と表示されます。         |
|     低額リース                |     リースが低額リースに分類される場合は、**はい** と表示されます。          |
|     当初使用権資産     |     当初認識仕訳からの使用権資産の元値を記載しています。      |
|     期首残高              |     **開始日** 時点での使用権資産の正味簿価額です。                         |
|     期間減価償却費    |     レポートに定義された日付の範囲内で取得した減価償却費の金額です。        |
|     減価償却累計額       |     **開始日** 時点での減価償却累計額です。                               |
|     減損                     |     レポートに定義された日付の範囲内で減損処理を行った金額です。               |
|     変更                  |     日付パラメータ間の使用権資産の調整金額です。                            |
|     正味帳簿価格                 |     使用権資産の期末正味簿価額で、**終了日** 時点での減価償却の累計額です。    |

## <a name="differences-report"></a>差異レポート
差異レポートには、リースのインポートフレームワークに読み込まれた情報と、現在システムに含まれている情報との違いが表示されます。 これにより、リースのインポート フレームワークを使用して、調整、更新、追加されたものを比較できます。  

レポートの値は、選択したリースによって異なります。 レポートでは、現在システムにあるものとステージング テーブルにあるものが異なるフィールドのみが表示されます。 元の値は、インポートプロセスが実行されたかどうかに応じて、現在システムに存在する値、または以前にシステムに存在していた値となります。 新しい値は、元の値を置き換えるステージング テーブルの値を示します。

## <a name="five-years-undiscounted-payment-forecast"></a>5 年割引前の支払予測
5 年間の割引前支払額予測レポートは、レポートのパラメーターで指定した日付から今後 5 年間に支払われる割引前のリース料の予測を表示します。 このレポートには、次のフィールドが含まれています。 

|     レポート フィールド         |     説明                                                                                       |
|-------------------------- |---------------------------------------------------------------------------------------------------    |
|     リースの説明     |     リース ヘッダーのリースの説明。                                                      |
|     リース ID              |     リースの固有 ID。                                                                              |
|     予約                  |     帳簿 ID に関連付けられているリース帳簿。                                                         |
|     分類        |     リースの分類。                                                                  |
|     転記階層         |     このリースが転記される階層。                                                         |
|     通貨              |     リースの通貨。                                                                        |
|     現行               |     レポートの日付けから 12 ヶ月以内に支払われるすべてのリース料の合計。          |
|     13-24 か月          |     レポートの日付けから 13 から 24 ヶ月以内に支払われるすべてのリース料の合計。           |
|     25-36 か月          |     レポートの日付けから 25 から 36 ヶ月以内に支払われるすべてのリース料の合計。           |
|     37-48 か月          |     レポートの日付けから 37 から 48 ヶ月以内に支払われるすべてのリース料の合計。           |
|     49-60 か月          |     レポートの日付けから 49 から 60 ヶ月以内に支払われるすべてのリース料の合計。           |
|     以降            |     レポートの日付けから 61 ヶ月またはそれ以降に支払われるすべてのリース料の合計。              |

## <a name="gaap-cash-flows-report"></a>GAAP キャッシュ フロー レポート
GAAP 開示レポートは 842-20-50-4 (g) (1) で指定されている米国 GAAP の開示要件を満たします。 このレポートを表示するには、**資産リース > 照会とレポート > 開示 > GAAP - キャッシュフロー** にアクセスします。 

|     レポート フィールド                                 |     説明                                                                                                                                               |
|------------------------------------------------   |-----------------------------------------------------------------------------  |
|     自 <br> 最終日                        |     レポートに含まれる情報の制限に使用する日付範囲を定義します。      |
|     法人                                  |     リースに関連付けられている法人。                                      |
|     リース タイプ                                    |     財務またはオペレーティングのいずれかのリースの分類。           |
|     リース ID                                      |     リースの固有 ID。                                                      |
|     リースの説明                             |     リース ヘッダーのリースの説明。                              |
|     リース帳簿                                    |     明細行が参照しているリース帳簿。                        |
|     転記階層                                 |     固定資産に関連付けられる各帳簿は、全体的な減価償却の目的を持つ特定の転記階層に対して設定されます。                          |
|     リース グループ                                   |     リースが結びつけられているグループ。                                     |
|     通貨                                      |     使用する取引通貨の略称。 すべてのレポートは、取引通貨をレポート通貨に変換します。                      |
|     ファイナンス リース - 運営キャッシュフロー         |     各日付パラメーターの間の償却スケジュールから転記された変動支払総額と利息支払総額の合計。        |
|     ファイナンス リース - ファイナンス キャッシュフロー           |     各日付パラメーター間の、償却スケジュールからの総元金支払の合計値。       |
|     オペレーティング リース - 運営キャッシュフロー       |     転記されたすべてのリース支払と、各日付パラメーターの間で転記された変動支払の合計値。            |

## <a name="lease-balances-forecast"></a>リース残高予測
リース残高の予測は、負債償却スケジュールと資産減価償却スケジュールから直接取得されます。 このレポートには、予測されたリース負債の予測金額と、それらのリースに対するすべての予測経費を含めた、一定期間の使用権資産が表示されます。 このレポートには、次のフィールドが含まれています。

|     レポート フィールド                 |     説明                                                                                                                                                                               |
|---------------------------------  |--------------------------------------------------------------------------------------------------------------------   |
|     期首残高             |     レポートの開始日を含む期間のリースの償還スケジュールの期首残高。            |
|     初度負債計上           |     リース開始日がレポートに定義された日付の範囲内の場合、この列には、当初の認識仕訳入力の負債勘定の値が表示されます。      |
|     支払利息                      |     レポートに定義された日付範囲内にあるリース料の合計。                               |
|     関心事項                      |     レポートに記載された日付範囲内にあるリース支払利息の合計。                    |
|     期末残高                |     **終了日** 時点でのリースの負債残高。                                                             |
|     短期負債          |     リースの償却スケジュールにおける、**終了日** 時点での短期債務額。                           |
|     長期負債           |     リースの償却スケジュールにおける、**終了日** 時点での長期債務額。                            |
|     期首正味簿価額      |     レポートの開始日を含む期間のリース資産減価償却表の「期首正味簿価額」を記載しています      |
|     初度負債計上           |     リース開始日がレポートの日付パラメーターの範囲内の場合、この列には、当初の認識仕訳の資産勘定の値が表示されます。            |
|     減価償却費          |     レポートにに記載された日付範囲内にあるリース減価償却の合計。                  |
|     期末残高                |     **終了日** 時点でのリースの資産残高。                                                              |
|     株主資本調整             |     期首残高から期首正味簿価額を差し引いた値。                                                             |

## <a name="lease-commencements-report"></a>リース開始レポート
リース開始レポートには、当初の負債と使用権資産の残高を含む、指定された日付の範囲内に開始したすべてのリースが表示されます。 このレポートには、次のフィールドが含まれています。 

|     レポート フィールド                 |     説明                                                                                       |
|---------------------------------  |---------------------------------------------------------------------------------------------------    |
|     開始日             |     そのリース バージョンに対して転記された当初の認識仕訳入力の日付。         |
|     初期負債額      |     当初の認識仕訳入力からの負債金額。                                  |
|     初期資産額          |     当初の認識仕訳入力からの資産金額。                                      |

## <a name="lease-modification-report"></a>リース変更レポート
リース変更レポートには、指定した日付範囲内で変更されたすべてのリースが表示されます。 また、リースを調整したユーザーと調整した負債の総額も含まれます。 このレポートには、次のフィールドが含まれています。 

|     レポート フィールド                 |     説明           |
|---------------------------------  |-------------------------  |
|     調整したユーザー                   |     リースを変更したユーザーのユーザー名。                                |
|     調整日               |     仕訳入力の修正が転記された日付。                        |
|     調整                   |     日付パラメーターの間で、リースの負債勘定に転記された調整仕訳入力の金額。 貸方は正の値として表示され、借方は負になります。       |
|     利益 (損失) 額            |     日付パラメーターの間で、リースの利益/損失勘定に転記された調整仕訳入力の金額。 貸方は正の値として表示され、借方は負になります。       |
|     利益剰余金             |     日付パラメーターの間で、リースの利益剰余金勘定に転記された調整仕訳入力の金額。      |
|     期末負債残高      |     リース調整日時点の債務残高。           |

## <a name="lease-movement-report"></a>リース移動レポート
リースの移動レポートは、各リースのリース負債の資産残高のロールフォワードレポートとして機能します。 このレポートでは、指定された期間のリースの負債取引を確認することができます。

|     レポート フィールド             |     説明                                               |
|----------------------------   |-------------------------------------------------------------- |
|     開始日         |     リースの最初のバージョンの開始日。    |
|     リース期間                |     リースの最も古いバージョンのリース期間。           |
|     支払残余        |     リースに対してまだ転記されていない支払の数。                       |
|     期首残高         |     レポートの開始日より前に発生したリースの負債に影響するすべてのトランザクションの合計。             |
|     初度負債計上       |     レポートに対して定義されている日付範囲内に転記されたリースの初期認識トランザクションの金額。     |
|     支払利息                  |     レポートに定義された日付の範囲内に転記されたリース支払トランザクションの合計。 この列の値は、支払によってリースの債務残高が減少しているため、マイナスの金額として表示されます。  |
|     関心事項                  |     レポートに定義された日付の範囲内で、リース料に対して転記れた未収利息の合計額を記載しています。            |
|     調整               |     レポートに定義された日付の範囲内で転記されたリース調整取引の合計。                               |
|     期末残高            |     レポートの **終了日** の時点で転記されたすべてのリースの負債トランザクションの残高。                  |

## <a name="lease-transactions-report"></a>リースのトランザクション レポート
リースト ランザクションの照会には、資産リースによって生成されたすべての仕訳入力が表示されます。 各仕訳入力は、元の帳簿 ID に関連付けられています。 これにより、対応するリースに仕訳入力を容易に関連付けることができます。 リース トランザクションの照会機能は、一般会計の **伝票トランザクション** ページと同様の方法で機能します。

## <a name="weighted-average-discount-rate-report"></a>加重平均割引率レポート
加重平均割引率レポートには、加重平均割引率が適用されており、ASC 842-20-50-4(g)(4) で定められている 米国 GAAP の開示要件を満たします。 このレポートを表示するには、**資産リース > 照会とレポート > 開示 > 加重平均割引率** にアクセスします。 このレポートには、次のフィールドが含まれています。 

|     レポート フィールド                     |     説明                                                           |
|------------------------------------   |------------------------------------------------------------------------   |
|     現在の日付                        |     このレポートには、**指定日付時点** パラメーターの日付以前に開始したすべてのリースが含まれます。 このレポートは、開示対象期間の最終日時点で実行する必要があります。      |
|     個法                      |     リースに関連付けられている法人。                           |
|     リース ID                          |     リースの固有 ID。                                                  |
|     リースの説明                 |     リース ヘッダーのリースの説明。                          |
|     予約                              |     表示されているリースの特定の帳簿タイプ。                                                                                                                                            |
|     転記階層                     |     固定資産に関連付けられる各帳簿は、全体的な減価償却の目的を持つ特定の転記階層に対して設定されます。      |
|     リース グループ                       |     リースが属するグループ。                                 |
|     割引率                     |     リース支払の割引に使用されるレート。                             |
|     通貨                          |     使用する取引通貨の略称。 すべてのレポートは、取引通貨をレポート通貨に変換します。  |
|     リース支払残余          |     日付 **時点** の支払スケジュールからの未払のリース支払の合計金額 。            |
|     加重支払残余       |     残りのリース支払に使用した割引率を乗じた額。   |
