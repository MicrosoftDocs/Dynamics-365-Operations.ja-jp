# <a name="manage-resource-competencies"></a><span data-ttu-id="bd99f-103">リソースのコンピテンシーの管理</span><span class="sxs-lookup"><span data-stu-id="bd99f-103">Manage resource competencies</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="bd99f-104">リソースのコンピテンシーは、リソース管理の重要な部分です。</span><span class="sxs-lookup"><span data-stu-id="bd99f-104">Resource competencies are an essential part of resource management.</span></span> <span data-ttu-id="bd99f-105">コンピテンシーを、適切なバランスのスキル、教育、証明書、およびプロジェクト経験のあるリソースを決定するためのベースラインとして使用できます。</span><span class="sxs-lookup"><span data-stu-id="bd99f-105">Competencies can be used as a baseline to determine resources that have the correct balance of skills, education, certification, and project experience.</span></span> <span data-ttu-id="bd99f-106">各リソースに対してこの情報を設定し、定期的に更新する必要があります。</span><span class="sxs-lookup"><span data-stu-id="bd99f-106">You should set up this information for each resource and update it on a regular basis.</span></span> <span data-ttu-id="bd99f-107">このようにして、プロジェクトのリソース割り当て時に特定のリソースのコンピテンシーを合わせると、機能を最大化できます。</span><span class="sxs-lookup"><span data-stu-id="bd99f-107">In this way, you can maximize capabilities when specific resource competencies are matched during project resource assignment.</span></span>

<span data-ttu-id="bd99f-108">[![スキル、証明書、教育、およびプロジェクト経験の例](./media/projectresourcing06-1024x383.jpg)](./media/projectresourcing06.jpg)</span><span class="sxs-lookup"><span data-stu-id="bd99f-108">[![Examples of skills, certifications, education, and project experience](./media/projectresourcing06-1024x383.jpg)](./media/projectresourcing06.jpg)</span></span>

<span data-ttu-id="bd99f-109">次の手順では、リソースのコンピテンシーを設定する方法について説明します。</span><span class="sxs-lookup"><span data-stu-id="bd99f-109">The following procedures explain how to set up some of the competencies for a resource.</span></span>

<span data-ttu-id="bd99f-110">作業者のコンピテンシーを設定するには、[人事管理] の**作業者**リスト ページ、または [プロジェクト管理および会計] で**リソース**リスト ページを使用します。</span><span class="sxs-lookup"><span data-stu-id="bd99f-110">To set up competencies for a worker, you can use either the **Workers** list page in Human resources or the **Resources** list page in Project management and accounting.</span></span> <span data-ttu-id="bd99f-111">次の手順で、[人事管理] の**作業者**リスト ページが使用されます。</span><span class="sxs-lookup"><span data-stu-id="bd99f-111">For the following procedures, the **Workers** list page in Human resources is used.</span></span>

## <a name="set-up-competencies-certificates"></a><span data-ttu-id="bd99f-112">コンピテンシーの設定: 証明書</span><span class="sxs-lookup"><span data-stu-id="bd99f-112">Set up competencies: Certificates</span></span>

1. <span data-ttu-id="bd99f-113">**作業者**リスト ページで、証明書情報を追加する作業者用の明細行を選択します。</span><span class="sxs-lookup"><span data-stu-id="bd99f-113">On the **Workers** list page, select the line for the worker to add certificate information for.</span></span>
2. <span data-ttu-id="bd99f-114">[アクション ウィンドウ] の、**作業者**タブの、**コンピテンシー**グループで、**証明書**を選択します。</span><span class="sxs-lookup"><span data-stu-id="bd99f-114">On the Action Pane, on the **Worker** tab, in the **Competencies** group, select **Certificates**.</span></span>
3. <span data-ttu-id="bd99f-115">**新規**を選択し、それから**証明書タイプ**フィールドで、**PMP** を選択します。</span><span class="sxs-lookup"><span data-stu-id="bd99f-115">Select **New**, and then, in the **Certificate type** field, select **PMP**.</span></span>
4. <span data-ttu-id="bd99f-116">**開始日**フィールドで、**10/1/2015** を選択し、**保存**を選択します。</span><span class="sxs-lookup"><span data-stu-id="bd99f-116">In the **Start date** field, select **10/1/2015**, and select **Save**.</span></span>

## <a name="set-up-competencies-skills"></a><span data-ttu-id="bd99f-117">コンピテンシーの設定: スキル</span><span class="sxs-lookup"><span data-stu-id="bd99f-117">Set up competencies: Skills</span></span>

1. <span data-ttu-id="bd99f-118">**作業者**リスト ページで、前の手順で使用した作業者がまだ選択されていることを確認します。</span><span class="sxs-lookup"><span data-stu-id="bd99f-118">On the **Workers** list page, make sure that the worker that you used in the previous procedure is still selected.</span></span> <span data-ttu-id="bd99f-119">次に、[アクション ウィンドウ] の、**作業者**タブの、**コンピテンシー**グループで、**スキル**を選択します。</span><span class="sxs-lookup"><span data-stu-id="bd99f-119">Then, on the Action Pane, on the **Worker** tab, in the **Competencies** group, select **Skills**.</span></span>
2. <span data-ttu-id="bd99f-120">**新規**を選択します。</span><span class="sxs-lookup"><span data-stu-id="bd99f-120">Select **New**.</span></span>
3. <span data-ttu-id="bd99f-121">**スキル**フィールドで、**プロジェクト管理**を選択します。</span><span class="sxs-lookup"><span data-stu-id="bd99f-121">In the **Skill** field, select **Project management**.</span></span>
4. <span data-ttu-id="bd99f-122">**レベル**フィールドで、**5 専門家**を選択します。</span><span class="sxs-lookup"><span data-stu-id="bd99f-122">In the **Level** field, select **5 Expert**.</span></span>
5. <span data-ttu-id="bd99f-123">**取得日**フィールドで、**1-/14/2014** を選択します。</span><span class="sxs-lookup"><span data-stu-id="bd99f-123">In the **Level date** field, select **1-/14/2014**.</span></span>
6. <span data-ttu-id="bd99f-124">**経験年数**フィールドに、**10** と入力します。</span><span class="sxs-lookup"><span data-stu-id="bd99f-124">In the **Years of experience** field, enter **10**.</span></span>
7. <span data-ttu-id="bd99f-125">**保存**を選択し、ページを閉じます。</span><span class="sxs-lookup"><span data-stu-id="bd99f-125">Select **Save**, and then close the page.</span></span>
