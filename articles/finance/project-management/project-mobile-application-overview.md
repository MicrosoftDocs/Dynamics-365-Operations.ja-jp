# <a name="project-mobile-applications-overview"></a>Project モバイル アプリケーションの概要

[!include [banner](../includes/banner.md)]

プロジェクトの時間に関連付けられている 3 つのモバイル アプリケーションが、Microsoft Dynamics 365 Project Timesheet、プロジェクト時間の入力、および確認用自分のタイムシート/タイムシート (モバイルに最適化) に対して使用可能です。

## <a name="project-timesheet-mobile-app"></a>プロジェクト タイムシート モバイル アプリ

Dynamics 365 プロジェクト タイムシート モバイル アプリを使用して、ユーザーは自分のモバイル デバイスでプロジェクトのタイムシートを送信および承認できます。 このモバイル アプリでは、Dynamics 365 Finance のプロジェクト管理および会計領域でタイムシート機能を表示します。 これはユーザーの生産性と効率を向上させるのに役立ち、プロジェクト タイムシートを適切なタイミングで入力し承認するのにも役立ちます。

## <a name="project-time-entry-workspace"></a>プロジェクト時間入力ワークスペース

プロジェクト時間入力ワークスペースにより、ユーザーはプロジェクトに対して入力でき、時間を節約できます。 ただし、ユーザーはプロジェクト タイムシートを送信または承認できません。 時間入力にのみ使用できます。 このモバイル ワークスペースは、Finance and Operations モバイル アプリケーションで共に使用するためのものです。

## <a name="my-timesheetstimesheets-for-my-review"></a>確認用自分のタイムシート/タイムシート

自分のタイムシート (モバイルに最適化) および確認用タイム シート (モバイルに最適化) は、従業員がモバイル デバイスでプロジェクト タイムシートを作成、送信、および承認できるようにするソリューションを構成します。 作業者は、アプリケーションの URL を通じて Finance の機能にアクセスできます。 このソリューションは、Finance and Operations モバイル アプリケーションを必要としません。
