# <a name="resource-fulfillment-for-planned-resources"></a><span data-ttu-id="9fa63-103">計画されているリソースのリソースの履行</span><span class="sxs-lookup"><span data-stu-id="9fa63-103">Resource fulfillment for planned resources</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="9fa63-104">プロジェクト マネージャは、プロジェクトの必要なリソース ロールを計画できます。</span><span class="sxs-lookup"><span data-stu-id="9fa63-104">A project manager can plan required resource roles for a project.</span></span> <span data-ttu-id="9fa63-105">リソース マネージャーは、**リソースの履行**ページで、これらの計画されているリソースを要求として参照し、実際のリソースを割り当てることができます。</span><span class="sxs-lookup"><span data-stu-id="9fa63-105">The resource manager will see these planned resources as requests on the **Resource fulfillment** page and can assign actual resources.</span></span>

1. <span data-ttu-id="9fa63-106">**すべてのプロジェクト**ページで **XYZ アップグレード フェーズ 2** プロジェクトを選択します。</span><span class="sxs-lookup"><span data-stu-id="9fa63-106">On the **All projects** page, select the **XYZ Upgrade Phase 2** project.</span></span>
2. <span data-ttu-id="9fa63-107">**プロジェクト**を選択し、次に**編集**を選択します。</span><span class="sxs-lookup"><span data-stu-id="9fa63-107">Select **Project**, and then select **Edit**.</span></span>
3. <span data-ttu-id="9fa63-108">**プロジェクト チームとスケジューリング**タブで、**追加**を選択します。</span><span class="sxs-lookup"><span data-stu-id="9fa63-108">On the **Project team and scheduling** tab, select **Add**.</span></span>
4. <span data-ttu-id="9fa63-109">**ロールの追加**ダイアログ ボックスで、**ソフトウェア開発者**ロールを選択します。</span><span class="sxs-lookup"><span data-stu-id="9fa63-109">In the **Add roles** dialog box, select the **Software developer** role.</span></span>
5. <span data-ttu-id="9fa63-110">**保存**を選択し、プロジェクト ページを閉じます。</span><span class="sxs-lookup"><span data-stu-id="9fa63-110">Select **Create**, and then close the project page.</span></span>
6. <span data-ttu-id="9fa63-111">**リソースの履行**ページで、**XYZ アップグレード プロジェクト フェーズ 2** プロジェクトの**ソフトウェア開発者 1** を選択します。</span><span class="sxs-lookup"><span data-stu-id="9fa63-111">On the **Resource fulfillment** page, select **Software developer 1** for the **XYZ Upgrade project Phase 2** project.</span></span>
7. <span data-ttu-id="9fa63-112">作業者を選択し、**割り当て**を選択します。</span><span class="sxs-lookup"><span data-stu-id="9fa63-112">Select a worker, and then select **Assign**.</span></span>
8. <span data-ttu-id="9fa63-113">**ソフトウェア開発者 1** の明細行が、**XYZ アップグレード プロジェクト フェーズ 2** プロジェクトに対して削除されたことを確認します。</span><span class="sxs-lookup"><span data-stu-id="9fa63-113">Verify that the line for **Software developer 1** has been removed for the **XYZ Upgrade project Phase 2** project.</span></span>
9. <span data-ttu-id="9fa63-114">**プロジェクト チームとスケジューリング**タブで、**XYZ アップグレード フェーズ 2** プロジェクトに対して、以前のステップで選択した作業者が、**ソフトウェア開発者**として追加されたことを確認します。</span><span class="sxs-lookup"><span data-stu-id="9fa63-114">On the **Project team and scheduling** tab, for the **XYZ Upgrade Phase 2** project, verify that the worker that you selected in the previous step has been added as **Software developer**.</span></span>
