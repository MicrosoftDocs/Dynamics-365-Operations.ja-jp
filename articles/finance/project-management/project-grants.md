# <a name="project-grants"></a>プロジェクト交付

[!include [banner](../includes/banner.md)]

補助金とは、特定の目的またはプロジェクトに対する金銭的支援です。 通常、補助金の使用には制限があります。 プロジェクト管理および会計で、補助金を入力、追跡し、プロジェクトおよびプロジェクト契約との関係を定義できます。 たとえば、次のタスクを実行できます。

- **プロジェクト** および **プロジェクト契約の詳細** ページへのリンクを介して、プロジェクトおよび資金調達ソースに補助金を関連付けます。
- ある状態から別の状態へ移動するに従い、補助金への変更を入力および追跡します。
- 原価、要求された金額、および支給される額を入力および追跡します。
- マスタ補助金を作成し、下位補助金をそれらと関連付けます。

詳細をすべて新しいレコードに入力して補助金を作成するか、既存の補助金の詳細をコピーし変更できます。

## <a name="create-a-new-grant"></a>新しい交付を作成

1. **プロジェクト管理および会計** \> **補助金** \> **補助金** の順に移動します。
2. **新規** \> **補助金** を選択します。
3. 補助金の詳細ページで、**一般** クイック タブの **補助金 ID** フィールドに、補助金の英数字の ID を入力します。
4. **補助金名** フィールドに、補助金の名前を入力します。
5. **説明** フィールドに、新規補助金に関する詳細を追加します。

    このページのその他のほとんどのフィールドはオプションであり、入力する情報の量は調整できます。

    補助金の詳細 ページの各クイック タブで、指定されている情報について、次の一覧で説明します。

    - **一般** – 補助金の名前、状態、説明、目的、および金額に関する詳細を入力します。
    - **連絡先情報** – 補助金を管理するスタッフ メンバーおよび部門の詳細、また補助金の交付元である補助金の顧客または組織を入力します。 組織が補助金を受け取る通過地点となる法人で、その後別の受け取り法人へ送信するかどうかも指定できます。 補助金を提供する組織の電話番号と電子メール アドレスなどの連絡先情報を追加するためには **追加** をクリックします。
    - **日付と期限** – 補助金と補助金申請に関連する日付を入力します。 たとえば、申請の期日、送信日、補助金が承認または拒否された日付などが含まれます。
    - **関連付けられたプロジェクトおよびプロジェクト契約** – **一般** クイック タブの **補助金の状態** フィールドが **有効** または **交付済み** に設定されている場合、このクイック タブ上でのみ入力できます。 関連するタスクを完了するには、次のいずれかのオプションを選択します。

        - **補助金の調達ソースを追加** - 補助金の新しい資金調達ソースを追加します。 すべての詳細をすぐに入力するか、既定の情報を使用して、後で更新できます。
        - **プロジェクト契約の追加** - プロジェクト契約情報を追加または更新します。
        - **プロジェクトの追加** – プロジェクトの詳細を追加または更新します。

    - **設定** - この情報が必要な場合は、照合資金に関する詳細を入力します。 補助金を与える組織の多くは、受給者が使用する自身のお金やリソースの特定額が、補助金で付与された額と一致することを要求することがあります。 **ローカル プロジェクトまたはトラッキング ID** フィールドに、プロジェクト ID とは異なる ID を入力できます。

        > [!NOTE]
        > 補助金の一部が別の組織に交付される場合は、**下位交付者** オプションを **はい** に設定します。 米国で交付された補助金については、補助金が州や連邦命令で交付されたものかどうかを指定できます。

    - **引き出しの詳細** – 補助金を引き出す、請求する、または費やすことができる頻度に関する情報を追加または更新します。

## <a name="create-a-new-grant-from-a-copy"></a>コピーからの新しい補助金の作成

1. **プロジェクト管理および会計** \> **補助金** \> **補助金** の順に移動します。
2. **新規** \> **補助金からコピー** の順に選択します。
3. 新しい補助金の英数字の ID と名前を入力し、**OK** を選択します。
4. 交付の詳細 ページで、コピーした補助金の詳細を確認し、必要な変更を行います。 ページのほとんどのフィールドはオプションです。

## <a name="view-or-modify-grant-details"></a>補助金の詳細の表示または変更

1. **プロジェクト管理および会計** \> **補助金** \> **補助金** の順に移動します。
2. 変更する補助金を選択します。
3. アクション ウィンドウで、**補助金** タブの **管理** グループで、**編集** を選択します。
4. 補助金の詳細を確認し、必要な変更を行います。
