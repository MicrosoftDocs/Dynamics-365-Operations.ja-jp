# <a name="transfer-a-quotation-to-a-project"></a>プロジェクトに見積書を転送する

[!include [banner](../includes/banner.md)]

プロジェクト見積のステータスが **確認済** の場合は、**プロジェクトに移動** ウィザードを使用して、一部またはすべての見積明細行を新規または既存のプロジェクトに転送できます。 

このウィザードを完了するには、次の手順に従います。

1. **プロジェクト管理と会計** > **見積書** > **プロジェクト見積** に移動します。
2. 転送する見積を選択し、**フォローアップ** タブの **変更** グループで 、**プロジェクトへ移動** を選択します。
3. **見積をプロジェクトに転送する** ウイザードに従って、見積をプロジェクトに関連付けて転送します。
