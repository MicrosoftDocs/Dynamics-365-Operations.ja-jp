# <a name="create-an-organization-report-hierarchy"></a>組織レポート階層の作成

[!include [banner](../../includes/banner.md)]

組織をレポートするためのレポート階層を作成するには、この手順を使用します。 この記録は、組織全体のレポートの構造が作成されるまで続行できるように、分析コード階層を使って説明することを目的とします。 この記録では、USP2 デモ データ会社を使用します。

1. [原価会計] > [分析コード] > [分析コード階層] に移動します。
2. [新規] をクリックします。
3. HierarchyTypeComboBox フィールドでは、「分析コード分類階層」を選択します。
    * [分析コード分類階層] を選択します。 [分析コード分類階層] タイプはルールの定義、およびレポート用に使用します。 コスト オブジェクト、コスト要素、および統計分析コードなどのすべての分析コードをサポートします。  
4. [作成] をクリックします。
5. [分析コード階層名] フィールドで、「組織 USP2」を入力します。
6. [分析コード] フィールドで、値を入力または選択します。
    * [コスト センター] を選択します。  
7. [保存] をクリックします。
8. [階層の表示] をクリックします。
9. [新規] をクリックします。
10. [ノード名] フィールドで、「CEO」を入力します。
11. [保存] をクリックします。
12. [新規] をクリックします。
13. [ノード名] フィールドで、「CEO コスト センター」を入力します。
14. [保存] をクリックします。
15. [新規] をクリックします。
16. [ノード名] フィールドで、「地域 East」を入力します。
17. [保存] をクリックします。
18. [新規] をクリックします。
19. 一覧で、選択された行をマークします。
20. [移動元分析コード メンバー] フィールドで、値を入力または選択します。
    * ノードに対応する分析コード メンバーを選択します。  
21. [保存] をクリックします。
22. ツリーで、「組織 USP2\CEO\CEO コスト センター」を選択します。
23. [新規] をクリックします。
24. [ノード名] フィールドで、「地域 West」を入力します。
25. [保存] をクリックします。
26. [新規] をクリックします。
27. 一覧で、選択された行をマークします。
28. [移動元分析コード メンバー] フィールドで、値を入力または選択します。
    * ノードに対応する分析コード メンバーを選択します。  
29. [保存] をクリックします。
30. ツリーで、「組織 USP2\CEO」を選択します。
31. [新規] をクリックします。
32. [ノード名] フィールドで、「CFO コスト センター」を入力します。
33. [保存] をクリックします。
34. [新規] をクリックします。
35. [ノード名] フィールドで、「マーケティング キャンペーン」を入力します。
36. [ノード名] フィールドで、「マーケティング キャンペーン」を入力します。
37. [保存] をクリックします。
38. [新規] をクリックします。
39. 一覧で、選択された行をマークします。
40. [移動元分析コード メンバー] フィールドで、値を入力または選択します。
    * ノードに対応する分析コード メンバーを選択します。  
41. [保存] をクリックします。
42. ツリーで、「組織 USP2\CEO\CFO コスト センター」を選択します。
43. [新規] をクリックします。
44. [ノード名] フィールドで、「取引の表示」を入力します。
45. [保存] をクリックします。
46. [新規] をクリックします。
47. 一覧で、選択された行をマークします。
48. [移動元分析コード メンバー] フィールドで、値を入力または選択します。
    * ノードに対応する分析コード メンバーを選択します。  
49. [保存] をクリックします。
50. ツリーで、「組織 USP2\CEO」を選択します。
51. [ノード名] フィールドで、「CIO コスト センター」を入力します。
52. [保存] をクリックします。
53. [新規] をクリックします。
54. [ノード名] フィールドで、「コール センター」を入力します。
55. [保存] をクリックします。
56. [新規] をクリックします。
57. 一覧で、選択された行をマークします。
58. [移動元分析コード メンバー] フィールドで、値を入力または選択します。
    * ノードに対応する分析コード メンバーを選択します。  
59. [保存] をクリックします。
