# <a name="set-up-withholding-tax-component-groups-for-the-tds-tax-type"></a><span data-ttu-id="952f6-103">TDS 税タイプに対する源泉徴収税コンポーネント グループの設定</span><span class="sxs-lookup"><span data-stu-id="952f6-103">Set up withholding tax component groups for the TDS tax type</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="952f6-104">このトピックでは、**賃料** や **契約社員** など、源泉徴収 (TDS) 税タイプの源泉徴収税コンポーネント グループを設定する方法について説明します。</span><span class="sxs-lookup"><span data-stu-id="952f6-104">This topic explains how to set up withholding tax component groups, such as **Rent** and **Contractor**, for the Tax Deducted at Source (TDS) tax type.</span></span>

1. <span data-ttu-id="952f6-105">**税 \> 設定 \> 源泉徴収税 \> 源泉徴収税のコンポーネント グループ** に移動します。</span><span class="sxs-lookup"><span data-stu-id="952f6-105">Go to **Tax \> Setup \> Withholding tax \> Withholding tax component groups**.</span></span>

    <span data-ttu-id="952f6-106">[![源泉徴収税コンポーネント グループ ページ](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span><span class="sxs-lookup"><span data-stu-id="952f6-106">[![Withholding tax component groups page](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span></span>

2. <span data-ttu-id="952f6-107">**税タイプ** フィールドで、**TDS** を選択して、TDS 税タイプの源泉徴収税コンポーネント グループを設定します。</span><span class="sxs-lookup"><span data-stu-id="952f6-107">In the **Tax type** field, select **TDS** to set up withholding tax component groups for the TDS tax type.</span></span>
3. <span data-ttu-id="952f6-108">アクション ウィンドウで **新規** を選択して、明細行を作成します。</span><span class="sxs-lookup"><span data-stu-id="952f6-108">On the Action Pane, select **New** to create a line.</span></span>
4. <span data-ttu-id="952f6-109">**源泉徴収税コンポーネント グループ** フィールドに、TDS コンポーネント グループの名前を入力します。</span><span class="sxs-lookup"><span data-stu-id="952f6-109">In the **Withholding tax component group** field, enter a name for the TDS component group.</span></span>
5. <span data-ttu-id="952f6-110">**説明** フィールドに説明を入力します。</span><span class="sxs-lookup"><span data-stu-id="952f6-110">In the **Description** field, enter a description.</span></span>
6. <span data-ttu-id="952f6-111">**一般** クイック タブの **ステータス** フィールドで、TDS コンポーネント グループの居住用ステータスを選択します。</span><span class="sxs-lookup"><span data-stu-id="952f6-111">On the **General** FastTab, in the **Status** field, select the residential status of the TDS component group.</span></span> <span data-ttu-id="952f6-112">**居住** および **非居住** オプションがあります。</span><span class="sxs-lookup"><span data-stu-id="952f6-112">The options are **Resident** and **Non-resident**.</span></span>
7. <span data-ttu-id="952f6-113">**セクション コード** フィールドで、TDS コンポーネント グループに適用されるセクション コードを入力します。</span><span class="sxs-lookup"><span data-stu-id="952f6-113">In the **Section code** field, enter the section code that is applied to the TDS component group.</span></span>
8. <span data-ttu-id="952f6-114">ページを閉じます。</span><span class="sxs-lookup"><span data-stu-id="952f6-114">Close the page.</span></span>
