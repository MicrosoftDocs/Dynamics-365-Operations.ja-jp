# <a name="invoice-capture-solution-configuration-groups"></a>Invoice Capture ソリューションの構成グループ

[!include [banner](../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]

ユーザーは、請求書フィールドのリストと法人の手動レビュー設定を、構成グループを使用して管理することができます。 ユーザーは、グループ内のさまざまな法人に対して請求書の構成を設定できます。 同じ構成グループに属するすべての法人は、同じ請求書フィールドと手動レビューの設定を使用します。

## <a name="manage-configuration-groups"></a>コンフィギュレーション グループの管理

アプリケーションを使用して構成グループを管理するには **設定** に移動し、**システム設定 \> 構成グループ コンポーネントの定義** を選択します。

**構成グループの定義** ページのメイン タブに、定義されている構成グループの一覧が表示されます。 既定の構成グループも表示されます。 それぞれの構成グループに対して、以下のアクションが可能です:

- **構成グループのコピー** - 既存のグループを複製することで、新しい構成グループを作成できます。 新しいグループの値は、**グループ名** および **グループの説明** 以外のすべてのフィールドで元のグループと同じになります。 構成グループが複製された後、**OK** を選択します。
- **構成グループの削除** - 構成グループを削除するには、構成グループを選択して、**削除** を選択します。

    > [!NOTE]
    > 既定の構成グループは削除できません。

- **構成グループの ID の編集** - 構成グループの ID を編集するには、**グループ ID** フィールドを選択して値を更新します。 グループ ID には、スペースや特殊文字を含めることはできません。また、20 文字以内で入力する必要があります。

    > [!NOTE]
    > **グループID** フィールド の値 は、一度だけ更新できます。

- **構成グループの説明の編集** - 構成グループの説明を編集するには、**説明** フィールドを選択して値を更新します。
- **手動による確認設定の変更** - ユーザーは、請求書を手動で確認する必要があるかどうかを決定できます。 手動によるレビューの設定を変更するには、**手動でレビューする** オプション を選択します。 次のオプションを使用できます。

    - **常に手動でレビューする** - 構成グループの請求書が常に手動で確認する必要がある場合は、このオプションを選択します。
    - **エラーおよび警告の場合** - エラー メッセージまたは警告メッセージを含む請求書で手動で確認する必要がある場合は、このオプションを選択します。
    - **エラーの場合** - エラー メッセージを含む請求書で手動で確認する必要がある場合は、このオプションを選択します。

- **信頼度スコア設定の変更** - 確信度スコアは、認識された請求書データの正確さを報告するために、Invoice Capture ソリューションが提供するメタデータです。 スコアが高い場合は、認識されるデータの精度が高くなります。 この構成により、ユーザーは、ユーザーは信頼度スコアに基づき、手動レビューが必要なタイミングを定義することができます。 ユーザーは、請求書の信頼度スコアのしきい値を変更できます。 信頼スコアの設定を更新するには、**信頼度スコア** フィールドを選択して値を更新します。

    信頼度スコアには次の 2 つの警告タイプがあります:

    - **警告** - 警告のしきい値に対する信頼度スコアが設定されている請求書フィールドは正しいと見なされます。 警告しきい値は、エラーのしきい値以上、100 未満である必要があります。
    - **エラー** - 信頼度スコアがエラーしきい値を下回る請求書フィールドは、失敗とみなされます。 エラー メッセージがユーザーに表示されます。 エラーのしきい値は 0 (ゼロ) 以上、警告のしきい未満である必要があります。 信頼度スコアが警告しきい値とエラーしきい値の間にある請求書フィールドには、警告メッセージが表示されます。

- **請求書フィールド** - 管理 : サイド バイ サイド ビューアーに表示される請求書フィールドのリストを、ユーザーが管理することができます。 **請求書フィールド管理** タブ で、歯車記号を選択し、追加する請求書フィールドを選択して、**保存** を選択します。 選択したフィールドがリストに追加されます。 一覧から請求フィールドを削除するには、このフィールドで **削除** を選択します。

### <a name="manage-file-filters-optional"></a>ファイル フィルターの管理 (オプション)

ファイル フィルターの管理を使用すると、ユーザーは、受信する請求書ファイルに対する追加のフィルターを定義できます。 フィルター基準を満たさないファイルは受信され、**受信したファイル** の一覧に表示されますが、ファイル検証エラーが表示されます。 この動作は、チャンネルに定義されたフィルターの動作とは異なります。 これらのフィルターでは、条件を満たさないファイルは全く受信されません。 ユーザーは受信したファイルを確認し、各ファイルが無効な請求書であるかどうかを判断し、ファイルを廃止したり、手動で認識して取り込まれた請求書に含めることができます。

Invoice Capture ソリューションがインストールされている場合は、既定のファイル フィルターが定義されます。 このファイル フィルターはグローバルです。 フィルターの設定を変更する場合は、既定のフィルターを更新できます。 フィールドが必須の場合は、**必須** を選択します。 

### <a name="accepted-file-size"></a>対応しているファイル サイズ

使用できるファイル サイズを選択できます。

> [!NOTE]
> 20,480 キロバイト (KB) よりも大きいファイルには対応していません。

### <a name="supported-file-types"></a>対応しているファイル タイプ

現在、次のファイル タイプに対応しています:

- PDF
- PNG
- JPG
- JPEG
- TIF
- TIFF
