# <a name="fixed-assets-integration"></a><span data-ttu-id="46e8c-104">固定資産の統合</span><span class="sxs-lookup"><span data-stu-id="46e8c-104">Fixed assets integration</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="46e8c-105">固定資産は、一般会計、在庫管理、売掛金勘定、および買掛金勘定と統合できます。</span><span class="sxs-lookup"><span data-stu-id="46e8c-105">Fixed assets can be integrated with General ledger, Inventory management, Accounts receivable, and Accounts payable.</span></span> <span data-ttu-id="46e8c-106">また、固定資産を発注書と統合するように設定することもできます。</span><span class="sxs-lookup"><span data-stu-id="46e8c-106">You can also set up Fixed assets so that it is integrated with purchase orders.</span></span>

<a name="general-ledger"></a><span data-ttu-id="46e8c-107">一般会計</span><span class="sxs-lookup"><span data-stu-id="46e8c-107">General ledger</span></span>
--------------

<span data-ttu-id="46e8c-108">一般会計では、すべての固定資産の金額は通常、財務報告に必要な複数の主勘定に集計されます。</span><span class="sxs-lookup"><span data-stu-id="46e8c-108">In General ledger, the value of all fixed assets is typically summarized in multiple main accounts that are required for financial reporting.</span></span> <span data-ttu-id="46e8c-109">ただし、**固定資産** ページでは、多くの固定資産レコードを作成できます。</span><span class="sxs-lookup"><span data-stu-id="46e8c-109">However, on the **Fixed assets** page, you can create many fixed asset records.</span></span> <span data-ttu-id="46e8c-110">これらのレコードには、取得価格、減価償却、および価額評価などの情報を含めることができます。</span><span class="sxs-lookup"><span data-stu-id="46e8c-110">These records can include information such as the acquisition price, depreciation, and valuation.</span></span> <span data-ttu-id="46e8c-111">固定資産のトランザクションを転記するたびに、該当する主勘定が更新されます。</span><span class="sxs-lookup"><span data-stu-id="46e8c-111">Each time that you post a transaction for a fixed asset, the appropriate main accounts are updated.</span></span> <span data-ttu-id="46e8c-112">固定資産の主勘定には常に固定資産の更新金額が表示されます。</span><span class="sxs-lookup"><span data-stu-id="46e8c-112">The main accounts for fixed assets always show the updated value of the fixed assets.</span></span>

<span data-ttu-id="46e8c-113">**固定資産転記プロファイル** ページで、固定資産帳簿トランザクションが転記される主勘定を定義します。</span><span class="sxs-lookup"><span data-stu-id="46e8c-113">On the **Fixed asset posting profiles** page, you define the main accounts that fixed asset book transactions are posted to.</span></span> <span data-ttu-id="46e8c-114">また、各主勘定に転記される固定資産トランザクション タイプも指定します。</span><span class="sxs-lookup"><span data-stu-id="46e8c-114">You also specify the types of fixed asset transactions that are posted to each main account.</span></span> <span data-ttu-id="46e8c-115">一般会計の固定資産に必要な詳細レベルに応じて、固定資産の主勘定のさまざまな組み合わせを作成できます。</span><span class="sxs-lookup"><span data-stu-id="46e8c-115">You can create various combinations of main accounts for fixed assets, depending on the level of detail that you want for fixed assets in the general ledger.</span></span> <span data-ttu-id="46e8c-116">主勘定は、トランザクション タイプ、帳簿、および他の主勘定に基づきます。</span><span class="sxs-lookup"><span data-stu-id="46e8c-116">Main accounts can be based on transaction types, books, and other main accounts.</span></span>

<span data-ttu-id="46e8c-126">得を転記したときの在庫の減少は、在庫管理の **転記** ページで設定した転記ルールによって制御されます。</span><span class="sxs-lookup"><span data-stu-id="46e8c-126">Posting rules that are set up on the **Posting** page in Inventory management control the decrease in inventory when an acquisition is posted.</span></span> <span data-ttu-id="46e8c-127">ただし、固定資産に関連した請求書を転記するときに必ずしも在庫が減少するとは限りません。</span><span class="sxs-lookup"><span data-stu-id="46e8c-127">However, you don’t always decrease inventory when you post invoices that are related to fixed assets.</span></span> <span data-ttu-id="46e8c-128">場合によっては、内部で使用するための固定資産を購入することがあります。</span><span class="sxs-lookup"><span data-stu-id="46e8c-128">In some cases, the fixed assets might be purchased for internal use.</span></span> <span data-ttu-id="46e8c-129">たとえば、販売部門用に購入するラップトップです。</span><span class="sxs-lookup"><span data-stu-id="46e8c-129">An example is a laptop that is purchased for the sales department.</span></span> <span data-ttu-id="46e8c-130">購買注文を処理するときは、再販と内部使用の両方の目的用に設定されている品目を使用できます。</span><span class="sxs-lookup"><span data-stu-id="46e8c-130">When you work with purchase orders, you can use items that are set up for both resale and internal use.</span></span> 

<span data-ttu-id="46e8c-131">品目グループの固定資産に対して特定の入庫勘定と払出勘定を使用する場合は、同じ在庫品目を内部の購入と再販のための在庫の両方に使用できます。</span><span class="sxs-lookup"><span data-stu-id="46e8c-131">If you use specific receipt and issue accounts on item groups for fixed assets, you can use the same inventory item both for internal purchases and as stock for resale.</span></span>
