# <a name="fixed-asset-management-workspace"></a>固定資産管理ワークスペース

[!include [banner](../includes/banner.md)]

**固定資産管理** ワークスペースでは、システムに入力されている固定資産に関連する情報が表示されます。 このワークスペースには、概要ビューと分析ビューが含まれます。 **自分の作業** タブには、概要タイル、固定資産の詳細、および現在の会社内の固定資産の関連情報が表示されます。 ワークスペースの Power BI 分析セクションに、分析を直接追加することもできます。 **分析 - すべての会社** タブでは、Microsoft Power BI の機能を使用して、すべての会社内の固定資産に関連付けられているビジュアルを表示します。

## <a name="my-work"></a>自分の作業

### <a name="summary"></a>概要

**集計** セクションのタイルでは、固定資産の概要を示します。 情報には、まだ取得されていない資産、現在の年度中に取得された資産、および現在の年度中に処分済の資産に関するメトリックが含まれています。 **集計** セクションにも、**固定資産** リストページ、バッチの償却提案、および固定資産仕訳帳にすばやく移動する機能があります。

### <a name="find-fixed-assets"></a>固定資産の検索

**固定資産を見つける** セクションでは、固定資産番号、グループ、名前、場所、または担当者を指定して、簡単に資産を検索することができます。 検索条件に一致するすべての資産は、一覧に表示されます。

リストから、次のページを表示できます。

 - 固定資産の **詳細** ページ
 - 固定資産に関連付けられているすべての帳簿の **帳簿** ページ
 - **固定資産の評価** ページ

### <a name="valuations"></a>評価

**固定資産の評価** ページは、1 つのページの固定資産に関する最も重要な情報、および固定資産に関連付けられているすべての帳簿の詳細について表示されます。 左上にある **残高** オプションは、固定資産に関連付けられている各帳簿の現在の評価を表示します。 集計の値を構成する詳細なトランザクションから値をドリルバックできます。 左上にある **プロファイル** オプションは、固定資産に関連付けられている各帳簿の償却スケジュールを表示します。

**固定資産の評価** ページは、**固定資産管理** ワークスペースまたは **固定資産** リスト ページからアクセスできます。

### <a name="related-information"></a>関連情報

**帳簿の設定** ページ、**固定資産トランザクションの照会** ページ、およびワークスペースの **関連情報** セクションのリンクから複数のレポートに直接移動できます。

### <a name="analytics--all-companies"></a>分析 - すべての会社

**分析** ページでは、システム内にすべての法人の固定資産に関する重要なメトリックスを提供します。 このタブへのアクセスは、すべての会社のセキュリティ権限の固定資産の表示によって制御されます。

次の表に、各レポートページで使用可能な表示機能を示します。

| レポート ページ            | 視覚化        |
|------------------------|----------------------|
| 固定資産の評価 | 正味帳簿価格の合計 |
| 固定資産の評価 | 固定資産グループからの正味帳簿価格 |
| 固定資産の評価 | 取得金額 |
| 固定資産の評価 | 処分金額 |
| 固定資産の評価 | 固定資産の詳細 |
| 評価マップ        | 正味帳簿価格の合計 |
| 評価マップ        | 固定資産の場所 |
| 評価マップ        | 固定資産の詳細 |

データを使用して分析を表示するには、まず、**エンティティ格納** ページで AssetTransactionMeasure 集計測定を更新する必要があります。
