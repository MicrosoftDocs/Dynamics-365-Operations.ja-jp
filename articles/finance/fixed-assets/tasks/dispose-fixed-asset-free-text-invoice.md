# <a name="dispose-of-a-fixed-asset-using-a-free-text-invoice"></a><span data-ttu-id="b1d81-103">自由書式の請求書を使用した固定資産の処分</span><span class="sxs-lookup"><span data-stu-id="b1d81-103">Dispose of a fixed asset using a free text invoice</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="b1d81-104">このトピックでは、自由書式の請求書を使用した固定資産の廃棄方法を説明します。</span><span class="sxs-lookup"><span data-stu-id="b1d81-104">This topic explains how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="b1d81-105">ナビゲーション ウィンドウで、**モジュール > 売掛金勘定 > 請求書 > すべての自由書式の請求書**の順に移動します。</span><span class="sxs-lookup"><span data-stu-id="b1d81-105">In the navigation pane, go to **Modules > Accounts receivable > Invoices > All free text invoices**.</span></span>
2. <span data-ttu-id="b1d81-106">**新規** を選択します。</span><span class="sxs-lookup"><span data-stu-id="b1d81-106">Select **New**.</span></span>
3. <span data-ttu-id="b1d81-107">**顧客口座**フィールドで、値を入力または選択します。</span><span class="sxs-lookup"><span data-stu-id="b1d81-107">In the **Customer account** field, enter or select a value.</span></span>
4. <span data-ttu-id="b1d81-108">既定の**請求書**日付を検証し、該当する場合は編集します。</span><span class="sxs-lookup"><span data-stu-id="b1d81-108">Validate the default **Invoice** date and edit if applicable.</span></span>
5. <span data-ttu-id="b1d81-109">**通貨**などの、残りの既定のヘッダー フィールドを検証し、該当する場合は編集します。</span><span class="sxs-lookup"><span data-stu-id="b1d81-109">Validate remaining default header fields, such as **Currency** and edit if applicable.</span></span>
6. <span data-ttu-id="b1d81-110">**請求明細行**セクションで、**説明**および**主勘定**フィールドを検証します。</span><span class="sxs-lookup"><span data-stu-id="b1d81-110">In the **Invoice lines** section, validate the **Description** and **Main account** fields.</span></span>
7. <span data-ttu-id="b1d81-111">既定の**売上税グループ**および**品目売上税グループ** フィールドを検証します。</span><span class="sxs-lookup"><span data-stu-id="b1d81-111">Validate the default **Sales tax group** and **Item sales tax group** fields.</span></span>
8. <span data-ttu-id="b1d81-112">**単価**または固定資産の売却の**金額**を入力します。</span><span class="sxs-lookup"><span data-stu-id="b1d81-112">Enter the **Unit price** or the **Amount** of the sale of the fixed asset.</span></span>
9. <span data-ttu-id="b1d81-113">**行の詳細**セクションを選択します。</span><span class="sxs-lookup"><span data-stu-id="b1d81-113">Select the **Line details** section.</span></span>  
10. <span data-ttu-id="b1d81-114">**固定資産**フィールドで、値を入力または選択します。</span><span class="sxs-lookup"><span data-stu-id="b1d81-114">Enter or select a value in the **Fixed asset** field.</span></span>
11. <span data-ttu-id="b1d81-115">**投稿**を選択します。</span><span class="sxs-lookup"><span data-stu-id="b1d81-115">Select **Post**.</span></span>
