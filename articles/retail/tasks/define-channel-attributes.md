# <a name="define-channel-attributes"></a>チャネル属性の定義

[!include[task guide banner](../includes/task-guide-banner.md)]

チャンネルおよび製品属性を追加して、チャンネルを強化します。 属性を定義する前に、組織の既定の階層にチャンネルを追加する必要があります。


## <a name="define-channel-product-attribute"></a>チャンネル製品属性を定義します。
1. [小売りとコマース] > [チャンネル設定] > [チャンネル カテゴリと製品属性] に移動します。
2. ツリーで、「Contoso Retail\Fashion\Houston」を選択します。
3. [編集] をクリックします。
4. [カテゴリ階層] フィールドで、値を入力または選択します。
    * ナビゲーション階層は、チャンネル内の製品を移動するために使用されます。  
5. [属性グループ] セクションを展開します。
6. [追加] をクリックします。
    * 属性グループは、製品の詳細を表示し、製品のフィルターを使用する時にも役立ちます。  
7. [名前] フィールドで値を入力または選択します。
8. [属性の表示] をクリックします。
9. [属性] セクションを展開します。
10. [追加] をクリックします。
11. 一覧で、選択された行をマークします。
12. \[AddBtn\](アドボタン) をクリックします。
13. [OK] をクリックします。
14. [保存] をクリックします。
15. ページを閉じます。
16. [製品] セクションを展開します。
17. [追加] をクリックします。
18. 一覧で、選択された行をマークします。
19. [追加] をクリックします。
    * 同様に、複数の製品を追加できます。  
20. [OK] をクリックします。
21. [属性] をクリックします。
22. [チャンネル メディア] セクションを展開します。
23. [編集] をクリックします。
24. [チャンネル] フィールドで、オプションを選択します。
25. [規定チェックボックス] を選択します。
26. [OK] をクリックします。
27. [チャンネルの製品属性] セクションを展開します。
28. [ブランド] フィールドに値を入力します。
    * 同様に、他の属性を設定できます。  
29. [保存] をクリックします。
30. ページを閉じます。
31. [保存] をクリックします。
32. [チャンネル更新の公開] をクリックします。
    * 公開された後、データはチャンネル テーブルと同期するために準備されます。 公開していないなら、変更されたデータはチャンネル テーブルと同期されません。  
33. [OK] をクリックします。
34. [保存] をクリックします。
