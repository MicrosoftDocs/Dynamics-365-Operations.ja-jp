# <a name="design-the-relationships-between-organizational-units"></a>組織単位間のリレーションシップのデザイン

[!include[task guide banner](../includes/task-guide-banner.md)]

この手順では、組織単位間の関係をデザインする方法を説明します。 関係を定義する前に新しい組織の目的を作成する必要があります。または既存の組織の目的を使用することができます。 この手順を完了するのに使用されるデモ データの会社は USRT です。 このタスクは、管理者ロールを対象としています。

1. [組織管理] > [組織] > [組織階層] の順に移動します。
2. [新規] をクリックします。
3. [名前] フィールドに値を入力します。
4. [目的の割り当て] をクリックします。
5. 一覧で、目的のレコードを見つけ、選択します。
6. [追加] をクリックします。
7. 一覧で、目的のレコードを見つけ、選択します。
8. [OK] をクリックします。
    * 組織の目的は、組織のために必要な数だけ選択できます。  
9. 一覧で、目的のレコードを見つけ、選択します。
10. [既定として設定] をクリックします。
11. ページを閉じます。
12. [保存] をクリックします。
13. [表示] をクリックします。
14. [編集] をクリックします。
15. [挿入] をクリックします。
16. [事業単位] をクリックします。
17. 一覧で、目的のレコードを見つけ、選択します。
18. 一覧で、選択された行のリンクをクリックします。
19. [挿入] をクリックします。
20. [小売チャンネル] をクリックします。
21. 一覧で、目的のレコードを見つけ、選択します。
22. 一覧で、選択された行のリンクをクリックします。
    * 組織単位は、必要な数だけ追加できます。  
23. [保存] をクリックします。
24. [閉じる] をクリックします。
25. [発行] をクリックして、ドロップ ダイアログを開きます。
26. [有効日] フィールドに日時を入力します。
27. [有効日] フィールドに日時を入力します。
28. [変更の説明] フィールドで、値を入力します。
29. [発行] をクリックします。
30. [閉じる] をクリックします。
