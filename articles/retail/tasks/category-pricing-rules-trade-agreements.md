# <a name="category-pricing-rules-to-create-trade-agreements"></a>売買契約を作成するためのカテゴリの価格決定ルール

[!include[task guide banner](../includes/task-guide-banner.md)]

この手順は、カテゴリ価格決定ルールを使用して販売価格の売買契約を作成する方法を示します。 このタスクの作成に使用するデモ データの会社は USRT です。 このタスクは [小売販売促進マネージャー ロール] を対象としています。

1. [価格設定および割引の管理] をクリックします。
2. [新規] をクリックします。
3. [カテゴリ価格ルール] をクリックします。
4. 一覧で、選択された行をマークします。
5. [会計コード] フィールドでオプションを選択します。
    * 「グループ」タイプのアカウント コードは [チャンネル]、[ロイヤルティ プログラム]、[カタログ] および [所属] 固有の販売価格の売買契約を設定するために使用されます。  
6. [勘定の選択] フィールドで値を入力または選択します。
7. [カテゴリ] フィールドで値を入力または選択します。
8. [金額 / パーセンテージ] フィールドに数値を入力します。
9. [丸めバージョン] フィールドで値を入力または選択します。
10. [売買契約の生成] をクリックします。
11. [次へ] をクリックします。
12. [開始日] フィールドに日付を入力します。
13. [終了日] フィールドで、日付を入力します。
14. [次を検索] フィールドで、[はい] を選択します。
15. [次へ] をクリックします。
16. [完了] をクリックします。
    * これにより [売買契約仕訳帳] が作成され、確認できるようになります。  
17. 一覧で、目的のレコードを見つけ、選択します。
    * [カテゴリ価格決定ルール] から作成された売買契約仕訳帳は転記されません。 それらを転記する前に生成される価格を確認および編集できます。  
18. [編集] をクリックします。
19. [通貨金額] フィールドに数値を入力します。
20. [転記] をクリックします。
21. [OK] をクリックします。
22. ページを閉じます。
23. ページを閉じます。
24. [カテゴリ価格のルール] タブをクリックします。
    * チャンネルの特定の [カテゴリ価格決定ルール] はこのリストに表示されます。
