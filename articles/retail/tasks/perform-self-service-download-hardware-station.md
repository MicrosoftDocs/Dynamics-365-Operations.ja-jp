# <a name="download-hardware-stations-by-using-self-service"></a>セルフサービスによるハードウェア ステーションのダウンロード

[!include [task guide banner](../includes/task-guide-banner.md)]

この手順では、あらかじめ定義された新しいハードウェア ステーションをダウンロードする方法を説明します。 この手順に使用するデモ データの会社は USRT です。 このタスクを開始する前に、新しいハードウェア ステーションのプロファイルとハードウェア ステーションを作成しておく必要があります。

1. [小売りとコマース] > [チャンネル] > [すべての小売店舗] に移動します。
2. 一覧でヒューストン店を選択します。
3. 一覧で、選択された行のリンクをクリックします。
4. [ハードウェア ステーション] セクションの展開を切り替えます。
5. 一覧で、目的のレコードを見つけ、選択します。
6. [ダウンロード] をクリックします。
