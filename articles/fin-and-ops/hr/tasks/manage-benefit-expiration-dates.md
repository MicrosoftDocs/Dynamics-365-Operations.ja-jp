# <a name="manage-benefit-expiration-dates"></a>給付金の有効期限の管理

[!include [task guide banner](../../includes/task-guide-banner.md)]

この手順では、給付金付を期限切れにするまたは延長する方法、および福利厚生に登録されている作業者の登録日を管理する方法について説明します。 この手順の作成に使用するデモ データの会社は USMF です。


## <a name="benefit-expiration-dates"></a>給付金の有効期限
1. [人事管理] > [福利厚生] > [福利厚生] の順に移動します。
2. [登録された作業者] 情報ボックスを展開します。
3. 一覧で、目的のレコードを見つけ、選択します。
4. [アクション] ウィンドウで、[福利厚生] をクリックします。
5. [給付金の期限切れと延長] をクリックします。
6. [新しい給付金の有効期限] フィールドに日付と時刻を入力します。
7. [期限切れ] をクリックします。
8. [アクション] ウィンドウで、[福利厚生] をクリックします。
9. [給付金の期限切れと延長の結果] をクリックします。
10. 一覧で、選択された行をマークします。
11. 一覧で、[対象の作業者リンク] をクリックします。
12. 一覧で、目的のレコードを見つけ、選択します。
13. クリックして、[個人番号] フィールドのリンク先を表示します。
14. [個人情報] セクションを展開します。
15. [福利厚生] をクリックします。
16. 一覧で、福利厚生を検索し、レコードを選択します。 新しい適用終了日に注意します。
