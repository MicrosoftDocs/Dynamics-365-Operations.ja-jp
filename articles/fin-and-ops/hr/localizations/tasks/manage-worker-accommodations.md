# <a name="manage-worker-accommodations"></a>作業者の特別設備の管理

[!include[task guide banner](../../../includes/task-guide-banner.md)]

この手順では、人間工学に基づいた椅子または定期的な休暇などの作業環境の特別設備タイプを設定するための手順を説明します。 また、作業者にこれらの特別設備タイプを割り当てる方法、および特別設備 タイプの承認または拒否方法を示します。 この手順の作成に使用するデモ データの会社は USMF です。


## <a name="setup-accommodations"></a>特別設備の設定
1. [人事管理] > [設定] > [特別設備 タイプ] の順に移動します。
2. [新規] をクリックします。
3. [特別設備 タイプ] フィールドに、特別設備の名前を入力します。 例: キーボード。
4. [説明] フィールドに、特別設備の説明を入力します。 例: 人間工学に基づいたキーボード。
5. [メモ] フィールドで、特別設備を明確にするのに役立つ情報を入力します。
6. [保存] をクリックします。
7. ページを閉じます。

## <a name="assign-accommodations"></a>特別設備の割り当て
1. [人事管理] > [作業者] > [従業員] の順に移動します。
2. リストから、従業員を選択します。
3. 特別設備を要求している従業員に関する詳細を表示するには、従業員の名前をクリックします。
4. [個人情報] クイック タブを展開します。
5. [特別設備] をクリックします。
6. [新規] をクリックします。
7. [特別設備 タイプ] フィールドで、適切な特別設備を選択します。 例: キーボード
8. [職務] フィールドに特別設備を要求する作業タスクを入力します。
9. [ステータス] フィールドに適切なステータスを入力します。 例: 適当
    * 次のステータスがあります。 要求済は、特別設備が作成されているが校閲されていないことを示します。 適当は、特別設備が妥当で許可されることを示します。 不適当な状態は、特別設備が雇用主に不適当な負担をかける可能性があり、拒否されていることを示します。 この例では、特別設備要求が最小であったため、要求はすぐに承認されました。  
10. [受入済] フィールドで、特別設備要求を受け入れた従業員を選択します。
11. [選択] をクリックします。
12. [受入日] フィールドに、特別設備要求が承受け入れられた日時を入力します。
13. [メモ] セクションを展開します。
14. [財務] のフィールドに、特別設備の影響を明確にするのに役立つ情報またはリソース コストを入力します。
    * 特別設備が拒否されている場合、[返信] フィールドは要求が拒否された理由を示すために使用できます。  
15. [保存] をクリックします。
