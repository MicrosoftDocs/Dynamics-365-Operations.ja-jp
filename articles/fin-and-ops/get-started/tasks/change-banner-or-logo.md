# <a name="change-the-banner-or-logo"></a><span data-ttu-id="d64c3-103">バナーまたはロゴの変更</span><span class="sxs-lookup"><span data-stu-id="d64c3-103">Change the banner or logo</span></span>

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="d64c3-104">次の手順は、システム管理者が法人用に表示するバナーまたはロゴ画像を更新するために使用できるステップを一覧にしています。</span><span class="sxs-lookup"><span data-stu-id="d64c3-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="d64c3-105">この手順の作成に使用するデモ データの会社は USMF です。</span><span class="sxs-lookup"><span data-stu-id="d64c3-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="d64c3-106">[組織管理] > [組織] > [法人] の順に移動します。</span><span class="sxs-lookup"><span data-stu-id="d64c3-106">Go to Organization administration > Organizations > Legal entities.</span></span>
2. <span data-ttu-id="d64c3-107">左側のリストで、バナーまたはロゴを更新する法人を選択します。</span><span class="sxs-lookup"><span data-stu-id="d64c3-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="d64c3-108">これが既に選択されている場合、次の手順を実行します。</span><span class="sxs-lookup"><span data-stu-id="d64c3-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="d64c3-109">[画像] セクションを展開します。</span><span class="sxs-lookup"><span data-stu-id="d64c3-109">Expand the Images section.</span></span> <span data-ttu-id="d64c3-110">これが既に展開されている場合、次の手順を実行します。</span><span class="sxs-lookup"><span data-stu-id="d64c3-110">If it is already expanded, go to the next step.</span></span>
4. <span data-ttu-id="d64c3-111">[編集] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="d64c3-111">Click Edit.</span></span>
5. <span data-ttu-id="d64c3-112">[ダッシュボードの会社の画像タイプ] フィールドで、「バナー」を選択します。</span><span class="sxs-lookup"><span data-stu-id="d64c3-112">In the Dashboard company image type field, select 'Banner'.</span></span>
6. <span data-ttu-id="d64c3-113">[変更] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="d64c3-113">Click Change.</span></span>
    * <span data-ttu-id="d64c3-114">ヒント : バナーの画像の理想的な解像度は 1920 x 281 ピクセルです。</span><span class="sxs-lookup"><span data-stu-id="d64c3-114">Tip: The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="d64c3-115">ロゴの画像の理想的な幅は 350 ピクセルです。</span><span class="sxs-lookup"><span data-stu-id="d64c3-115">The ideal width for a logo image is 350 pixels.</span></span>  
    * <span data-ttu-id="d64c3-116">アップロードが完了すると、[ダッシュボードの画像] セクションに新しいバナーまたはロゴの画像が表示されているはずです。</span><span class="sxs-lookup"><span data-stu-id="d64c3-116">When the upload is complete, you should see your new banner or logo image in the Dashboard Image section.</span></span>  
7. <span data-ttu-id="d64c3-117">[保存] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="d64c3-117">Click Save.</span></span>
8. <span data-ttu-id="d64c3-118">[共通] > [共通] > [既定のダッシュボード] に移動します。</span><span class="sxs-lookup"><span data-stu-id="d64c3-118">Go to Common > Common > Default dashboard.</span></span>
    * <span data-ttu-id="d64c3-119">ダッシュボードに新しいバナーまたはロゴの画像が表示されているはずです。</span><span class="sxs-lookup"><span data-stu-id="d64c3-119">You should see the new banner or logo image on the dashboard.</span></span>  
    * <span data-ttu-id="d64c3-120">会社のピッカーを使用して、バナーをアップロードしたものに、会社を変更する必要があるかもしれません。</span><span class="sxs-lookup"><span data-stu-id="d64c3-120">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
