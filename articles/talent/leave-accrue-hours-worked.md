# <a name="accrue-time-off-based-on-hours-worked"></a><span data-ttu-id="c9623-103">作業時間に基づいて休暇を見越計上する</span><span class="sxs-lookup"><span data-stu-id="c9623-103">Accrue time off based on hours worked</span></span>

## <a name="overview"></a><span data-ttu-id="c9623-104">概要</span><span class="sxs-lookup"><span data-stu-id="c9623-104">Overview</span></span>

<span data-ttu-id="c9623-105">時給の従業員を持つ組織は、組織での勤続年数ではなく、作業時間に基づいて休暇を付与することができます。</span><span class="sxs-lookup"><span data-stu-id="c9623-105">Organizations with hourly employees can award time off based on hours worked instead of tenure with the organization.</span></span> <span data-ttu-id="c9623-106">作業時間のデータは通常、時間と出勤に関するシステムに格納されます。</span><span class="sxs-lookup"><span data-stu-id="c9623-106">The hours worked data is typically stored in a time and attendance system.</span></span> 

## <a name="leave-plans"></a><span data-ttu-id="c9623-107">休暇計画</span><span class="sxs-lookup"><span data-stu-id="c9623-107">Leave plans</span></span>

<span data-ttu-id="c9623-108">休暇計画では、見越計上タイプは勤続月数か作業時間のどちらでも可能です。</span><span class="sxs-lookup"><span data-stu-id="c9623-108">Within the leave plan, the accrual type can either be months of service or hours worked.</span></span> <span data-ttu-id="c9623-109">作業時間を選択した場合は、見越計上に使用する時間のタイプは 2 つあります: 通常および残業です。</span><span class="sxs-lookup"><span data-stu-id="c9623-109">When hours worked is selected, there are two types of hours to use for the accural: regular and overtime.</span></span>

<span data-ttu-id="c9623-110">作業時間を使用する休暇計画を設定するには、次の手順を実行します:</span><span class="sxs-lookup"><span data-stu-id="c9623-110">To set up a leave plan to use hours worked, follow these steps:</span></span>

1. <span data-ttu-id="c9623-111">**休暇計画**のページで、**新しい計画の作成**をクリックします。</span><span class="sxs-lookup"><span data-stu-id="c9623-111">On the **Leave and absence plans** page, click **Create new plan**.</span></span>
2. <span data-ttu-id="c9623-112">休暇計画の名前を入力します。</span><span class="sxs-lookup"><span data-stu-id="c9623-112">Enter a name for your leave plan.</span></span>
3. <span data-ttu-id="c9623-113">計画の見越計上の発生頻度を選択します。</span><span class="sxs-lookup"><span data-stu-id="c9623-113">Select the accrual frequency for the plan.</span></span>
5. <span data-ttu-id="c9623-114">計画の開始日を選択します。</span><span class="sxs-lookup"><span data-stu-id="c9623-114">Select the start date for the plan.</span></span>
6. <span data-ttu-id="c9623-115">見越計上期間の基準を選択し、該当する場合は、従業員固有の日付を選択します。</span><span class="sxs-lookup"><span data-stu-id="c9623-115">Choose the accrual period basis and select the employee-specific date, if applicable.</span></span>
7. <span data-ttu-id="c9623-116">見越計上スケジュールに関しては、見越計上タイプの**作業時間**を選択します。</span><span class="sxs-lookup"><span data-stu-id="c9623-116">For the accrual schedule, select **Hours worked** for the accrual type.</span></span>
8. <span data-ttu-id="c9623-117">見越計上に使用する時間のタイプを選択します。</span><span class="sxs-lookup"><span data-stu-id="c9623-117">Select the type of hours to be used for the accrual.</span></span>
9. <span data-ttu-id="c9623-118">作業時間数、関連付けられている見越計上金額、最低限残高、および最大繰越または交付金額を入力します。</span><span class="sxs-lookup"><span data-stu-id="c9623-118">Enter the number of hours worked and the associated accrual amount, minimum balance, and maximum carryforward or grant amount.</span></span>

<span data-ttu-id="c9623-119">作業時間計画の見越計上処理では、見越計上期間の基準と同時に見越計上の頻度を使用して、見越計上する時間を決定します。</span><span class="sxs-lookup"><span data-stu-id="c9623-119">Accrual processing for hours worked plans uses the accrual frequency, along with the accrual period basis, to determine the hours to be accrued.</span></span>
