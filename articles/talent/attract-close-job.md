# <a name="close-a-job-in-attract"></a>Attract のジョブを閉じる

[!include [banner](includes/banner.md)]

Microsoft Dynamics 365 Talent: Attract、**ジョブがキャンセルされました** または **職位は採用済みです** のいずれかのジョブを閉じることができます。 ジョブはいつでもキャンセルできます。

職位が入力されたことを示す場合、応募者はその職務の申し出を受諾する必要があります。

## <a name="cancel-a-job"></a>ジョブのキャンセル

Attract のジョブはいつでもキャンセルできます。

1. Attract の **ジョブ** タブで、キャンセルするジョブを選択します。

2. **ジョブを閉じる** を選択します。

   ![Attract のジョブを閉じる](./media/attract-close-job.png)

3. **このジョブを閉じる** ダイアログ ボックスで、**キャンセルされたジョブ** を選択してからコメントを追加し、**ジョブを閉じる** を選択します。

   ![職位は採用済としてジョブを閉じる](./media/attract-close-job-as-cancelled.png)

## <a name="close-a-job-as-position-filled"></a>職位は採用済としてジョブを閉じる

少なくとも 1 つの見込顧客がオファー ステージを完了した後に、**職位は採用済み** として、ジョブを閉じることができます。

1. Attract の **ジョブ** タブで、キャンセルするジョブを選択します。

2. **ジョブを閉じる** を選択します。

   ![Attract のジョブを閉じる](./media/attract-close-job.png)

3. **このジョブを閉じる** ダイアログ ボックスで、**職位は採用済み** を選択し、オファーを受けた候補者を選びコメントを追加してから **ジョブを閉じる** を選択します。

   ![職位は採用済としてジョブを閉じる](./media/attract-close-job-as-position-filled.png)

   > [!NOTE]
   > 少なくとも 1 つの見込顧客がオファー ステージを完了したとき、**職位は採用済み** として、ジョブを閉じることしかできません。 職務アプリケーションにて **オファー承諾済み** が選択されていない場合、またはオファーの **承諾済み** の状態が表示されていない場合、**職務は採用済み** オプションは **理由** ドロップダウンに表示されません。
