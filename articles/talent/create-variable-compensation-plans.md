[!include[banner](includes/banner.md)]

### <a name="describe-vesting-rules"></a><span data-ttu-id="29b49-113">給付ルールの説明</span><span class="sxs-lookup"><span data-stu-id="29b49-113">Describe vesting rules</span></span>

## <a name="variable-compensation-plans"></a><span data-ttu-id="29b49-118">変動報酬プラン</span><span class="sxs-lookup"><span data-stu-id="29b49-118">Variable compensation plans</span></span>
<span data-ttu-id="29b49-119">**変動報酬プラン**には、登録された従業員の変動報酬計算のルール、計算方法、および既定値が含まれています。</span><span class="sxs-lookup"><span data-stu-id="29b49-119">The **variable compensation plan** contains the rules, calculation methods, and default values for the calculation of variable compensation for enrolled employees.</span></span> <span data-ttu-id="29b49-120">変動報酬プランを作成する場合、変動報酬タイプを設定する必要があります。</span><span class="sxs-lookup"><span data-stu-id="29b49-120">When you create a variable compensation plan, you must set the variable compensation type.</span></span> <span data-ttu-id="29b49-121">変動報酬タイプは、システムが報奨を通貨金額または単位数のどちらで計算するかを決定します。</span><span class="sxs-lookup"><span data-stu-id="29b49-121">The variable compensation type determines whether the system calculates a currency amount or a number of units as the award.</span></span> <span data-ttu-id="29b49-122">また、次の計算方法を設定する必要があります。</span><span class="sxs-lookup"><span data-stu-id="29b49-122">You must also set the calculation method:</span></span>

-   <span data-ttu-id="29b49-125">**複合** – 報奨金額は、プロセス イベントで指定されたサイクル開始日と終了日の間の従業員固有の固定報酬支払レートごとに計算されます。</span><span class="sxs-lookup"><span data-stu-id="29b49-125">**Composite** – An award amount is calculated for each unique fixed compensation pay rate that the employee had between the cycle start date and the cycle end date on the process event.</span></span> <span data-ttu-id="29b49-126">次に、このレートは最終的な報奨を決定するために合計されます。</span><span class="sxs-lookup"><span data-stu-id="29b49-126">The rates are then added together to determine the final award.</span></span> <span data-ttu-id="29b49-127">たとえば、従業員が、サイクル中に異なる支払レートがある別の職位に異動する場合があります。</span><span class="sxs-lookup"><span data-stu-id="29b49-127">For example, during the cycle, an employee transferred to a different position that had a different pay rate.</span></span> <span data-ttu-id="29b49-128">この場合、変動報奨は、従業員が各支払レートを使用していた時間に合わせて調整されます。</span><span class="sxs-lookup"><span data-stu-id="29b49-128">In this case, the variable award is adjusted for the length of time that the employee had each pay rate.</span></span>

<span data-ttu-id="29b49-129">変動報奨金額は、従業員の通常の基準収入の割合または一連の単位数に基づきます。</span><span class="sxs-lookup"><span data-stu-id="29b49-129">The amount of the variable award can be based on either a percentage of the employee's regular base earnings or a set number of units.</span></span>

-   <span data-ttu-id="29b49-130">既定の割合を入力し、基準が従業員の固定支払レートか従業員の報酬レベルの標準職務給かを指定するために、[基準の割合] オプションを選択します。</span><span class="sxs-lookup"><span data-stu-id="29b49-130">Select the **Percent of basis** option to enter a default percentage, and specify whether the basis should be the employee's fixed pay rate or the control point for the employee's compensation level.</span></span> <span data-ttu-id="29b49-131">報酬レベルは、従業員の職務に設定されます。</span><span class="sxs-lookup"><span data-stu-id="29b49-131">The compensation level is set on the employee's job.</span></span> <span data-ttu-id="29b49-132">報酬構造の基準点の 1 つは、固定報酬プランの標準職務給として設定できます。</span><span class="sxs-lookup"><span data-stu-id="29b49-132">One of the reference points from the compensation structure can be set as the control point on the fixed compensation plan.</span></span> <span data-ttu-id="29b49-133">システムは、従業員の職務の報酬レベルを使用して、従業員の報酬レベルの標準職務給金額を検索する従業員の固定報酬プランに表示される標準職務給とその報酬レベルを相互参照します。</span><span class="sxs-lookup"><span data-stu-id="29b49-133">The system will use the compensation level from the employee's job and cross-reference it with the control point that is listed on the employee's fixed compensation plan to find the control point amount for the employee's compensation level.</span></span> <span data-ttu-id="29b49-134">標準職務給金額は、報奨の基準として、従業員の固定支払レートの代わりに使用されます。</span><span class="sxs-lookup"><span data-stu-id="29b49-134">The control point amount will then be used instead of the employee's fixed pay rate as the basis for the award.</span></span>
-   <span data-ttu-id="29b49-135">報酬プランが現金以外の場合 (たとえば、40 USD の価値がある株 200 単位) は、既定の単位数、各単位の値、単位値の通貨を入力し、報酬プランが賞金の場合は単位数を入力するために、[単位数] オプションを選択します。</span><span class="sxs-lookup"><span data-stu-id="29b49-135">Select the **Number of units** option to enter a default number of units, the value of each unit and the currency of the unit value if the compensation plan is for a non-cash award (for example, 200 units of stock that are valued at 40 USD), or just the number of units if the compensation plan is for a cash award.</span></span> <span data-ttu-id="29b49-136">賞金の場合、従業員は、固定報酬プラン (たとえば、1 USD を 500 単位) に使用するために指定された通貨の単位数を受け取ります。</span><span class="sxs-lookup"><span data-stu-id="29b49-136">For a cash award, the employee will receive the specified number of units of the currency that is used for his or her fixed compensation plan (for example, 500 units of 1 USD).</span></span> <span data-ttu-id="29b49-137">1 対 1 の関係管理が、単位数と単位値の間に直接 1 対 1 のマッピングがあるかどうかを示すために使用されます。</span><span class="sxs-lookup"><span data-stu-id="29b49-137">The one-to-one relationship control can be used to indicate whether there is a direct one-to-one mapping between the number of units and the unit value.</span></span> <span data-ttu-id="29b49-138">単位数を使用して現金ベース プランの変動報酬プランを作成する場合、このオプションは自動的に [有] に固定され、単位値は **1.0000** になります。</span><span class="sxs-lookup"><span data-stu-id="29b49-138">When you create a variable compensation plan for a cash-based plan by using the number of units, this option is automatically locked to **Yes**, and the unit value is **1.0000**.</span></span>

<span data-ttu-id="29b49-140">[レバレッジ] では、従業員の部門の業績に基づいて従業員の報酬を調整することができます。</span><span class="sxs-lookup"><span data-stu-id="29b49-140">**Leverage** lets you to adjust an employee's award, based on the performance of the employee's department.</span></span> <span data-ttu-id="29b49-141">性能測定方法は、[部門] ページの [関連するフォーム] &gt; [報酬] &gt; [業績] で、各部門に設定できます。</span><span class="sxs-lookup"><span data-stu-id="29b49-141">Performance metrics can be set for each department on the **Departments** page, under **Related forms** &gt; **Compensation** &gt; **Performance**.</span></span> <span data-ttu-id="29b49-142">その部門の従業員が受け取る報奨は、部門の業績を示す [目標達成率] フィールドの値により決まります。</span><span class="sxs-lookup"><span data-stu-id="29b49-142">The award that employees in that department receive depends on the value of the **Percent of target achieved** field, which indicates the department's performance:</span></span>
