# <a name="create-a-hiring-team"></a><span data-ttu-id="276fb-103">採用チームの作成</span><span class="sxs-lookup"><span data-stu-id="276fb-103">Create a hiring team</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="276fb-104">Microsoft Dynamics 365 Talent: Onboard では、採用チームを作成できます。</span><span class="sxs-lookup"><span data-stu-id="276fb-104">In Microsoft Dynamics 365 Talent: Onboard, you can create hiring teams.</span></span> <span data-ttu-id="276fb-105">その後、各チームに研修用ガイドとテンプレートを割り当てることができます。</span><span class="sxs-lookup"><span data-stu-id="276fb-105">You can then assign onboarding guides and templates to each team.</span></span>

## <a name="create-a-hiring-team"></a><span data-ttu-id="276fb-106">採用チームの作成</span><span class="sxs-lookup"><span data-stu-id="276fb-106">Create a hiring team</span></span>

1. <span data-ttu-id="276fb-107">左側のメニューで、**チーム**を選択します。</span><span class="sxs-lookup"><span data-stu-id="276fb-107">On the left menu, select **Teams**.</span></span>
2. <span data-ttu-id="276fb-108">**チーム**で、**追加** (プラス記号 \[**+**\]) タイルを選択します。</span><span class="sxs-lookup"><span data-stu-id="276fb-108">Under **Teams**, select the **Add** (plus sign \[**+**\]) tile.</span></span>
3. <span data-ttu-id="276fb-109">**新しいチームの作成**ダイアログ ボックスの**チーム名**で、採用チームの名前を入力します。</span><span class="sxs-lookup"><span data-stu-id="276fb-109">In the **Create a new team** dialog box, under **Team name**, enter a name for the hiring team.</span></span>

    ![[<span data-ttu-id="276fb-110">新しいチームをオンボードで作成する</span><span class="sxs-lookup"><span data-stu-id="276fb-110">Creating a new team in Onboard</span></span>](./media/onboard-create-team.png)](./media/onboard-create-team.png)

4. <span data-ttu-id="276fb-111">**チーム メンバーを選択**で、各チーム メンバーの名前または電子メール アドレスを入力します。</span><span class="sxs-lookup"><span data-stu-id="276fb-111">Under **Choose team members**, enter the name or email address of each team member.</span></span>

    <span data-ttu-id="276fb-112">チーム メンバーを削除するには、ボックス下の名前の横に表示される **X** を選択します。</span><span class="sxs-lookup"><span data-stu-id="276fb-112">To remove team members, select the **X** that appears next to their name under the box.</span></span>

5. <span data-ttu-id="276fb-113">新しいチーム メンバーに電子メールを送信するには、**新しいメンバーに電子メールを送信**オプションをオンにします。</span><span class="sxs-lookup"><span data-stu-id="276fb-113">To email new team members, turn on the **Send an email to the new members** option.</span></span>
6. <span data-ttu-id="276fb-114">**作成**を選択します。</span><span class="sxs-lookup"><span data-stu-id="276fb-114">Select **Create**.</span></span>
7. <span data-ttu-id="276fb-115">新しいチーム メンバーに電子メールを送信するように指定した場合は、次のダイアログ ボックスで電子メールを編集し、**完了**を選択します。</span><span class="sxs-lookup"><span data-stu-id="276fb-115">If you indicated that you want to email new team members, edit the email in the next dialog box, and then select **Done**.</span></span>
