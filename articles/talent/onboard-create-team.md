# <a name="create-a-hiring-team-by-using-dynamics-365-for-talent-onboard"></a><span data-ttu-id="76c60-103">Dynamics 365 for Talent: Onboard を使用した採用チームの作成</span><span class="sxs-lookup"><span data-stu-id="76c60-103">Create a hiring team by using Dynamics 365 for Talent: Onboard</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="76c60-104">Microsoft Dynamics 365 for Talent: Onboard では、採用チームを作成できます。</span><span class="sxs-lookup"><span data-stu-id="76c60-104">In Microsoft Dynamics 365 for Talent: Onboard, you can create hiring teams.</span></span> <span data-ttu-id="76c60-105">その後、各チームに研修用ガイドとテンプレートを割り当てることができます。</span><span class="sxs-lookup"><span data-stu-id="76c60-105">You can then assign onboarding guides and templates to each team.</span></span>

## <a name="create-a-hiring-team"></a><span data-ttu-id="76c60-106">採用チームの作成</span><span class="sxs-lookup"><span data-stu-id="76c60-106">Create a hiring team</span></span>

1. <span data-ttu-id="76c60-107">左側のメニューで、**チーム**を選択します。</span><span class="sxs-lookup"><span data-stu-id="76c60-107">On the left menu, select **Teams**.</span></span>
2. <span data-ttu-id="76c60-108">**チーム**で、**追加** (プラス記号 \[**+**\]) タイルを選択します。</span><span class="sxs-lookup"><span data-stu-id="76c60-108">Under **Teams**, select the **Add** (plus sign \[**+**\]) tile.</span></span>
3. <span data-ttu-id="76c60-109">**新しいチームの作成**ダイアログ ボックスの**チーム名**で、採用チームの名前を入力します。</span><span class="sxs-lookup"><span data-stu-id="76c60-109">In the **Create a new team** dialog box, under **Team name**, enter a name for the hiring team.</span></span>

    ![[<span data-ttu-id="76c60-110">Onboard で新しいチームを作成する</span><span class="sxs-lookup"><span data-stu-id="76c60-110">Creating a new team in Onboard</span></span>](./media/onboard-create-team.png)](./media/onboard-create-team.png)

4. <span data-ttu-id="76c60-111">**チーム メンバーを選択**で、各チーム メンバーの名前または電子メール アドレスを入力します。</span><span class="sxs-lookup"><span data-stu-id="76c60-111">Under **Choose team members**, enter the name or email address of each team member.</span></span>

    <span data-ttu-id="76c60-112">チーム メンバーを削除するには、ボックス下の名前の横に表示される **X** を選択します。</span><span class="sxs-lookup"><span data-stu-id="76c60-112">To remove team members, select the **X** that appears next to their name under the box.</span></span>

5. <span data-ttu-id="76c60-113">新しいチーム メンバーに電子メールを送信するには、**新しいメンバーに電子メールを送信**オプションをオンにします。</span><span class="sxs-lookup"><span data-stu-id="76c60-113">To email new team members, turn on the **Send an email to the new members** option.</span></span>
6. <span data-ttu-id="76c60-114">**作成**を選択します。</span><span class="sxs-lookup"><span data-stu-id="76c60-114">Select **Create**.</span></span>
7. <span data-ttu-id="76c60-115">新しいチーム メンバーに電子メールを送信するように指定した場合は、次のダイアログ ボックスで電子メールを編集し、**完了**を選択します。</span><span class="sxs-lookup"><span data-stu-id="76c60-115">If you indicated that you want to email new team members, edit the email in the next dialog box, and then select **Done**.</span></span>

## <a name="assign-onboarding-guides-to-a-hiring-team"></a><span data-ttu-id="76c60-116">研修用ガイドを採用チームに割り当てる</span><span class="sxs-lookup"><span data-stu-id="76c60-116">Assign onboarding guides to a hiring team</span></span>

2. <span data-ttu-id="76c60-118">チームを選択します。</span><span class="sxs-lookup"><span data-stu-id="76c60-118">Select the team.</span></span>

    ![[<span data-ttu-id="76c60-120">チームへの研修用ガイドの追加</span><span class="sxs-lookup"><span data-stu-id="76c60-120">Adding onboarding guides to a team</span></span>](./media/onboard-add-guides-to-team.png)](./media/onboard-add-guides-to-team.png)

4. <span data-ttu-id="76c60-121">チームに割り当てる各研修用ガイドのチェック ボックスをオンにし、**追加**を選択します。</span><span class="sxs-lookup"><span data-stu-id="76c60-121">Select the check box for each onboarding guide that you want to assign to the team, and then select **Add**.</span></span>

    ![[<span data-ttu-id="76c60-122">研修用ガイドを選択し、チームに追加する</span><span class="sxs-lookup"><span data-stu-id="76c60-122">Selecting the onboarding guides to add to the team</span></span>](./media/onboard-select-guides.png)](./media/onboard-select-guides.png)
