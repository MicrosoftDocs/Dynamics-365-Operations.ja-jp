# <a name="share-content-with-other-contributors-in-dynamics-365-talent---onboard"></a><span data-ttu-id="9d47e-103">Dynamics 365 Talent - Onboard で他の投稿者とコンテンツを共有する</span><span class="sxs-lookup"><span data-stu-id="9d47e-103">Share content with other contributors in Dynamics 365 Talent - Onboard</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="9d47e-104">Microsoft Dynamics 365 Talent: Onboard では、研修用ガイドおよびテンプレートを他の投稿者と共有して、ドキュメントを確認したり変更を加えることができます。</span><span class="sxs-lookup"><span data-stu-id="9d47e-104">Microsoft Dynamics 365 Talent: Onboard lets you share onboarding guides and templates with other contributors so that they can review the documents and make changes.</span></span>

## <a name="share-an-onboarding-guide-with-other-contributors"></a><span data-ttu-id="9d47e-105">他の投稿者と研修用ガイドを共有する</span><span class="sxs-lookup"><span data-stu-id="9d47e-105">Share an onboarding guide with other contributors</span></span>

1. <span data-ttu-id="9d47e-106">左側のメニューで、**ガイド**を選択してから、共有するガイドを選択します。</span><span class="sxs-lookup"><span data-stu-id="9d47e-106">On the left menu, select **Guides**, and then select the guide that you want to share.</span></span>
2. <span data-ttu-id="9d47e-107">ページの右上隅にある**詳細**ボタン (縦の 3 つの点) を選択してから、**共有**を選択します。</span><span class="sxs-lookup"><span data-stu-id="9d47e-107">Select the **More** button (the three vertical dots) in the upper-right corner of the page, and then select **Share**.</span></span>

    ![[<span data-ttu-id="9d47e-108">他の投稿者と研修用ガイドを共有する</span><span class="sxs-lookup"><span data-stu-id="9d47e-108">Sharing an onboarding guide with other contributors</span></span>](./media/onboard-share-guide.png)](./media/onboard-share-guide.png)

3. <span data-ttu-id="9d47e-109">**共有先**で、受信者がガイドを変更できるようにする必要がある場合は、**これらのユーザーは編集できますか**オプションを**オン**に設定します。</span><span class="sxs-lookup"><span data-stu-id="9d47e-109">Under **Share with**, if recipients should be able to revise the guide, set the **Can these people make edits?** option to **On**.</span></span> <span data-ttu-id="9d47e-110">ガイドを共有したことを受信者に電子メールで通知する必要がある場合は、**電子メール通知を受信者に送信する**オプションを**オン**に設定します。</span><span class="sxs-lookup"><span data-stu-id="9d47e-110">If recipients should be notified by email that you've shared the guide with them, set the **Send email notification to recipients** option to **On**.</span></span>

    ![[<span data-ttu-id="9d47e-111">研修用ガイドのアクセスおよび電子メールの基本設定を設定する</span><span class="sxs-lookup"><span data-stu-id="9d47e-111">Setting access and email preferences for an onboarding guide</span></span>](./media/onboard-share-guide-with-contributors.png)](./media/onboard-share-guide-with-contributors.png)

4. <span data-ttu-id="9d47e-112">名前または電子メール アドレスを入力し、ドロップダウン リストで目的の受信者を選択します。</span><span class="sxs-lookup"><span data-stu-id="9d47e-112">Enter a name or email address, and select the desired recipient in the drop-down list.</span></span>
5. <span data-ttu-id="9d47e-113">**アクセスの管理**タブで、受信者がガイドで実行できる操作を確認または変更できます。</span><span class="sxs-lookup"><span data-stu-id="9d47e-113">On the **Manage access** tab, you can review or change what recipients can do with the guide.</span></span> <span data-ttu-id="9d47e-114">受信者のアクセスを変更するには、受信者名の横にあるドロップダウン リストで、必要に応じて**表示可能**、**編集可能**、または**削除**を選択します。</span><span class="sxs-lookup"><span data-stu-id="9d47e-114">To change a recipient's access, in the drop-down list next to the recipient's name, select **Can View**, **Can edit**, or **Remove**, as appropriate.</span></span>
6. <span data-ttu-id="9d47e-115">受信者の追加または変更が完了したら、**更新**を選択します。</span><span class="sxs-lookup"><span data-stu-id="9d47e-115">When you've finished adding or modifying recipients, select **Update**.</span></span>

2. <span data-ttu-id="9d47e-118">**自分のテンプレート**で、共有するテンプレートの右上隅にある**詳細**ボタン (省略記号 \[**...**\]) を選択してから、**共有**を選択します。</span><span class="sxs-lookup"><span data-stu-id="9d47e-118">Under **My templates**, select the **More** button (the ellipsis \[**...**\]) in the upper-right corner of the template that you want to share, and then select **Share**.</span></span>

    <span data-ttu-id="9d47e-119">[![他の投稿者とテンプレートを共有する](./media/onboard-share-template.png)](./media/onboard-share-template.png)</span><span class="sxs-lookup"><span data-stu-id="9d47e-119">[![Sharing a template with other contributors](./media/onboard-share-template.png)](./media/onboard-share-template.png)</span></span>

3. <span data-ttu-id="9d47e-120">**受信者**で、受信者がテンプレートを変更できるようにする必要がある場合は、**これらのユーザーは編集できますか**オプションを**オン**に設定します。</span><span class="sxs-lookup"><span data-stu-id="9d47e-120">Under **Recipient**, if recipients should be able to revise the template, set the **Can these people make edits?** option to **On**.</span></span> <span data-ttu-id="9d47e-121">テンプレートを共有したことを受信者に電子メールで通知する必要がある場合は、**電子メール通知を受信者に送信する**オプションを**オン**に設定します。</span><span class="sxs-lookup"><span data-stu-id="9d47e-121">If recipients should be notified by email that you've shared the template with them, set the **Send email notification to recipients** option to **On**.</span></span>

    [!テンプレートのアクセスおよび電子メールの基本設定を設定する]<span data-ttu-id="9d47e-122">(./media/onboard-share-template-access-and-email.png)](./media/onboard-share-template-access-and-email.png)</span><span class="sxs-lookup"><span data-stu-id="9d47e-122">(./media/onboard-share-template-access-and-email.png)](./media/onboard-share-template-access-and-email.png)</span></span>

4. <span data-ttu-id="9d47e-123">名前または電子メール アドレスを入力し、ドロップダウン リストで目的の受信者を選択します。</span><span class="sxs-lookup"><span data-stu-id="9d47e-123">Enter a name or email address, and select the desired recipient in the drop-down list.</span></span>
5. <span data-ttu-id="9d47e-124">**アクセスの管理**タブで、受信者がテンプレートで実行できる操作を確認または変更できます。</span><span class="sxs-lookup"><span data-stu-id="9d47e-124">On the **Manage access** tab, you can review or change what recipients can do with the template.</span></span> <span data-ttu-id="9d47e-125">受信者のアクセスを変更するには、受信者名の横にあるドロップダウン リストで、必要に応じて**表示可能**、**編集可能**、または**削除**を選択します。</span><span class="sxs-lookup"><span data-stu-id="9d47e-125">To change a recipient's access, in the drop-down list next to the recipient's name, select **Can View**, **Can edit**, or **Remove**, as appropriate.</span></span>
