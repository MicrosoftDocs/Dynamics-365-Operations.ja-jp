# <a name="get-support-for-microsoft-dynamics-365-talent"></a>Microsoft Dynamics 365 Talent に関するサポートの利用

このトピックでは、Dynamics 365 Talent: Attract と Dynamics 365 Talent: Onboard のヘルプを入手する方法について説明します。

## <a name="support-for-talent"></a>Talent に関するサポート

Talent のサポートを得るには、次の表にあるリソースを使用します。 これらのリソースは、トラブルシューティング時に従うよう推奨される順序で一覧表示されています。

| タスク | 詳細情報 |
|------|------------------|
| コミュニティに質問します。 | [Dynamics 365 Talent コミュニティ](https://community.dynamics.com/365/talent) に移動し、質問に関するヘルプを入手します。 回答を検索するには、**質問**のとなりの検索フィールドに質問を入力します。 質問するには、**質問する**を選択します。 |
| クラウドを利用したサポート ツールの使用。 | [Microsoft Dynamics Lifecycle Services (LCS)](https://lcs.dynamics.com/) において、プロジェクトに移動し、**サポート**タイルを選択します。 そこから、2 つの方法で問題を送信することができます。<ul><li>**アクティブな問題** タブで、送信する問題を選択し、**Microsoft に送信**を選択します。</li><li>**Microsoft に送信** タブで、**新しいインシデントを送信** をクリックし、新しい問題を送信します。</li></ul> |
| アプリ内のサポートを取得します。 | ページの右上隅にある**ヘルプ** ボタン (**?**) を選択します。 **問題をレポート**を選択します。 問題は LCS の**アクティブな問題**上で報告され、管理者は社内サポートを提供するか Microsoft に問題を送信するかを決定できます。 (オンボードのスタンドアロン バージョンを使用している場合、問題は Microsoft に直接送信されます。) |
| Talent のライセンスに関する質問のヘルプを入手します。 | パートナー、または Microsoft 販売担当者に問い合わせます。 |
| Talent 内の新機能および機能を要求する。 | Talent のメイン ページで、**ヘルプ** ボタン (**?**) を選択してから、**アイデア**を選択します。 また、[Dynamics 365 アプリケーションのアイデア](https://experience.dynamics.com/ideas/) にアクセスし、既存のアイデアの表示、検索、または投票をしたり、新しいアイデアを追加したりすることもできます。 |

## <a name="support-for-the-onboard-stand-alone-app"></a>Onboard のスタンドアロン アプリのサポート

Onboard のスタンドアロン アプリのサポートを得るには、次の表のリソースを使用します。 これらのリソースは、トラブルシューティング時に従うよう推奨される順序で一覧表示されています。 これらのサポート オプションはすべて、ページの右上隅にある**ヘルプ** ボタン (**?**) を選択することによって参照できます。

| タスク | 詳細情報 |
|------|------------------|
| コミュニティに質問します。 | [Dynamics 365 Talent コミュニティ](https://community.dynamics.com/365/talent) に移動し、ヘルプを入手します。 回答を検索するには、**質問**のとなりの検索フィールドに質問を入力します。 質問するには、**質問する**を選択します。 |
| 問題を報告します。 | ページの右上隅にある**ヘルプ** ボタン (**?**) を選択し、**問題の報告**を選択します。 問題の説明を入力し、その問題をフォローアップしたい場合は、電子メール アドレスを含めます。 |
| Talent のライセンスに関する質問のヘルプを入手します。 | パートナー、または Microsoft 販売担当者に問い合わせます。 |
| Onboard 内の新機能および機能を要求する。 | **ヘルプ** ボタン (**?**) を選択し、**アイデア**を選択します。 また、[Dynamics 365 Talent: Onboard のアイデア](https://experience.dynamics.com/ideas/categories/?forum=569a7fb2-8327-e911-a95a-000d3a4f3883&forumName=Dynamics%20365%20for%20Talent%3A%20Onboard) にアクセスし、既存のアイデアの表示、検索、または投票をしたり、新しいアイデアを追加したりすることもできます。 |
