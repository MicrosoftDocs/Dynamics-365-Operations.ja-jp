# <a name="get-support-for-talent"></a>Talent 用サポートの入手

[!include [banner](includes/banner.md)]

このトピックでは Talent のヘルプを入手する方法について説明します: Core HR、Attract および Onboard。

<a name="contact-support"></a>サポートへの問い合わせ
---------------

Talent に関する技術的な質問がある場合は、このトピックにある情報を使用して Microsoft にお問い合わせください。

### <a name="talent-support---core-hr"></a>Talent サポート - Core HR

Talent のサポートを得るには、次の表にあるリソースを使用します。 これらのリソースは、トラブルシューティングの際に従うことを推奨する順序で一覧表示されています。

| **タスク**                                                | **詳細情報**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
|---------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| コミュニティに質問する                                      | [コミュニティに質問する](https://community.dynamics.com/365/talent) ページに移動して質問のヘルプを検索し、Core HR 内のトピックを検索する “ヘルプ“ オプションを使用します。                                                                                                                                                                                                                                                                                                                                                                                  |
| クラウドを利用したサポート ツールの使用                     | [Lifecycle Services (LCS)](https://lcs.dynamics.com/) では、**サポート**タイルはサポート インシデント管理の支援ツールです。 Microsoft に直接問題を送信するには、LCS プロジェクトの**サポート**タイルに移動、または LCS の**チケット セクションのサポート**を使用して**クラウドを利用したサポート**をクリックします。 そこからは、2 つの方法で問題を送信することができます。 1: LCS を通して問題を送信する場合、**アクティブな問題**タブで問題を選択し、次に **Microsoft に送信**をクリックします。 2: **Microsoft に送信**タブで、**新しいインシデントを送信**をクリックして新しい問題を送信します。 |
| Talent ライセンスに関する質問のヘルプを入手します。         | パートナー、または Microsoft 販売担当者に問い合わせます。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| Talent 内で新機能および機能の要求 | アプリケーションのメイン Talent ページで、**?** アイコンをクリックして**アイデア**を選択、または、[アイデア](https://powerusers.microsoft.com/t5/Ideas-for-Human-Resources/idb-p/HumanResources) リンクを使用して表示、検索、投票、または新しいアイデアを追加します。                                                                                                                                                                                                                                                                                                                      |

### <a name="talent-support--attract-and-onboard"></a>Talent サポート – Attract および Onboard

Attract および Onboard のサポートを得るには、次の表のリソースを使用します。 これらのリソースは、トラブルシューティングの際に従うことを推奨する順序で一覧表示されています。 すべてのサポート オプションは、**?** アイコン、製品の右上隅エリアにの下にあります。

| **タスク**                                                                | **詳細情報**                                                                                                                                                                                                                                                                           |
|-------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| コミュニティに質問する                                                      | [コミュニティに質問する](https://community.dynamics.com/365/talent) ページに移動して質問のヘルプを検索します。                                                                                                                                                                                |
| 問題をレポート                                                        | メニューの **?** アイコンを選択し、問題を報告するオプションを選択します。 説明に問題を入力し、報告している問題のフォローアップしたい場合は、電子メール アドレスを含めます。                                                                                           |
| Talent ライセンスに関する質問のヘルプを入手します。                         | パートナー、または Microsoft 販売担当者に問い合わせます。                                                                                                                                                                                                                                        |
| Attract および Onboard 内で新機能および機能の要求 | **?** アイコンをクリックして**アイデア**を選択、または次のリンクを使用して表示、検索、投票、または新しいアイデアを追加します: [Attract](https://powerusers.microsoft.com/t5/Ideas-for-Attract/idb-p/Attract) または [Onboard](https://powerusers.microsoft.com/t5/Ideas-for-Onboard/idb-p/Onboard)。 |
