# <a name="working-time-calendars"></a><span data-ttu-id="f0b3f-103">作業時間カレンダー</span><span class="sxs-lookup"><span data-stu-id="f0b3f-103">Working time calendars</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="f0b3f-104">作業時間カレンダーを使用すると、組織内で従業員が作業する時間および日数を含むカレンダーを作成できます。</span><span class="sxs-lookup"><span data-stu-id="f0b3f-104">The working time calendar enables you to create a calendar with the hours and days that employees work in your organization.</span></span> <span data-ttu-id="f0b3f-105">カレンダーでは、既定による時間数または日数で休暇申請プロセスを合理化します。</span><span class="sxs-lookup"><span data-stu-id="f0b3f-105">Calendars streamline the time off request process by default in the hours or days.</span></span> <span data-ttu-id="f0b3f-106">従業員が休暇申請を送信する場合、作業時間カレンダーを通じて処理されるため、休日および休業日を考慮する必要はありません。</span><span class="sxs-lookup"><span data-stu-id="f0b3f-106">When an employee submits a time off request, they don't have to worry about holidays and closures, which is handled for them through the working time calendar.</span></span>

## <a name="setting-up-a-working-time-calendar"></a><span data-ttu-id="f0b3f-107">作業時間カレンダーの設定</span><span class="sxs-lookup"><span data-stu-id="f0b3f-107">Setting up a working time calendar</span></span>

<span data-ttu-id="f0b3f-108">カレンダーには生成詳細、含めたい日数および時間、カレンダーの日数、その日数に対する作業時間、および登録された従業員が含まれます。</span><span class="sxs-lookup"><span data-stu-id="f0b3f-108">Calendars include generation details, the days and hours you want to included, the days of the calendar, working times for those days as well as enrolled employees.</span></span> 

<span data-ttu-id="f0b3f-109">カレンダーを設定するには、次の手順に従います。</span><span class="sxs-lookup"><span data-stu-id="f0b3f-109">To set up a calendar, follow these steps:</span></span>

1. <span data-ttu-id="f0b3f-110">**組織管理**ページで、**カレンダー**をクリックします。</span><span class="sxs-lookup"><span data-stu-id="f0b3f-110">On the **Organization Administration** page, click **Calendars**.</span></span>

2. <span data-ttu-id="f0b3f-111">アクション ウィンドウで、**新規**を選択して名前と説明を入力します。</span><span class="sxs-lookup"><span data-stu-id="f0b3f-111">On the Action Pane, select **New** and enter a name and description.</span></span>

3. <span data-ttu-id="f0b3f-112">組織に対する作業日数を選択し、作業時間を入力します。</span><span class="sxs-lookup"><span data-stu-id="f0b3f-112">Choose the work days for your organization and enter work time.</span></span>

4. <span data-ttu-id="f0b3f-113">**追加**ボタンを使用して、休日および休業日を追加します。</span><span class="sxs-lookup"><span data-stu-id="f0b3f-113">Add Holidays and closures by using the **Add** button.</span></span>

5. <span data-ttu-id="f0b3f-114">米国の休日または銀行の休業日など、休日および休業日の名前と説明を入力します。</span><span class="sxs-lookup"><span data-stu-id="f0b3f-114">Enter the name and description for the holidays and closures, such as US holidays or Bank holidays.</span></span> <span data-ttu-id="f0b3f-115">休日および休業日の日付を入力します。</span><span class="sxs-lookup"><span data-stu-id="f0b3f-115">Enter the dates for the holidays and closures.</span></span> <span data-ttu-id="f0b3f-116">休日および休業日リストを保存し、ページを閉じます。</span><span class="sxs-lookup"><span data-stu-id="f0b3f-116">Save the Holidays and closures list and close the page.</span></span>

6. <span data-ttu-id="f0b3f-117">ドロップダウン メニューから、休日および休業日リストを選択します。</span><span class="sxs-lookup"><span data-stu-id="f0b3f-117">Select the holidays and closures list from the drop-down menu.</span></span>

7. <span data-ttu-id="f0b3f-118">従業員に昼食時間、休憩時間などの非作業時間がある場合、それらも追加します。</span><span class="sxs-lookup"><span data-stu-id="f0b3f-118">If your employees have non-work time, such as lunches or breaks, add those as well.</span></span> <span data-ttu-id="f0b3f-119">**非作業時間**を選択し、名前と時間範囲を入力します。</span><span class="sxs-lookup"><span data-stu-id="f0b3f-119">Select **Non-work time** and enter the name and time range.</span></span> <span data-ttu-id="f0b3f-120">ページを閉じます。</span><span class="sxs-lookup"><span data-stu-id="f0b3f-120">Close the page.</span></span> 

8. <span data-ttu-id="f0b3f-121">**追加**をクリックして、カレンダーに非作業時間を追加します。</span><span class="sxs-lookup"><span data-stu-id="f0b3f-121">Click **Add** to add the non-work time to your calendar.</span></span>

9. <span data-ttu-id="f0b3f-122">**日数**タブで、**生成**を選択して、カレンダーに日数を生成します。</span><span class="sxs-lookup"><span data-stu-id="f0b3f-122">On the **Days** tab, select **Generate** to generate the days in your calendar.</span></span> <span data-ttu-id="f0b3f-123">カレンダーに対する日付範囲を入力します。</span><span class="sxs-lookup"><span data-stu-id="f0b3f-123">Enter the date range for the calendar.</span></span> <span data-ttu-id="f0b3f-124">日数および作業時間は、選択した日付と組み合わせて生成オプションの下で定義される作業日および作業時間に基づいて生成されます。</span><span class="sxs-lookup"><span data-stu-id="f0b3f-124">The days and working times are generated based on the work days and times defined under the generation options in conjunction with the dates selected.</span></span>

10. <span data-ttu-id="f0b3f-125">カレンダーを従業員に割り当てるには、アクション ウィンドウで**従業員に割り当て**を選択します。</span><span class="sxs-lookup"><span data-stu-id="f0b3f-125">To assign a calendar to employees, select **Assign to employees** in the Action Pane.</span></span> <span data-ttu-id="f0b3f-126">このカレンダーを割り当てたい従業員を選択してから、**割り当て**をクリックします。</span><span class="sxs-lookup"><span data-stu-id="f0b3f-126">Select the that employees you would like to assign this calendar to, and then click **Assign**.</span></span>

<span data-ttu-id="f0b3f-127">従業員には割り当てられたカレンダーは必要ありません。</span><span class="sxs-lookup"><span data-stu-id="f0b3f-127">Employees aren't required to have calendars assigned.</span></span> <span data-ttu-id="f0b3f-128">作業時間カレンダーが定義されている場合、休暇日は自動的に申請から除外されます。</span><span class="sxs-lookup"><span data-stu-id="f0b3f-128">If there is a working time calendar defined, off days are automatically excluded from the request.</span></span> <span data-ttu-id="f0b3f-129">時間数または日数による期間の既定値は、カレンダーで定義されている作業時間になります。</span><span class="sxs-lookup"><span data-stu-id="f0b3f-129">The amount, in hours or days, defaults to the working times defined in the calendar.</span></span> <span data-ttu-id="f0b3f-130">従業員に割り当てられたカレンダーがない場合、すべての日数は休暇に対して利用可能で、休暇の期間は申請の既定ではありません。</span><span class="sxs-lookup"><span data-stu-id="f0b3f-130">If an employee doesn't have a calendar assigned, all days are available for time off and the amount of time off is not the default for the request.</span></span>
