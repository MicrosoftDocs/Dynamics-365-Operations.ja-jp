# <a name="working-time-calendars"></a>作業時間カレンダー

[!include [banner](includes/banner.md)]

作業時間カレンダーを使用すると、組織内で従業員が作業する時間および日数を含むカレンダーを作成できます。 カレンダーでは、既定による時間数または日数で休暇申請プロセスを合理化します。 従業員が休暇申請を送信する場合、作業時間カレンダーを通じて処理されるため、休日および休業日を考慮する必要はありません。

## <a name="setting-up-a-working-time-calendar"></a>作業時間カレンダーの設定

カレンダーには生成詳細、含めたい日数および時間、カレンダーの日数、その日数に対する作業時間、および登録された従業員が含まれます。 

カレンダーを設定するには、次の手順に従います。

1. **組織管理**ページで、**カレンダー**をクリックします。

2. アクション ウィンドウで、**新規**を選択して名前と説明を入力します。

3. 組織に対する作業日数を選択し、作業時間を入力します。

4. **追加**ボタンを使用して、休日および休業日を追加します。

5. 米国の休日または銀行の休業日など、休日および休業日の名前と説明を入力します。 休日および休業日の日付を入力します。 休日および休業日リストを保存し、ページを閉じます。

6. ドロップダウン メニューから、休日および休業日リストを選択します。

7. 従業員に昼食時間、休憩時間などの非作業時間がある場合、それらも追加します。 **非作業時間**を選択し、名前と時間範囲を入力します。 ページを閉じます。 

8. **追加**をクリックして、カレンダーに非作業時間を追加します。

9. **日数**タブで、**生成**を選択して、カレンダーに日数を生成します。 カレンダーに対する日付範囲を入力します。 日数および作業時間は、選択した日付と組み合わせて生成オプションの下で定義される作業日および作業時間に基づいて生成されます。

10. カレンダーを従業員に割り当てるには、アクション ウィンドウで**従業員に割り当て**を選択します。 このカレンダーを割り当てたい従業員を選択してから、**割り当て**をクリックします。

従業員には割り当てられたカレンダーは必要ありません。 作業時間カレンダーが定義されている場合、休暇日は自動的に申請から除外されます。 時間数または日数による期間の既定値は、カレンダーで定義されている作業時間になります。 従業員に割り当てられたカレンダーがない場合、すべての日数は休暇に対して利用可能で、休暇の期間は申請の既定ではありません。
