<span data-ttu-id="23190-365">Talent から Finance and Operations にデータを統合する場合は、統合が ID に基づいてレコードを一致させようとします。</span><span class="sxs-lookup"><span data-stu-id="23190-365">When integrating data from Talent to Finance and Operations, the integration will attempt to match records based on the ID.</span></span> <span data-ttu-id="23190-366">一致が発生した場合、Finance and Operations のデータは、Talent の値で上書きされます。</span><span class="sxs-lookup"><span data-stu-id="23190-366">If the match occurs, the data in Finance and Operations will be overwritten with the values in Talent.</span></span> <span data-ttu-id="23190-367">ただし、論理的にさまざまなレコードが存在し、Talent または Finance and Operations のいずれかでそれぞれの番号順序に基づいて同一の ID が生成された場合に、問題が発生する場合があります。</span><span class="sxs-lookup"><span data-stu-id="23190-367">However, an issue may occur if logically these are different records and the same ID was generated in either Talent or Finance and Operations based on the respective number sequence.</span></span>
