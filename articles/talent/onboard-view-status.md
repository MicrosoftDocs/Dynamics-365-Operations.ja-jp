# <a name="view-the-status-of-onboarding-employees-and-your-tasks"></a>研修中の従業員の状態とタスクの表示

[!include [banner](includes/banner.md)]

Microsoft Dynamics 365 for Talent: Onboard により、新規採用者の研修活動の進行状況を表示することができます。 また、研修ガイドから割り当てられているタスクを表示することもできます。

## <a name="view-the-status-of-onboarding-employees"></a>研修中の従業員の状態の表示

1. 左側のメニューで、**ガイド**を選択します。 送信したすべての研修ガイドの進行状況の概要が表示されます。

    ![[研修ガイドの状態の表示](./media/onboard-guide-status.png)](./media/onboard-guide-status.png)

2. ドリルダウンするには、表示するガイドを選択します。
3. **活動**タブを選択して、新規採用により完了した活動を表示します。

    ![[新規採用により完了した活動の表示](メディア/オンボードステータスアクティビティ.png)](./media/onboard-status-activities.png)

## <a name="view-onboarding-tasks-that-are-assigned-to-you"></a>自分に割り当てられている研修タスクの表示

1. 左側のメニューで、**タスク**を選択します。 従業員に送信された研修ガイドから、自分に割り当てられているすべての研修タスクの概要を確認することができます。
2. ドリルダウンするには、必要なタスクを選択します。
3. 必要に応じて新規採用者とフォローアップします。
4. タスクが完了したら、円内にチェック マークを追加して、完了としてマークします。
