# <a name="employment-verification-i9-verification"></a>雇用検証 i9 検証

[!include [banner](../../../includes/banner.md)]


[!INCLUDE [PEAP](../../../../../includes/peap-1.md)]

連邦移民改革管理法 (IRCA) により、米国の雇用主は、新しく雇用した従業員の雇用適格性の状態を確認する必要があります。 この手順では、I-9 の検証に必要なドキュメントを記録する手順を説明します。 この手順では、USMF というデモ会社を使用します。

1. **人事管理 \> 作業者 \> 従業員** の順に移動します。
2. クイック フィルターを使用して、レコードを見つけます。 たとえば、**Vince (ヴィンス)** という値を含む **名前** フィールドをフィルター処理します。
3. 従業員を選択します。 たとえば、**Vince Prado (ヴィンス・プラド)** を選択します。
4. **個人情報** クイックタブ選択します。
5. **ID 番号** を選択します。
6. **新規** を選択します。
7. 記録する ID タイプを選択します。 たとえば、**パスポート** を選択します。
8. **数** フィールドに値を入力します。
9. **プライマリ** フィールドで、**はい** を選択します。
10. **説明** フィールドに、ID レコードの簡単な説明を入力します。
11. **発行機関** フィールドで、作業者の ID のフォームを発行した機関を選択します。 たとえば、**政府** を選択します。
12. 発行機関が作業者の ID のフォームを発行した日付を入力します。 たとえば、**2011/02/15** (2011 年 2 月 15 日) と入力します。
13. ID のフォームの期限が切れる日付を入力します。 たとえば、**2021/2/15** (2021 年 2 月 15 日) と入力します。
14. **保存** を選択します。
15. ページを閉じます。
16. **雇用** タブを選択します。
17. **I-9** を選択します。
18. **新規** を選択します。
19. **就業資格** フィールドで、オプションを選択します。

    従業員が米国の市民権または国籍を持っていない場合は、従業員の居住外国人番号または入国番号を入力する必要があります。

20. **GroupListA** オプションを選択します。

    選択するリストは、従業員が提出した識別フォームにより決まります。 従業員は、リスト A から 1 つのドキュメント、またはリスト B とリスト C の両方から 1 つのドキュメントを提出しなければなりません。例えば、従業員がパスポートを提供した場合、リスト A を選択することができます。しかし、従業員が運転免許証と社会保障カードのみを提供した場合、リスト B およびリスト C を選択する必要があります。

21. **I-9 ドキュメント タイプ**  フィールドで、従業員が提出したドキュメントのタイプを選択します。
22. **ドキュメント番号** フィールドで、値を入力または選択します。
23. **保存** を選択します。
