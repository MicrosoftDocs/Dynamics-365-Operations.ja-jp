# <a name="enter-applicant-and-application-data-manually"></a>申請者およびアプリケーション データの手動入力

> [!NOTE]
> この記事の採用機能は採用プロジェクトと呼ばれ、応募者、応募、採用プロジェクトに焦点を当てます。  


この手順では、手動で申請者とその申請に関する情報を管理する方法について説明します。 申請者についての個人情報、面接の日時、能力、宿舎施設のリクエストを入力および管理できます。 また、雇用申請者の申請状態を更新し、申請者とやり取りを行う書状や電子メールを作成できます。 申請者レコードを作成すると、その申請者の人物のレコードはグローバル アドレス帳で作成されます。 この手順の作成に使用するデモ データの会社は **USMF** です。

## <a name="create-a-new-applicant-record"></a>新しいレコードの作成

1. **人事管理 \> 採用 \> 申請者 \> 申請者** の順に移動します。
2. **新規** を選択します。
3. **名** フィールドに値を入力します。
4. **姓** フィールドに値を入力します。

    可能な場合、追加の申請者情報を入力できます。 たとえば、この情報には、応募者の最高学位、現在の職位、以前の雇用者などが含まれます。

5. **連絡先情報** セクションを展開します。
6. **追加** を選択します。
7. **説明** フィールドに、**電子メール通知** を入力します。
8. **タイプ** フィールドで、オプションを選択します。
9. **連絡先の電話番号/住所** フィールドに、値を入力します。

    この電子メール アドレスは申請者との電子メール通信を生成するのに使用されるようになります。

10. **追加** を選択します。
11. **説明** フィールドで値を入力します。
12. **連絡先の電話番号/住所** フィールドに、値を入力します。

    このフィールドを使用して、必要に応じて申請者に関する追加情報を入力します。 たとえば、この情報には、申請者の生年月日、民族的背景、性別、配偶者の有無などが含まれます。

13. アクション ペインで、**能力** を選択します。

    スキル、職務経験、教育、テスト、または資格証明書を含む申請者の能力プロファイルを入力できます。 この情報をもとに、申請者のスキルを、自社のデータで定義されている職務に関連するスキルにマッピングすることができます。

## <a name="create-an-application-for-the-applicant"></a>申請者のアプリケーションの作成

1. **申請** を選択します。
2. **新規** を選択します。
3. **採用プロジェクト** フィールドで、ドロップ ダウン矢印を選択し、検索を開きます。

    採用プロジェクトを選択することによって、申請者は、その採用プロジェクトに含まれている特定の募集と関連付けられます。

4. 一覧で、目的のレコードを見つけ、選択します。
5. 一覧で、選択した行のリンクを選択します。

    既定では、ジョブと部門は選択した採用プロジェクトに基づいています。

6. **保存** を選択します。

    申請を保存した後は、ドキュメントを関連付けできます。 これらのドキュメントには、申請者の経歴、受賞歴、カバーレターなどが含まれます。
