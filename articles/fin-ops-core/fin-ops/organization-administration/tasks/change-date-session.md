# <a name="change-the-date-for-a-session"></a>セッションの日付変更

[!include [banner](../../includes/banner.md)]


[!INCLUDE [PEAP](../../../../includes/peap-1.md)]

この記事では、セッションの日付を変更する方法について説明します。 既定では、仕訳入力や元伝票を入力および転記するときに、現在の日付が使用されます。 現在のセッションに使用される日付を変更できます。 仕訳入力や元伝票の日付をさかのぼるには、この機能を必要に応じて使用します。

1. ナビゲーション ウィンドウで、**モジュール > 共通 > 共通 > セッションの日時** の順に移動します。
2. **日付** フィールドに日付を入力します。
3. **OK** を選択します。
