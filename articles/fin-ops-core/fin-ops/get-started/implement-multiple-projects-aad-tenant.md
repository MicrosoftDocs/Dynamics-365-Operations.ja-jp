## <a name="disadvantages-of-multiple-lcs-projects"></a><span data-ttu-id="9e72d-110">複数の LCS プロジェクトの欠点</span><span class="sxs-lookup"><span data-stu-id="9e72d-110">Disadvantages of multiple LCS projects</span></span>

- <span data-ttu-id="9e72d-114">統合は各 LCS プロジェクトで構成される必要があります。</span><span class="sxs-lookup"><span data-stu-id="9e72d-114">Integrations must be configured in each LCS project.</span></span>
- <span data-ttu-id="9e72d-115">各 LCS プロジェクトには、個別の自分のデータベースの持ち込み (BYOD) インスタンスが必要です。</span><span class="sxs-lookup"><span data-stu-id="9e72d-115">Each LCS project requires a separate Bring your own database (BYOD) instance</span></span>
