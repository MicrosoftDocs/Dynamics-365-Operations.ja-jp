# <a name="keyboard-shortcuts"></a><span data-ttu-id="c5aa0-103">キーボード ショートカット</span><span class="sxs-lookup"><span data-stu-id="c5aa0-103">Keyboard shortcuts</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="c5aa0-104">次のキーボード ショートカットは、Finance and Operations アプリでデータを短時間および効率的にに入力するのに役立ちます。</span><span class="sxs-lookup"><span data-stu-id="c5aa0-104">The following keyboard shortcuts can help you quickly and efficiently enter data in Finance and Operations apps .</span></span>

> [!NOTE]
> <span data-ttu-id="c5aa0-105">ここで説明するショートカット キーは、米国のキーボード レイアウトのものです。</span><span class="sxs-lookup"><span data-stu-id="c5aa0-105">The keyboard shortcuts described here refer to the United States keyboard layout.</span></span> <span data-ttu-id="c5aa0-106">他のキーボード レイアウト キーは、米国のキーボード キーとまったく同じではない場合があります。</span><span class="sxs-lookup"><span data-stu-id="c5aa0-106">Keys on other keyboard layouts might not correspond exactly to the keys on a US keyboard.</span></span>

<span data-ttu-id="c5aa0-107">このページのショートカットの一部は *キー コード* です。つまり、必要なアクションをトリガーするために個別に押す必要がある 2 つの連続するキーの組み合わせのセット (コンマで区切ぎる) が必要です。</span><span class="sxs-lookup"><span data-stu-id="c5aa0-107">Some of the shortcuts on this page are *key chords*, meaning they require two consecutive sets of key combinations (separated by a comma) that need to be independently pressed to trigger the required action.</span></span> <span data-ttu-id="c5aa0-108">たとえば、ショートカット "Alt + M、A" は、ユーザーは最初に "Alt+M" を押し、キーを放してから、"A" を押す必要があります。</span><span class="sxs-lookup"><span data-stu-id="c5aa0-108">For example, the shortcut "Alt+M,A" requires the user to first press "Alt+M", release the keys, and then press "A".</span></span>
