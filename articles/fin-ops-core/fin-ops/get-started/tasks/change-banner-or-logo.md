# <a name="change-the-banner-or-logo"></a><span data-ttu-id="bef13-103">バナーまたはロゴの変更</span><span class="sxs-lookup"><span data-stu-id="bef13-103">Change the banner or logo</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="bef13-104">次の手順は、システム管理者が法人用に表示するバナーまたはロゴ画像を更新するために使用できるステップを一覧にしています。</span><span class="sxs-lookup"><span data-stu-id="bef13-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="bef13-105">この手順の作成に使用するデモ データの会社は USMF です。</span><span class="sxs-lookup"><span data-stu-id="bef13-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="bef13-106">**ナビゲーション ウィンドウ > モジュール > 組織管理 > 組織 > 法人**の順に移動します。</span><span class="sxs-lookup"><span data-stu-id="bef13-106">Go to **Navigation pane > Modules > Organization administration > Organizations > Legal entities**.</span></span>
2. <span data-ttu-id="bef13-107">左側のリストで、バナーまたはロゴを更新する法人を選択します。</span><span class="sxs-lookup"><span data-stu-id="bef13-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="bef13-108">これが既に選択されている場合、次の手順を実行します。</span><span class="sxs-lookup"><span data-stu-id="bef13-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="bef13-109">**ダッシュボードの画像**タブを展開します。</span><span class="sxs-lookup"><span data-stu-id="bef13-109">Expand **Dashboard image** tab.</span></span>
4. <span data-ttu-id="bef13-110">**変更**を選択します。</span><span class="sxs-lookup"><span data-stu-id="bef13-110">Select **Change**.</span></span>
    
    <span data-ttu-id="bef13-111">バナーの画像の理想的な解像度は 1920 x 281 ピクセルです。</span><span class="sxs-lookup"><span data-stu-id="bef13-111">The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="bef13-112">ロゴの画像の理想的な幅は 350 ピクセルです。</span><span class="sxs-lookup"><span data-stu-id="bef13-112">The ideal width for a logo image is 350 pixels.</span></span>
    
5. <span data-ttu-id="bef13-113">**保存** を選択します。</span><span class="sxs-lookup"><span data-stu-id="bef13-113">Select **Save**.</span></span>
6. <span data-ttu-id="bef13-114">**ナビゲーション ウィンドウ > モジュール > 共通 > 共通 > 既定のダッシュボード**に移動します。</span><span class="sxs-lookup"><span data-stu-id="bef13-114">Go to **Navigation pane > Modules > Common > Common > Default dashboard**.</span></span> <span data-ttu-id="bef13-115">ダッシュボードに新しいバナーまたはロゴの画像が表示されているはずです。</span><span class="sxs-lookup"><span data-stu-id="bef13-115">You should see the new banner or logo image on the dashboard.</span></span>  
    
    <span data-ttu-id="bef13-116">会社のピッカーを使用して、バナーをアップロードしたものに、会社を変更する必要があるかもしれません。</span><span class="sxs-lookup"><span data-stu-id="bef13-116">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
