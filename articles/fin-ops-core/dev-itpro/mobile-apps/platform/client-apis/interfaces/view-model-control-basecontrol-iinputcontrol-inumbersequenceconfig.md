### <a name="datatype"></a>dataType

データ型は、参照ページで番号順序が編集可能かどうかを検索するために使用されます。


### <a name="referencepagename"></a>referencePageName

referencePageName: string

番号順序が編集可能かどうかを定義するページのページ名。
