# <a name="create-a-record-template-to-facilitate-data-entry"></a>データ入力を容易にするレコード テンプレートの作成

[!include [banner](../../includes/banner.md)]

このトピックでは、レコード テンプレートの作成して、よく使用されるフィールド値を新しいレコードごとに明示的に入力する必要がないようにする方法を示します。 この手順では、固定資産に追加する必要がある新しいラップトップのレコードを新規作成します。 この手順では、USMF というサンプル会社を使用します。

1. ナビゲーション ウィンドウで、**モジュール > 固定資産 > 固定資産 > 固定資産**の順に移動します。
2. **新規** を選択します。
3. **固定資産グループ** フィールドで、値を入力または選択します。
4. **名前**フィールドに値を入力します。 たとえば、**企業潜在顧客のラップトップ**と入力します。  
5. **検索名**フィールドに値を入力します。 たとえば、**ラップトップ**と入力します。  
6. **技術情報**セクションを展開します。
7. **メーカー**、**モデル**、および**年式**フィールドに、値を入力します。
8. アクション ウィンドウで、**オプション**を選択します。
9. **レコード情報**を選択します。
10. **ユーザー テンプレート**を選択します。
11. **名前**フィールドに値を入力します。
12. **説明**フィールドで、値を入力します。
13. **OK** を選択します。
14. **閉じる**を選択します。
