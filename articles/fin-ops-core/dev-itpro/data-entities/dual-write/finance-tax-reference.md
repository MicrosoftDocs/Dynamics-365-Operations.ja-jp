# <a name="access-to-finance-and-tax-reference-data"></a>財務および税参照データへのアクセス

[!include [banner](../../includes/banner.md)]



すべてのビジネスでは、会計暦年、取引が行われる通貨、ビジネスを運営する金銭が出入りする口座、税率、および送金為替などの、財務データの基本セットが使用されます。 このデータは財務と運用アプリ内にあります。 ただし、これは Dataverse に公開されているので、 Customer Engagement アプリは財務および税データの単一のソースを持つことができます。 この方法により、データはビジネス エコシステム全体で統一されます。

財務および税データは、次のマッピングを使用して統合されます。

+ [統合された元帳](ledger-mapping.md)
+ [統合された税マスター](tax-mapping.md)
