<span data-ttu-id="e1100-106">コンフィギュレーション データ パッケージは、Microsoft Dynamics Lifecycle Services (LCS) からプロセス データ パッケージとして入手できます。</span><span class="sxs-lookup"><span data-stu-id="e1100-106">Configuration data packages are available as process data packages from Microsoft Dynamics Lifecycle Services (LCS).</span></span> <span data-ttu-id="e1100-107">これらのデータ パッケージは、実装の再現性を向上させ、構成を高速化させます。</span><span class="sxs-lookup"><span data-stu-id="e1100-107">These data packages can help improve the repeatability of implementations and accelerate the configuration.</span></span>

<span data-ttu-id="e1100-108">データ パッケージにはコンフィギュレーション エンティティ スプレッドシートが含まれています。</span><span class="sxs-lookup"><span data-stu-id="e1100-108">Data packages contain configuration entity spreadsheets.</span></span> <span data-ttu-id="e1100-109">これらのエンティティ スプレッドシートには、初期のゴールデン ビルドを作成するために使用できるベスト プラクティス データが含まれています。</span><span class="sxs-lookup"><span data-stu-id="e1100-109">These entity spreadsheets contain best practice data that you can use to create an initial golden build.</span></span> <span data-ttu-id="e1100-110">データ パッケージ内のデータ エンティティも適切に順序付けされ、1 回のクリックによるインポートを成功させるのに役立ちます。</span><span class="sxs-lookup"><span data-stu-id="e1100-110">The data entities in the data packages are also sequenced appropriately to help guarantee a successful single-click import of the data.</span></span> 

<span data-ttu-id="e1100-111">エンティティのスプレッドシートには、次の 3 種類のデータが含まれます。</span><span class="sxs-lookup"><span data-stu-id="e1100-111">The entity spreadsheets include three types of data:</span></span>

- <span data-ttu-id="e1100-112">**業務データ** - スプレッドシートには、中規模の貿易会社または小売企業の標準的なビジネス データが含まれています。</span><span class="sxs-lookup"><span data-stu-id="e1100-112">**Business data** – The spreadsheet contains standard business data for a mid-sized trade or retail company.</span></span> <span data-ttu-id="e1100-113">このデータは、ベスト プラクティスとビジネス標準を組み合わせて、構成の始点として使用できます。</span><span class="sxs-lookup"><span data-stu-id="e1100-113">This data combines best practices and business standards that can be used as a starting point for your configuration.</span></span>
- <span data-ttu-id="e1100-114">**サンプル データ** – スプレッドシートには、ビジネス特有のデータの例として使用できるデータが含まれています。</span><span class="sxs-lookup"><span data-stu-id="e1100-114">**Sample data** – The spreadsheet contains data that can be used as an example for business-specific data.</span></span> <span data-ttu-id="e1100-115">このデータは例としてインポートして使用できますが、個々の業務内容で変更する必要があります。</span><span class="sxs-lookup"><span data-stu-id="e1100-115">This data can be imported and used as an example, but it must be changed for individual business practices.</span></span>
- <span data-ttu-id="e1100-116">**データなし** – スプレッドシートにデータが含まれていません。</span><span class="sxs-lookup"><span data-stu-id="e1100-116">**No data** – The spreadsheet doesn't contain any data.</span></span> <span data-ttu-id="e1100-117">製品のいくつかの領域は、各業務とその業務内容に固有です。</span><span class="sxs-lookup"><span data-stu-id="e1100-117">Several areas of the product are unique to each business and its business practices.</span></span> <span data-ttu-id="e1100-118">これらの領域は組織専用に設定する必要があります。</span><span class="sxs-lookup"><span data-stu-id="e1100-118">These areas must be configured specifically for the organization.</span></span> <span data-ttu-id="e1100-119">これらのスプレッドシートは、必要に応じて組織のために見直し、更新する必要があります。</span><span class="sxs-lookup"><span data-stu-id="e1100-119">These spreadsheets should be reviewed and updated for the organization as appropriate.</span></span>

<span data-ttu-id="e1100-120">パッケージデータで、エンティティ スプレッドシートに含まれるデータのタイプの詳細については、このトピックの [データ パッケージ](#data-packages-system) セクションを参照してください。</span><span class="sxs-lookup"><span data-stu-id="e1100-120">For more information about the type of data that is included in each entity spreadsheet in the data packages see the [Data packages](#data-packages-system) section of this topic.</span></span> <span data-ttu-id="e1100-121">データ パッケージをインポートする前に個々のスプレッドシートを修正するか、提供されるときにデータ パッケージをインポートしてからシステムでデータを更新することができます。</span><span class="sxs-lookup"><span data-stu-id="e1100-121">You can modify individual spreadsheets before you import the data packages, or you can import the data packages as they have been supplied and then update your data in the system.</span></span>

## <a name="using-configuration-data-packages"></a><span data-ttu-id="e1100-122">コンフィギュレーション データ パッケージの使用</span><span class="sxs-lookup"><span data-stu-id="e1100-122">Using configuration data packages</span></span>
<span data-ttu-id="e1100-123">LCS から構成データ パッケージにアクセスすることができます。</span><span class="sxs-lookup"><span data-stu-id="e1100-123">You can access configuration data packages from LCS.</span></span> <span data-ttu-id="e1100-124">それらを LCS 環境に適用するか、手動でインポートすることができるようにダウンロードすることができます。</span><span class="sxs-lookup"><span data-stu-id="e1100-124">You can either apply them to an LCS environment, or download them so that you can manually import them.</span></span>

1. <span data-ttu-id="e1100-125">LCS プロジェクトを開いて、アセット ライブラリを開きます。</span><span class="sxs-lookup"><span data-stu-id="e1100-125">Open your LCS project, and open the Asset library.</span></span>
2. <span data-ttu-id="e1100-126">アセット タイプの一覧で、**プロセス データ パッケージ**を選択します。</span><span class="sxs-lookup"><span data-stu-id="e1100-126">In the list of asset types, select **Process data package**.</span></span>

<span data-ttu-id="e1100-130">この時点で**消費**関数を使用してプロセス データ パッケージを LCS 環境に適用することができます。</span><span class="sxs-lookup"><span data-stu-id="e1100-130">At this point, you can use the **Consume** function to apply the process data package to an LCS environment.</span></span> 

<span data-ttu-id="e1100-131">また、個々のデータ パッケージ ファイルを **データ パッケージ** 領域からダウンロードすることができます。</span><span class="sxs-lookup"><span data-stu-id="e1100-131">You can also download the individual data package files from the **Data package** area.</span></span> <span data-ttu-id="e1100-132">**データ管理**ワークスペースを使用して、LCS からデータ パッケージをインポートします。</span><span class="sxs-lookup"><span data-stu-id="e1100-132">Use the **Data management** workspace to import the data packages from LCS.</span></span> <span data-ttu-id="e1100-133">構成のインポートおよびエクスポート方法の詳細については、 [会社間または法人間での構成データのコピー](copy-configuration.md) を参照してください。</span><span class="sxs-lookup"><span data-stu-id="e1100-133">For more information about how to import and export configurations, see [Copy configuration data between companies or legal entities overview](copy-configuration.md).</span></span>
