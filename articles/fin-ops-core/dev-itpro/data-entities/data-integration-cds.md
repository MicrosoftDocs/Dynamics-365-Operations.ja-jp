# <a name="data-integration-using-common-data-service-overview"></a>Common Data Service を使用したデータ統合の概要

[!include [banner](../includes/banner.md)]

Common Data Service を使用することにより、Finance and Operations アプリと Dynamics 365 Sales との間でのデータ フローを有効にすることができます。 たとえば、Sales の顧客情報は、Finance and Operations アプリに流れることができます。 手動でデータを移動したり、統合ツールでサード パーティのデータを使用する必要はありません。

Common Data Service データ統合の詳細については、Microsoft Power Apps ドキュメントの [アプリ用 Common Data Service へのデータ統合](https://docs.microsoft.com/powerapps/administrator/data-integrator) を参照してください。

Common Data Serviceを使用する例については、 [見込顧客から収益化へ](../../../supply-chain/sales-marketing/prospect-to-cash.md) を参照してください。
