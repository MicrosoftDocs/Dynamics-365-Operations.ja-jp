# <a name="get-support-for-finance-and-operations-apps-or-lifecycle-services-lcs"></a>Finance and Operations アプリまたは Lifecycle Services (LCS) に関するサポートを受ける

[!include [banner](../includes/banner.md)]

このトピックでは、Finance and Operations アプリまたは Microsoft Dynamics Lifecycle Services (LCS) のヘルプを入手する方法について説明します。 

<table>
<thead>
<tr>
<th>タスク</th>
<th>詳細</th>
</tr>
</thead>
<tbody>
<tr>
<td>コミュニティに質問します。</td>
<td><a href="https://community.dynamics.com/">Dynamics 365 コミュニティ</a> ページに移動して、Microsoft Dynamics コミュニティからの質問のヘルプを入手します。</td>
</tr>
<tr>
<td>ライセンスに関する質問のヘルプを入手します。</td>
<td>パートナー、または Microsoft 販売担当者に問い合わせます。</td>
</tr>
<tr>
<td><strong>問題検索</strong> ツールの使用。</td>
<td><a href="https://lcs.dynamics.com/">LCS</a> で <strong>問題検索</strong> ツールを使用して、Microsoft サポート技術情報 (KB) の記事、修正プログラム、および 報告された問題の回避策をすぐに検索します。 特定の機能領域でどの報告された問題が修正処理中か、どの問題がすでに修正済みかを確認できます。 詳細については、<a href="issue-search-lcs.md">問題検索 (Lifecycle Services, LCS)</a> を参照してください。</td>
</tr>
<tr>
<td>アプリ内のサポートを取得します。</td>
<td>アプリの右上隅にある <strong>ヘルプ</strong> ボタン (<strong>?</strong>) を選択し、<strong>サポート</strong> を選択します。 問題は LCS の <strong>アクティブな問題</strong> タブで報告されます。 そこで、管理者は社内サポートを提供するか、Microsoft に問題を送信するかを決定できます。</td>
</tr>
<tr>
<td>Microsoft サポート チームとのサポート チケットを開きます。</td>
<td><a href="https://lcs.dynamics.com/">LCS</a> では、<strong>サポート</strong> タイルは、サポート インシデントの管理に役立つツールを開きます。 Microsoft に直接問題を送信するには、LCS プロジェクトで <strong>サポート</strong> タイルを選択します。 その後、次の 2 つの方法で問題を送信できます:
<ul>
<li><strong>アクティブな問題</strong> タブで問題を選択し、次に <strong>Microsoft に送信</strong> を選択します。</li>
<li><strong>Microsoftに送信済み</strong> タブで、<strong>インシデントの送信</strong> を選択し、画面の指示に従ってインシデントを送信します。 インシデントを送信すると、ケースに割り当てられている Microsoft サポート エンジニアから、電子メール メッセージが届きます。</li>
</ul>
</td>
</tr>
<tr>
<td>新機能および機能を要求します。</td>
<td><a href="https://experience.dynamics.com/ideas/">Dynamics 365 アプリケーションのアイデア</a> にアクセスし、既存のアイデアを表示、検索、または投票したり、新しいアイデアを追加したりすることもできます。</td>
</tr>
</tbody>
</table>
