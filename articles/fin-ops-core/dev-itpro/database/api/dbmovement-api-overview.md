# <a name="database-movement-api"></a>データベース移動 API

[!include [banner](../../includes/banner.md)]

データベース移動アプリケーション プログラミング インターフェイス (API) は、Microsoft Dynamics 365 環境のデータ ライフサイクルを管理するために使用される RESTful エンドポイントです。 これは、環境間でのデータベースのコピー、およびデータベース バックアップの一覧表示とダウンロードのために現在使用できる、バージョン管理された一連の機能を提供します。 後のリリースでは、更にサポートされているアクションの数が追加されます。

## <a name="what-is-supported-by-the-database-movement-api"></a>データベース移動 API では何がサポートされていますか。

データベース移動 API では、次の Dynamics 365 サービスに対する RESTful エンドポイントが公開されます。

* Dynamics 365 Finance
* Dynamics 365 Supply Chain Management
* Dynamics 365 Commerce
