# <a name="ABS">ABS ER 関数</a>

[!include [banner](../includes/banner.md)]

`ABS` 関数は、指定された数の絶対値 (係数) を*実数*値として返します。 つまり、符号のない数値を返します。

## <a name="syntax"></a>構文

```
ABS (number)
```

## <a name="arguments"></a>引数

`number`: *実数*

係数を求める数値。

## <a name="return-values"></a>戻り値

*実績*

結果数値。

## <a name="example"></a>例

`ABS (-1)` は、**1** を返します。
