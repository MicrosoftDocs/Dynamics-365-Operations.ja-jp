# <a name="ScreenDestinationType">画面出力先</a>

[!include [banner](../includes/banner.md)]

送信ドキュメントを生成するようにコンフィギュレーションされている電子申告 (ER) 形式の各フォルダーまたはファイル コンポーネントに対して、画面出力先をコンフィギュレーションできます。 送信先の設定に基づき、生成されたドキュメントはプレビュー用に別のブラウザー タブで開かれます。

**有効** を **はい** に設定すると、出力のプレビューが作成されます。 ブラウザー ウィンドウで直接 XML、TXT、PDF などの一部のファイル タイプを表示できます。 Microsoft Excel や Word などの他のファイル タイプについては、Microsoft Office Online サービスが使用されます。
