# <a name="list-of-er-functions-in-the-record-category"></a>レコード カテゴリ内の ER 関数のリスト

[!include [banner](../includes/banner.md)]

電子申告 (ER) レコード関数を使用して、*コンテナー (レコード)* データ タイプのデータ ソースから情報を抽出し、操作を実行できます。 このトピックでは、これらの関数の概要を示します。

## <a name="list-of-supported-functions"></a>サポートされている関数のリスト

| 職務 | 説明 |
|----------|-------------|
| [NullContainer](er-functions-record-nullcontainer.md) | この関数は、指定されたリストまたはレコードと同じ構造を持つ *コンテナー (レコード)* 値を返します。 |
| [EmptyRecord](er-functions-record-emptyrecord.md) | この関数は、指定されたリストまたはレコードと同じ構造を持つ *コンテナー (レコード)* 値を返します。 |
