# <a name="class-referencenode"></a>クラス ReferenceNode

[!include [banner](../../includes/banner.md)]

```xpp
class ReferenceNode extends TreeNode
```

ReferenceNode クラスは、X++ にカーネル クラスを公開する参照 AOT ノードを表します。

## <a name="remarks"></a>備考

このクラスは、AOT 参照ノードの下のノードを表します。 このクラスでは、X++ コードから参照されるマネージド アセンブリについて説明します。
