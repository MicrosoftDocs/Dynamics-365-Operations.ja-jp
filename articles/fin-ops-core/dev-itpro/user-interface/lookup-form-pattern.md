# <a name="lookup-form-pattern"></a><span data-ttu-id="8e5ae-103">ルックアップのフォーム パターン</span><span class="sxs-lookup"><span data-stu-id="8e5ae-103">Lookup form pattern</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="8e5ae-104">このトピックでは、ルックアップ フォームのパターンについて説明します。</span><span class="sxs-lookup"><span data-stu-id="8e5ae-104">This topic provides information about the Lookup form pattern.</span></span> <span data-ttu-id="8e5ae-105">標準フレームワークによって指定されたルックアップで正しいデータが提供されない場合や、データの高度な視覚化が必要な場合は、カスタム ルックアップ フォームを使用する必要があります。</span><span class="sxs-lookup"><span data-stu-id="8e5ae-105">Custom lookup forms should be used when a standard framework-provided lookup would not provide the correct data, or when advanced visualization of the data is required.</span></span>

<a name="usage"></a><span data-ttu-id="8e5ae-106">用途</span><span class="sxs-lookup"><span data-stu-id="8e5ae-106">Usage</span></span>
-----

<span data-ttu-id="8e5ae-107">カスタム ルックアップ フォームは、標準フレームワークによって指定されたルックアップ (通常はテーブル定義で定義された AutoLookup フィールド グループを使用して生成される)、が正しいデータを提供しない場合、またはデータの高度な視覚化が必要な場合に使用する必要があります。</span><span class="sxs-lookup"><span data-stu-id="8e5ae-107">Custom lookup forms should be used when a standard framework-provided lookup (which is typically generated by using the AutoLookup field group that is defined on the table definition), would not provide the correct data, or when advanced visualization of the data is required.</span></span> <span data-ttu-id="8e5ae-108">このドキュメントでは、3 つのパターンについて説明します。</span><span class="sxs-lookup"><span data-stu-id="8e5ae-108">Three patterns are described in this document:</span></span>

-   <span data-ttu-id="8e5ae-109">**基本のルックアップ** – これは、リストまたはツリーが 1 つだけの基本のルックアップ パターンです。オプションのカスタム フィルターとアクションもあります。</span><span class="sxs-lookup"><span data-stu-id="8e5ae-109">**Lookup basic** – This is the basic Lookup pattern that has just one list or tree, and also optional custom filters and actions.</span></span>
-   <span data-ttu-id="8e5ae-110">**タブ付きのルックアップ** – このルックアップ パターンは、ルックアップの複数のビューをユーザーが利用できるようにする場合に使用されます。</span><span class="sxs-lookup"><span data-stu-id="8e5ae-110">**Lookup w/tabs** – This Lookup pattern is used when more than one view of the lookup can be made available to the user.</span></span> <span data-ttu-id="8e5ae-111">タブ キャプションは表示されません。</span><span class="sxs-lookup"><span data-stu-id="8e5ae-111">Tab captions aren't shown.</span></span> <span data-ttu-id="8e5ae-112">代わりに、タブはコンボ ボックスを通じて選択されます。</span><span class="sxs-lookup"><span data-stu-id="8e5ae-112">Instead, the tab is selected through a combo box.</span></span>
-   <span data-ttu-id="8e5ae-113">**プレビュー付きのルックアップ** – このより高度なルックアップ パターンにより、ルックアップ グリッドの現在のレコードのプレビューが有効になります。</span><span class="sxs-lookup"><span data-stu-id="8e5ae-113">**Lookup w/preview** – This more advanced Lookup pattern enables a preview of the current record in the lookup grid.</span></span>
