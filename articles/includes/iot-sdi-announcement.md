> [!IMPORTANT]
> この記事で説明されている IoT インテリジェンス機能は、間もなくバージョン 2.0 にアップデートされ、同時に *Sensor Data Intelligence* という名前に変更されます。 この新しいバージョンは現在、Supply Chain Management のバージョン 10.0.30 以降でプレビュー機能として提供されています。
> 
> Sensor Data Intelligence では、必要な Azure コンポーネントが Microsoft に管理されるのではなく、独自の Azure サブスクリプションに展開されるという新しい展開モデルを導入しています。 この新しいモデルによって、機能のカスタマイズやサード パーティ製システムとの統合の機能が強化されます。 また、この新しいバージョンでは資産管理の 2 つの新しい IoT シナリオへのサポートが導入されます。
>
> このような変更が控えているため、既存の IoT インテリジェンス機能に基づく新たなプロジェクトの開始はお勧めしません。 システムに IoT 機能を追加することに関心をお持ちの場合は、代わりにまずプレビュー バージョンの Sensor Data Intelligence を使用することをお勧めします。 Sensor Data Intelligence の詳細と使用方法については、[Sensor Data Intelligence のホーム ページ](/dynamics365/supply-chain/sensor-data-intelligence/sdi-home-page)をご覧ください。
