# <a name="dynamics-ax-intelligent-data-management-framework-idmf"></a>Dynamics AX Intelligent Data Management Framework (IDMF)

[!include [banner](../../includes/banner.md)]

The Microsoft Dynamics AX Intelligent Data Management Framework (IDMF) を使用すると、システム管理者が Microsoft Dynamics AX インストールのパフォーマンスを最適化できます。 IDMF は、Microsoft Dynamics AX アプリケーションの正常性の評価と現在の使用パターンの分析を行い、データベース サイズの削減に役立ちます。

IDMF が移動しました。 Microsoft Dynamics Lifecycle Services のダウンロード可能なツール セクションから利用できます。 [Lifecycle Services に移動](https://lcs.dynamics.com).

### <a name="what-has-changed"></a>変更点

The Microsoft Dynamics AX Intelligent Data Management Framework (IDMF) を使用すると、システム管理者が Microsoft Dynamics AX インストールのパフォーマンスを最適化できます。 IDMF は、Microsoft Dynamics AX アプリケーションの正常性の評価と現在の使用パターンの分析を行い、データベース サイズの削減に役立ちます。 IDMF の 2014 年 10 月のリリースでは、Microsoft Dynamics AX 2012 R3 のサポートが追加されました。 IDMF の 2014 年 7 月のリリースでは、次の機能が変更されました。

-   AX 2012 R2 のサポートが追加されました。
-   Microsoft Dynamics AX 3.0 および Microsoft Dynamics AX 4.0 のサポートが削除されました。
-   システム要件が更新されました。
-   スレッドのサポートが追加されました。
-   **ByFiscalYear** テンプレートは、非実稼働環境のテスト後に計画的にのみ実行する必要があるため、コンフィギュレーション可能で、既定では非表示です。 これにより生産のデータベースから財務トランザクションが削除されます。

[サービスのダウンロード](http://go.microsoft.com/fwlink/?LinkID=228149) ページに移動します。
