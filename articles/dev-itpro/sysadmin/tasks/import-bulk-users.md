# <a name="import-users-in-bulk"></a>一括でユーザーをインポートする

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

システム管理者はこの手順を使用して、Azure Active Directory から多数のユーザーをインポートすることができます。


## <a name="run-as-a-batch-job"></a>バッチ ジョブとして実行
1. [システム管理] > [ユーザー] > [ユーザー] の順に移動します。
2. [バッチ インポート] をクリックします。
3. [バックグラウンドで実行] セクションを展開します。
4. [バッチ処理] フィールドで [はい] を選択します。
5. [タスクの説明] フィールドでは、値を入力します。
6. [バッチ グループ] フィールドで、値を入力または選択します。
    * これは、オプションのステップです。  
7. [個人] フィールドでは、[はい] を選択します。
    * これは、オプションのステップです。  
8. [重要なジョブ] フィールドでは、[はい] を選択します。
    * これは、オプションのステップです。  
9. [監視カテゴリ] フィールドでは、オプションを選択します。
10. [OK] をクリックします。

## <a name="run-in-a-sandbox-environment"></a>サンドボックス環境で実行
1. [バッチ インポート] をクリックします。
2. [OK] をクリックします。
