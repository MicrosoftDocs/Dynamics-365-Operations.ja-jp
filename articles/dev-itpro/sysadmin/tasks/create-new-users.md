# <a name="create-new-users"></a>新規ユーザーの作成

[!include [task guide banner](../../includes/task-guide-banner.md)]

ユーザーは、ジョブの実行のためにシステムへアクセスする必要がある、組織の内部従業員、または外部顧客や仕入先です。 システム管理者は、この手順を実行してシステムにユーザーを追加できます。 この手順の作成に使用するデモ データの会社は USMF です。 


## <a name="add-a-new-user"></a>新しいユーザーを追加する
1. **ナビゲーション ウィンドウ > モジュール > システム管理 > ユーザー > ユーザー**の順に移動します。
2. **アクション ウィンドウ**で、**新規**をクリックします。
3. **ユーザー ID** フィールドで、値を入力します。 ユーザーを示す固有の ID を入力します。 ユーザー ID が必要です。  
4. **ユーザー名**フィールドに値を入力します。 ユーザーの名前を入力します。  
5. **ドメイン** フィールドに値を入力します。 ユーザーのドメインを入力します。  
6. **エイリアス** フィールドに値を入力します。 ユーザーのエイリアスを入力します。  
7. **会社**フィールドで、ドロップ ダウン ボタンをクリックし、ルックアップを開きます。
8. 一覧で、目的のレコードを見つけ、選択します。 
9. **ユーザーのロール** セクションで、**ロールの割り当て**をクリックします。
10. 一覧で、目的のレコードを見つけ、選択します。
11. **OK**をクリックします。
12. **保存**をクリックします。

## <a name="import-users"></a>ユーザーのインポート
1. **アクション ウィンドウ**で、**ユーザーのインポート**をクリックします。
2. 一覧で、選択された行をマークします。
3. **ユーザーのインポート**をクリックします。
4. **閉じる**をクリックします。
