        <target logoport:matchpercent="101" state="translated" state-qualifier="leveraged-tm">たとえば、クライアントのユーザー アクションは、すぐに送信されるまたは後で送信されるようにキューに入れられるかのいずれかであるサーバーへのサービス コールを作成します。</target></trans-unit>
