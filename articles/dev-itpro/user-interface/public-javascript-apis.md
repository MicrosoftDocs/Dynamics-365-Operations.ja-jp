# <a name="extensible-controls---public-javascript-apis"></a>拡張可能なコントロール - パブリック JavaScript API

[!include [banner](../includes/banner.md)]

このトピックでは、拡張可能なコントロールで使用できるパブリック JavaScript API に関するドキュメントの入手先を示します。 

拡張可能なコントロールが将来中断されることを最小にするために、拡張可能なコントロールで使用できる JavaScript アプリケーショ ンプログラミング インターフェイス (API) をパブリックなものと非パブリックなものとを区別するための努力がなされています。 コントロール作成者は、**非公開 API は今後のリリースで削除または変更できるため**、公開 API のみを使用するようにする必要があります。 計画された変更の 1 つは、アクセス レベルを明確に表示すために非公開の API の名前の接頭語にアンダースコアを付けることです。 パブリック Api の完全なセット ドキュメントについては、[Extensible Controls - Public JavaScript APIs](https://mbs.microsoft.com/Files/public/CS/AX/DynamicsAX_JavaScript_API_Documentation_Update2_07_2016.pdf)を参照してください。
