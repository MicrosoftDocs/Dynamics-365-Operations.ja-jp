# <a name="data-integration-using-common-data-service-overview"></a>Common Data Service を使用したデータの統合の概要

[!include [banner](../includes/banner.md)]

Common Data Service を使用して、Microsoft Dynamics 365 for Finance and Operations と Dynamics 365 for Sales 間のデータの流れを有効にできます。 たとえば、Microsoft Dynamics 365 for Sales 内の顧客情報は、Microsoft Dynamics 365 for Finance and Operations Enterprise Edition に流れることができます。 手動でデータを移動したり、統合ツールでサード パーティのデータを使用する必要はありません。

Common Data Service データ統合の詳細については、PowerApps ドキュメントの[アプリ用 Common Data Service にデータを統合](https://docs.microsoft.com/powerapps/administrator/data-integrator) を参照してください。

Common Data Service を使用する例については、[見込顧客から入金のシナリオのコンフィギュレーション](../../supply-chain/sales-marketing/prospect-to-cash.md) を参照してください。
