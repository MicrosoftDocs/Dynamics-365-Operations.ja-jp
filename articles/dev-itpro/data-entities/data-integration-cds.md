# <a name="data-integration-by-using-common-data-service"></a>Common Data Service を使用したデータの統合

[!include [banner](../includes/banner.md)]

Common Data Service を使用して、Microsoft Dynamics 365 for Finance and Operations と Dynamics 365 for Sales の間のデータの流れを有効にすることができます。 たとえば、Microsoft Dynamics 365 for Sales での顧客情報は、Microsoft Dynamics 365 for Finance and Operations Enterprise Edition に流れます。 手動でデータを移動したり、統合ツールでサード パーティのデータを使用する必要はありません。

Common Data Service データ統合機能の詳細については、Common Data Service ドキュメントの [Dynamics 365 データ統合](/common-data-service/entity-reference/dynamics-365-integration) を参照してください。

Common Data Service を使用する例については、[見込顧客から入金のシナリオのコンフィギュレーション](../../supply-chain/sales-marketing/prospect-to-cash.md) を参照してください。
