# <a name="create-a-record-template-to-facilitate-data-entry"></a>データ入力を容易にするレコード テンプレートの作成

[!include [task guide banner](../../includes/task-guide-banner.md)]

この手順は、よく使用されるフィールド値を新しいレコードごとに明示的に入力する必要をなくなるようにする、レコード テンプレートの作成方法を示します。 この手順では、固定資産に追加する必要がある新しいラップトップのレコードを新規作成します。 この手順では、USMF というサンプル会社を使用します。

1. [固定資産] > [固定資産] > [固定資産] に移動します。
2. [新規] をクリックします。
3. [固定資産グループ] フィールドで、値を入力または選択します。
4. [名前] フィールドに値を入力します。
    * たとえば、「企業潜在顧客のラップトップ」と入力します。  
5. [検索名] フィールドに値を入力します。
    * たとえば、「ラップトップ」と入力します。  
6. [技術情報] セクションを展開します。
7. [製造業者] フィールドに値を入力します。
8. [モデル] フィールドに値を入力します。
9. [年始] フィールドに値を入力します。
10. [アクション] ウィンドウで、[オプション] をクリックします。
11. [レコード情報] をクリックします。
12. [ユーザー テンプレート] をクリックします。
13. [名前] フィールドに値を入力します。
    * たとえば、「企業のラップトップ」と入力します。  
14. [説明] フィールドに値を入力します。
    * たとえば、「企業のラップトップ」と入力します。  
15. [OK] をクリックします。
16. [閉じる] をクリックします。
