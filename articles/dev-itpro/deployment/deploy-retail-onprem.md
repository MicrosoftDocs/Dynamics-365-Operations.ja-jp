## <a name="overview"></a><span data-ttu-id="f5e59-105">概要</span><span class="sxs-lookup"><span data-stu-id="f5e59-105">Overview</span></span>

<span data-ttu-id="f5e59-106">オンプレミス環境では、小売チャネル機能は Retail Store Scale Unit による使用のみ有効です。</span><span class="sxs-lookup"><span data-stu-id="f5e59-106">Retail channel functionality, in an on-premises environment, is enabled exclusively via use of Retail Store Scale Unit.</span></span> <span data-ttu-id="f5e59-107">Retail Store Scale Unit の概要については、[Retail Store Scale Unit](../../retail/dev-itpro/retail-store-system-begin.md) をご覧ください。</span><span class="sxs-lookup"><span data-stu-id="f5e59-107">For an overview of Retail Store Scale Unit, see [Retail Store Scale Unit](../../retail/dev-itpro/retail-store-system-begin.md).</span></span>

> [!Note]
