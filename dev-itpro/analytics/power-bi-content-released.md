# <a name="power-bi-content-recently-released-for-dynamics-365-for-operations"></a>Dynamics 365 for Operations 用に最近リリースされた Power BI コンテンツ

[!include[banner](../includes/banner.md)]

"[!include[banner](../includes/banner.md)]"


このトピックでは、Dynamics 365 for Operations 用に最近リリースされた Power BI コンテンツを一覧表示します。

---------------------------------------------------

| [Power BI コンテンツ]       | 説明                                                                                                                                                                                                                                                                                                                                                                                   | [詳細情報]                                                                                                         |
|------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| 倉庫パフォーマンス  | この Microsoft Power BI コンテンツは、倉庫と工程マネージャーが重要な受信、送信、および在庫指標を監視するのに役立ちます。 倉庫管理、製品、およびその他の Microsoft Dynamics 365 for Operations からのトランザクション データを使用して、倉庫パフォーマンスおよび仕入先、製品グループと製品、サイトと倉庫の内訳の両方の集計ビューを提供します。 | [倉庫パフォーマンス Power BI コンテンツ](warehouse-power-bi-content.md) |

## <a name="power-bi-content-that-was-released-in-january-2017"></a>2017 年 1 月にリリースされた Power BI コンテンツ
| [Power BI コンテンツ]                  | 説明                                                                                                                                                                                                                                      | [詳細情報]                                                                                                                           |
|-----------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------|
| 購買先支出の分析           | この Power BI コンテンツは、購買部門のマネージャと予算を担当しているマネージャに役立ちます。 購買先支出を監視するために設計されました。                                                                                       | [Power BI コンテンツでの購買先支出分析](purchase-content-pack-for-power-bi.md)         |
| 原価会計分析          | この Power BI コンテンツは、原価の管理者または組織の原価管理を担当者に役立ちます。 これには、実際原価、予算原価、変動予算原価ごとの原価、規模、原価率などの主なメトリックスが含まれています。 | [原価会計分析の Power BI コンテンツ](cost-accounting-analysis-content-pack.md) |
| 与信および回収管理 | この Power BI コンテンツにより、与信および取立マネージャーは与信限度額、クレジット エクスポージャ、顧客残高、顧客支払の延滞、支払約束、および主なパフォーマンス測定を分析できます。                                               |                                                                                                                                      |
