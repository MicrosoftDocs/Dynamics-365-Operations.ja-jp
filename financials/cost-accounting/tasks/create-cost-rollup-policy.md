# <a name="create-a-cost-rollup-policy"></a>原価ロールアップ ポリシーの作成

[!include[task guide banner](../../includes/task-guide-banner.md)]

この手順では、原価ロールアップ ポリシーとポリシーのルールを作成する方法を示します。 この手順の作成に使用されたデモ データの会社は、USP2 です。


## <a name="create-a-policy"></a>ポリシーの作成
1. [原価会計] > [ポリシー] > [原価ロールアップ ポリシー] へ移動します。
2. [新規] をクリックします。
3. [ポリシー名] フィールドで、値を入力します。
4. [説明] フィールドに値を入力します。
5. [原価オブジェクト分析コード階層] フィールドで、値を入力または選択します。
    * [原価ロールアップ CC] を選択します。  
6. [原価要素分析コード階層] フィールドで、値を入力または選択します。
    * [原価ロールアップ CC] を選択します。  
7. [保存] をクリックします。

## <a name="create-rules-for-the-cost-rollup-policy"></a>原価ロールアップ ポリシーのルールを作成
1. [新規] をクリックします。
2. 一覧で、選択された行をマークします。
3. [原価オブジェクト分析コード階層ノード] フィールドで、値を入力または選択します。
    * 007 を選択します。  
4. [原価要素分析コード階層ノード] フィールドで、値を入力または選択します。
    * [原価ロールアップ CE] を選択します。  
5. [第 2 原価要素] フィールドで、値を入力または選択します。
    * この例の場合、コスト センターに第 2 原価要素 [CC-007] を位置づけます。  
6. [新規] をクリックします。
7. 一覧で、選択された行をマークします。
8. [原価オブジェクト分析コード階層ノード] フィールドで、値を入力または選択します。
    * 008 を選択します。  
9. [原価要素分析コード階層ノード] フィールドで、値を入力または選択します。
    * [原価ロールアップ CE] を選択します。  
10. [第 2 原価要素] フィールドで、値を入力または選択します。
    * この例の場合、コスト センターに第 2 原価要素 [CC-008] を位置づけます。  
11. [新規] をクリックします。
12. 一覧で、選択された行をマークします。
13. [原価オブジェクト分析コード階層ノード] フィールドで、値を入力または選択します。
    * 009 を選択します。  
14. [原価要素分析コード階層ノード] フィールドで、値を入力または選択します。
    * [原価ロールアップ CE] を選択します。  
15. [第 2 原価要素] フィールドで、値を入力または選択します。
    * この例の場合、コスト センターに第 2 原価要素 [CC-009] を位置づけます。  
    * 全てのコスト センターに、対応する第 2 原価要素が位置づけされるまで続けます。  
16. [保存] をクリックします。
