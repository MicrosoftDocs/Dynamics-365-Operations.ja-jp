## <a name="introduction"></a>はじめに
携帯電話ワークスペースを制御する原価、予算原価に対して実績原価を比較して原価部門の現在のパフォーマンスの簡単なビューを提供します。 個々の原価要素のステータスにすることができます。

### <a name="example"></a>例

従業員は、国際会議に招待が表示されます。 コンフィギュレーションはすべての旅行経費をカバーする必要があります。 従業員は、が会議への出席します。いければと彼のマネージャを入力する必要があります。 マネージャは、その会議への出席する従業員の予算があるかどうかを確認する彼の携帯電話の携帯電話ワークスペースを制御する原価を開きます。

### <a name="data-security"></a>データ セキュリティ

原価の管理ワークスペースのデータは、ユーザーの資格情報を保護されます。 原価部門のマネージャがただ自身の原価部門のデータを表示することができます。 アクセス レベルのセキュリティは、原価計算モジュール内で管理されます。 原価会計係は、原価計算モジュールの携帯電話ワークスペース コンフィギュレーションを制御する原価を定義します。 ワークスペースは、工程のアプリケーションのMicrosoft Dynamics 365に公開されると、アクションは者のモバイル アプリケーションの場合は、Dynamics 365 for Operationsで使用できます。 これは組織内のすべての原価部門のマネージャが同じフォームのデータを表示することができます。

## <a name="get-started"></a>はじめに
モバイル デバイスの原価管理の携帯電話アプリケーションを使用して開始するには、次の手順に従います。

1.  携帯電話アプリケーションの店舗で、工程のアプリケーションのMicrosoft Dynamics 365をダウンロードしてインストールします。
2.  デバイスのアプリケーションを開始します。
3.  Dynamics 365 for Operations URLを入力します。
4.  に署名するための会社を入力します。 たとえば、入力しますUSMF ** **。
5.  ユーザーが署名する、最初に、作業勘定、Microsoft Dynamics 365のユーザー名とパスワードの入力が求められます。 資格情報を入力します。 署名した後に、会社については、ワークスペースが表示されます。

携帯電話アプリケーション ワークスペースを表示するには、工程のアプリケーション365 for Operationsに必要なワークスペースを公開する必要があります。

1.  Dynamics 365 for Operationsを開始します。
2.  **システム** ** ** &gt; 設定、管理実行 &gt; **システム パラメータ**。
3.  [**携帯電話アプリケーションを管理します。**。
4.  携帯電話プラットフォームに発行する場合は、ワークスペース**原価の管理**を選択します。
5.  選択します**発行ワークスペース**。
6.  発行されたワークスペースを表示するデバイスを更新します。
