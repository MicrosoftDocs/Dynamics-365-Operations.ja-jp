# <a name="intercompany-accounting-setup"></a>会社間会計の設定

このトピックでは、日次仕訳帳、仕入先の請求仕訳帳、支払仕訳帳など、元帳配賦および財務仕訳帳の会社間仕訳帳を使用できるように、会社間会計の設定方法を説明します。

会社間仕訳帳は、日次仕訳帳に対して、仕入先の請求仕訳帳、元帳配賦、集中支払など、さまざまなシナリオで作成できます。 これらのシナリオを有効にするには、会社間会計を設定する必要があります。

## <a name="define-main-accounts"></a>重要な勘定を定義します。
最初に、貸し勘定項目および借り勘定項目に使用する会社間主勘定を作成する必要があります。 会社間勘定項目の調整および削除を簡略化するために、会社ごとに固有の主勘定を使用することをお勧めします。 会社間関係を識別すると、取引相手または対応側の分析コードを使用する会社間会計に定義されている重要な勘定の固定分析コードにこの分析コードを定義できます。 重要な勘定を設定すると、**主要な勘定**ページで、[**主要な勘定タイプ**フィールドを**貸借対照表**設定する必要があります。

## <a name="define-journal-names"></a>仕訳帳名を定義します。
次に、仕訳帳名を定義する必要があります。 ** **仕訳帳名**ページで、[**仕訳帳タイプ**フィールドを**毎日設定します。 会社間会計に対して特定の仕訳帳名を使用することをお勧めします。
