# <a name="import-currency-exchange-rates"></a>現在の通貨為替レートをインポート

法人が外貨で請求書を受け取った場合は、国内通貨で外貨を変換する必要があります。 これは異なる通貨に関する最新の為替レートが必要であることを意味します。 このトピックでは、欧州中央銀行、ロシア連邦中央銀行のような為替レートのプロバイダによってインターネットに発行された取扱い領域は外部の参照率をインポートするに必要な設定の概要と処理を提供します。

次のセクションでは、取扱い領域外率のインポートを設定して処理するために使用する情報の流れについて説明します。

## <a name="configure-an-exchange-rate-provider"></a>為替レートのプロバイダを設定します。
為替レートをインポートする前に、為替レートを提供するプロバイダに必要な情報を設定する必要があります。 為替レートのプロバイダを選択、**為替レートのプロバイダをコンフィギュレーションします。**ページを使用します。 ある為替レートのプロバイダでは、Microsoft Dynamics 365にデモ データが含まれます。 次の表で、このページのコントロールについて説明します。

|           |                                                                                                                                                                                                                             |
|-----------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Field** | **Description**                                                                                                                                                                                                             |
| **Name**  | 為替レート プロバイダーの名前。                                                                                                                                                                                     |
| **キー**   | プロバイダーで必須となる各コンフィギュレーション情報の固有 ID。 この情報は、為替レートのプロバイダに対して自動的に**追加します。**ボタンをクリックするか、追加するに追加されます。 |
| **Value** | 各キーの情報。 この情報は、為替レートのプロバイダに**追加します。**ボタンをクリックするか、追加するに追加されます。                                                                                         |

## <a name="import-currency-exchange-rates"></a>現在の通貨為替レートをインポート
為替レートのプロバイダ ソースから為替レートをインポートして**通貨為替レート**ページで設定します。 為替レートをインポートする、インポート**通貨の為替レート**ページを使用します。 次の表に、説明を正常にインポート プロセスを完了するために必要なフィールド示します。

|                                        |                                                                                                                                                                                                                                                                                                                                                                             |
|----------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Field**                              | **Description**                                                                                                                                                                                                                                                                                                                                                             |
| **Exchange rate type**                 | 為替レートのタイプ。                                                                                                                                                                                                                                                                                                                                                      |
| **Exchange rate provider**             | 為替レートのプロバイダ。                                                                                                                                                                                                                                                                                                                                                  |
| **Import as of**                       | このパラメータは今日の日付、または日付範囲にインポートするかどうかを管理します。 日付範囲を使用する場合は、開始日と終了日を入力、または選択します。                                                                                                                                                                                                                |
| **Create necessary currency pairs**    | このチェック ボックスはインポートされる通貨のペアがない場合、通貨のペアの自動作成を管理します。 このオプションは、プロバイダでは使用できませんかもしがあります。                                                                                                                                                                                               |
| **Override existing exchange rates**   | このチェック ボックスは、特定の日付の為替レートが既に存在すると通貨のペアの既存の為替レートの更新を管理します。 このチェック ボックスをオフにすると、特定の日付の為替レートが別の為替レートがある場合はインポートされません。                                                                                       |
| **Prevent import on national holiday** | このチェック ボックスは関する休暇の日付の為替レートのインポートを管理します。 たとえば、このチェック ボックスを選択し、為替レートのプロバイダとして、欧州中央銀行を使用すると、現在の法人に関連する関する休暇の為替レートは更新されません。 このオプションは、プロバイダでは使用できませんかもしがあります。 |
