# <a name="financial-period-close-workspace"></a>財務期間終了ワークスペース

この記事は、財務期間終了ワークスペースおよび関連するコンフィギュレーションの概要を提供します。

財務期間終了ワークスペース

** [財務期間**ワークスペースは、会社、面積、個人に財務の決算処理を追跡することができます。 ** [財務期間**ワークスペースのビューで、最後のスケジュールのすべてのタスク、ステータス、または自分に割り当てられて、タスクに表示されます。 

ワークスペースの上部による最後のスケジュールを選択する必要があります。 ワークスペースに表示されるすべてのデータは、選択した最後のスケジューリングで、フィルター処理されます。

### <a name="summary-tiles"></a>概要タイル

[**概要**] タイルは、プロセスの概要、および決算プロセスを順調に進める助けになるインジケーターを示します。 未払今日今日の期日後の、残りのタスクなタスク、タスクを表示になりますが、できますが、プロセスの依存関係、すべての残りのタスクに対して請求。 この情報は、選択した最後のスケジュールに含まれるすべての会社のためです。

### <a name="tasks-and-status-section"></a>タスクと状態のセクション

**タスクおよびステータス**セクションで、全体的な最後のスケジューリングのステータスは複数の方法で中断されます: 会社によるステータス領域、ステータス、および担当する従業員のステータス。 今日の対象となる、またはタスクできます。最後のスケジュールのすべてのタスク、カードの一覧の先頭にフィルタの変更によって期日経過に、そのタスクのステータスを表示できます。 特定の会社のステータスを表示するには、会社のフィルタを選択できます。 各タブにはステータス完了し、あるタスクの数の両方の割合を示します。 選択したカードによって詳細な作業一覧をフィルタ処理するカードまたは**表示の詳細**工程をクリックします。 

最後のタブに詳細な作業一覧を使用します。 このリストは、必要なタスクのみを示すように、すべての仕事リストを表示およびフィルタ処理することができます。 複数の方法で作業一覧をフィルタ処理できます。 たとえば、タスクの期日、関連する会社および関連の領域フィルタ処理することができます。 また、作業一覧に完了したタスクを表示するか、または非を選択できます。 

2 つのインジケーターがタスクに使用されます。

-   感嘆符のアイコンはタスクが期限日付であることを示します。 期日経過するタスクに、期日は、赤色で強調表示されます。
-   錠前のアイコンはタスクがまだ完了していない他のタスクに依存することを示します。 依存関係によってブロックされているタスクが完了としてマークすることはできません。 **一定の依存関係**工程を使用してタスクの依存関係を設定できます。

タスクの名前、工程のページまたはユーザーがタスクを完了することを結合方法別のWebページのMicrosoft Dynamics 365にハイパーリンクです。 **使用してタスク リンク**タスクを編集、または作成すると、このフィールドにはハイパーリンクを設定できます。 

タスクに**添付ファイル**工程を使用してファイル、メモ、画像、URLを関連付けることができます。 たとえば、タスクの一部として追加する特定の作業に関するコメントを使用すると、関連するタスクに印刷されたレポートをファイルには、仕訳帳番号を表します。 アイコンがタスクの**添付ファイル**列に添付ファイルがある表示されます。 

**完全なタスク**オプションは、作業の完了後に手動で選択する必要があります。 タスクが完了済としてマークすると、**完了した日付**このフィールドは、現在の日時に自動的に更新されます。 依存関係のインジケータが、必要に応じて更新されます。

## <a name="all-financial-period-close-tasks-list-page"></a>すべての財務期間終了タスク リスト ページ

### <a name="resources"></a>リソース

**リソース**]タブで、決算処理に関連する従業員を定義します。 最後の作業を担当した従業員が最初に次に割り当てる必要があります。 従業員のワークスペースのビューを指定する必要があります。  次のオプションを使用できます。

-   **割り当てられたタスクのみ** – ユーザーには、その従業員に割り当てられているタスクのみが表示されます。
-   **すべてのタスクと状態** – ユーザーには、すべての決算タスクと全体的なプロセスの状態が表示されます。

割り当てられたタスクのみを表示するアクセス許可があるユーザーは、タスク リストへのタスクの追加、タスクの編集、またはタスク リストからタスクを削除することはできません。

### <a name="closing-schedules"></a>決算スケジュール

精算する特定の財務期間によって決算のテンプレートを割り当てる場合でも、最後のスケジューリングを使用します。 テンプレートからのタスクは指定した期間で自動的に生成され、新しい最後のスケジューリングは、ワークスペースに追加されます。 新しい最後のスケジュールを作成すると、**期間の終了日などの決算テンプレートに割り当てられた相対的な期日に基づいて、最後のタスクの実際の期日を、確認、**フィールドが使用されます。 

適切なタスクのスケジューリングで使用される作業日を表示するには最後のスケジュールにカレンダーを割り当てます。 特定のカレンダーを指定しない場合、タスクの期日は、すべての曜日を使用します。 

[最終スケジュールと関連付けられる会社を定義する必要があります。 テンプレート タスクが複数の会社に割り当てられている場合、別のタスクが最後のスケジュールにある作成され、テンプレート タスクに割り当てられた各会社に対して。 

最後のスケジューリングが完了した後、**決算**オプションを選択します。 タスクの履歴は**すべての財務期間の最後タスク**リスト ページから使用できますが、最後のスケジューリングは、ワークスペースから削除されます。 最後のスケジューリングと同様に**決算**マークされると、タスクを追加できませんし、タスクを編集はありませんが、タスクを削除されません。
