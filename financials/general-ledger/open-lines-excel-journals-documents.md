# <a name="publish-journal-lines-and-documents-from-excel"></a>Excelからの払出の仕訳帳明細行とドキュメント

このトピックでは、Microsoft Excelから一般仕訳帳の明細行を入力し発行する方法を説明します。 [入力するタイプのトランザクションで使用できるさまざまなテンプレートに関する情報が含まれます。

ユーザーはMicrosoft Excelから財務仕訳帳の明細行を入力し、発行できます。 ユーザーが仕訳帳を作成した後、** Excelの未処理の明細行**ボタンは使用できるテンプレートを表示します。 テンプレートは勘定タイプの任意の組み合わせを仕訳帳でがされなくてサポートが特定のシナリオをサポートするように設計されています。 次の表に、サポートするか、を使用して、勘定タイプを示します。テンプレート。
|                          |                                                                                                                         |                                                                                         |
|--------------------------|-------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------|
| **Template**             | **サポートされる勘定タイプ**                                                                                             | **テンプレートへのアクセス方法を**                                                          |
| 仕訳元帳明細行     | 勘定: 勘定科目、顧客、仕入先、銀行の相手勘定: 勘定科目、顧客、仕入先、会社間でサポートされている取引します。       | 一般仕訳帳                                                                         |
| 仕入帳         | 勘定: 仕入先の相手勘定: 会社間元帳ではサポートされていません。                                                    | APの仕入帳                                                                     |
| 請求仕訳帳          | 勘定: 仕入先の相手勘定: 会社間元帳がサポートされています。                                                      | AP 請求仕訳帳                                                                      |
| 仕入先請求書           |                                                                                                                         | 仕入先請求書                                                                          |
| 顧客請求書仕訳帳 | 勘定: 顧客の相手勘定: 会社間元帳がサポートされています。                                                     | 一般仕訳帳                                                                         |
| 自由書式の請求書        |                                                                                                                         | **自由書式の請求書**ページで、[** Excelで開きます。** (Microsoft Officeのアイコン)。 |
| 固定資産仕訳帳     | 元帳、銀行、顧客、または仕入先への資産。 会社間サポートされていません。                                               | 固定資産仕訳帳                                                                     |
| 仕入先支払仕訳帳   | 勘定: 仕入先の相手勘定: 元帳、会社間銀行がサポートされます。                                                 | 仕入先支払仕訳帳                                                                  |

行が発行されるときに財務仕訳帳で設定されたルールに準拠していることを確認するために、検証されます。 行が発行した後で、ユーザーは、Microsoft Dynamics 365の伝票を編集、または転記できます。 

テンプレートに財務分析コードを追加するには、追加の変更が必要です。 詳細については、" "を参照してください[Microsoft Excelテンプレート追加します。\dev-itpro\financial-dimensions\add-dimensions-excel-templates] () に分析コードを。 分析コードのエンティティに追加された後、Excelデザイナで使用できる、テンプレートに追加できます。
