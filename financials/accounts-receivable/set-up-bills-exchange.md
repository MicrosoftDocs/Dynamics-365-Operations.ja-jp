# <a name="set-up-bills-of-exchange"></a>受取手形の設定

このトピックでは、為替手形を設定するための手順を説明します。

為替手形は別の従業員が会社に関係者 (通常は銀行)、指定された金額を支払うように指定する顧客から書き込まれるか、または電子注文です。 為替手形を販売注文請求書または自由書式の請求書の支払に使用する場合は、顧客勘定の貸方に記入します。 この貸方は、顧客が銀行に為替手形を支払うまで、為替手形で担保が付けられます。 通常、期日を為替手形による請求書を選択します。 銀行から為替手形の受取済通知を受領したら、為替手形を決済できます。 以下の時点で銀行から為替手形を振り出すことができます:

-   期日になったとき。 この方法は、取立依頼"と呼びますの所プラクティスと呼ばれます。
-   顧客に対して設定された支払条件で指定された割引日の期日まで。 トランザクションを転記すると、割引額が費用勘定に転記されます。 残りの金額は銀行が顧客から支払を受けるまで負債になります。 この方法は、割引依頼"と呼びますの所プラクティスと呼ばれます。

## <a name="set-up-posting-profiles-for-bills-of-exchange"></a>為替手形の転記プロファイルの設定
**顧客転記プロファイル**、取立用の為替手形、為替手形受取拒否、送金、割引に送金に使用する転記プロファイルを設定するには、次のページを使用します。 **集計勘定では**集計勘定]フィールドで、為替手形金額を転記する場合に選択します。 この勘定は、為替手形トランザクション タイプによって、借方転記されるか、または貸方に転記されます:
-   為替手形の場合、この勘定は、割引依頼または取立依頼を転記する際為替手形を転記するときは、信じましたときに借方に転記されます。
-   為替手形の受取拒否の場合、この勘定では、為替手形の受取拒否を転記するときに借方に転記されます。
-   取立依頼の場合、この勘定では、取立依頼を転記するときに借方に転記されます。
-   割引依頼の場合、この勘定では、割引依頼を転記するときに借方に転記されます。

では**精算勘定**現金勘定]フィールドで、為替手形金額を転記する場合に選択します。 この勘定では、為替手形を決済するときに借方に記入されます。 では**売上税前払**為替手形が前払に使用されると集計勘定]フィールドで、売上税金額を転記する場合に選択します。 では**割引勘定負債**勘定]フィールドで、割引依頼の割引金額を転記する場合に選択します。 この勘定では、割引依頼を転記するときに貸方に転記されます。

## <a name="set-up-accounts-receivable-parameters-for-bills-of-exchange"></a>為替手形の売掛金勘定パラメータを設定します。
**売掛金勘定パラメータ**ページで、為替手形の既定の転記プロファイルが**元帳および売上税**タブで入力します。 番号順序は**番号順序**]タブで定義されます。
為替手形の仕訳帳名の設定
------------------------------------------

**仕訳帳名**ページで、為替手形に使用するには、少なくとも5つの仕訳帳名を作成します。 仕訳帳タイプを次に示します。:
-   **顧客振出為替手形** –は、為替手形振出仕訳帳の仕訳帳名を作成します。
-   **為替手形受取拒否顧客** –は、為替手形受取拒否仕訳帳の仕訳帳名を作成します。
-   **顧客が為替手形を再振出** –は、為替手形再振出仕訳帳の仕訳帳名を作成します。
-   **顧客銀行送金為替** –は、送金仕訳帳の仕訳帳名を作成します。
-   **顧客決済為替手形** –は、為替手形決済仕訳帳の仕訳帳名を作成します。

各為替手形仕訳帳の仕訳伝票]ページで、**為替手形**タブの為替手形に関する情報を入力します。 為替手形仕訳帳明細行が転記された後、**為替手形仕訳帳照会**ページと**為替手形統計**ページに表示できます。
為替手形の支払方法の設定
-----------------------------------------------

**支払方法を**ページ設定為替手形に対して少なくとも一つの支払方法。 複数の銀行との取引がある場合、各銀行が為替手形に必要な送金形式に対応する支払方法を設定します。
為替手形の支払手数料の設定
-----------------------------------------

支払手数料は、顧客からの支払の回収プロセスに関連付けられた請求金額です。 複数の支払手数料設定行は支払手数料関連付けることができます。 支払手数料の既定の金額の計算方法を制御するのに設定明細行を使用できます。 たとえば、支払方法、支払仕様、通貨、および期間を設定する明細行を作成できます。 また、日付間隔に基づいて、割合または金額の設定明細行を作成できます。 たとえば、支払によってが期限超過となる時間に基づく利率を設定できます。 各送金のタイプの銀行手数料、手数料が、**などのコレクション**または**割引**、各送金のタイプの別の支払手数料の行を設定する。
銀行送金ファイルの送金手数料の設定
------------------------------------------------

**銀行口座**ページでは、生成された各送金ファイルのその銀行手数料送金手数料を設定できます。 送金手数料は送金が確認され、実際の手数料金額が判明した時点で転記されます。 送金手数料は、顧客から回収され、仕訳帳明細行に関連付けられる支払手数料とは異なります。
為替手形のドキュメント レイアウトの設定
---------------------------------------------

**銀行口座**ページで、**設定**クリックし、印刷された為替手形文書を生成する銀行口座ごとに必要なドキュメント レイアウトを指定します。
