# <a name="set-up-asset-retirement-obligation"></a>資産の退職の作業を設定します。

日本では、資産除去債務 (ARO) は法的な除去責務のある固定資産に対して認識されます。 この記事は、ARO の負債がどのように認識、償却、および未払となるか、および固定資産と ARO の負債が除去される方法について説明します。

資産の退職の関税(ARO)の耐用年数、資産の退職の原価を配分使用されます。 ARO は、固定資産の取得または構築する際に最初に負債として認識されます。 AROの職責を耐用年数の開始時に資産に要される見積退職の現在の価値と等しくなります。 ARO 負債を固定資産の取得原価に追加すると、ARO の負債原価は固定資産の耐用年数で償却されます。 ARO 負債の利息費用は、最終的な ARO 負債が当初の見積除去原価に到達するように、特定の割引率での未払費用になります。 固定資産が除去されると、ARO 負債の勘定に支払額が転記されます。 元の見積からの違いがない限り、これ以外の経費は転記されません。 次の表に、ライフ サイクルの実例を示します。

| 品目        | コンテンツ                                                                                                                                                                                   |
|-------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 前提 | 取得原価: **10,000** (残存価額 0) 耐用期間: **5** 減価償却方法: **定額** 見積除去原価: **1,000 ** 割引率: **3%**                  |
| ステップ１      | 借方: 設備 **10,863** 貸方: 他の支払 **10,000** 貸方: ARO の負債 **863**                                                                                                |
| ステップ２\*    | 借方: 減価償却費用 **2,173** 貸方: 減価償却累計額 (FA) **2,000** 貸方: 減価償却累計額 (ARO) **173**                                                     |
| 手順3\*    | 借方: 支払利息 **27** 貸方: ARO の負債 **27**                                                                                                                              |
| ステップ 4      | 借方: 減価償却累計額 (FA) **10,000** 借方: 減価償却累計額 (ARO) **863** 貸方: 設備 **10,863 ** 借方: ARO 負債 **1,000** 貸方: 他の支払 **1,000** |

> [!NOTE]
>\*、手順2と3を複数回固定資産の耐用年数全体に繰り返されます。

## <a name="setup-information"></a>設定情報
ARO を使用するには、次の設定手順を完了する必要があります。

-   **固定資産パラメータ**ページの既定の書籍や理由コードと番号順序などの基本的な固定資産パラメータを設定します。
-   **固定資産グループ**ページの固定資産グループを定義します。
-   減価償却の会計カレンダーを設定します。
-   ARO金額を計算し、現在の市場割引率を使用する割引率のスケジュールを設定します。
-   資産に使用するAROのタイプを指定して行う頻度ARO金額の変更が転記されるかを指定します。
-   AROの見積退職の原価計画を設定、資産の耐用年数の各会計期間のARO金額をシミュレートします。
-   **大文字で入力された資産の退職の関税**と**資産の退職の職務-増大の転記プロファイルを** **経費**ドキュメント タイプ設定します。
-   トランザクション タイプは、固定資産の転記時にトランザクション金額を取得する勘定を設定します。**処分のARO **
