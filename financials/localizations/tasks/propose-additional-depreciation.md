<span data-ttu-id="934db-104">日本では、特定の条件下で割増償却が許可されています。</span><span class="sxs-lookup"><span data-stu-id="934db-104">In Japan, an additional depreciation is permitted under certain conditions.</span></span> 



<span data-ttu-id="934db-105">この手順では、割増償却の提案方法を説明します。</span><span class="sxs-lookup"><span data-stu-id="934db-105">Use this procedure to learn how to propose additional depreciation.</span></span> <span data-ttu-id="934db-106">始める前に、固定資産が取得され、通常の減価償却が転記されていることを確認します。</span><span class="sxs-lookup"><span data-stu-id="934db-106">Before you begin, be sure that the fixed asset has been acquired and the ordinary depreciation has been posted.</span></span>  <span data-ttu-id="934db-107">また、[固定資産コンフィギュレーション キー] が選択されていることを確認します。</span><span class="sxs-lookup"><span data-stu-id="934db-107">Also, be sure that the Fixed Assets configuration key is selected.</span></span>



<span data-ttu-id="934db-108">この手順では、JPMF デモ会社のデータを使用します。</span><span class="sxs-lookup"><span data-stu-id="934db-108">This procedure uses the JPMF demo company data.</span></span>

5. <span data-ttu-id="934db-113">[提案] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="934db-113">Click Proposals.</span></span>
6. <span data-ttu-id="934db-114">[特別償却提案] クリックします。</span><span class="sxs-lookup"><span data-stu-id="934db-114">Click Extraordinary depreciation proposal.</span></span>

10. <span data-ttu-id="934db-119">[OK] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="934db-119">Click OK.</span></span>
11. <span data-ttu-id="934db-120">[OK] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="934db-120">Click OK.</span></span>
12. <span data-ttu-id="934db-121">[転記] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="934db-121">Click Post.</span></span>
