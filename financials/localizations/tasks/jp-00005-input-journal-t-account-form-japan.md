# <a name="input-journal-in-t-account-form-japan"></a><span data-ttu-id="3ee96-103"> T 勘定フォームでの仕訳帳入力 (日本)</span><span class="sxs-lookup"><span data-stu-id="3ee96-103">Input journal in T account form (Japan)</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="3ee96-104">日本では、仕訳入力には一般的に T 字勘定形式が使用されます。</span><span class="sxs-lookup"><span data-stu-id="3ee96-104">In Japan, the T-account format is commonly used to input journal entries.</span></span> 



<span data-ttu-id="3ee96-105">この手順では、T 字勘定形式を使用した仕訳入力の作成について説明します。</span><span class="sxs-lookup"><span data-stu-id="3ee96-105">This procedure walks you through creating a journal entry using the T-account format.</span></span> 



<span data-ttu-id="3ee96-106">この手順では、JPMF デモ会社のデータを使用します。</span><span class="sxs-lookup"><span data-stu-id="3ee96-106">This task uses the JPMF demo company data.</span></span>

1. <span data-ttu-id="3ee96-107">[総勘定元帳] > [総勘定元帳] > [仕訳入力] > [一般仕訳帳] の順に移動します。</span><span class="sxs-lookup"><span data-stu-id="3ee96-107">Go to General ledger > Journal entries > General journals.</span></span>
2. <span data-ttu-id="3ee96-108">[新規] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="3ee96-108">Click New.</span></span>
3. <span data-ttu-id="3ee96-109">[名前] フィールドに値を入力します。</span><span class="sxs-lookup"><span data-stu-id="3ee96-109">In the Name field, type a value.</span></span>

5. <span data-ttu-id="3ee96-111">[T 字勘定仕訳入力 (JP)] をクリックします。</span><span class="sxs-lookup"><span data-stu-id="3ee96-111">Click T-account (JP).</span></span>

9. <span data-ttu-id="3ee96-115">[説明] フィールドで値を入力します。</span><span class="sxs-lookup"><span data-stu-id="3ee96-115">In the Description field, type a value.</span></span>
10. <span data-ttu-id="3ee96-116">[借方] フィールドに数値を入力します。</span><span class="sxs-lookup"><span data-stu-id="3ee96-116">In the Debit field, enter a number.</span></span>
11. <span data-ttu-id="3ee96-117">[日付] フィールドに日付を入力します。</span><span class="sxs-lookup"><span data-stu-id="3ee96-117">In the Date field, enter a date.</span></span>
12. <span data-ttu-id="3ee96-118">[勘定] フィールドで、任意の値を指定します。</span><span class="sxs-lookup"><span data-stu-id="3ee96-118">In the Account field, specify the desired values.</span></span>
13. <span data-ttu-id="3ee96-119">[説明] フィールドに値を入力します。</span><span class="sxs-lookup"><span data-stu-id="3ee96-119">In the Description field, type a value.</span></span>
