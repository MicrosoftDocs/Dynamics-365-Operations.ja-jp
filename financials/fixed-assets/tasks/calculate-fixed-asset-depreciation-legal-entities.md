# <a name="calculate-fixed-asset-depreciation-across-legal-entities"></a>法人間での固定資産減価償却の計算

[!include[task guide banner](../../includes/task-guide-banner.md)]

固定資産が割り当てられている固定資産グループを変更する場合にこの手順を使用します。 固定資産は正しい固定資産グループに割り当てる必要があります。 固定資産グループは、照会およびレポートの作成、新しい固定資産の設定、元帳の統合、固定資産トランザクションの適切な勘定科目への転記に使用されます。

このレコードでは、USMF デモ会社を使用します。

1. [固定資産] > [固定資産] > [固定資産] に移動します。
2. 固定資産グループを変更するために、固定資産グループを選択します。
3. [固定資産グループの変更] をクリックします。
4. [新規グループ] フィールドで、値を入力または選択します。
5. [新しい固定資産番] オプションを [はい] に設定して、固定資産番号を選択した固定資産に割り当てます。
    * 新しい固定資産番号オプションが [はい] に設定されている場合、固定資産番号フィールドが使用できるようになります。   固定資産に自動番号付けが設定されている場合、次に使用可能な固定資産番号がフィールドに表示されます。 数値は変更できます。   手動での番号付けが固定資産に対して設定されている場合、このフィールドは空白で、新しい固定資産番号を入力する必要があります。  
6. [OK] をクリックします。
7. [はい] をクリックします。
